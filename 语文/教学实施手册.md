# 语文教学实施手册

> 从90分到105分，每次课做什么、做到什么程度算过关、用什么材料、怎么检验
> 创建：2026-04-08

---

## 一、整体框架

| 阶段 | 课时 | 核心任务 | 内容 | 过关标准 |
|------|------|---------|------|---------|
| 第一阶段 | 12次 | 文言文基础（120实词） | 每次课5个高频实词，放高考真题语境考 | 120实词掌握≥90%，古今异义全对 |
| 第二阶段 | 4次 | 作文模板实操 | 五段式议论文模板+二元立意+素材运用 | 独立写1篇47分+的完整作文 |
| 第三阶段 | 4次 | 诗歌鉴赏+阅读技巧 | 3大题型模板+论述类/实用类阅读技巧 | 诗歌选择题对，主观题能写3个得分点 |
| 第四阶段 | 4次 | 语言文字运用专项+名句默写 | 成语+病句+衔接+图文转换+默写 | 语言运用14分+/20分，默写4分+/5分 |
| 第五阶段 | 持续到高考 | 综合模拟+查漏补缺 | 每周1套高考语文卷 | 稳定105分+ |

**总共约24次专项课**（不含日常词汇和默写），加上每天持续进行的实词积累和名句默写。

**注意：** 语文在日程表中没有固定大块时间，以碎片化穿插为主。文言文实词已在日常进行（每次5个），作文/诗歌/语言运用需安排专题课时。

## 二、考试概况

### 试卷结构（新高考Ⅰ卷）
| 题型 | 分值 | 新灵当前 | 新灵目标 | 提分空间 |
|------|------|---------|---------|---------|
| 信息类文本阅读（3选择） | 9分 | 4-6分 | 6分 | +2 |
| 文学类文本阅读（1选择+2主观） | 14分 | 6-8分 | 8分 | +1 |
| 文言文阅读（3选择+1翻译） | 19分 | 8-10分 | 12分 | +3 |
| 古代诗歌阅读（1选择+1主观） | 11分 | 4-5分 | 6分 | +2 |
| 名句默写 | 5分 | 2-3分 | 4分 | +2 |
| 语言文字运用 | 20分 | 10-12分 | 14分 | +3 |
| 作文 | 60分 | 40-43分 | 47分 | +5 |
| **合计** | **150分** | **~90分** | **~105分** | **+15分** |

### 得分策略
- **最容易提分：** 作文（+5分）、语言文字运用（+3分）、文言文（+3分）
- **日常积累型：** 实词、名句默写 → 每天5-10分钟，不用大块时间
- **模板速成型：** 诗歌鉴赏、语言文字运用 → 学了就能拿分
- **作文是提分大头：** 从43分到47分，靠模板+素材，不需要天赋

| 模块 | 高考分值 | 课时 | 优先级 | 理由 |
|------|---------|------|--------|------|

## 四、第一阶段：文言文实词（12次，日常穿插）

> 文言文实词是语文的"基本功"，每天5个，不需要专门课时，放在日常零碎时间。
> 已经完成20个（4组），继续推进。

### 教学方式
- 每次5个高频实词
- **必须放到高考真题语境中考**，不单独问"这个字什么意思"
- 古今异义词重点标注（已积累7个：走、去、兵、适、寻、再、曾）
- 每次新课前先滚动复习上次的词

### 120实词分组规划

| 组别 | 实词 | 重点标注 |
|------|------|---------|
| 第5组 | 就、怜、弥、莫、乃 | "乃"多义（于是/才/是/竟然） |
| 第6组 | 期、穷、求、胜、使 | "穷"古今异义（不是贫穷，是走不通/极点） |
| 第7组 | 是、速、涕、图、亡 | "涕"古今异义（不是鼻涕，是眼泪） |
| 第8组 | 恶、悉、相、谢、信 | "信"多义（确实/书信/信用/伸） |
| 第9组 | 兴、修、阳、要、宜 | "阳"古义（山的南面/水的北面） |
| 第10组 | 遗、贻、阴、右、再造 | "阴"古义（山的北面/水的南面） |
| 第11组 | 质、治、诸、族、卒 | "治"多义（治理/医治/研究/安定） |
| 第12组 | 走、左、坐、被、从 | "坐"古义（因为/犯罪/座） |
| 第13-24组 | 剩余72个高频实词 | 按高考频率选编 |

> 注：第5-24组的具体词目在教学过程中逐次编写，参考高考历年真题的高频词统计。

### 过关标准
- 每组5个词，测试8-10道题，正确率≥80%
- 古今异义词**必须全对**（这是高考必考陷阱）
- 滚动复习：每隔3组回头考之前的词，确保没忘

### 笔记本内容
每次课后的笔记格式：
```
第X组实词
1. XX：①___ ②___ ③___
   ⚠️ 古今异义！现代___，古文___
2. ...
```
古今异义单独汇总在笔记本最后一页。

## 五、第二阶段：作文模板实操（4次专题课）

> 新灵已学过五段式模板和二元立意写法，有素材（刘备三顾茅庐、鲁迅弃医从文）。
> 但**没有完整写过一篇**，这是本阶段的核心目标。

### 第1次课：模板复习+片段训练

**教学目标：** 巩固五段式模板，各段落独立写出合格片段

**教学内容：**
1. 复习五段式结构（10分钟）
```
五段式议论文模板：
┌─────────────────────────────┐
│ 第一段：引（引材料+亮观点）   │ 80-100字
│ 第二段：论A（正面论证）       │ 150-200字
│ 第三段：论B（反面/深化论证）   │ 150-200字
│ 第四段：合（辩证/联系现实）    │ 100-150字
│ 第五段：结（重申论点+升华）    │ 80-100字
│ 合计：约600-750字             │
└─────────────────────────────┘
```

2. 开头段写法训练（15分钟）
   - "引材料"三步法：概述材料→提炼关键词→亮明论点
   - 练习：给3个作文题，每个写一个开头段（80-100字）

3. 论证段写法训练（20分钟）
   - 论证段公式：`分论点 + 事例（一句话） + 分析（为什么支持论点） + 小结`
   - 论A（正面）和论B（反面/深化）的区别
   - 练习：给1个论点，分别写论A段和论B段

4. 结尾段写法训练（10分钟）
   - 结尾公式：`重申论点 + 升华/号召`
   - 练习：写2个结尾段

**过关标准：**
- 能说出五段式每段的功能和字数
- 开头段能"引材料+亮观点"，不绕弯
- 论证段有"分论点+事例+分析"三要素

**测试方式：** 给一个作文题（如"坚持与放弃"），限时写开头段+1个论证段，检查结构是否完整

### 第2次课：二元立意实操

**教学目标：** 掌握二元对立题目的写法，A面+B面+合二为一

**教学内容：**
1. 二元立意题目特征（5分钟）
   - 常见二元词：快与慢、得与失、坚持与放弃、个人与集体
   - 高考最爱考：两个看似矛盾的概念如何统一

2. 二元立意结构（15分钟）
```
开头：引材料，提出"A和B都很重要，关键在于____"
论A：A的价值（为什么需要A）
论B：B的价值（为什么也需要B）
合：什么情况下用A，什么情况下用B（辩证统一）
结：总结，联系自身/时代
```

3. 辩证段落写法（20分钟）
   - "合"段是最难的，也是最提分的
   - 公式：`A和B不是对立的，而是____。当____时，我们更需要A；当____时，我们更需要B。`
   - 练习：给2组二元词，各写一个"合"段

4. 素材运用训练（15分钟）
   - 已有素材回顾：刘备三顾茅庐（坚持）、鲁迅弃医从文（放弃/选择）
   - 新素材补充2个：
     - 屠呦呦（坚持→青蒿素，196次失败后成功）
     - 苏炳添（突破自我→9秒83，挑战年龄极限）
   - 练习：用4个素材各写1个论证片段

**过关标准：**
- 能区分"单一立意"和"二元立意"的题目
- 辩证段能写出"A不是否定B"的逻辑
- 能灵活运用至少3个素材

**测试方式：** 给一个二元立意题目，写出完整的提纲（5段各1句话概括），检查逻辑链

### 第3次课：完整作文实操（限时训练）

**教学目标：** 60分钟内完成一篇完整的600-750字议论文

**教学内容：**
1. 选一个高考真题作文题（5分钟）
   - 推荐：2023或2024年全国卷二元立意题
   
2. 限时写作（60分钟）
   - 前5分钟：审题+列提纲（5段各写1句话）
   - 中45分钟：写作
   - 后10分钟：检查+修改
   - **衍全程不插手，只计时**

3. 写完后点评（20分钟）
   - 按评分标准打分：
     - 论点是否明确（20分）
     - 结构是否完整（20分）
     - 论证是否有力（20分）
     - 语言是否通顺（20分）
     - 字数是否达标（20分）
   - 指出具体问题，标注改进点

**过关标准：**
- 60分钟内完成≥600字
- 论点明确，不跑题
- 五段结构完整

### 第4次课：作文修改+第二篇

**教学目标：** 根据反馈修改第一篇，再独立写第二篇

**教学内容：**
1. 修改第一篇作文（20分钟）
   - 根据上次点评的3个最严重问题修改
   - 对比修改前后

2. 写第二篇完整作文（60分钟）
   - 选不同类型的题目（可以是单一立意）
   - 同样限时

3. 自评+点评（15分钟）
   - 新灵先自评：觉得自己哪里写得好，哪里不满意
   - 衍再点评，对比两篇的进步

**过关标准（本阶段总过关）：**
- 独立写出2篇完整作文
- 第二篇比第一篇有明显进步
- 按高考评分标准，至少1篇能达到45分+

### 笔记本内容
```
作文模板笔记：
1. 五段式结构（每段功能+字数）
2. 二元立意公式
3. 论证段公式：分论点+事例+分析+小结
4. 辩证段公式：A不是否定B，当…时需要A，当…时需要B
5. 素材卡片（每个素材：人物+事件+可论证的方向）
```

## 六、第三阶段：诗歌鉴赏+阅读技巧（4次专题课）

### 第1次课：诗歌鉴赏—形象与意象

**教学目标：** 能识别诗歌中的意象，分析诗歌形象

**教学内容：**
1. 意象分类速查（15分钟）
```
常见意象及对应情感：
- 月亮 → 思乡、思人
- 柳 → 离别（"柳"谐音"留"）
- 菊花 → 隐逸、高洁
- 梅花 → 坚强、不屈
- 梧桐 → 悲秋、愁苦
- 杜鹃 → 悲苦、思归
- 大雁 → 思乡、传书
- 酒 → 豪情/愁绪
- 落花 → 伤春、时光流逝
```

2. 形象分析模板（15分钟）
```
人物形象题答题公式：
① 这是一首___诗（送别/边塞/田园/咏物…）
② 诗中描写了___的形象（概括特点）
③ 通过___（具体诗句）可以看出
④ 表达了诗人___的情感

景物形象题答题公式：
① 描绘了___的画面（概括）
② 用了___等意象
③ 营造了___的氛围
④ 表达了___的情感
```

3. 练习（25分钟）
   - 2道高考真题：1道人物形象、1道景物形象
   - 先让新灵自己写，再对照标准答案修改

**过关标准：** 能说出10个常见意象的含义，形象题能写3个得分点

### 第2次课：诗歌鉴赏—手法与情感

**教学目标：** 能识别常见表现手法，情感分析题能拿满分

**教学内容：**
1. 常见表现手法（20分钟）
```
必背手法（高考最爱考）：
- 对比：两个事物/场景放一起比
- 衬托：用甲衬托乙（正衬/反衬）
- 借景抒情：不直接说感情，用景物暗示
- 虚实结合：现实+想象/回忆
- 用典：借用历史故事或前人诗句
- 动静结合：动态描写+静态描写
- 白描：不加修饰，简洁勾勒
- 象征：用具体事物代表抽象概念
```

2. 手法题答题模板（10分钟）
```
手法题答题公式（4步）：
① 指出手法：这首诗用了___手法
② 结合诗句：在"___"这句中，…
③ 分析效果：这样写___（突出了/增强了/营造了…）
④ 表达情感：表达了诗人___的情感
```

3. 情感分析题（15分钟）
```
情感题常见类型：
- 忧国忧民（杜甫专用）
- 离别思念（送别诗/羁旅诗）
- 归隐田园（山水田园诗）
- 怀古伤今（咏史诗）
- 壮志难酬（边塞/怀才不遇）

答题公式：
① 这是一首___诗（判断类型→缩小情感范围）
② 诗人通过___（意象/事件/手法）
③ 表达了___的情感（要具体，不能只写"热爱自然"）
```

4. 练习（15分钟）
   - 2道高考真题：1道手法题、1道情感题
   - 诗歌选择题1道（找错误选项）

**过关标准：** 
- 能识别6种以上表现手法
- 手法题能按4步公式作答
- 情感题能写出具体情感，不笼统

### 第3次课：论述类/实用类阅读技巧

**教学目标：** 信息类文本选择题对2/3，主观题能拿基础分

**教学内容：**
1. 选择题解题三步法（15分钟）
```
论述类文本选择题做对的方法：
1. 先读题干，明确问什么
2. 回原文定位（找题干中的关键词）
3. 逐字对比选项和原文

常见陷阱：
- 偷换概念（换个词但意思变了）
- 以偏概全（"都"换成"有些"）
- 无中生有（原文没提到的）
- 因果倒置（A导致B，说成B导致A）
- 绝对化（"一定""必须"等词要警惕）
```

2. 主观题答题技巧（15分钟）
```
论述类主观题公式：
① 找原文相关段落
② 提取关键句（通常是段首/段尾）
③ 整理归纳，分点作答
④ 每点：原文关键词 + 自己的归纳
```

3. 练习（25分钟）
   - 1篇完整信息类文本（3道选择+1道主观）
   - 限时25分钟
   - 做完后逐题复盘

**过关标准：** 3道选择题对2道，主观题能写2个有效得分点

### 第4次课：文学类文本阅读技巧

**教学目标：** 小说/散文阅读的选择题能对，主观题不丢基础分

**教学内容：**
1. 小说阅读答题模板（20分钟）
```
人物形象题：性格特点 + 具体表现 + 作用
环境描写题：自然环境/社会环境 + 营造氛围 + 推动情节 + 烘托人物
情节作用题：照应前文 + 推动发展 + 深化主题
主题理解题：表层含义 + 深层含义 + 现实意义
```

2. 散文阅读答题模板（15分钟）
```
散文核心：形散神不散
- "形散"：材料多、角度多
- "神不散"：围绕一个中心

答题要点：
① 抓住线索（人/事/物/情感）
② 找到文眼（关键句/主旨句）
③ 分析手法（对比/象征/借物喻人）
```

3. 练习（20分钟）
   - 1篇小说阅读（1选择+2主观）
   - 限时20分钟

**过关标准：** 选择题对，主观题每题能写2-3个得分点

### 笔记本内容
```
诗歌鉴赏笔记：
1. 常见意象10个+对应情感
2. 手法题4步公式
3. 情感题5大类型

阅读技巧笔记：
1. 选择题5大陷阱
2. 小说4类题答题模板
3. 散文"形散神不散"分析法
```

## 七、第四阶段：语言文字运用+名句默写（4次专题课）

### 第1次课：成语运用

**教学目标：** 成语选择题稳定对，不因望文生义丢分

**教学内容：**
1. 高考高频成语分类（20分钟）
```
易望文生义的成语（最常考）：
- 不以为然：不是"不在乎"，是"不认为对"
- 不刊之论：不是"不能发表"，是"不可修改的好文章"
- 侧目而视：不是"看不起"，是"恐惧愤恨"
- 鼎足之势：三方对峙，不是"稳固"
- 七月流火：天气转凉，不是"天热"
- 无所不为：什么都干（贬义），不是"能力强"

易用错对象的成语：
- 美轮美奂：形容建筑，不能形容舞蹈/风景
- 相敬如宾：形容夫妻，不能形容同学/同事
- 破镜重圆：夫妻复合，不能形容朋友和好

易褒贬误用的成语：
- 处心积虑：贬义（不是"认真思考"）
- 始作俑者：贬义（不是"创始人"）
- 叹为观止：褒义（不是"看傻了"）
```

2. 成语题解法（10分钟）
   - 先看语境是褒是贬
   - 看主语是人还是物
   - 逐个排除

3. 练习（20分钟）
   - 10道高考成语真题
   - 每道标出考点（望文生义/对象错误/褒贬误用）

**过关标准：** 10道对8道

### 第2次课：病句辨析

**教学目标：** 病句选择题能识别常见语病

**教学内容：**
1. 六大常见语病（25分钟）
```
高考必考语病（按频率排序）：
1. 语序不当
   - 定语位置错：→ "一本鲁迅写的书"不是"鲁迅写的一本的书"
   - 关联词位置错：主语相同→关联词在主语后
   
2. 搭配不当
   - 主谓不搭：→ "速度提高了"不是"速度扩大了"
   - 动宾不搭：→ "发扬传统"不是"发扬力量"
   
3. 成分残缺或赘余
   - 缺主语：→ "通过…使…"（删掉"通过"或"使"）
   - 赘余：→ "大约…左右"（删掉一个）
   
4. 结构混乱
   - 句式杂糅：→ "原因是…造成的"（选一个）
   
5. 表意不明
   - 歧义：→ "两个学生的家长"（谁的家长？）
   
6. 不合逻辑
   - 自相矛盾：→ "几乎全部"（选一个）
   - 否定不当：→ "防止不发生"（去掉"不"）
```

2. 病句题解题步骤（10分钟）
   - 先通读找"别扭"的地方
   - 缩句（去掉修饰语看主干）
   - 按六大语病逐项检查

3. 练习（20分钟）
   - 10道高考病句真题
   - 每道标出语病类型

**过关标准：** 10道对8道，能说出错误类型

### 第3次课：语言连贯（衔接）+图文转换

**教学目标：** 衔接题和图文转换题稳定拿分

**教学内容：**
1. 衔接排序题技巧（20分钟）
```
解题步骤：
1. 先看首尾句（确定话题起点和结尾）
2. 找关联词（"因此""但是""此外"→判断前后逻辑）
3. 找代词（"这""其"→指代上文内容）
4. 找时间顺序/空间顺序
5. 代入验证，读一遍看通顺不
```

2. 图文转换题技巧（15分钟）
```
两种题型：
① 徽标类（Logo/标志）
   - 描述构图要素（有什么）
   - 描述构图特点（怎么排列）
   - 揭示寓意（象征什么）

② 图表类（柱状图/饼图/折线图）
   - 描述趋势/变化
   - 找关键数据（最高/最低/转折点）
   - 归纳结论
```

3. 练习（20分钟）
   - 3道衔接排序题
   - 2道图文转换题

**过关标准：** 衔接题对2/3，图文转换能得70%的分

### 第4次课：名句默写+综合语言运用

**教学目标：** 默写稳拿4分+，语言运用综合训练

**教学内容：**
1. 名句默写重点篇目（15分钟）
```
高考必背篇目（选最常考的）：
- 《劝学》荀子 → "学不可以已""故木受绳则直"
- 《师说》韩愈 → "师者，所以传道受业解惑也"
- 《赤壁赋》苏轼 → "寄蜉蝣于天地，渺沧海之一粟"
- 《念奴娇·赤壁怀古》苏轼 → "大江东去，浪淘尽"
- 《永遇乐·京口北固亭怀古》辛弃疾 → "千古江山，英雄无觅"
- 《琵琶行》白居易 → "同是天涯沦落人，相逢何必曾相识"
- 《阿房宫赋》杜牧 → "后人哀之而不鉴之，亦使后人而复哀后人也"
- 《登高》杜甫 → "无边落木萧萧下，不尽长江滚滚来"
- 《蜀道难》李白 → "蜀道之难，难于上青天"
```

2. 默写练习（10分钟）
   - 默写以上篇目的关键句
   - 标出易错字（如"受"业不是"授"业，"萧萧"不是"潇潇"）

3. 语言运用综合训练（30分钟）
   - 1组完整的高考语言运用题（成语+病句+衔接+图文转换+其他）
   - 限时25分钟
   - 做完后逐题复盘

**过关标准：**
- 默写5个句子写对4个（含无错别字）
- 语言运用综合正确率≥70%

### 笔记本内容
```
语言文字运用笔记：
1. 易望文生义成语20个
2. 六大语病类型+例句
3. 衔接排序5步法
4. 图文转换2种题型模板
5. 默写易错字汇总
```

## 八、第五阶段：综合模拟（持续到高考）

### 执行方式
- 每周1套高考语文真题/模拟卷
- 限时150分钟完整做
- 做完后逐题对答案，分析失分原因
- 把失分的知识点加入下周复习重点

### 重点复盘方向
| 题型 | 常见失分原因 | 补救方法 |
|------|-------------|---------|
| 选择题 | 没回原文对比 | 强制先定位再判断 |
| 文言文翻译 | 实词不认识 | 滚动复习120实词 |
| 诗歌主观题 | 答非所问/太笼统 | 用模板分点作答 |
| 作文 | 跑题/结构散 | 审题5分钟+列提纲 |
| 默写 | 错别字 | 每天抄写1遍易错句 |

### 过关标准
- 模拟卷总分稳定≥105分
- 作文≥45分
- 文言文≥12分
- 语言运用≥14分

## 十、教学注意事项（针对新灵特点）

1. **作文是新灵的舒适区** — 她有想法，但结构散。重点是给框架，不是教"怎么想"
2. **文言文是新灵的薄弱点** — 她对"记住了"判断偏乐观，测试要严格，至少8-10道确认
3. **诗歌鉴赏要让她先说感受** — 她腼腆但感受力不差，先让她说，再引导她用术语表达
4. **语言运用多出题** — 新灵原话"文科类要多出题"，成语和病句至少10道才放过
5. **默写不能只背不写** — 很多字"会背但写错"，必须手写确认
6. **作文点评要具体** — 不说"写得不错"，要说"第二段的分论点很清楚，但分析部分少了为什么"
