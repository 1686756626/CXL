## 一、整体框架

| 阶段 | 课时 | 核心任务 | 内容 | 过关标准 |
|------|------|---------|------|---------|

## 二、考试概况

### 试卷结构
| 题型 | 题量 | 分值 | 新灵目标 |
|------|------|------|---------|
| 单选题 | 15-16题 | 45-48分 | 30分（对10个） |
| 非选择题（材料分析题） | 3-4题 | 52-55分 | 30分 |
| **合计** | | **100分** | **60分卷面→赋分75+** |

### 得分策略
- **选择题：** 人文地理选择题用模板排除，自然地理选择题抓关键信息
- **非选择题：** 分点答题，用地理术语，每点不超过2行
- **核心原则：** 会做的题不丢分，不会的题写模板关键词蹭分

## 三、第一阶段：人文地理（8次课）

> 人文地理是地理中最容易拿分的部分，答题高度模板化。先把模板背熟，再练真题。

### 第1次课：农业区位因素分析（上）

**教学目标：** 掌握农业区位分析模板，能默写自然+社会经济因素

**教学内容：**
1. 农业区位分析模板（核心中的核心）

```
自然因素：
- 气候：光照、热量、降水、昼夜温差、雨热同期
- 地形：平原/盆地/河谷，地形平坦
- 水源：临近河流/湖泊，灌溉水源充足
- 土壤：黑土/冲积土，土壤肥沃

社会经济因素：
- 市场：人口稠密/城市密集，市场广阔
- 交通：铁路/公路/水运便利，便于农产品运输
- 政策：国家政策支持
- 科技：农业科技发达/机械化程度高
- 劳动力：人口稠密，劳动力丰富廉价
- 工业基础：工业发达，能提供农机/化肥
```

2. 中国典型农业区域特征（必背）

```
东北平原：黑土肥沃，地广人稀，适合大规模机械化；制约=热量不足（一年一熟）
华北平原：地势平坦，光热充足；制约=水源不足，旱涝盐碱风沙
南方平原：水热充足，土壤肥沃；制约=洪灾
西北灌溉农业：光照强，昼夜温差大（瓜果甜）；制约=缺水
青藏河谷农业：光照足，昼夜温差大，单产高品质好；制约=热量不足
```

**测试方式：**
- 默写农业区位模板（自然4要素+社会6要素）
- 给一道高考真题（如"分析新疆种植棉花的区位条件"），要求按模板分点作答
- **过关标准：** 模板默写不漏项，真题至少答出4个得分点

**笔记本内容：**
```
地理·农业区位分析模板
自然：气候（光热水温差）+ 地形（平坦）+ 水源（河流）+ 土壤（肥沃）
社会经济：市场 + 交通 + 政策 + 科技 + 劳动力 + 工业基础
```

### 第3次课：工业区位因素分析（上）

**教学目标：** 掌握工业区位分析模板

**教学内容：**
1. 工业区位分析模板

```
自然因素：
- 土地：平坦开阔，地价低
- 水源：临近河流，水源充足
- 原料：接近原料产地
- 能源：接近能源产地（煤、石油、水能）

经济因素：
- 市场：人口稠密，市场广阔
- 交通：铁路/公路/港口/航空，交通便利
- 劳动力：丰富（劳动密集型）或素质高（技术密集型）
- 科技：靠近高校/科研院所，技术水平高
- 工业基础：工业发达，协作条件好
- 农业：靠近农业产区，原料供应充足

社会因素：
- 政策：国家政策支持/税收优惠
- 国防/安全/个人偏好

环境因素（必考）：
- 污染大气的工厂→主导风向下方或最小风频上风向
- 污染水源的工厂→河流下游
- 远离居民区和农田
```

2. 五种工业指向型类型

```
原料指向型：制糖、水产品加工 → 靠近原料地
市场指向型：啤酒、家具、面包 → 靠近市场
动力指向型：炼铝 → 靠近能源
劳动力指向型：纺织、服装 → 靠近廉价劳动力
技术指向型：电子、航天 → 靠近科技中心
```

**过关标准：** 能区分5种指向型，能默写工业区位模板

**笔记本内容：**
```
地理·工业区位分析模板
自然：土地+水源+原料+能源
经济：市场+交通+劳动力+科技+工业基础
社会：政策+国防
环境：风向+水源+居民区距离
五种指向型：原料/市场/动力/劳动力/技术
```

### 第5次课：城市区位+交通区位

**教学目标：** 掌握城市区位和交通区位模板

**教学内容：**

1. 城市区位模板

```
自然因素：
- 地形：平原（城市最优选）、河谷、盆地
- 气候：温暖湿润、适宜居住
- 河流：供水+运输+军事防御（河流交汇处/河流弯曲处/河口）

社会经济因素：
- 资源：矿产资源开发形成城市（大庆、攀枝花）
- 交通：铁路枢纽、港口城市
- 政治：省会、首都
- 宗教/历史/科技
```

2. 交通区位模板

```
自然因素：
- 地形：平原好修，山地沿等高线/开隧道
- 地质：避开断层
- 水文：少跨河流/沼泽
- 气候：注意暴雨、冻土

社会经济因素：
- 人口城市：连接人口密集区
- 经济：带动沿线经济发展
- 资源：开发沿线资源
- 科技：技术保障

选线原则：路程最短+造价最低+效益最大+少占耕地
```

**过关标准：** 两套模板都能默写核心要素

**笔记本内容：**
```
地理·城市区位：地形+气候+河流+资源+交通+政治
地理·交通区位：地形+地质+水文+人口+经济+资源
```

### 第6次课：人口与城市化

**教学目标：** 掌握人口迁移推拉力模型和城市化分析框架

**教学内容：**

1. 人口分布与迁移

```
人口分布影响因素：
- 自然：气候（温暖湿润）、地形（平原）、水源（河流）、土壤
- 经济：经济发展水平、城市化水平

人口迁移推拉力模型：
- 推力（迁出地）：经济落后、就业少、收入低、环境差、战争
- 拉力（迁入地）：经济发达、就业多、收入高、环境好、教育好
```

2. 城市化

```
城市化表现：城市人口增加、城市用地规模扩大、劳动力从第一产业向二三产业转移

城市化的问题：
- 环境污染（大气/水/噪声/固废）
- 交通拥堵
- 住房紧张
- 就业困难

解决措施：
- 发展卫星城/新城
- 改善城市交通
- 治理污染
- 合理规划城市功能分区
```

**过关标准：** 能用推拉力模型分析一道人口迁移题，能列出城市化问题和解决措施

## 四、第二阶段：自然地理（5次课）

> 自然地理比人文难，但高考只考重点。不追全覆盖，只抓高频考点。

### 第9次课：气候类型及其成因

**教学目标：** 能识别主要气候类型，能说出发因

**教学内容：**

1. 必须掌握的气候类型（只背8种高频的）

```
热带雨林气候：全年高温多雨 → 赤道低气压带控制
热带草原气候：全年高温，干湿季分明 → 赤道低压/信风交替
热带季风气候：全年高温，雨旱季分明 → 海陆热力性质差异+气压带风带移动
热带沙漠气候：全年炎热干燥 → 副热带高气压带/信风控制
亚热带季风气候：夏季高温多雨，冬季温和少雨 → 海陆热力性质差异
地中海气候：夏季炎热干燥，冬季温和多雨 → 副高和西风交替控制
温带季风气候：夏季高温多雨，冬季寒冷干燥 → 海陆热力性质差异
温带海洋性气候：全年温和湿润 → 终年西风控制
```

2. 气候判断三步法

```
步骤1：看最冷月均温
  >15°C → 热带
  0~15°C → 亚热带
  <0°C → 温带

步骤2：看降水量和季节分配
  年雨型 → 温海/热带雨林
  夏雨型 → 季风/热带草原
  冬雨型 → 地中海
  少雨型 → 热带沙漠

步骤3：定位（大陆东岸/西岸/内陆）
```

**过关标准：** 给气温降水数据能判断气候类型，能说出成因

**笔记本内容：**
```
地理·气候判断三步法
1. 最冷月均温定热量带（>15热/0-15亚热/<0温）
2. 降水分配定雨型（年雨/夏雨/冬雨/少雨）
3. 海陆位置定具体类型

8种气候+成因必背
```

### 第10次课：大气受热过程+热力环流

**教学目标：** 理解大气受热三环节，能用原理解释现象

**教学内容：**

1. 大气受热过程（三个环节）

```
太阳暖大地：太阳辐射（短波）→ 穿过大气（被吸收/反射/散射削弱）→ 到达地面
大地暖大气：地面吸收升温 → 地面辐射（长波）→ 大气中CO₂/水汽吸收
大气还大地：大气逆辐射 → 把热量还给地面 → 保温作用
```

2. 用原理解释现象（必考）

```
为什么高原昼夜温差大？
→ 海拔高，空气稀薄
→ 白天：削弱作用弱，太阳辐射强，气温高
→ 夜晚：保温作用弱，大气逆辐射弱，散热快，气温低

为什么晴天昼夜温差大？
→ 晴天云少
→ 白天：削弱少，气温高
→ 夜晚：大气逆辐射弱，气温低

温室大棚原理？
→ 玻璃/薄膜让太阳短波进入
→ 阻止地面长波散出
→ 增强保温效果
```

3. 热力环流

```
根本原因：地面冷热不均
热的地方 → 气流上升 → 近地面低压 → 高空高压
冷的地方 → 气流下沉 → 近地面高压 → 高空低压
水平方向：高压→低压（形成风）
```

**过关标准：** 能画出大气受热过程示意图，能用原理解释3种现象

### 第11次课：锋面系统

**教学目标：** 区分冷锋/暖锋/准静止锋，判断过境天气

**教学内容：**

```
冷锋：冷气团主动向暖气团移动
→ 过境前：暖气团控制，气温高
→ 过境时：大风、降温、阴雨（暴雨/大雪）
→ 过境后：冷气团控制，气温降低，气压升高，天气转晴
→ 举例：北方冬春沙尘暴、冬季寒潮、夏季暴雨

暖锋：暖气团主动向冷气团移动
→ 过境前：冷气团控制，气温低
→ 过境时：连续性降水（雨区在锋前）
→ 过境后：暖气团控制，气温升高，气压降低，天气转晴

准静止锋：冷暖气团势均力敌，来回摆动
→ 江淮准静止锋（6月）：梅雨
→ 昆明准静止锋（冬）：贵阳阴雨，昆明晴朗
→ 天山准静止锋（冬）：新疆北部降雪
```

**判断口诀：**

```
冷锋：冷气团主动推 → 雨在锋后 → 大风降温
暖锋：暖气团主动推 → 雨在锋前 → 连续性降水
准静止锋：来回摆 → 持续阴雨
```

**过关标准：** 给天气变化描述能判断锋面类型，能说出现实中3个例子

### 第12次课：地形与河流

**教学目标：** 掌握外力作用和河流水文特征分析

**教学内容：**

1. 外力作用

```
风化：岩石破碎（物理/化学/生物）
侵蚀：
  流水侵蚀→V型谷/峡谷（上游）
  风力侵蚀→雅丹地貌/戈壁（西北）
  冰川侵蚀→U型谷/角峰/冰斗（高海拔）
  海浪侵蚀→海蚀崖/海蚀柱
搬运：流水/风力搬运泥沙
堆积：
  流水堆积→冲积平原/三角洲/冲积扇（中下游）
  风力堆积→沙丘/黄土高原
  冰川堆积→冰碛地貌
```

2. 河流水文特征

```
必须记住的分析框架：
- 流量：大/小（受气候和流域面积影响）
- 汛期：长/短，有无春汛+夏汛
- 含沙量：大/小（受植被覆盖和土质影响）
- 结冰期：有/无（受气温影响）
- 流速：快/慢（受地形落差影响）
- 水能：丰富/贫乏（流量×落差）
```

**过关标准：** 能说出4种外力作用对应的典型地貌，能用框架分析河流特征

## 五、第三阶段：答题综合训练（3次课）

### 第14次课：答题规范专项

**教学目标：** 养成分点答题习惯，掌握阅卷得分技巧

**教学内容：**

1. 答题三大原则

```
原则1：分点作答，标序号（①②③④）
原则2：每点一行，先写结论再写分析
原则3：用地理术语，不用大白话
```

2. 常见题型答题框架

```
原因分析题：自然原因（气候+地形+水文）+ 人文原因（经济+社会+政策）
影响评价题：有利影响 + 不利影响（辩证分析）
措施建议题：工程措施 + 生物措施 + 管理措施
差异比较题：从XX角度比较A和B的不同
```

3. 做真题2道，严格按格式写答案

### 第15次课：区域发展+资源调配

**教学目标：** 掌握区域分析和资源调配的答题角度

**教学内容：**

1. 区域发展分析框架

```
自然地理特征：位置+地形+气候+河流+土壤+植被
人文地理特征：人口+城市+农业+工业+交通
区域发展方向：因地制宜，发挥优势
```

2. 资源跨区域调配（模板）

```
调出地（西部/北方）：
  有利：资源转经济优势，带动就业
  不利：生态破坏、移民安置

调入地（东部/南方）：
  有利：缓解资源短缺，促进经济发展
  不利：可能带来环境问题

经典案例：
  西气东输：新疆天然气→上海
  南水北调：长江水→华北/西北
  西电东送：西部水电火电→东部
```

**过关标准：** 能独立分析一道资源调配大题

## 六、过关验证机制

### 每次课的标准流程

```
1. 快速复习上节课内容（5分钟）→ 默写核心模板
2. 新知识讲解（20-25分钟）→ 结合例子讲解
3. 当堂练习（20分钟）→ 做真题或模拟题
4. 总结+笔记本（10分钟）→ 明确要记什么
```

### 过关三步走

```
学（讲模板+举例）→ 练（做真题）→ 测（独立分析）
```

### 没过关怎么处理

1. 重记模板，隔天再测
2. 第二次还错 → 回到知识讲解部分重新理解
3. 第三次还错 → 降低难度，先做简单题建立信心
