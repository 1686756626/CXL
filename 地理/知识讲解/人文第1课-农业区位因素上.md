## 一、什么是"区位"？（3分钟）

"区位"就是"为什么建在这里"。

举例：
> 为什么江西种水稻，新疆种棉花？
> → 因为江西降水多、气温合适；新疆日照强、温差大。
> 这就是"区位因素"——决定某个农业活动放在哪里的因素。

## 二、农业区位分析模板（核心！必须背熟，20分钟）

### 自然因素

```
气候：
- 光照 → 光照充足→作物长得好（新疆瓜果甜）
- 热量 → 热量够→可以种多季（南方一年两三熟）
- 降水 → 降水适量→不旱不涝（太多太少都不行）
- 昼夜温差 → 温差大→糖分积累多（新疆葡萄甜）
- 雨热同期 → 降水和高温同季（中国季风区的优势）

地形：
- 平原 → 利于耕作、机械化
- 山区 → 适合发展林业、畜牧业
- 坡度 → 坡度大不宜耕作（水土流失）

土壤：
- 肥力 → 土壤肥沃→产量高（东北黑土）
- 酸碱度 → 南方红壤偏酸，北方黄土偏碱
- 类型 → 水稻需要黏性土，花生需要沙质土

水源：
- 河流、湖泊、地下水 → 灌溉条件
- 干旱地区水源是决定性因素（绿洲农业）
```

### 社会经济因素

```
市场：
- 市场需求量决定农业类型和规模
- 城市近郊→蔬菜、花卉、乳畜（新鲜、易坏，要离市场近）
- 偏远地区→粮食、棉花（耐储运）

交通：
- 交通运输便利→农产品可以运出去卖
- 冷链运输→扩大了生鲜农产品的销售范围

政策：
- 政府补贴、价格保护、土地政策
- 例：粮食安全政策→保护耕地红线

劳动力：
- 劳动力数量→劳动密集型农业（水稻种植需要大量人工）
- 劳动力素质→影响农业技术水平

技术：
- 种子改良、温室大棚、机械化了
- 冷藏保鲜→扩大销售范围
- 无土栽培→减少对土壤的依赖
```

## 四、经典案例（10分钟，帮助理解）

### 案例1：新疆棉花为什么好？

```
自然因素：
- 光照：晴天多，光照充足 → 棉花光合作用强
- 热量：夏季气温高 → 棉花生长周期够
- 温差：昼夜温差大 → 棉花品质好
- 水源：有高山冰雪融水 → 灌溉水源（绿洲农业）

社会经济因素：
- 政策：国家棉花基地政策支持
- 劳动力：劳动力充足
- 市场：国内外市场需求大
```

### 案例2：为什么城郊种蔬菜不种水稻？

```
→ 市场因素决定！
- 蔬菜：需求量大、易腐烂、要新鲜 → 必须离市场近
- 水稻：耐储运 → 可以在远离城市的地方种植
- 城郊地价高 → 种高附加值的蔬菜、花卉更划算
```

## 五、易考辨析（选择题陷阱）

### 陷阱1：自然因素 vs 社会经济因素
- 气候、地形、土壤、水源 → **自然**
- 市场、交通、政策、技术、劳动力 → **社会经济**
- ❌ "技术进步改良了土壤" → 技术是社会经济因素，但土壤是自然因素

### 陷阱2：主导因素判断
- 新疆绿洲农业 → **水源**（没有水什么都种不了）
- 城郊蔬菜基地 → **市场**（因为靠近城市需求大）
- 东北大豆 → **土壤**（黑土肥沃）
- 海南热带水果 → **气候**（热量充足）

**判断方法：** 哪个因素没有就不行，哪个就是主导因素。

### 陷阱3：区位因素的变化
- 自然因素相对稳定（不能轻易改变）
- 社会经济因素变化快（市场变了、技术进步了）
- **大题常考：** "随着交通冷藏技术发展，XX农业区位发生了什么变化？"
  → 交通改善+冷藏技术 → 农产品可以运更远 → 区位选择更灵活

## 七、测试题（至少8道）

### 选择题（4道）

**1.** 新疆能种植优质棉花的主要自然原因是（）
A. 劳动力丰富
B. 光照充足，昼夜温差大
C. 降水丰富
D. 政策支持

**2.** 城郊农业以蔬菜、花卉为主，主要考虑的区位因素是（）
A. 气候
B. 地形
C. 市场
D. 土壤

**3.** 下列属于社会经济区位因素的是（）
A. 光照
B. 降水
C. 交通
D. 地形

**4.** 随着冷藏保鲜技术的发展，对农业区位影响最大的是（）
A. 气候条件改善
B. 农产品销售范围扩大
C. 土壤肥力提高
D. 劳动力需求减少

### 简答题（4道）

**5.** 分析东北平原种植大豆的有利自然条件（至少4点）。

**6.** 为什么城郊种蔬菜而不是粮食？（从市场和地价角度分析）

**7.** "新疆的绿洲农业，主导区位因素是水源。"请解释为什么。

**8.** 读以下材料，分析该地发展农业的有利条件：
> 某地位于亚热带季风气候区，年降水量1200mm，地势平坦，土壤肥沃，靠近大城市，高速公路通过。

---

### 答案

1. **B**（劳动力、政策是社会经济因素，新疆降水少不选C）
2. **C**（城郊靠近城市市场，蔬菜需求大、要新鲜）
3. **C**（交通是社会经济因素，ABD都是自然因素）
4. **B**（冷藏技术→生鲜能运更远→销售范围扩大）
5. ①地形：平原广阔，地势平坦，利于耕作；②土壤：黑土广布，土壤肥沃；③气候：夏季热量充足，雨热同期；④水源：河流多，灌溉水源充足
6. 市场因素：蔬菜需求量大、易腐烂，必须靠近消费市场。地价因素：城郊地价高，种蔬菜等高附加值农产品更划算。
7. 新疆气候干旱，降水稀少。没有水源（高山冰雪融水），根本无法进行农业生产。因此水源是最关键的、没有就不行的因素。
8. 自然：①气候：亚热带季风，雨热同期；②地形：平坦利于耕作；③土壤：肥沃；④水源：降水1200mm充足。社会经济：⑤市场：靠近大城市；⑥交通：高速公路便利。
