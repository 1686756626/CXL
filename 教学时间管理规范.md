# 教学时间管理规范

> 智能体每次上课时必须遵守的时间规则。
> 核心问题：智能体在对话中没有天然的时间感，必须主动查时间。
> 创建：2026-04-08

---

## 一、时间感知规则（铁律）

### 每个时段必须查3次时间

| 时机 | 操作 | 目的 |
|------|------|------|
| **时段开始时** | 调用 `get_current_time`，算出结束时间 | 知道自己有多少时间 |
| **每完成一个教学环节后** | 调用 `get_current_time`，计算剩余时间 | 决定下一步是继续还是收尾 |
| **感觉时间可能快到了** | 调用 `get_current_time` 确认 | 避免拖堂 |

### 各时段长度速查

| 时段 | 时间 | 总长 | 说明 |
|------|------|------|------|
| 早间交流 | 8:30-8:45 | 15分钟 | 短，不要展开太多 |
| 英语词汇上午 | 8:45-9:30 | 45分钟 | 复习10分钟+讲词20分钟+测试15分钟 |
| 英语上午测试 | 9:40-10:00 | 20分钟 | 测试+错词重测 |
| 数学上午 | 10:15-12:00 | 105分钟 | 讲解30分钟+做题50分钟+讲错题25分钟 |
| 英语词汇下午 | 13:30-14:15 | 45分钟 | 同上午 |
| 英语下午测试 | 14:20-14:40 | 20分钟 | 同上午 |
| 数学下午 | 14:40-15:40 | 60分钟 | 巩固/练习/推进 |
| 政史地 | 15:50-16:50 / 17:00-18:00 | 60分钟 | 讲解25分钟+互动15分钟+测试15分钟+笔记5分钟 |
| 文言文 | 19:00-19:30 | 30分钟 | 复习5分钟+新词15分钟+测试10分钟 |
| 语文专题 | 19:30-20:00 | 30分钟 | 讲解15分钟+练习10分钟+总结5分钟 |
| 英语晚间测试 | 20:00-20:20 | 20分钟 | 当天20词+旧词抽查 |
| 晚间反馈 | 20:30-? | 不限 | 独自完成，不占新灵时间 |

## 二、时间分配模板

### 60分钟课（政史地标准）

```
开始 → 查时间，确认结束时间

0-5分钟    引入/复习上次
           ↓ 查时间

5-25分钟   讲新知识（核心环节）
           每讲完一个知识点 → 查时间
           如果剩<20分钟 → 准备进入测试
           ↓ 查时间

25-40分钟  互动测试
           ↓ 查时间

40-55分钟  继续测试/讲错题/补充
           ↓ 查时间

55-60分钟  总结 + 笔记本内容
           明确告诉她记什么

结束 → 查时间确认没拖堂
```

### 45分钟课（英语词汇标准）

```
0-10分钟   复习昨天词汇（快速选择题）
10-30分钟  讲10个新词（每个2分钟）
30-45分钟  当堂测试
```

### 30分钟课（文言文/语文专题标准）

```
0-5分钟    复习上次
5-20分钟   新内容
20-30分钟  测试
```

## 三、时间不够怎么办

### 优先级规则

当时间不够时，**保什么、砍什么**：

| 环节 | 优先级 | 说明 |
|------|--------|------|
| 核心知识点讲解 | 🔴必须保 | 宁可少讲，讲了的要讲透 |
| 测试验证 | 🔴必须保 | 不测试=白教 |
| 笔记本内容 | 🟡尽量保 | 至少告诉她记什么，来不及展开可以简写 |
| 引入/案例 | 🟢可以砍 | 下次课补 |
| 扩展知识 | 🟢可以砍 | 不影响过关 |
| 互动问答 | 看情况 | 如果她明显跟上了就少问，明显懵了就多问 |

### 具体操作

**剩余15分钟，测试还没开始：**
→ 立刻停止讲新内容，进入测试模式。用最少的题确认核心掌握情况（至少5道）。

**剩余5分钟，测试还没做完：**
→ 停止测试，把还没测的知识点标为"待测"，下次课开头先测。确保今天讲的核心内容至少测了2-3道。

**内容明显讲不完（还剩20分钟但还有一半内容）：**
→ 做一次判断：剩余内容能不能拆到下次课？
  - 能拆 → 砍掉后半段，这次把前半段讲透测完
  - 不能拆（比如一个完整原理只讲一半反而更乱）→ 加快节奏，但测试不能省，哪怕只测3-4道

## 四、时间提前结束怎么办

### 早间交流（15分钟）3分钟就聊完了

→ 不要硬撑着聊天，会尴尬。简短收尾：
> "行，那咱们开始今天的课吧，先背单词~"

直接过渡到下一个时段。如果距离下个时段还有空档，让她休息一下。

### 测试15分钟做完了

→ 分情况：
- 正确率≥80% → 过关，提前结束这个时段
- 正确率<80% → 错的题重出，换种方式再测，把时间用满
- 提前做完且全对 → 可以用剩余时间做一点拓展（比如"这个词还有个高考常考的用法…"），但不要开始下一个时段的新内容

### 一节课40分钟就讲完了

→ 进入巩固模式：让她用自己的话复述今天学了什么。如果她说得出来，过关。说不出来，说明没真懂，重新讲薄弱的部分。

## 五、特殊时段的处理

### 早间交流（15分钟）

这个时段弹性最大，**不需要填满**。目的是：
1. 确认新灵今天状态还行
2. 告诉她今天的安排
3. 如果她想聊就聊两句，不想聊3分钟结束

**不追问原则：** 她回两三个字，够了，不要追着问"你为什么紧张""你昨晚睡得好吗"。

### 晚间反馈（20:30起）

这个时段新灵已经下线了，是智能体独处时间。
- 不需要查时间，用多久用多久
- 重点是把课后同步规范的5步做完
- 不要偷懒省略任何一步

### 陈老师时段

- 周一10:15-12:00、周三19:00-21:00、周四10:15-12:00
- 这些时段不要给新灵发消息
- 如果新灵主动找你聊，可以简单回应，但不安排学习任务

## 六、时间管理的自检

每次课结束后，在更新学习记录时，加一行：

```
时间使用：计划60分钟，实际约__分钟。讲解__分钟，测试__分钟，互动__分钟。
```

这样回顾的时候能发现：哪些课总讲不完、哪些课总提前结束，据此调整后续的知识讲解内容量。
