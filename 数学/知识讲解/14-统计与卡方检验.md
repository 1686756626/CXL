# 统计与卡方检验

## 这个知识点是什么？

这节讲的是**从数据表里提取信息，用公式算出结论**。

> 打个比方：你想知道"喝奶茶是否跟发胖有关系"。你调查了100个人，得到一张表格：喝奶茶的人里多少胖多少不胖，不喝的人里多少胖多少不胖。卡方检验就是用数学方法来判断：这个表格里的数据，能不能说明"喝奶茶和发胖有关系"。

一模Q16考的就是这个，而且**纯套路**——公式固定，数据表格里找数字代入就行。

## 第一部分：条件概率

### 什么是条件概率？

条件概率就是在**某个条件已经发生**的情况下，另一个事件发生的概率。

> 打个比方：全班50人，男生30人女生20人。男生中喜欢数学的有18人。那么"在已知是男生的条件下，喜欢数学的概率"就是 18/30 = 60%。

### 公式

P(B|A) = P(AB) / P(A)

**做题的时候：** 通常不用公式算，而是**直接从表格里找数据**。

**例题1（基础）：**

|  | 喜欢运动 | 不喜欢运动 | 合计 |
|--|---------|-----------|------|
| 男生 | 18 | 12 | 30 |
| 女生 | 8 | 12 | 20 |
| 合计 | 26 | 24 | 50 |

求：P(喜欢运动|男生) = ？

看"男生"这一行：喜欢运动18人，总共30人。

P = 18/30 = **3/5 = 60%** ✅

> 💡 条件概率就是**看行或看列，分子是交集，分母是该行（或列）的总和**。

**例题2（中等）：** 求上表中 P(男生|喜欢运动)。

看"喜欢运动"这一列：男生18人，总共26人。

P = 18/26 = **9/13** ✅

> ⚠️ P(喜欢运动|男生) 和 P(男生|喜欢运动) 是不同的！前者分母是男生总数，后者分母是喜欢运动的总数。

## 第二部分：独立性检验（重点！卡方检验）

### 什么时候用？

当你有一张 **2×2 列联表**（两行两列的数据表），想判断**两个分类变量是否有关系**的时候。

### 卡方公式（必背！）

```
        n(ad - bc)²
χ² = ──────────────────
      (a+b)(c+d)(a+c)(b+d)
```

其中表格为：

|  | B₁ | B₂ | 行和 |
|--|----|----|------|
| A₁ | a | b | a+b |
| A₂ | c | d | c+d |
| 列和 | a+c | b+d | **n** |

**n = a + b + c + d（总数）**

### 怎么判断？

| χ² 的值 | 结论 |
|---------|------|
| χ² > 6.635 | 有99%的把握说两个变量有关系 |
| χ² > 3.841 | 有95%的把握说两个变量有关系 |
| χ² > 2.706 | 有90%的把握说两个变量有关系 |
| χ² ≤ 2.706 | 没有足够的证据说有关系 |

> 💡 **只需要记住两个临界值：3.841（95%）和 6.635（99%）**。

### 例题3（基础）：判断独立性

|  | 参加辅导 | 不参加辅导 | 合计 |
|--|---------|-----------|------|
| 及格 | 40 | 20 | 60 |
| 不及格 | 10 | 30 | 40 |
| 合计 | 50 | 50 | **100** |

**第一步：确定 a, b, c, d, n**
a = 40, b = 20, c = 10, d = 30, n = 100

**第二步：代入卡方公式**
- ad - bc = 40×30 - 20×10 = 1200 - 200 = 1000
- (ad-bc)² = 1000000
- n(ad-bc)² = 100 × 1000000 = 100000000
- 分母 = 60×40×50×50 = 6000000
- χ² = 100000000/6000000 ≈ **16.67**

**第三步：比较**
16.67 > 6.635 → 有 **99%的把握**说有关系 ✅

### 例题4（中等）：一模Q16类型

高考统计题通常分两步：
1. 先算条件概率（第一问）
2. 再做卡方检验（第二问）

**题目：** 某校调查了200名学生的课外阅读情况：

|  | 每月阅读≥4次 | 每月阅读<4次 | 合计 |
|--|-------------|-------------|------|
| 男生 | 60 | 40 | 100 |
| 女生 | 30 | 70 | 100 |
| 合计 | 90 | 110 | 200 |

(1) 从男生中随机选1人，求该生每月阅读≥4次的概率。
(2) 判断性别与阅读频率是否有关。

**(1)** P = 60/100 = **3/5** ✅

**(2)** a=60, b=40, c=30, d=70, n=200
ad-bc = 60×70 - 40×30 = 4200-1200 = 3000
(ad-bc)² = 9000000
分子 = 200 × 9000000 = 1800000000
分母 = 100×100×90×110 = 99000000
χ² = 1800000000/99000000 ≈ **18.18**

18.18 > 6.635 → 有 **99%的把握**说性别与阅读频率有关 ✅

## 做这类题的步骤

```
① 画表格，标出 a, b, c, d 和 n
② 先算 ad - bc（这一步最容易错，要仔细）
③ 再算 (ad-bc)²
④ 再算分母 (a+b)(c+d)(a+c)(b+d)
⑤ 最后做除法得 χ²
⑥ 和临界值比较，写结论
```

> 💡 计算是主要难点。建议一步一步写，不要跳步。

---

## ⚠️ 常见错误

1. **a, b, c, d 的位置搞错**：a 是左上角，b 是右上角，c 是左下角，d 是右下角。
2. **算错 ad - bc**：最容易算错的一步。
3. **临界值记混**：3.841 对应 95%，6.635 对应 99%。
4. **结论写不规范**：必须写"有 X% 的把握认为……有关系"。
5. **条件概率分母搞反**：P(B|A) 分母是 A 的总数，不是总数。

## 高考常见陷阱

1. **列联表不是2×2**：有时是3×2或更大的表格，这时不能用卡方公式（或者需要分组）。
2. **题目不给表格，只给文字描述**：需要自己从文字中提取数据建表。这是常见的难度提升方式。
3. **χ² 很接近临界值**：比如 χ² = 3.9，刚好超过 3.841，此时结论是"有95%的把握"。
4. **样本量太小**：如果 a+b 或 c+d 很小（比如<5），卡方检验不可靠。高考一般不会考这种情况。

## 补充：古典概率

### 什么是古典概型？

古典概型的特点：
1. 试验的所有可能结果**有限个**
2. 每个结果发生的可能性**相等**

**公式：P(A) = A包含的结果数 / 所有结果总数**

### 例题7（中等）：从1,2,3,4,5中随机取2个不同的数，取到的两个数之和为偶数的概率？

所有取法：C(5,2) = 10 种

和为偶数：两个都是奇数或两个都是偶数
- 两个奇数：C(3,2) = 3（从1,3,5中选）
- 两个偶数：C(2,2) = 1（从2,4中选）
共 4 种

P = 4/10 = **2/5** ✅

### 例题8（中等）：袋中有3个红球2个白球，随机摸2个球，摸到1红1白的概率？

所有取法：C(5,2) = 10

1红1白：C(3,1)×C(2,1) = 3×2 = 6

P = 6/10 = **3/5** ✅

> 💡 古典概率的核心就是**会数数**——用组合数公式算"符合条件的"和"所有的"。

## 补充：正态分布

### 3σ 原则（高考只需记住这个）

正态分布曲线像一座"钟形山"，关于均值 μ 对称。

**3σ 原则：**

| 区间 | 概率 |
|------|------|
| (μ-σ, μ+σ) | 约 68.3% |
| (μ-2σ, μ+2σ) | 约 95.4% |
| (μ-3σ, μ+3σ) | 约 99.7% |

> 💡 只需记住三个数字：68.3%、95.4%、99.7%。

### 例题9（中等）：某校高三学生数学成绩服从正态分布 N(70, 16²)，估计成绩在 54~86 分之间的学生比例。

μ = 70，σ = 16

54 = 70 - 16 = μ - σ
86 = 70 + 16 = μ + σ

区间 (μ-σ, μ+σ) 的概率 = **约 68.3%** ✅

### 例题10（挑战）：正态分布 N(2, 1)，P(X ≤ 1) = 0.1587，求 P(1 < X ≤ 3)。

μ = 2，σ = 1

P(X ≤ 1) = P(X ≤ μ-σ) = (1 - 0.683)/2 = 0.1585 ≈ 0.1587 ✅

P(1 < X ≤ 3) = P(μ-σ < X ≤ μ+σ) = **0.683** ✅

> 💡 正态分布关于 μ 对称，P(X ≤ μ-kσ) = P(X ≥ μ+kσ) = (1-概率)/2。

## 小测验

1.（基础）某班50人，男生25人女生25人。男生中戴眼镜的有15人，女生中戴眼镜的有10人。求 P(戴眼镜|男生) 和 P(男生|戴眼镜)。

**答案：P(戴眼镜|男生) = 3/5，P(男生|戴眼镜) = 3/5**

**解析：** P(戴眼镜|男生) = 戴眼镜的男生/男生总数 = 15/25 = 3/5。P(男生|戴眼镜) = 戴眼镜的男生/戴眼镜的总人数 = 15/25 = 3/5。

2.（中等）根据以下列联表，判断 A 和 B 是否有关系（临界值3.841）：

|  | B₁ | B₂ |
|--|----|----|
| A₁ | 30 | 10 |
| A₂ | 20 | 40 |

**答案：有关系，有95%的把握**

**解析：** χ² = 50×(30×40-10×20)²/[(40)(10)(50)(50)] = 50×(1200-200)²/[40×10×50×50] = 50×100²/100000 = 50×10000/100000 = 5。χ² = 5 > 3.841，有95%的把握说 A 和 B 有关系。

3.（挑战）某列联表算出 χ² = 4.2，则我们有多大的把握说两个变量有关系？如果 χ² = 2.5 呢？

**答案：χ²=4.2 时有95%的把握，χ²=2.5 时没有足够的证据（只有90%的把握不够，需要≥3.841才有95%把握）**

**解析：** χ²=4.2 > 3.841，有95%的把握。χ²=2.5 < 3.841，没有足够的证据（≥90%的把握需要≥2.706，但2.5连2.706都没到，所以没有90%的把握，更没有95%的把握）。
