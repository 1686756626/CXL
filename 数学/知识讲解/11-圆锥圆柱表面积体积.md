# 圆锥圆柱表面积体积

## 这个知识点是什么？

这节就是**背公式+用公式+从图形中提取信息**。

> 打个比方：你知道长方形面积 = 长 × 宽、长方体体积 = 长 × 宽 × 高。圆柱和圆锥也一样，就是记几个公式，然后往里代数字。一模Q4考的就是圆锥侧面积，纯套公式题。

## 必背公式

### 圆柱（像易拉罐）

| 名称 | 公式 | 符号说明 |
|------|------|----------|
| 侧面积 | S侧 = 2πrh | r = 底面半径，h = 高 |
| 全面积 | S全 = 2πr² + 2πrh | 两个底 + 侧面 |
| 体积 | V = πr²h | 底面积 × 高 |

> 💡 圆柱的侧面展开是一个**长方形**，长 = 底面周长 2πr，宽 = 高 h，所以侧面积 = 2πrh。

### 圆锥（像冰淇淋甜筒）

| 名称 | 公式 | 符号说明 |
|------|------|----------|
| 侧面积 | S侧 = πrl | r = 底面半径，l = 母线长 |
| 全面积 | S全 = πr² + πrl | 底面 + 侧面 |
| 体积 | V = (1/3)πr²h | 柱体的1/3 |

> 💡 圆锥的侧面展开是一个**扇形**，半径 = 母线 l，弧长 = 底面周长 2πr，所以侧面积 = (1/2)×弧长×半径 = (1/2)×2πr×l = πrl。

### 球

| 名称 | 公式 |
|------|------|
| 表面积 | S = 4πR² |
| 体积 | V = (4/3)πR³ |

> 💡 球的表面积公式记忆：球的表面是"4个半圆面"，S = 4 × πR²。

## 三个关键字母的关系

### 圆柱：r 和 h（没有 l）

圆柱的"母线"就是高 h，所以只需要知道 r 和 h。

### 圆锥：r、h、l 三者通过轴截面三角形联系

圆锥的**轴截面**（过轴切一刀）是一个**等腰三角形**：
- 底边 = 2r（底面直径）
- 两条腰 = l（母线）
- 高 = h（从顶点到底面的垂线）

所以：**r² + h² = l²**（勾股定理！）

> 💡 这是最重要的关系。考试给你的条件可能不会直接给出 r 和 l，而是需要通过轴截面来求。

## 做题的时候怎么用？

### 例题1（基础，一模Q4！）

> 某圆锥的轴截面是一个斜边长为4的等腰直角三角形，则此圆锥的侧面积为（　）
> A. 4√2π　　B. 8√2π　　C. 16√2π　　D. (4√2+4)π

**第一步：画轴截面，确定三角形的结构**

等腰直角三角形，斜边 = 4。

在圆锥轴截面中：
- 底边 = 2r（直径）
- 两腰 = l（母线）
- 高 = h

**高考默认：等腰直角三角形中，直角在顶点（两腰的夹角处）。**

所以两腰 = l = 4/√2 = 2√2（等腰直角三角形中，腰 = 斜边/√2）
底边 = 2r = 4（直角三角形中，斜边在底面）
r = 2

**第二步：套公式**

侧面积 = πrl = π × 2 × 2√2 = **4√2π**

**答案：A** ✅

**检验：** h = l/√2 = 2√2/√2 = 2，验证 r² + h² = 4 + 4 = 8 = l² = 8 ✅

### 例题2（基础）：圆柱全面积

> 已知圆柱的底面半径为3，高为4，求全面积。

底面积 = πr² = 9π（两个底面 = 18π）
侧面积 = 2πrh = 2π × 3 × 4 = 24π
全面积 = 18π + 24π = **42π** ✅

### 例题3（基础）：球的体积

> 已知球的表面积为 16π，求体积。

S = 4πR² = 16π → R² = 4 → R = 2
V = (4/3)πR³ = (4/3)π × 8 = **32π/3** ✅

### 例题4（中等）：展开图问题

> 圆锥的侧面展开图是一个半径为6、圆心角为120°的扇形，求这个圆锥的体积。

**第一步：从展开图提取信息**

扇形半径 = l = 6（母线长）
扇形弧长 = 2πl × (120°/360°) = 2π × 6 × 1/3 = 4π

底面周长 = 弧长 = 4π → 2πr = 4π → r = 2

**第二步：求 h**

r² + h² = l² → 4 + h² = 36 → h² = 32 → h = 4√2

**第三步：求体积**

V = (1/3)πr²h = (1/3)π × 4 × 4√2 = **16√2π/3** ✅

### 例题5（挑战）：组合体

> 一个圆柱和一个圆锥同底等高，底面半径为3，高为4。求它们的全面积之差。

**圆柱全面积：**
S柱全 = 2πr² + 2πrh = 2π×9 + 2π×3×4 = 18π + 24π = 42π

**圆锥：**
l = √(r²+h²) = √(9+16) = 5
S锥全 = πr² + πrl = 9π + 15π = 24π

**全面积之差 = 42π - 24π = 18π** ✅

### 球的切接问题（高考常考！）

"球的内切"和"球的内接"是两类不同的问题：

- **球内切于几何体**：球和几何体的每个面都相切。球心到每个面的距离都等于球半径 R。
- **球内接于几何体**（几何体外接球）：球的半径 = 球心到每个顶点的距离 = R。

**例题6（中等）：** 一个球内切于正方体，正方体棱长为 a，求球的半径 R。

球和正方体的6个面都相切 → 球心到每个面的距离 = R

正方体的对面距离 = a（棱长）

球心在正方体中心，到每个面的距离 = a/2

**答案：R = a/2** ✅

> 💡 正方体内切球直径 = 棱长，R = a/2。

**例题7（中等）：** 一个球内切于圆柱，圆柱的底面半径为3，高为4，求球的半径 R。

球内切于圆柱 → 球和圆柱的上下底面相切，球和圆柱的侧面也相切。

球心在圆柱中心（上下底面中间）。

球的直径 = 圆柱的高 = 4... 不对，还要满足侧面相切。

球和侧面相切 → 球的半径 = 圆柱底面半径？不对。

让我重新想。球内切于圆柱：
- 球和上下底面相切 → 球心到上底面距离 = R，到下底面距离 = R → 2R ≤ h
- 球和侧面相切 → 球心到侧面的距离 = R。球心到侧面的距离 = 圆柱底面半径 r = 3

所以 R = r = 3？但 h = 4，2R = 6 > 4，球比圆柱还高，这不可能。

这说明**球不能同时和上下底面及侧面都相切**（除非 2R = h = 2r）。

实际上，"球内切于圆柱"的条件是 2R = h 且 R = r。这里 h=4, r=3，不满足，所以球**不能**内切于这个圆柱。

换一个正确的题：

**例题7（中等）：** 一个球内切于圆柱，圆柱的底面半径为3，高为6，求球的表面积。

内切条件：R = r = 3，且 2R = h = 6 ✅（满足）

球的表面积 = 4πR² = 4π×9 = **36π** ✅

> 💡 球内切于圆柱的条件：**2R = h = 2r**，即圆柱高等于直径，底面半径等于球半径。

**例题8（挑战）：** 一个球内接于正三棱锥（球和三棱锥的所有顶点都在球面上），正三棱锥的底面边长为 6，侧棱长为 4。求球的半径 R。（了解即可，这道题偏难）

这道题需要用空间直角坐标系或几何方法，对于33分阶段的学生，**记住结论就好**：正方体内接球 R = (√3/2)a，正四面体内接球 R = (√6/4)a。

> 💡 对于球的切接问题，记住几个常见结论比现场推导更实用。

**常见结论速查：**

| 几何体 | 内切球半径 R | 外接球半径 R |
|--------|-------------|-------------|
| 正方体（棱长a） | a/2 | (√3/2)a |
| 正四面体（棱长a） | (√6/12)a | (√6/4)a |
| 圆柱（r, h=2r） | r | — |

## 做这类题的步骤

```
① 画图，标出 r、h、l
② 用勾股定理 r² + h² = l² 求出未知量
③ 选对公式，代入计算
④ 注意：题目问的是侧面积、全面积还是体积？
```

⚠️ **第四步最重要！** 很多同学算了半天，结果题目问侧面积你算了全面积。

---

## ⚠️ 常见错误

1. **r 和 l 搞混**：r 是底面半径，l 是母线（从顶点到底面圆周的距离），不是一回事！
2. **侧面积和全面积搞混**：全面积 = 底面积 + 侧面积，看清楚题目问什么。
3. **圆锥体积忘除以3**：圆锥体积 = (1/3)πr²h，那个 1/3 经常被忘掉。
4. **不会从轴截面找信息**：圆锥的 r、h、l 全部藏在轴截面三角形里，学会画图很重要。
5. **展开图弧长和底面周长的关系搞错**：扇形弧长 = 圆锥底面周长，这是连接展开图和原几何体的桥梁。

## 高考常见陷阱

1. **题目问的是"侧面积"不是"全面积"**：最经典的坑，看了A算成B。
2. **轴截面三角形的直角位置**：等腰直角三角形的直角可能在顶点也可能在底角，要看清楚。
3. **球的内切/外接**：球内切于几何体 → 球心到各面的距离=R；球外接 → 球心到各顶点的距离=R。这类题用构造直角三角形的方法做。
4. **单位换算**：题目给的是直径，你用半径算，或者反过来。

## 小测验

1.（基础）已知圆锥的底面半径为3，母线长为5，求侧面积和全面积。

**答案：侧面积 = 15π，全面积 = 24π**

**解析：** 侧面积 = πrl = π×3×5 = 15π。全面积 = 底面积+侧面积 = πr²+πrl = 9π+15π = 24π。

2.（中等）已知圆柱的侧面展开图是一个长为8π、宽为6的长方形，求这个圆柱的体积。（提示：长 = 底面周长，宽 = 高）

**答案：体积 = 96π**

**解析：** 底面周长 = 8π → 2πr = 8π → r = 4。高 h = 6。体积 = πr²h = π×16×6 = 96π。

3.（挑战）已知圆锥的轴截面是一个等边三角形（边长为6），求体积。（提示：等边三角形的高 = 边长×√3/2）

**答案：体积 = 18√3π**

**解析：** 轴截面是等边三角形（边长6），所以母线 l = 6，底面半径 r = 3。高 h = 6×√3/2 = 3√3。体积 = (1/3)πr²h = (1/3)π×9×3√3 = 9π√3。
