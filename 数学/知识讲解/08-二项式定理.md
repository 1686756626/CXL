# 二项式定理

## 这个知识点是什么？

二项式定理讲的是：**像 (a+b)ⁿ 这样的式子展开后，每一项长什么样**。

好消息是：**二项式定理套路非常固定**，记住公式和做题流程，考试就是送分。

> 打个比方：二项式定理就像一个"模具"，你把数字塞进去，按流程操作，答案就出来了。不需要创造力，只需要熟练度。

## 知识结构图

```
二项式定理

│   ├── (a+b)ⁿ = Σ C(n,k)·aⁿ⁻ᵏ·bᵏ

│   └── C(n,k) = n!/(k!(n-k)!)
│
├── 组合数
│   ├── C(n,0)=1, C(n,1)=n, C(n,n)=1
│   ├── C(n,k) = C(n,n-k)（对称性）
│   └── 杨辉三角（背到n=6）
│
├── 高频考法
│   ├── 求某一项的系数（令指数=目标值求k）
│   ├── 求常数项（令指数=0）
│   ├── 求系数之和（令x=1）
│   └── 求二项式系数最大的项
│
├── 进阶技巧
│   ├── 区分"二项式系数"和"项的系数"

│   └── 赋值法（x=0, x=1, x=-1）
│
└── 关联知识点
    ├── 组合计数
    ├── 多项式（三项展开先用二项式展开两项，再展开第三项）
    └── 数列（杨辉三角与数列关系）
```

## 核心公式（必背）

### 二项式定理

(a + b)ⁿ = C(n,0)·aⁿ + C(n,1)·aⁿ⁻¹·b + C(n,2)·aⁿ⁻²·b² + ... + C(n,n)·bⁿ

### 通项公式（做题的核心！）

**Tₖ₊₁ = C(n,k) · aⁿ⁻ᵏ · bᵏ**

这里：
- **k 的范围**：0, 1, 2, ..., n
- **a 的指数** = n - k
- **b 的指数** = k

### 杨辉三角（背到 n=6）

```
n=0:         1
n=1:       1   1
n=2:     1   2   1
n=3:   1   3   3   1
n=4: 1   4   6   4   1
n=5: 1  5  10  10  5  1
n=6: 1  6  15  20  15  6  1
```

## 做题的时候怎么用？

### 题型一：求展开式中某一项的系数（最常考！）

**例题1（中等，一模Q12类型）：** 求 (x² - 2x)⁴ 展开式中 x⁶ 的系数。

**第一步：写通项**
Tₖ₊₁ = C(4,k) · (x²)⁴⁻ᵏ · (-2x)ᵏ = C(4,k) · (-2)ᵏ · x⁸⁻ᵏ

**第二步：令 x 的指数 = 6**
8 - k = 6 → **k = 2**

**第三步：代入算系数**
C(4,2) · (-2)² = 6 × 4 = **24** ✅

> 💡 三步走：写通项 → 令指数=目标值求k → 代入算系数。

### 题型二：求常数项

"常数项"就是 **x 的指数为 0** 的项。

**例题2（基础）：** (x + 1/x)⁶ 展开式中的常数项。

Tₖ₊₁ = C(6,k) · x⁶⁻ᵏ · x⁻ᵏ = C(6,k) · x⁶⁻²ᵏ

令 6 - 2k = 0 → k = 3

常数项 = C(6,3) = **20** ✅

### 题型三：求系数之和

**例题3（基础）：** (2x + 1)⁵ 展开式中，所有项的系数之和。

**令 x = 1！**

(2×1 + 1)⁵ = 3⁵ = **243** ✅

> 💡 **求系数之和 → 令 x = 1**。这个技巧考试非常管用。

### 题型四：二项式系数 vs 项的系数（高考陷阱）

- **二项式系数** = C(n,k)，只看 n 和 k
- **项的系数** = C(n,k) × 其他系数

**例题4（中等）：** (x - 2)⁵ 展开式中：
- 二项式系数最大的是：C(5,2) = C(5,3) = **10**（k=2 和 k=3）
- k=2 的项的实际系数：C(5,2)×(-2)² = 10×4 = **40**
- k=3 的项的实际系数：C(5,3)×(-2)³ = 10×(-8) = **-80**

> ⚠️ 二项式系数始终为正，但项的系数可能是负数！

### 题型五：赋值法求特殊和（挑战）

**例题5（挑战）：** (x + 1)⁵ 展开式中，奇数项的系数之和是多少？

**令 x = 1：** 系数总和 = 2⁵ = 32
**令 x = -1：** 奇数项系数 - 偶数项系数 = (-1+1)⁵ = 0

设奇数项系数和为 S₁，偶数项系数和为 S₂：
S₁ + S₂ = 32
S₁ - S₂ = 0

**S₁ = 16** ✅

> 💡 赋值法口诀：**x=1 求总和，x=-1 求奇偶差**。

## 做题流程总结

```
① 写通项：Tₖ₊₁ = C(n,k) · aⁿ⁻ᵏ · bᵏ
② 找 k：令 x 的指数 = 题目要求的值，解出 k
③ 算答案：把 k 代入，算 C(n,k) 和其他系数
④ 检查：k 是否为正整数？是否在 0~n 范围内？
```

---

## ⚠️ 常见错误

1. **漏掉 b 的负号**：b = -2x 时 (-2)ᵏ 的符号要算对
2. **分不清"二项式系数"和"项的系数"**：考试常挖这个坑
3. **通项公式写错指数**：a 的指数是 n-k，b 的指数是 k
4. **组合数算错**：背熟杨辉三角，常用值直接查表

## 高考常见陷阱

1. **k 的范围检查**：算出的 k 不是正整数说明那一项不存在
2. **三项展开**：如 (1+x+x²)⁵，不能直接用二项式定理，需要分组：(1+x+x²)⁵ = [(1+x)+x²]⁵ 先展开再展开内层
3. **a 或 b 含多个 x**：如 (x²-1/x)⁴，a=x², b=-1/x，通项里 x 的指数要合并
4. **二项式系数最大 vs 项的系数最大**：有负号时两者不同

## 小测验

1.（基础）求 (x + 1)⁵ 展开式中 x³ 的系数。

**答案：10**

**解析：** 通项 Tₖ₊₁ = C(5,k) · x⁵⁻ᵏ · 1ᵏ = C(5,k) · x⁵⁻ᵏ。令 5-k = 3 → k = 2。系数 = C(5,2) = 10。

2.（中等）求 (x - 1/x)⁴ 展开式中的常数项。

**答案：6**

**解析：** 通项 Tₖ₊₁ = C(4,k) · x⁴⁻ᵏ · (-1/x)ᵏ = C(4,k) · (-1)ᵏ · x⁴⁻²ᵏ。令 4-2k = 0 → k = 2。常数项 = C(4,2) · (-1)² = 6。

3.（挑战）求 (2x - 1)⁶ 展开式中所有偶数项的系数之和。（提示：令 x=1 和 x=-1）

**答案：365**

**解析：** 设奇数项系数和为 S₁，偶数项系数和为 S₂。令 x=1：(2-1)⁶ = 1 = S₁ + S₂。令 x=-1：(-2-1)⁶ = 729 = S₂ - S₁。两式相加：2S₂ = 730 → S₂ = 365。
