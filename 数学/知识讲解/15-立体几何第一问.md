# 立体几何第一问

## 这个知识点是什么？

立体几何解答题一般有2-3问。**第一问通常要求你证明线面平行或线面垂直**，这是套路最固定、最容易拿分的一问。

> 打个比方：立体几何第一问就像是证明题里的"送分题"。你只需要背几个判定定理，然后在图上找条件、套格式写证明过程就行了。

## 方法一：纯几何证明

### 证明线面平行

**判定定理：** 如果平面外一条直线平行于平面内的一条直线，那么这条直线平行于这个平面。

```
a ∥ b，b ⊂ α，a ⊄ α → a ∥ α
```

**证明格式：**
```
因为 b ∥ a（已知或已证）
又 b ⊂ 平面α
a ⊄ 平面α
所以 a ∥ 平面α
```

### 证明线面垂直

**判定定理：** 如果一条直线垂直于平面内的两条相交直线，那么这条直线垂直于这个平面。

```
l ⊥ a，l ⊥ b，a ∩ b = O，a ⊂ α，b ⊂ α → l ⊥ α
```

**证明格式：**
```
因为 l ⊥ a（已知或已证）
又 l ⊥ b（已知或已证）
a ∩ b = O
a ⊂ 平面α，b ⊂ 平面α
所以 l ⊥ 平面α
```

## 方法二：空间向量法（推荐掌握！）

> 💡 **33分的同学优先掌握向量法。** 几何法需要空间想象力，向量法只需要算数。

### 建坐标系三步走

```
① 选一个顶点作为原点 O
② 选三条互相垂直的棱作为 x, y, z 轴
③ 标出各点坐标
```

### 求平面的法向量

设平面上有两个不平行向量 u⃗ = (a₁, b₁, c₁) 和 v⃗ = (a₂, b₂, c₂)，设法向量 n⃗ = (x, y, z)：

```
n⃗ · u⃗ = a₁x + b₁y + c₁z = 0 ... ①
n⃗ · v⃗ = a₂x + b₂y + c₂z = 0 ... ②
```

解这个方程组，令某个未知数为1（比如 z=1），求出 x 和 y。

### 向量法证明线面垂直

**条件：** 直线的方向向量 n⃗_line 与平面的法向量 n⃗_plane 平行（成比例关系）

**例题1（基础）：** 正方体 ABCD-A₁B₁C₁D₁（棱长2），证明 AC₁ ⊥ 平面 A₁BD。

**建系：** A 为原点
A=(0,0,0), B=(2,0,0), C=(2,2,0), D=(0,2,0)
A₁=(0,0,2), B₁=(2,0,2), C₁=(2,2,2), D₁=(0,2,2)

**第一步：AC₁ 的方向向量**
AC₁⃗ = C₁ - A = **(2, 2, 2)**

**第二步：求平面 A₁BD 的法向量**
平面 A₁BD 内的两个向量：
A₁B⃗ = (2, 0, -2)
A₁D⃗ = (0, 2, -2)

设法向量 n⃗ = (x, y, z)：
2x - 2z = 0 → x = z
2y - 2z = 0 → y = z

取 z = 1 → n⃗ = **(1, 1, 1)**

**第三步：验证**
AC₁⃗ = (2,2,2) = 2 × (1,1,1) = 2·n⃗

方向向量与法向量成比例 → AC₁ ⊥ 平面 A₁BD ✅

### 向量法证明线面平行

**条件：** ① 方向向量⊥法向量（点乘=0）② 直线上至少一点不在平面内

**例题2（基础）：** 在正方体 ABCD-A₁B₁C₁D₁（棱长2）中，证明 AB₁ ∥ 平面 A₁C₁D。

**建系：** A 为原点
A=(0,0,0), B=(2,0,0), C=(2,2,0), D=(0,2,0)
A₁=(0,0,2), B₁=(2,0,2), C₁=(2,2,2), D₁=(0,2,2)

**第一步：AB₁ 的方向向量**
AB₁⃗ = B₁ - A = **(2, 0, 2)**

**第二步：求平面 A₁C₁D 的法向量**

平面 A₁C₁D 内两个向量：
A₁C₁⃗ = (2, 2, 0)
A₁D⃗ = (0, 2, -2)

设法向量 n⃗ = (x, y, z)：
n⃗ · A₁C₁⃗ = 2x + 2y = 0 → x = -y
n⃗ · A₁D⃗ = 2y - 2z = 0 → y = z

取 y = 1 → n⃗ = **(-1, 1, 1)**

**第三步：验证点乘**
AB₁⃗ · n⃗ = 2×(-1) + 0×1 + 2×1 = -2 + 2 = **0** ✅

**第四步：验证 AB₁ 不在平面 A₁C₁D 内**

A(0,0,0) 不在平面 A₁C₁D 上（平面过 A₁, C₁, D 三点，A 的 z 坐标为0，而平面内的点 z 坐标在 0~2 之间且不全为0）

**结论：AB₁ ∥ 平面 A₁C₁D** ✅

**例题3（中等，纯几何法）：** 在四棱锥 P-ABCD 中，底面 ABCD 是平行四边形，E、F 分别是 PA、PC 的中点。证明：EF ∥ 平面 ABCD。

**证明：**
连接 AC。
在 △PAC 中，E、F 分别是 PA、PC 的中点
所以 EF 是 △PAC 的中位线
EF ∥ AC

又 AC ⊂ 平面 ABCD
EF ⊄ 平面 ABCD（E 在 PA 上，F 在 PC 上，都不在底面）
所以 **EF ∥ 平面 ABCD** ✅

> 💡 纯几何法的关键是找到**面内一条线和面外一条线平行**，常用中位线、平行四边形性质。

## 向量法总结

### 证明线面垂直的正确步骤

```
① 建坐标系，标出各点坐标
② 求直线的方向向量 n⃗
③ 求平面内两个不平行向量
④ 验证 n⃗ 与这两个向量都垂直（点乘都为0）
⑤ 得出结论：直线垂直于平面
```

### 证明线面平行的正确步骤

```
① 建坐标系，标出各点坐标
② 求直线的方向向量 n⃗
③ 求平面的法向量 m⃗
④ 验证 n⃗ · m⃗ = 0（方向向量垂直于法向量）
⑤ 验证直线上至少有一点不在平面内
⑥ 得出结论：直线平行于平面
```

> 💡 **最常用的几何体是四棱锥**（PA⊥底面ABC...），这种天然有垂直关系，建坐标系很方便。

## ⚠️ 常见错误

1. **法向量求错**：解方程组时容易算错，一定要代回两个方程验证。
2. **点乘算错**：正负号、乘法最容易出错。
3. **忘记验证"直线不在平面内"**：证明线面平行时，如果直线就在平面内，那就不是平行了。
4. **建坐标系选错原点**：选一个方便计算（有多个0坐标）的点作为原点。

## 高考常见陷阱

1. **坐标系建错**：三条轴不互相垂直，或者原点选得不好导致坐标计算复杂。
2. **混淆"方向向量"和"法向量"**：方向向量沿直线方向，法向量垂直于平面。
3. **线面平行证明不完整**：只验证了点乘=0，忘了验证线不在面内。
4. **题目不给直角**：如果没有现成的直角（如正方体），需要自己构造或找垂直关系来建系。

## 小测验

1.（基础）在正方体 ABCD-A₁B₁C₁D₁（棱长1）中，写出 A、B、C、D、A₁、B₁、C₁、D₁ 的坐标。

**答案：A(0,0,0), B(1,0,0), C(1,1,0), D(0,1,0), A₁(0,0,1), B₁(1,0,1), C₁(1,1,1), D₁(0,1,1)**

**解析：** A 放在原点，AB 沿 x 轴，AD 沿 y 轴，AA₁ 沿 z 轴。每个顶点按棱长1依次写出坐标。

2.（中等）在四棱锥 P-ABCD 中，底面是边长2的正方形，PA⊥底面，PA=2。用向量法证明 PA ⊥ BD。

**答案：PA⃗·BD⃗ = 0，所以 PA⊥BD**

**解析：** 建系：A 为原点，AB 沿 x 轴，AD 沿 y 轴，AP 沿 z 轴。PA⃗ = (0,0,2)，B(2,0,0)，D(0,2,0)，BD⃗ = D-B = (-2,2,0)。PA⃗·BD⃗ = 0×(-2)+0×2+2×0 = 0，所以 PA⊥BD。

3.（挑战）上题中，求平面 PBC 的法向量。（提示：用 PB⃗ 和 PC⃗ 列方程组）

**答案：法向量为 (1,1,2)**

**解析：** P(0,0,2)，B(2,0,0)，C(2,2,0)。PB⃗ = (2,0,-2)，PC⃗ = (2,2,-2)。设法向量 n⃗ = (x,y,z)，则 n⃗·PB⃗ = 2x-2z=0 → z=x，n⃗·PC⃗ = 2x+2y-2z=0。代入 z=x：2x+2y-2x=0 → y=0。法向量 (x,0,x)，取 x=1 得 (1,0,1)。验证：(1,0,1)·(2,0,-2)=2-2=0✓，(1,0,1)·(2,2,-2)=2-2=0✓。答案：**(1,0,1)** 或其任意倍数。
