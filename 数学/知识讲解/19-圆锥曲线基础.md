# 圆锥曲线基础（椭圆双曲线离心率）

## 这个知识点是什么？

这节不是讲圆锥曲线的全部（联立方程、弦长、面积等大题我们不碰），只讲**离心率**——这是圆锥曲线在高考里最常考的基础知识点，几乎每年填空题都会出一道。

> 打个比方：离心率就像描述"圆有多不圆"。离心率越接近0越接近圆，越大越"扁"。高考就考你"知道这个概念+会算"。

## 知识结构图

```
圆锥曲线基础
├── 椭圆
│   ├── 标准方程：x²/a² + y²/b² = 1（a>b>0）
│   ├── c² = a² - b²
│   ├── 离心率：e = c/a（0 < e < 1）
│   └── e 越大越扁，e 越小越圆
│
├── 双曲线
│   ├── 标准方程：x²/a² - y²/b² = 1（a>0, b>0）
│   ├── c² = a² + b²
│   ├── 离心率：e = c/a（e > 1）
│   └── e 越大开口越大
│
├── 抛物线
│   ├── 标准方程：y² = 2px（p>0）
│   ├── 焦点：(p/2, 0)
│   └── 准线：x = -p/2
│
├── 高频考法
│   ├── 求离心率（给方程求e）
│   ├── 给条件求离心率（给a,b,c的关系）
│   ├── 椭圆/双曲线判定
│   └── 焦点三角形相关
│
└── 关联知识点
    ├── 向量（焦点距离用坐标算）
    ├── 三角函数（焦点三角形中用余弦定理）
    └── 函数（离心率范围）
```

## 第一部分：椭圆

### 标准方程

**x²/a² + y²/b² = 1**（a > b > 0）

- a：半长轴（长的一半）
- b：半短轴（短的一半）
- c：半焦距（焦点到中心的距离）
- 关系：**c² = a² - b²**

### 离心率

**e = c/a**

因为 c < a，所以 **0 < e < 1**。

e 越接近 0 → 越像圆
e 越接近 1 → 越扁

### 例题1（基础）：椭圆 x²/9 + y²/4 = 1 的离心率是多少？

a² = 9 → a = 3
b² = 4 → b = 2
c² = a² - b² = 9 - 4 = 5 → c = √5

e = c/a = √5/3 ✅

### 例题2（中等）：椭圆经过点 (2, √3)，离心率为 √3/2，求椭圆方程。

e = c/a = √3/2，设 a = 2k，c = √3 k
c² = a² - b² → 3k² = 4k² - b² → b² = k² → b = k

椭圆方程：x²/4k² + y²/k² = 1

代入 (2, √3)：4/4k² + 3/k² = 1 → 1/k² + 3/k² = 1 → 4/k² = 1 → k² = 4 → k = 2

a = 4, b = 2

方程：**x²/16 + y²/4 = 1** ✅

## 第二部分：双曲线

### 标准方程

**x²/a² - y²/b² = 1**（a > 0, b > 0）

- a：半实轴
- b：半虚轴
- c：半焦距
- 关系：**c² = a² + b²**（注意：和椭圆不同，是加号！）

### 离心率

**e = c/a**

因为 c > a，所以 **e > 1**。

### 例题3（基础）：双曲线 x²/4 - y²/5 = 1 的离心率是多少？

a² = 4 → a = 2
b² = 5
c² = a² + b² = 4 + 5 = 9 → c = 3

e = c/a = 3/2 ✅

### 例题4（中等）：双曲线的离心率为 2，一个焦点为 (3, 0)，求双曲线方程。

c = 3，e = c/a = 2 → a = c/2 = 3/2

c² = a² + b² → 9 = 9/4 + b² → b² = 27/4

方程：**x²/(9/4) - y²/(27/4) = 1**

化简（两边乘4）：**4x²/9 - 4y²/27 = 1** ✅

## 第三部分：抛物线（了解即可）

### 标准方程

**y² = 2px**（p > 0，开口向右）

- 焦点：(p/2, 0)
- 准线：x = -p/2

p 的含义：焦点到准线的距离。

### 例题5（基础）：抛物线 y² = 8x 的焦点坐标和准线方程。

2p = 8 → p = 4

焦点：**(2, 0)**
准线：**x = -2** ✅

## 做这类题的步骤

```
椭圆：
① 从方程读出 a² 和 b²
② 用 c² = a² - b² 求 c
③ e = c/a

双曲线：
① 从方程读出 a² 和 b²
② 用 c² = a² + b² 求 c
③ e = c/a

注意：椭圆 c² = a² - b²（减），双曲线 c² = a² + b²（加）！
```

---

## ⚠️ 常见错误

1. **椭圆和双曲线的 c² 公式搞混**：椭圆是 c² = a² - b²（减），双曲线是 c² = a² + b²（加）
2. **a 和 b 的大小关系**：椭圆 a > b，双曲线 a 和 b 没有大小限制
3. **离心率范围**：椭圆 0 < e < 1，双曲线 e > 1
4. **方程写错加减号**：椭圆是 + 号（x²/a² + y²/b² = 1），双曲线是 - 号

## 高考常见陷阱

1. **方程中 a² 和 b² 的位置**：x²/a² + y²/b² = 1 中，x 下面的 a² 是长轴（大），y 下面的 b² 是短轴（小）。但如果方程写成 x²/4 + y²/9 = 1，那 a²=9（在 y 下面），说明焦点在 y 轴上
2. **双曲线的渐近线**：x²/a² - y²/b² = 1 的渐近线是 y = ±(b/a)x，高考偶尔会考
3. **椭圆上一点到两焦点距离之和 = 2a**：这个性质填空题偶尔考
4. **"离心率增大"的含义**：椭圆 e 增大→更扁；双曲线 e 增大→开口更大

## 小测验

1.（基础）求椭圆 x²/25 + y²/9 = 1 的离心率和焦点坐标。

**答案：e = 4/5，焦点坐标 (±4, 0)**

**解析：** a²=25, b²=9, c²=a²-b²=16, c=4。e = c/a = 4/5。焦点在 x 轴上：(±4, 0)。

2.（中等）已知双曲线的一个顶点为 (2,0)，离心率为 3/2，求双曲线方程。

**答案：x²/4 - 5y²/4 = 1**

**解析：** 顶点 (2,0) → a=2。e=c/a=3/2 → c=3。c²=a²+b² → 9=4+b² → b²=5。方程：x²/4 - y²/5 = 1。

3.（挑战）椭圆 x²/a² + y²/b² = 1（a>b>0）的离心率为 1/2，F₁、F₂ 为焦点，P 为椭圆上一点，且 ∠F₁PF₂ = 60°，求△F₁PF₂ 的面积。（提示：用 |PF₁|+|PF₂|=2a 和余弦定理）

**答案：S = √3 b²**

**解析：** 设 |PF₁|=m, |PF₂|=n，则 m+n=2a。余弦定理：|F₁F₂|²=m²+n²-2mn·cos60° → 4c²=m²+n²-mn。又 m²+n²=(m+n)²-2mn=4a²-2mn。代入：4c²=4a²-2mn-mn=4a²-3mn → 3mn=4(a²-c²)=4b² → mn=4b²/3。S=(1/2)mn·sin60°=(1/2)·(4b²/3)·(√3/2)=√3b²/3。e=1/2 时 c=a/2, b²=a²-c²=3a²/4, 所以 b²=3a²/4。代入：S=√3·(3a²/4)/3=√3a²/4。也可以直接写 S=√3·b²/3。

**修正答案：S = √3·b²/3**（验算：m+n=2a, mn=4b²/3, S=½·mn·sin60°=½·(4b²/3)·(√3/2)=√3b²/3）
