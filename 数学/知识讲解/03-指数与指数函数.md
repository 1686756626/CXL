# 指数与指数函数

## 这个知识点是什么？

"指数"就是"几次方"的意思。2³ = 8，这里3就是指数。指数函数就是把"底数固定、指数当变量"的函数——比如 y = 2ˣ。

> 打个比方：指数就像"复利"。存100块钱，每年利息10%，一年后110，两年后121，三年后133.1……增长越来越快。指数函数就是描述这种"越涨越猛"或"越降越猛"的数学工具。

## 第一部分：指数运算法则（必背！）

| 法则 | 公式 | 例子 |
|------|------|------|
| 同底相乘 | aᵐ·aⁿ = aᵐ⁺ⁿ | 2³·2⁴ = 2⁷ = 128 |
| 同底相除 | aᵐ÷aⁿ = aᵐ⁻ⁿ | 2⁵÷2² = 2³ = 8 |
| 幂的幂 | (aᵐ)ⁿ = aᵐⁿ | (2³)² = 2⁶ = 64 |
| 0次方 | a⁰ = 1 | 5⁰ = 1 |
| 负指数 | a⁻ⁿ = 1/aⁿ | 2⁻³ = 1/8 |
| 乘积的幂 | (ab)ⁿ = aⁿ·bⁿ | (2·3)² = 4·9 = 36 |

> ⚠️ **注意：aᵐ + aⁿ 不能合并！** 2³ + 2⁴ = 8 + 16 = 24，不是 2⁷。

### 例题1（基础）：化简

**(a)** (2³)² ÷ 2⁴ = 2⁶ ÷ 2⁴ = 2² = **4** ✅

**(b)** 3⁻² × 3⁵ = 3³ = **27** ✅（负指数直接加）

**(c)** (2·5)² = 4·25 = **100** ✅

### 例题2（中等）：根式与指数互化

**(a)** √9 = 9^(1/2) = **3** ✅

**(b)** ∛(8) = 8^(1/3) = **2** ✅

**(c)** 4^(3/2) = (√4)³ = 2³ = **8** ✅

> 💡 a^(m/n) 就是先开n次根号再m次方，或者先m次方再开n次根号，结果一样。

## 第二部分：指数函数 y = aˣ

### 关键性质

| | a > 1 | 0 < a < 1 |
|--|-------|-----------|
| 图像 | 从左下到右上 | 从左上到右下 |
| 过点 | (0, 1) | (0, 1) |
| 单调性 | 单调递增 | 单调递减 |
| x→+∞ | y→+∞ | y→0 |
| x→-∞ | y→0 | y→+∞ |

> 💡 无论 a>1 还是 0<a<1，图像**一定过点 (0,1)**，因为 a⁰ = 1。

### 例题3（中等）：比较大小

**(a)** 2^(1.5) 和 2^(2.3)

底数相同（都是2，>1），指数大的更大。**2^(1.5) < 2^(2.3)** ✅

**(b)** (1/2)^(0.3) 和 (1/2)^(0.8)

底数相同（都是1/2，<1），指数大的反而更小。**(1/2)^(0.3) > (1/2)^(0.8)** ✅

**(c)** 3^(0.5) 和 2^(0.8)

底数不同，指数也不同。找中间量：3^(0.5) = √3 ≈ 1.732，2^(0.8)：2 = 2^(1) > 2^(0.8) > 2^(0.5) = √2 ≈ 1.414，所以 2^(0.8) 在 1.414 和 2 之间。**3^(0.5) < 2^(0.8)** ✅

### 例题4（中等）：解指数不等式

**2ˣ > 8**

8 = 2³，所以 2ˣ > 2³。因为底数 2 > 1（递增），指数不等号方向不变。

**x > 3** ✅

**(1/2)ˣ < 1/8**

1/8 = (1/2)³，所以 (1/2)ˣ < (1/2)³。底数 1/2 < 1（递减），不等号方向**反转**。

**x > 3** ✅

> ⚠️ **底数大于1，不等号不变；底数小于1，不等号反转！** 这个最容易搞错。

### 例题5（中等）：指数函数图像识别

y = 2ˣ 和 y = (1/2)ˣ 的图像有什么关系？

(1/2)ˣ = 2⁻ˣ，所以 y = (1/2)ˣ 的图像就是 y = 2ˣ 关于**y轴对称**的图像 ✅

### 例题6（中等，综合）：已知 2ˣ ≤ 16，求 x 的范围。

16 = 2⁴，2ˣ ≤ 2⁴，底数2>1递增，**x ≤ 4** ✅

---

## ⚠️ 常见错误

1. **aᵐ + aⁿ 合并**：2³ + 2⁴ ≠ 2⁷，只能算出 8+16=24
2. **负指数搞错**：a⁻² = 1/a²，不是 -a²
3. **指数不等式方向**：底数<1时不等号要反转
4. **0次方忘记**：任何非零数的0次方都是1（0⁰无意义）

## 高考常见陷阱

1. **底数为负数时**：(-2)² = 4，但(-2)^(1/2) 无意义（偶数次根号下不能是负数）
2. **比较大小找错中间量**：3^(0.5)和2^(0.8)比较时，不能直接用1做中间量（两个都比1大）
3. **指数函数过(0,1)不是过(1,0)**：a⁰=1，所以横坐标0、纵坐标1
4. **分数指数和根式混淆**：(-1)^(1/2) 无意义，但 (-1)^(1/3) = -1（奇数次根号可以有负数）

## 小测验

1.（基础）化简：2⁵ × 2⁻² ÷ 2³

**答案：1**

**解析：** 2⁵ × 2⁻² ÷ 2³ = 2^(5+(-2)-3) = 2⁰ = 1。

2.（中等）解不等式：3ˣ ≥ 27

**答案：x ≥ 3**

**解析：** 27 = 3³，3ˣ ≥ 3³，底数3>1递增，x ≥ 3。

3.（挑战）比较 (1/3)^(0.4) 和 (1/3)^(0.6) 的大小。

**答案：(1/3)^(0.4) > (1/3)^(0.6)**

**解析：** 底数1/3<1，递减函数，指数越大值越小。0.4 < 0.6，所以 (1/3)^(0.4) > (1/3)^(0.6)。
