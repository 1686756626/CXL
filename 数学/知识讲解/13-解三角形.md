# 解三角形

## 这个知识点是什么？

"解三角形"就是**已知三角形的某些信息（边或角），求出其他的边和角**。

> 打个比方：你知道三角形的两条边和一个角，想求第三条边。就像你知道出发地和目的地之间的部分信息，要算出剩余的。两个工具就能搞定：**正弦定理**和**余弦定理**。

你一模Q15第一问做对了，说明有基础。现在要巩固到**每次都能做对**，并且拿下第二问。

## 知识结构图

```
解三角形
├── 正弦定理 a/sinA = b/sinB = c/sinC = 2R
│   ├── 适用：两边一对角、两角一边
│   ├── 注意：可能有两个解（钝角情况）
│   └── 推论：a/sinA = b/sinB → a=bsinA/sinB
│
├── 余弦定理 a²=b²+c²-2bc·cosA
│   ├── 适用：三边求角、两边夹角求第三边
│   ├── 变形：cosA=(b²+c²-a²)/(2bc)
│   └── 特殊情况：c²=a²+b²（勾股定理）
│
├── 面积公式 S=(1/2)ab·sinC
│
├── 解题策略
│   ├── 正弦定理和余弦定理结合使用
│   ├── 算出一个新量→加入已知→再用
│   └── 恒等变换：tan→sin/cos→化简
│
└── 关联知识点
    ├── 三角函数公式（sin²+cos²=1等）
    ├── 基本不等式（面积最值）
    └── 向量数量积（a·b=|a||b|cosθ）
```

## 两个核心工具

### 正弦定理

```
a/sinA = b/sinB = c/sinC = 2R
```

**什么时候用：**
- 知道**两边一对角**，或**两角一边**
- 需要把边和角互化

**例题1（基础）：** △ABC 中，a = 4, A = 30°, B = 45°，求 b。

a/sinA = b/sinB
4/sin30° = b/sin45°
4/(1/2) = b/(√2/2)
8 = b × √2
b = 8/√2 = **4√2** ✅

### 余弦定理

```
a² = b² + c² - 2bc·cosA
cosA = (b² + c² - a²) / (2bc)
```

**什么时候用：**
- 知道**三边**求角
- 知道**两边和夹角**（SAS）求第三边

**例题2（基础）：** △ABC 中，a = 3, b = 5, C = 60°，求 c。

c² = 9 + 25 - 2×3×5×cos60° = 34 - 15 = 19
c = **√19** ✅

### 面积公式

```
S = (1/2)ab·sinC = (1/2)bc·sinA = (1/2)ac·sinB
```

> 💡 知道两边和夹角就能算面积。

## 做题的时候怎么用？

### 例题3（中等，一模Q15！）

> 已知△ABC 的角 A,B,C 所对的边分别为 a,b,c，且 c = 2，b/a = 2sinB/cosA。
> (1) 求 tan A；
> (2) 若△ABC 的面积为 2，求 a 和 cos C。

**第一问：求 tan A**

已知：b/a = 2sinB/cosA

用正弦定理 b = 2RsinB, a = 2RsinA：
b/a = sinB/sinA

所以 sinB/sinA = 2sinB/cosA

约去 sinB（sinB ≠ 0）：
1/sinA = 2/cosA → cosA = 2sinA → tanA = 1/2 ✅

**第二问：求 a 和 cos C**

由 tanA = 1/2，构造直角三角形：
对边=1，邻边=2，斜边=√5 → **sinA = 1/√5 = √5/5，cosA = 2/√5 = 2√5/5**

已知面积 S = 2：
S = (1/2)bc·sinA = 2
bc·(√5/5) = 4
bc = 4√5

又 c = 2，所以 b = 4√5/2 = 2√5

用余弦定理求 a：
a² = b² + c² - 2bc·cosA
= (2√5)² + 4 - 2×2√5×2×(2√5/5)
= 20 + 4 - 16√5×√5/5
= 24 - 16×5/5
= 24 - 16 = 8
**a = 2√2** ✅

求 cosC：
cosC = (a² + b² - c²)/(2ab)
= (8 + 20 - 4)/(2×2√2×2√5)
= 24/(8√10) = 3/√10 = **3√10/10** ✅

### 例题4（中等）：判断三角形形状

> △ABC 中，若 a² = b² + c² + bc，则角 A 等于多少？

用余弦定理：a² = b² + c² - 2bc·cosA

题目给 a² = b² + c² + bc，所以：
b² + c² + bc = b² + c² - 2bc·cosA
bc = -2bc·cosA
cosA = -1/2
**A = 120°** ✅

### 例题5（挑战）：面积最大值

> △ABC 中，a = 2, B = 60°，求面积 S 的最大值。

S = (1/2)ac·sinB = (1/2)×2×c×sin60° = (√3/2)c

要使 S 最大，需要 c 最大。

**方法：利用正弦定理**

由正弦定理：c/sinC = a/sinA → c = 2sinC/sinA

又 A = 180° - B - C = 120° - C

当 C = 90° 时 sinC = 1（最大），此时 A = 30°

c = 2×sin90°/sin30° = 2/(1/2) = 4

**S最大 = (√3/2)×4 = 2√3** ✅

> 💡 已知一边和一个角，求面积最大值：让对角取90°（此时sin最大），用正弦定理求出最大边长。

## 解三角形通用策略

```
① 把已知条件全部列出来
② 看看能直接用什么定理
③ 算出一个新量后，加入已知条件，再看能用什么
④ 重复直到求出所有要求的量
```

---

## ⚠️ 常见错误

1. **正弦定理和余弦定理选错**：已知两边夹角用余弦，已知两边对角用正弦。
2. **特殊角的三角函数值记错**：sin30°=1/2, cos60°=1/2, sin45°=cos45°=√2/2。
3. **正弦定理求角有两个解**：当已知两边和其中一边的对角，且对角为锐角时，可能有两个解（一个锐角三角形、一个钝角三角形），要检查。
4. **计算错误**：解三角形计算量大，每一步都要仔细。

## 高考常见陷阱

1. **正弦定理的两解问题**：sinA = k（k<1）时，A 可能有两个值（互补），需要根据实际判断是否都成立。
2. **tan 求出后要还原**：tanA = 1/2 不要写成 sinA = 1/2！tan = sin/cos，要构造直角三角形求 sin 和 cos。
3. **面积公式选错形式**：S = (1/2)ab·sinC 需要知道两边和**夹角**，不是任意角。

## 小测验

1.（基础）△ABC 中，a = 5, b = 7, C = 60°，求 c 和面积 S。

**答案：c = √39，S = (35√3)/4**

**解析：** 余弦定理 c² = a²+b²-2ab·cosC = 25+49-70×(1/2) = 39 → c = √39。面积 S = (1/2)ab·sinC = (1/2)×5×7×(√3/2) = 35√3/4。

2.（中等）△ABC 中，A = 30°, B = 45°, a = 6，求 b 和 c。

**答案：b = 6√2，c = 3√6 + 3√2**

**解析：** 正弦定理：b/sinB = a/sinA → b = 6×sin45°/sin30° = 6×(√2/2)/(1/2) = 6√2。C = 180°-30°-45° = 105°。c = 6×sin105°/sin30° = 12×sin105°。sin105° = sin(60°+45°) = sin60°cos45°+cos60°sin45° = (√3/2)(√2/2)+(1/2)(√2/2) = √6/4+√2/4。所以 c = 12×(√6+√2)/4 = 3√6+3√2。

3.（挑战）△ABC 中，a² = b² + c² + √3·bc，求角 A。（提示：用余弦定理比较）

**答案：A = 150°**

**解析：** 余弦定理 a² = b²+c²-2bc·cosA。题目给出 a² = b²+c²+√3·bc。对比：b²+c²-2bc·cosA = b²+c²+√3·bc → -2cosA = √3 → cosA = -√3/2 → A = 150°。
