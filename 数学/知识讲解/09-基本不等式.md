# 基本不等式

## 这个知识点是什么？

基本不等式是高中数学里一个非常重要的工具，它可以帮你**在已知某些约束条件下，求出某个表达式的最大值或最小值**。

> 打个比方：你有一根长度固定的绳子，要围成一个长方形。什么形状时面积最大？答案是正方形。基本不等式说的就是这个道理——**两个数之和固定时，乘积最大的是两数相等**。

## 知识结构图

```
基本不等式
├── 核心：a² + b² ≥ 2ab（a, b ∈ R）
│
├── 常用变形
│   ├── a + b ≥ 2√(ab)（a, b > 0）—— 最常用
│   ├── a² + b² ≥ (a+b)²/2
│   ├── 1/a + 1/b ≥ 4/(a+b)（a,b > 0）
│   └── a + 1/a ≥ 2（a > 0）
│
├── "1"的代换（高考最爱考）
│   └── a+b=1 时 ab≤1/4, a²+b²≥1/2
│
├── 技巧：拆项凑定值
│   ├── 凑"和"为定值 → 求"积"的最值
│   └── 凑"积"为定值 → 求"和"的最值
│
└── 关联知识点
    ├── 函数最值（与函数结合求值域）
    ├── 多选题（判断选项对错）
    ├── 解三角形（面积最值）
    └── 数列（前n项和的估计）
```

## 核心公式（必背）

### 公式一：a² + b² ≥ 2ab

**使用条件：** a 和 b 都是实数，等号成立当且仅当 **a = b**

> 💡 记忆方式：两个数的平方和，一定大于等于它们乘积的2倍。两个数越接近，等号越可能成立。

**例题1（基础）：** 已知 x > 0，求 x + 1/x 的最小值。

**解题：**
x + 1/x = (√x)² + (1/√x)²
≥ 2 × √x × (1/√x)    ← 用 a² + b² ≥ 2ab
= 2 × 1 = 2

等号成立：√x = 1/√x → x = 1

**答案：最小值是 2**，当 x = 1 时取到 ✅

### 公式二（最重要）：a + b ≥ 2√(ab)（a, b > 0）

这是公式一的变体，更常用。

**使用条件：** a > 0 且 b > 0，等号成立当且仅当 **a = b**

> 💡 两个正数之和，一定大于等于它们乘积的2倍开根号。

### 公式三："1"的代换（高考最爱考！）

当题目告诉你 **a + b = 1**（或某个常数）时，可以推导出 ab 的最大值。

**推导：**
a + b = 1
ab ≤ ((a+b)/2)² = (1/2)² = 1/4

> 💡 记住这个结论：**两个正数之和为1时，乘积最大为 1/4**。

**例题2（基础）：** 已知 a > 0, b > 0，且 a + b = 1，则 ab 的最大值是（　）

直接套公式：ab ≤ (1/2)² = 1/4

**答案：最大值是 1/4** ✅

### 重要变形（全部要会！）

**变形1：a² + b² ≥ (a+b)²/2**

推导：a² + b² ≥ 2ab → 2(a²+b²) ≥ a² + 2ab + b² = (a+b)²

**结论：** a + b 固定时，a² + b² 的最小值 = (a+b)²/2，当且仅当 a = b。

**变形2：1/a + 1/b ≥ 4/(a+b)（a, b > 0）**

推导：
1/a + 1/b = (a+b)/(ab)
又 ab ≤ (a+b)²/4 → 1/ab ≥ 4/(a+b)²
所以 (a+b)/(ab) ≥ (a+b) × 4/(a+b)² = 4/(a+b)

> 💡 当 a+b=1 时，1/a + 1/b ≥ 4，等号成立当 a=b=1/2。

**变形3：a + 1/a ≥ 2（a > 0）**

这就是公式二直接用，令 b = 1/a。等号成立当 a = 1。

**变形4：a² + 1/a² ≥ 2（a ≠ 0）**

令 b = 1/a²，直接用 a² + b ≥ 2√(a² × 1/a²) = 2。

### 常见条件下的结论

| 条件 | 结论 | 等号成立条件 |
|------|------|-------------|
| a + b = 1 (a,b>0) | ab ≤ 1/4 | a = b = 1/2 |
| a + b = 1 (a,b>0) | a² + b² ≥ 1/2 | a = b = 1/2 |
| a + b = 1 (a,b>0) | 1/a + 1/b ≥ 4 | a = b = 1/2 |
| a + b = 2 (a,b>0) | ab ≤ 1 | a = b = 1 |
| a + b = c (a,b>0) | ab ≤ c²/4 | a = b = c/2 |

## 核心技巧：拆项凑定值

用基本不等式求最值的关键：**先保证能用，再保证等号成立**。

"能用"的前提是**两项乘积或两项之和是定值（常数）**。如果不是定值，就需要"拆项"或"配系数"来凑。

### 凑定值的方法

**方法一：拆开常数**

**例题3（中等）：** 已知 x > 0，求 x + 9/x 的最小值。

x + 9/x ≥ 2√(x × 9/x) = 2√9 = 6

等号成立：x = 9/x → x = 3

**答案：最小值是 6** ✅

**方法二：分子分母同除**

**例题4（中等）：** 已知 x > 0, y > 0，且 2x + y = 1，求 xy 的最大值。

xy = x · y
要凑成"和为定值"的形式。已知 2x + y = 1。

xy = (1/2) × 2x × y
又 (2x)(y) ≤ ((2x+y)/2)² = (1/2)² = 1/4
所以 xy ≤ (1/2) × 1/4 = 1/8

等号成立：2x = y，且 2x + y = 1 → 2x + 2x = 1 → x = 1/4, y = 1/2

**答案：最大值是 1/8** ✅

**方法三：已知 ax+by=1，求 1/x+1/y 的最小值**

**例题5（中等）：** 已知 a > 0, b > 0，且 2a + 3b = 1，求 1/a + 1/b 的最小值。

这里不能直接用基本不等式，因为 1/a + 1/b 的两项乘积不是定值。

**技巧：用已知条件"乘"进去**

1/a + 1/b = (2a + 3b)(1/a + 1/b)
= 2a × 1/a + 2a × 1/b + 3b × 1/a + 3b × 1/b
= 2 + 2a/b + 3b/a + 3
= 5 + 2a/b + 3b/a

现在对 2a/b + 3b/a 用基本不等式：
2a/b + 3b/a ≥ 2√(2a/b × 3b/a) = 2√6

等号成立：2a/b = 3b/a → 2a² = 3b² → a/b = √(3/2)

所以 1/a + 1/b ≥ 5 + 2√6

**答案：最小值是 5 + 2√6** ✅

> 💡 这道题的方法叫"乘1法"——把已知条件（等于1的那个式子）乘上去展开，再逐项用基本不等式。

**方法四：配系数**

**例题6（挑战）：** 已知 x > 1/2，求 f(x) = x + 1/(2x-1) 的最小值。

直接用基本不等式的话，x × 1/(2x-1) 不是定值。需要配系数。

令 f(x) = x + 1/(2x-1) = (1/2)(2x-1) + 1/2 + 1/(2x-1)
= 1/2 + [(2x-1)/2 + 1/(2x-1)]

对 [(2x-1)/2 + 1/(2x-1)] 用基本不等式：
设 t = 2x-1 > 0（因为 x > 1/2）
t/2 + 1/t ≥ 2√(t/2 × 1/t) = 2√(1/2) = √2

等号成立：t/2 = 1/t → t² = 2 → t = √2 → x = (1+√2)/2

f(x) ≥ 1/2 + √2

**答案：最小值是 1/2 + √2** ✅

> 💡 配系数的关键：把 x 变成 (2x-1) 的形式，让分母和分子能约掉一部分，凑成定值。

## 做题的时候怎么用？

### 题型一：求最值（选择题/填空题）

**例题7（中等，多选题类型）：** 已知 a > 0, b > 0, a + b = 1，下列说法正确的是（　）
A. ab ≥ 1/4
B. ab ≤ 1/4
C. a² + b² ≥ 1
D. a² + b² ≥ 1/2

逐个判断：
- A：ab ≤ 1/4，所以 ab ≥ 1/4 **错** ❌
- B：ab ≤ 1/4，**对** ✅
- C：a² + b² ≥ (a+b)²/2 = 1/2，但 a² + b² 不一定 ≥ 1。反例：a=0.9, b=0.1，a²+b²=0.82 < 1 ❌
- D：a² + b² ≥ 1/2，**对** ✅

**答案：BD** ✅

### 题型二：与指数对数结合

**例题8（挑战）：** 已知 a > 0 且 a ≠ 1，求 log₂a + log_a2 的最小值。

令 t = log₂a（t ≠ 0），则 log_a2 = 1/log₂a = 1/t。

log₂a + log_a2 = t + 1/t

当 t > 0（即 a > 1）时：t + 1/t ≥ 2
当 t < 0（即 0 < a < 1）时：t + 1/t ≤ -2（两个负数，和为负）

所以 t + 1/t 的最小值 = **-2**（当 t = -1 即 a = 1/2 时取到）

**答案：最小值是 -2** ✅

> ⚠️ 注意：t + 1/t 在 t > 0 时最小值为2，在 t < 0 时最大值为-2，**全局最小值是 -2**。

## 做基本不等式题的三步走

```
① 检查条件：a 和 b 是不是正数？
② 选公式：求最值用 a+b≥2√(ab)，判断用 a²+b²≥2ab
③ 验等号：等号能不能成立？什么时候成立？
```

⚠️ **第三步最容易被忽略！** 如果等号不成立（比如两个数不能相等），那这个最小值/最大值就取不到。

---

## ⚠️ 常见错误

1. **忘记检查正负条件**：a + b ≥ 2√(ab) 要求 a, b > 0。如果 a = -1, b = -2，直接套就错了。
2. **忘记写等号成立条件**：考试要求写完整过程，"当 a = b 时取等号"一定要写。
3. **混淆最大和最小**：和固定时，积有最大值；积固定时，和有最小值。别搞反。
4. **乘积/和不是定值就直接套**：比如求 x + 1/(x+1) 的最小值，x 和 1/(x+1) 的乘积不是定值，直接套会出错。要先凑定值。
5. **多选题看漏选项**：基本不等式多选题经常有2-3个正确选项，逐个判断不要漏。

## 高考常见陷阱

1. **"恰好取到"陷阱**：算出最小值是6，但等号成立的条件 a=2 不在题目定义域内，那就取不到6。一定要检查！
2. **多次使用基本不等式**：有时需要连续用两次（如"乘1法"），每次都要分别验证等号条件，且**两次的等号条件要能同时满足**。
3. **多选题中的反向命题**：比如"ab ≤ 1/4"是对的，但"ab ≥ 1/4"是错的，不要看到不等式就选。
4. **与对数结合时忽略定义域**：log₂a 要求 a > 0，a ≠ 1。

## 小测验

1.（基础）已知 x > 0，求 x + 9/x 的最小值，并指出等号成立的条件。

**答案：最小值 = 6，当 x = 3 时等号成立**

**解析：** x + 9/x ≥ 2√(x · 9/x) = 2√9 = 6。等号条件：x = 9/x → x² = 9 → x = 3（x>0取正值）。

2.（中等）已知 a > 0, b > 0，且 3a + 2b = 1，求 ab 的最大值。（提示：配系数）

**答案：最大值 = 1/24**

**解析：** 1 = 3a + 2b ≥ 2√(6ab)，所以 6ab ≤ 1/4 → ab ≤ 1/24。等号条件：3a = 2b → a = 2/12 = 1/6, b = 3/12 = 1/4。验证：3×(1/6)+2×(1/4) = 1/2+1/2 = 1 ✅。

3.（挑战）已知 x > 0, y > 0，且 x + 2y = 1，求 1/x + 1/y 的最小值。（提示：乘1法）

**答案：最小值 = 3 + 2√2**

**解析：** 1/x + 1/y = (x+2y)(1/x + 1/y) = 1 + x/y + 2y/x + 2 = 3 + x/y + 2y/x。x/y + 2y/x ≥ 2√(x/y · 2y/x) = 2√2。所以 1/x + 1/y ≥ 3 + 2√2。等号条件：x/y = 2y/x → x² = 2y² → x = √2 y。代入 x+2y=1：√2y+2y=1 → y = 1/(2+√2)。
