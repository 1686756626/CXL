# 分段函数与不等式

## 这个知识点是什么？

"分段函数"就是**一个函数，在不同的范围内用不同的公式**。

> 打个比方：出租车计费。3公里内10块钱起步价，超过3公里后每公里2块钱。这就是一个分段函数：
> - x ≤ 3 时，费用 = 10
> - x > 3 时，费用 = 10 + 2(x-3)

分段函数本身不难，难的是**解分段函数的不等式**——要在每一段里分别解，最后合并结果。这就是一模Q5考的内容。

## 第一部分：分段函数基础

### 怎么读分段函数？

```
         ┌ 2⁻ˣ,     x ≤ -1
f(x) = {
         └ -x²+4x+1, x > -1
```

这读作：
- 当 x ≤ -1 时，f(x) = 2⁻ˣ
- 当 x > -1 时，f(x) = -x² + 4x + 1

> 💡 两个分段之间的"分界点"是 x = -1。注意一个用 ≤（包含-1），另一个用 >（不包含-1），这样每个x值都只属于其中一段，不会重复也不会遗漏。

### 求分段函数的值

**例题1（基础）：** 用上面的 f(x)，求 f(-2)、f(-1) 和 f(0)。

- f(-2)：-2 ≤ -1，所以用第一段：f(-2) = 2⁻² = 1/4 = 0.25
- f(-1)：-1 ≤ -1（注意≤包含），所以用第一段：f(-1) = 2⁻¹ = 1/2
- f(0)：0 > -1，所以用第二段：f(0) = -0² + 4×0 + 1 = 1

> ⚠️ 分界点归哪一段要看清楚！用 ≤ 的一段包含分界点。

### 画分段函数的图像（重要辅助工具！）

画分段函数图像的方法：**每一段单独画，然后拼在一起**。

**例题2（中等）：** 画 f(x) = { |x|, x < 2; x²-2, x ≥ 2 } 的图像。

**第一段 x < 2：f(x) = |x|**
- x < 0 时，|x| = -x（一条向右下的直线）
- 0 ≤ x < 2 时，|x| = x（一条向右上的直线）
- 在 x = 2 处：f(2) 不属于第一段（x < 2），但趋近于 |2| = 2

**第二段 x ≥ 2：f(x) = x² - 2**
- 这是一个开口向上的抛物线
- 在 x = 2 处：f(2) = 4 - 2 = 2

> 💡 两段在 x = 2 处的值都是 2 → 图像在这里**连续**（没有断点）。

如果两段在分界点处值不同，图像就会有一个"跳跃"（断点）。

## 第二部分：解分段函数不等式（核心考点！）

### 方法：分段解，再合并

**解 f(x) < c 的步骤：**

```
① 看分界点把数轴分成几段
② 在每一段里，把对应的公式代入不等式，分别解
③ 把每一段的解集与该段的条件取交集
④ 所有交集取并集（合在一起）就是最终解集
```

> 💡 **关键一步：** 每段解出的解集，一定要和该段的条件取交集！这是最容易犯错的地方。

### 例题3（基础，一模Q5！）

> 已知 f(x) = { 2⁻ˣ, x ≤ -1; -x²+4x+1, x > -1 }，则不等式 f(x) < 1 的解集为？

**段一：x ≤ -1 时，f(x) = 2⁻ˣ < 1**

2⁻ˣ < 1 = 2⁰

底数 2 > 1，指数函数单调递增，所以 -x < 0 → x > 0。

与条件 x ≤ -1 取交集：{x > 0} ∩ {x ≤ -1} = **空集**

**段二：x > -1 时，f(x) = -x² + 4x + 1 < 1**

-x² + 4x < 0
x² - 4x > 0（两边乘-1，不等号变号）
x(x - 4) > 0
解集：(-∞, 0) ∪ (4, +∞)

与条件 x > -1 取交集：
(-∞, 0) ∩ (-1, +∞) = **(-1, 0)**
(4, +∞) ∩ (-1, +∞) = **(4, +∞)**

**最终解集 = (-1, 0) ∪ (4, +∞)**

**答案：D** ✅

### 例题4（中等）：含参数的分段函数不等式

> f(x) = { x+1, x ≤ 1; x²-2x+2, x > 1 }，求 f(x) ≤ 3 的解集。

**段一：x ≤ 1 时，x + 1 ≤ 3 → x ≤ 2**

与 x ≤ 1 取交集：x ≤ 1

**段二：x > 1 时，x² - 2x + 2 ≤ 3**

x² - 2x - 1 ≤ 0

用求根公式：x = (2 ± √(4+4))/2 = (2 ± 2√2)/2 = 1 ± √2

1 - √2 ≈ -0.41，1 + √2 ≈ 2.41

所以 x² - 2x - 1 ≤ 0 的解集：[1-√2, 1+√2]

与 x > 1 取交集：(1, 1+√2]

**最终解集 = (-∞, 1] ∪ (1, 1+√2] = (-∞, 1+√2]**

**答案：(-∞, 1+√2]** ✅

> 💡 这里两段刚好"无缝衔接"，最终可以合并成一个区间。

### 例题5（挑战）：分段函数与单调性结合

> f(x) = { -x²+2x, x ≤ 0; ln(x+1), x > 0 }，求 f(x) > 0 的解集。

**段一：x ≤ 0 时，-x² + 2x > 0**

-x² + 2x > 0 → x² - 2x < 0 → x(x-2) < 0 → 0 < x < 2

与 x ≤ 0 取交集：空集

**段二：x > 0 时，ln(x+1) > 0**

ln(x+1) > 0 → x + 1 > 1 → x > 0

与 x > 0 取交集：x > 0

**最终解集 = (0, +∞)** ✅

## 解题模板

遇到解分段函数不等式 f(x) > c（或 < c、≥ c、≤ c），就按这个格式写：

```
f(x) < c 的解集：

段一（x ≤ k）：代入对应公式 → 解不等式 → 得到解集 S₁ → S₁ ∩ {x ≤ k} = T₁
段二（x > k）：代入对应公式 → 解不等式 → 得到解集 S₂ → S₂ ∩ {x > k} = T₂

最终解集 = T₁ ∪ T₂
```

## 补充：绝对值不等式

### 基本公式（两个，必背！）

**|x| < a（a > 0）⟹ -a < x < a**

> 例：|x| < 3 ⟹ -3 < x < 3

**|x| > a（a > 0）⟹ x < -a 或 x > a**

> 例：|x| > 2 ⟹ x < -2 或 x > 2

> 💡 记忆方法：|x|<a 是"靠近0"，所以夹在中间；|x|>a 是"远离0"，所以跑到两边。

### |ax+b| 型绝对值不等式

方法：**设 t = ax+b，先解 |t| 的不等式，再解 ax+b 的不等式。**

**例题：解 |2x-1| < 3**

第一步：2x-1 对应 t，|t| < 3 → -3 < t < 3 → -3 < 2x-1 < 3

第二步：解双重不等式
-3 < 2x-1 → 2x > -2 → x > -1
2x-1 < 3 → 2x < 4 → x < 2

**答案：-1 < x < 2** ✅

### 绝对值和集合结合

> A = {x | |2x-1| < 3}，B = {x | x²-1 ≤ 0}
> 求 A ∩ B

先解 A：|2x-1| < 3 → **-1 < x < 2**，A = (-1, 2)
再解 B：x²-1 ≤ 0 → (x+1)(x-1) ≤ 0 → **-1 ≤ x ≤ 1**，B = [-1, 1]

A ∩ B = **(-1, 1]** ✅

## ⚠️ 常见错误

1. **忘记和分段条件取交集**：在段二解出 x < 0，但忘了段二的条件是 x > -1，结果多写了 (-∞, -1] 这部分。
2. **分界点归属搞错**：一个分段用 ≤，另一个用 >，不要两个都用 ≤ 或者两个都用 >。
3. **乘以负数忘记变号**：解 -x² + 4x < 0 时，两边乘-1变成 x² - 4x > 0，不等号要变号！
4. **不等式方向搞混**：f(x) < 1 和 f(x) > 1 是完全不同的题，仔细看题目。
5. **解二次不等式时求根公式算错**：判别式、根号里的数，每一步都要仔细。

## 高考常见陷阱

1. **分界点处的函数值**：分段函数在分界点处的值取决于该点属于哪一段，经常考选择题让你判断选项。
2. **图像断点**：如果两段在分界点处的值不同，函数图像有跳跃。选择题会利用这一点设置选项。
3. **结合绝对值**：|x| 本身就是分段函数，高考经常把绝对值和分段函数结合考。
4. **端点值验证**：解不等式时，端点值（开区间还是闭区间）经常是选择题的区分点。

## 小测验

1.（基础）已知 f(x) = { x+1, x ≤ 1; 2x-1, x > 1 }，求 f(1)、f(2) 和 f(-1)。

**答案：f(1) = 2，f(2) = 3，f(-1) = 0**

**解析：** x=1 ≤ 1，用第一段：f(1) = 1+1 = 2。x=2 > 1，用第二段：f(2) = 2×2-1 = 3。x=-1 ≤ 1，用第一段：f(-1) = -1+1 = 0。

2.（中等）已知 f(x) = { x+1, x ≤ 1; 2x-1, x > 1 }，解不等式 f(x) > 3。

**答案：x > 2**

**解析：** 分段解。x ≤ 1 时：x+1 > 3 → x > 2，但 x ≤ 1，无交集，无解。x > 1 时：2x-1 > 3 → x > 2。答案：**x > 2**。

3.（挑战）已知 f(x) = { |x|, x < 2; x²-2, x ≥ 2 }，解不等式 f(x) < 3。

**答案：-3 < x < 2 或 2 ≤ x < √5，合并为 -3 < x < √5**

**解析：** x < 2 时：|x| < 3 → -3 < x < 3，与 x < 2 取交集 → -3 < x < 2。x ≥ 2 时：x²-2 < 3 → x² < 5 → -√5 < x < √5，与 x ≥ 2 取交集 → 2 ≤ x < √5。合并：-3 < x < √5。
