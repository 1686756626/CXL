# 导数第一问（求导+单调区间+切线）

## 这个知识点是什么？

导数就是**研究函数变化快慢**的工具。高考里导数题一般有2-3问，**第一问几乎都是求导+判断单调区间或求切线方程**，套路非常固定。

> 打个比方：导数就像汽车的"速度表"——告诉你函数在每个点是"上升中"（导数>0）还是"下降中"（导数<0），还是"到了顶峰/谷底"（导数=0）。

## 知识结构图

```
导数第一问
├── 基本求导公式（必背）
│   ├── (xⁿ)' = nxⁿ⁻¹
│   ├── (sinx)' = cosx, (cosx)' = -sinx
│   ├── (eˣ)' = eˣ, (aˣ)' = aˣlna
│   ├── (lnx)' = 1/x
│   └── 常数导数为0：(C)' = 0
│
├── 求导法则
│   ├── (f±g)' = f'±g'
│   ├── (fg)' = f'g+fg'（乘法）
│   ├── (f/g)' = (f'g-fg')/g²（除法）
│   └── 链式法则：[f(g(x))]' = f'(g(x))·g'(x)
│
├── 导数的应用
│   ├── 求单调区间：f'(x)>0递增, f'(x)<0递减
│   ├── 求极值：f'(x)=0的根附近导数变号
│   └── 求切线方程：y-f(x₀)=f'(x₀)(x-x₀)
│
├── 高频考法

```

## 第一部分：基本求导公式（必背！）

| 函数 f(x) | 导数 f'(x) | 记忆技巧 |
|-----------|-----------|----------|
| xⁿ | nxⁿ⁻¹ | 指数下来当系数，指数减1 |
| sin x | cos x | 正弦变余弦 |
| cos x | -sin x | 余弦变负正弦（加负号！） |
| eˣ | eˣ | 不变 |
| aˣ | aˣ ln a | 多乘个lna |
| ln x | 1/x | 对数的导数是倒数 |
| C（常数） | 0 | 常数不变化 |

### 例题1（基础）：求下列函数的导数

**(a)** f(x) = x³ - 3x² + 2x - 1

f'(x) = 3x² - 6x + 2 ✅

**(b)** f(x) = 2sin x + cos x

f'(x) = 2cos x - sin x ✅

**(c)** f(x) = x² · eˣ（用乘法法则）

f'(x) = 2x·eˣ + x²·eˣ = **eˣ(x² + 2x)** ✅

## 第二部分：求单调区间

### 步骤

```
① 求导 f'(x)
② 解 f'(x) = 0 找临界点
③ 列表或画数轴，判断各区间 f'(x) 的正负
④ 写出单调递增/递减区间
```

### 例题2（中等）：求 f(x) = x³ - 3x 的单调区间

**第一步：求导**
f'(x) = 3x² - 3 = 3(x+1)(x-1)

**第二步：解 f'(x) = 0**
x = -1 或 x = 1

**第三步：判断正负**

| 区间 | (-∞, -1) | (-1, 1) | (1, +∞) |
|------|---------|---------|---------|
| f'(x) = 3(x+1)(x-1) | + | - | + |

**第四步：结论**
递增区间：**(-∞, -1) 和 (1, +∞)**
递减区间：**(-1, 1)** ✅

> 💡 开口向上的抛物线（f'(x)=3x²-3），在两个根之间为负，两侧为正。

### 例题3（中等，含参）：f(x) = x³ - ax² + 1，已知 f(x) 在 x=1 处取极值，求 a 并判断单调性。

f'(x) = 3x² - 2ax

x=1 处取极值 → f'(1) = 0 → 3 - 2a = 0 → **a = 3/2**

f'(x) = 3x² - 3x = 3x(x-1)

| 区间 | (-∞, 0) | (0, 1) | (1, +∞) |
|------|---------|---------|---------|
| f'(x) | + | - | + |

递增：(-∞, 0) 和 (1, +∞)
递减：(0, 1)

x=0 处取极大值，x=1 处取极小值 ✅

## 第三部分：求切线方程

### 切线方程公式

在 x = x₀ 处的切线方程：**y - f(x₀) = f'(x₀)(x - x₀)**

### 例题4（基础）：求 f(x) = x² - 2x + 1 在 x = 2 处的切线方程。

f(2) = 4 - 4 + 1 = 1（切点）
f'(x) = 2x - 2
f'(2) = 2（斜率）

切线方程：y - 1 = 2(x - 2) → y = **2x - 3** ✅

### 例题5（中等，已知切线过某点）：求 f(x) = x³ - 3x 的过点 (0, -2) 的切线方程。

设切点为 (a, f(a))，则：
f(a) = a³ - 3a（切点在曲线上）
f'(a) = 3a² - 3（切线斜率）

切线方程：y - (a³-3a) = (3a²-3)(x - a)

过 (0, -2)：-2 - (a³-3a) = (3a²-3)(0-a)
-2 - a³ + 3a = -a(3a²-3)
-2 - a³ + 3a = -3a³ + 3a
-2 - a³ + 3a + 3a³ - 3a = 0
2a³ - 2 = 0
a³ = 1 → a = 1

f(1) = 1 - 3 = -2，f'(1) = 3 - 3 = 0

切线方程：y - (-2) = 0·(x-1) → **y = -2** ✅

> 💡 切线过某点时，这个点**不一定在曲线上**！要先设切点，再用条件求出切点。

## ⚠️ 常见错误

1. **求导公式记错**：cos x 的导数是 -sin x（有负号），别忘
2. **单调区间写成不等式**：递增区间写 (-∞, -1) 不是 x < -1
3. **切线点不是已知点**：过点求切线时，已知点不一定在曲线上
4. **含参求导后忘讨论**：a 的值不同可能导致临界点个数不同

## 高考常见陷阱

1. **乘法法则和链式法则混淆**：(fg)' = f'g+fg'，不是 (fg)' = f'g'
2. **f'(x₀)=0 不一定是极值**：比如 f(x)=x³，f'(0)=0 但 x=0 不是极值点（拐点）
3. **eˣ 的导数是 eˣ 不是 xeˣ⁻¹**：指数函数和幂函数求导公式不同
4. **切线斜率可能为0**：水平切线 y=常数 也是有效的切线

## 小测验

1.（基础）求 f(x) = x³ - 6x² + 9x 的单调区间。

**答案：递增区间 (-∞, 1) 和 (3, +∞)，递减区间 (1, 3)**

**解析：** f'(x) = 3x²-12x+9 = 3(x-1)(x-3)。f'(x)>0 → x<1 或 x>3 → 递增。f'(x)<0 → 1<x<3 → 递减。

2.（中等）求 f(x) = ln x 在 x=1 处的切线方程。

**答案：y = x - 1**

**解析：** f(1) = ln1 = 0（切点），f'(x) = 1/x，f'(1) = 1（斜率）。切线方程：y-0 = 1·(x-1) → y = x-1。

3.（挑战）已知 f(x) = x³ - 3x，求过点 (2, 2) 的切线方程。

**答案：y = 9x - 16**

**解析：** 先验证 (2,2) 是否在曲线上：f(2) = 8-6 = 2 ✅，在曲线上。所以 (2,2) 就是切点，不需要设变量。f'(x) = 3x²-3，f'(2) = 12-3 = 9（斜率）。切线方程：y-2 = 9(x-2) → y = 9x-16。
