# 数列基础

## 这个知识点是什么？

"数列"就是**按一定规律排列的一串数**。

> 打个比方：1, 3, 5, 7, 9, ... 这是一个数列——每次加2。2, 4, 8, 16, 32, ... 也是一个数列——每次乘2。

高考里考最多的两种数列是**等差数列**（每次加一样的）和**等比数列**（每次乘一样的）。一模Q17考的就是等比数列。

## 知识结构图

```
数列基础
├── 等差数列
│   ├── 定义：aₙ - aₙ₋₁ = d（常数）
│   ├── 通项公式：aₙ = a₁ + (n-1)d
│   ├── 求和公式：Sₙ = n(a₁+aₙ)/2
│   ├── 性质：aₙ - aₘ = (n-m)d
│   └── 等差中项：2aₙ = aₙ₋₁ + aₙ₊₁
│
├── 等比数列
│   ├── 定义：aₙ/aₙ₋₁ = q（常数）
│   ├── 通项公式：aₙ = a₁·qⁿ⁻¹
│   ├── 求和公式：Sₙ = a₁(1-qⁿ)/(1-q)
│   ├── 性质：aₙ/aₘ = qⁿ⁻ᵐ
│   └── 等比中项：aₙ² = aₙ₋₁·aₙ₊₁
│
├── 高频考法
│   ├── 知条件求通项（列方程组）
│   ├── 求前n项和
│   ├── 等差等比判定
│   └── 与绝对值结合的求和（分类讨论）
│
└── 关联知识点
    ├── 函数（数列是定义在正整数上的函数）
    ├── 不等式（数列不等式证明）
    └── 概率（等可能事件的概率=符合条件项数/总项数）
```

## 等差数列

### 必背公式

| 公式 | 内容 |
|------|------|
| 通项公式 | aₙ = a₁ + (n-1)d |
| 求和公式 | Sₙ = n(a₁ + aₙ) / 2 |
| 求和公式（变形） | Sₙ = na₁ + n(n-1)d / 2 |
| 性质 | aₙ - aₘ = (n-m)d |

### 怎么理解通项公式？

aₙ = a₁ + (n-1)d 的意思是：
- 第1项 = a₁
- 第2项 = a₁ + d（加了一次公差）
- 第n项 = a₁ + (n-1)d（加了n-1次公差）

### 怎么理解求和公式？

Sₙ = n(a₁ + aₙ) / 2 = **（首项 + 末项）× 项数 / 2**

> 💡 头尾配对，每对的和都一样（首+末=第二+倒数第二=...），共 n/2 对。

## 等比数列

### 必背公式

| 公式 | 内容 |
|------|------|
| 通项公式 | aₙ = a₁ · qⁿ⁻¹ |
| 求和公式 | Sₙ = a₁(1 - qⁿ) / (1 - q)，q ≠ 1 |
| 求和公式（q=1时） | Sₙ = na₁ |
| 性质 | aₙ / aₘ = qⁿ⁻ᵐ |

---

## 做题的时候怎么用？

### 题型一：已知条件求通项公式（一模Q17类型）

**例题1（中等，类似一模Q17）：** 等比数列 {aₙ} 的公比 q 为整数，且 a₁ + a₄ = 52，a₂ + a₃ = -12。求通项公式。

**第一步：用通项公式列出方程组**

a₁ + a₁q³ = a₁(1 + q³) = 52 ... ①
a₁q + a₁q² = a₁q(1 + q) = -12 ... ②

**第二步：求 q**

由 ①÷②：a₁(1+q³) / [a₁q(1+q)] = 52/(-12)

化简 1+q³ = (1+q)(1-q+q²)：

(1+q)(1-q+q²) / [q(1+q)] = -13/3

1-q+q² = -13q/3

3(1-q+q²) = -13q

3 - 3q + 3q² = -13q

3q² + 10q + 3 = 0

(3q+1)(q+3) = 0

q = -1/3 或 q = -3

题目说 q 是整数 → **q = -3** ✅

**第三步：求 a₁**

代入 ②：a₁ × (-3)(1-3) = a₁ × 6 = -12 → **a₁ = -2**

**第四步：写通项公式**

aₙ = **-2 · (-3)ⁿ⁻¹** ✅

### 题型二：等差数列求通项

**例题2（基础）：** 等差数列 {aₙ} 中，a₃ = 7, a₇ = 19，求通项公式。

用性质：a₇ - a₃ = 4d = 19 - 7 = 12 → d = 3
a₁ = a₃ - 2d = 7 - 6 = 1

aₙ = 1 + (n-1)×3 = **3n - 2** ✅

### 题型三：求和

**例题3（基础）：** 等比数列 a₁ = 1, q = 2，求 S₅。

S₅ = 1×(1-32)/(1-2) = (-31)/(-1) = **31** ✅

验证：1 + 2 + 4 + 8 + 16 = 31 ✅

**例题4（中等）：** 等比数列 {aₙ} 中，a₃ = 2, S₃ = 6，求 a₁ 和 q。

a₃ = a₁q² = 2 ... ①
S₃ = a₁(1-q³)/(1-q) = a₁(1+q+q²) = 6 ... ②

由 ①：a₁ = 2/q²

代入 ②：(2/q²)(1+q+q²) = 6
2(1+q+q²) = 6q²
2 + 2q + 2q² = 6q²
4q² - 2q - 2 = 0
2q² - q - 1 = 0
(2q+1)(q-1) = 0

q = -1/2 或 q = 1

q=1 时 S₃ = 3a₁ = 6 → a₁ = 2，但 a₃ = 2，a₁=a₂=a₃=2 ✅（常数列）
q=-1/2 时 a₁ = 2/(1/4) = 8

两组解都成立：**a₁=2, q=1** 或 **a₁=8, q=-1/2** ✅

### 题型五（挑战）：等比数列与绝对值结合

**例题5（挑战）：** 等比数列 {aₙ} 中，a₁ = 2, q = -2，求 |a₁|+|a₂|+...+|a₅|。

先算各项：a₁=2, a₂=-4, a₃=8, a₄=-16, a₅=32

|a₁|+|a₂|+|a₃|+|a₄|+|a₅| = 2+4+8+16+32 = 62

更一般地：{|aₙ|} 是首项为 |a₁|、公比为 |q| 的等比数列。

|a₁| = 2, |q| = 2
S₅ = 2(1-32)/(1-2) = 62 ✅

> 💡 绝对值数列求和时，{|aₙ|} 是一个新的等比数列，公比 = |q|。

### 题型六：已知 Sₙ 求 aₙ（高考必考！）

这是数列题里最高频的考点之一。题目给你前n项和 Sₙ 的表达式，让你求通项 aₙ。

**核心公式：**

```
当 n ≥ 2 时：aₙ = Sₙ - S(n-1)
当 n = 1 时：a₁ = S₁
```

⚠️ **必须验证 n=1 时公式是否一致！** 如果把 n=1 代入 aₙ = Sₙ - S(n-1) 算出的值不等于 S₁，就要**分段写**通项。

**例题6（基础）：** 已知 Sₙ = n² + n，求 aₙ。

**第一步：n=1 时**
a₁ = S₁ = 1² + 1 = 2

**第二步：n ≥ 2 时**
aₙ = Sₙ - S(n-1)
= (n² + n) - [(n-1)² + (n-1)]
= n² + n - (n² - 2n + 1 + n - 1)
= n² + n - n² + 2n - 1 - n + 1
= 2n

**第三步：验证**
当 n=1 时，aₙ = 2×1 = 2 = a₁ = S₁ ✅ 一致！

**答案：aₙ = 2n** ✅

> 💡 验证通过就不需要分段，直接写 aₙ = 2n (n ≥ 1)。

**例题7（中等）：** 已知 Sₙ = 2ⁿ - 1，求 aₙ。

**第一步：n=1 时**
a₁ = S₁ = 2¹ - 1 = 1

**第二步：n ≥ 2 时**
aₙ = Sₙ - S(n-1)
= (2ⁿ - 1) - (2ⁿ⁻¹ - 1)
= 2ⁿ - 2ⁿ⁻¹
= 2ⁿ⁻¹(2 - 1)
= 2ⁿ⁻¹

**第三步：验证**
当 n=1 时，aₙ = 2⁰ = 1 = a₁ = S₁ ✅ 一致！

**答案：aₙ = 2ⁿ⁻¹** ✅（这就是公比 q=2 的等比数列）

**例题8（挑战）：** 已知 Sₙ = n² - n + 1，求 aₙ。

**第一步：n=1 时**
a₁ = S₁ = 1 - 1 + 1 = 1

**第二步：n ≥ 2 时**
aₙ = Sₙ - S(n-1)
= (n² - n + 1) - [(n-1)² - (n-1) + 1]
= n² - n + 1 - (n² - 2n + 1 - n + 1 + 1)
= n² - n + 1 - n² + 2n - 1 + n - 1 - 1
= 2n - 2

**第三步：验证**
当 n=1 时，aₙ = 2×1 - 2 = 0 ≠ a₁ = 1 ❌ 不一致！

**需要分段写：**
a₁ = 1
aₙ = 2n - 2（n ≥ 2）

**答案：aₙ = { 1, n=1; 2n-2, n≥2 }** ✅

> ⚠️ 这道题就是"验证不通过"的典型情况。考试时不验证直接写 aₙ = 2n-2 就丢分了！

## ⚠️ 常见错误

1. **等比数列求和忘写 q ≠ 1 的条件**：当 q = 1 时，Sₙ = na₁，不能套通用公式。
2. **通项公式指数搞错**：aₙ = a₁·qⁿ⁻¹，是 n-1 不是 n！
3. **正负号搞错**：等比数列公比可以是负数（比如 q = -3），计算时注意 (-3)ⁿ 的正负号交替。
4. **方程组解错**：求 q 的过程中容易算错，做完要代回去验证。
5. **等比求和分子分母写反**：Sₙ = a₁(1-qⁿ)/(1-q)，不是 a₁(qⁿ-1)/(q-1)（虽然数学上等价，但前者更不容易出错）。

## 高考常见陷阱

1. **q = 1 的特殊情况**：等比数列求和时，题目可能暗示 q = 1（比如数列是常数列），此时不能套公式。
2. **q 为负数时的求和**：q < 0 时，数列各项正负交替，求和时要小心符号。
3. **"前n项和"vs"前n项的绝对值之和"**：一模Q17第二问考的就是这个区别，是常见考点。
4. **等差等比的混合判断**：有些数列既不是等差也不是等比，但隔项是等差或等比。

## 小测验

1.（基础）等差数列 {aₙ} 中，a₁ = 3, d = 4，求 a₁₀ 和 S₁₀。

**答案：a₁₀ = 39，S₁₀ = 210**

**解析：** a₁₀ = a₁ + 9d = 3 + 36 = 39。S₁₀ = 10(a₁+a₁₀)/2 = 10×(3+39)/2 = 210。

2.（中等）等比数列 {aₙ} 中，a₁ = 2, q = -3，求 a₄ 和 S₄。

**答案：a₄ = -54，S₄ = -40**

**解析：** a₄ = a₁q³ = 2×(-27) = -54。S₄ = a₁(1-q⁴)/(1-q) = 2×(1-81)/(1+3) = 2×(-80)/4 = -40。

3.（挑战）等差数列 {aₙ} 中，a₂ = 5, a₆ = 13，S₁₀ = ？（提示：先用 aₙ-aₘ=(n-m)d 求 d 和 a₁）

**答案：S₁₀ = 190**

**解析：** a₆ - a₂ = 4d = 13-5 = 8 → d = 2。a₁ = a₂ - d = 5-2 = 3。S₁₀ = 10×3 + 10×9×2/2 = 30+90 = 190。或者 S₁₀ = 10(a₁+a₁₀)/2 = 10×(3+21)/2 = 120... 不对，a₁₀ = 3+9×2 = 21，S₁₀ = 10×24/2 = 120。

**修正答案：S₁₀ = 120**
