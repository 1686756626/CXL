# 函数性质（奇偶性+单调性）

## 这个知识点是什么？

函数性质就是研究**函数图像有什么规律**——图像对称不对称（奇偶性）、上升还是下降（单调性）。高考选择题Q6几乎必考。

> 打个比方：奇偶性就是看图像"照镜子"——偶函数照的是y轴镜子（左右对称），奇函数照的是原点镜子（绕原点转180°和原来一样）。

## 知识结构图

```
函数性质
├── 奇偶性
│   ├── 偶函数：f(-x) = f(x)，关于y轴对称
│   ├── 奇函数：f(-x) = -f(x)，关于原点对称
│   ├── 判断方法：代入-x看等不等
│   └── 前提：定义域关于原点对称
│
├── 单调性
│   ├── 单调递增：x增大时y增大
│   ├── 单调递减：x增大时y减小
│   ├── 求导法：f'(x)>0递增，f'(x)<0递减
│   ├── 定义法：任取x₁<x₂，比较f(x₁)和f(x₂)
│   └── 与分段函数结合：分段判断
│
├── 应用
│   ├── 比较大小（利用单调性）
│   ├── 求最值（单调区间端点）
│   ├── 解不等式（转化为单调性判断）
│   └── 奇偶性+单调性综合
│
├── 高频考法
│   ├── 判断奇偶性（代入验证）
│   ├── 判断单调性（求导或定义法）
│   ├── 比较函数值大小
│   └── 已知奇偶性求参数
│
└── 关联知识点
    ├── 分段函数（07）
    ├── 导数（15）
    └── 三角函数（04，sin是奇函数，cos是偶函数）
```

## 第一部分：奇偶性

### 判断奇偶性的步骤

```
① 检查定义域是否关于原点对称（不对称就不是奇函数也不是偶函数）
② 计算 f(-x)
③ 比较 f(-x) 和 f(x)：
   - f(-x) = f(x) → 偶函数
   - f(-x) = -f(x) → 奇函数
   - 都不满足 → 非奇非偶
```

> ⚠️ 定义域关于原点对称是**前提条件**！比如 f(x) = x²（x > 0），定义域 (0,+∞) 不关于原点对称，所以不是偶函数。

### 例题1（基础）：判断奇偶性

**例题1a：** f(x) = x³，判断奇偶性。

f(-x) = (-x)³ = -x³ = -f(x) ✅

**奇函数** ✅

**例题1b：** f(x) = x² + 1，判断奇偶性。

f(-x) = (-x)² + 1 = x² + 1 = f(x) ✅

**偶函数** ✅

**例题1c：** f(x) = x² + x，判断奇偶性。

f(-x) = x² - x

f(-x) ≠ f(x)（因为 x²-x ≠ x²+x），f(-x) ≠ -f(x)（因为 x²-x ≠ -x²-x）

**非奇非偶** ✅

### 例题2（中等，带绝对值）：f(x) = |x - 1| + |x + 1|

f(-x) = |-x - 1| + |-x + 1| = |x + 1| + |x - 1| = f(x)

**偶函数** ✅

### 奇偶性的性质

- 偶函数的图像关于 **y 轴对称**
- 奇函数的图像关于 **原点对称**
- f(0) = 0 是奇函数的**必要条件**（如果0在定义域内）
- 奇函数+奇函数 = 奇函数，偶函数+偶函数 = 偶函数

## 第二部分：单调性

### 两种判断方法

**方法一：求导法（推荐，速度快）**

```
① 求导 f'(x)
② 解 f'(x) > 0 → 递增区间
③ 解 f'(x) < 0 → 递减区间
```

**方法二：定义法（求导不方便时用）**

```
① 任取 x₁ < x₂（在定义域内）
② 计算 f(x₂) - f(x₁)
③ 判断符号：正→递增，负→递减
```

### 例题3（基础，求导法）：判断 f(x) = x² - 2x 的单调性

f'(x) = 2x - 2

f'(x) > 0 → 2x - 2 > 0 → x > 1 → 在 **(1, +∞) 递增**
f'(x) < 0 → x < 1 → 在 **(-∞, 1) 递减** ✅

### 例题4（中等，分段函数）：判断分段函数的单调性

> f(x) = { x² + 1, x ≤ 0; 2^x, x > 0 }，判断单调性。

**分段判断：**

x ≤ 0 时：f(x) = x² + 1，f'(x) = 2x < 0 → 在 (-∞, 0] **递减**
x > 0 时：f(x) = 2^x，f'(x) = 2^x ln2 > 0 → 在 (0, +∞) **递增**

**检查衔接点：** x = 0 处
lim(x→0⁻) f(x) = 0² + 1 = 1
f(0) = 1
lim(x→0⁺) f(x) = 2⁰ = 1

左端递减到 f(0)=1，右端从 f(0)=1 开始递增 → **整体不是单调函数**（先减后增）

但递减区间是 (-∞, 0]，递增区间是 [0, +∞) ✅

## 第三部分：应用

### 题型一：比较大小

**例题5（中等）：** 已知 f(x) 是奇函数，在 (0, +∞) 上单调递减，且 f(2) = -3，则 f(-3) 与 f(-1) 的大小关系是？

奇函数：f(-x) = -f(x)
f(x) 在 (0, +∞) 递减 → 在 (-∞, 0) 也递减（奇函数在对称区间单调性相同）

在 (-∞, 0) 上递减：-3 < -1 → f(-3) > f(-1)

或者用奇函数性质：f(-3) = -f(3), f(-1) = -f(1)
(0,+∞)递减：f(3) < f(1) → -f(3) > -f(1) → f(-3) > f(-1) ✅

### 题型二：利用单调性解不等式

**例题6（挑战）：** 已知 f(x) 是定义在 R 上的偶函数，在 (-∞, 0] 上递减，且 f(3) = 1，解不等式 f(x) ≥ 1。

偶函数在 (0, +∞) 上递减（对称区间单调性相反）

f(x) ≥ 1 = f(3)

在 (0, +∞) 上递减：f(x) ≥ f(3) → x ≤ 3

在 (-∞, 0) 上递减：f(x) = f(-x) ≥ 1 → f(-x) ≥ f(3)
(-∞, 0) 上递减 → -x ≤ 3 → x ≥ -3

**答案：-3 ≤ x ≤ 3** ✅

## ⚠️ 常见错误

1. **忘记检查定义域**：定义域不对称的函数一定不是奇/偶函数
2. **奇函数 f(0)=0 忘记验证**：如果0在定义域内但f(0)≠0，一定不是奇函数
3. **单调区间写错端点**：递减到x=1和递增从x=1开始，x=1是分界点
4. **分段函数忘记检查衔接点**：两段在分界点的值要连续才能整体判断

## 高考常见陷阱

1. **"定义域对称"是隐含条件**：题目说"函数f(x)是奇函数"就隐含定义域关于原点对称
2. **奇函数在对称区间单调性相同，偶函数相反**：f(x)在(0,+∞)递增→奇函数在(-∞,0)也递增，偶函数在(-∞,0)递减
3. **f(x)单调递增不能推出f(x)是奇/偶函数**：单调性和奇偶性是两个独立性质
4. **抽象函数题**：题目不给你具体表达式，只说"f(x)是奇函数且f(1)=2"，这种要用性质推理

## 小测验

1.（基础）判断下列函数的奇偶性：(a) f(x) = x³-3x (b) f(x) = x⁴+2x² (c) f(x) = x+1

**答案：(a) 奇函数 (b) 偶函数 (c) 非奇非偶**

**解析：** (a) f(-x) = (-x)³-3(-x) = -x³+3x = -(x³-3x) = -f(x)，奇函数。(b) f(-x) = x⁴+2x² = f(x)，偶函数。(c) f(-x) = -x+1，既不等于 f(x)=x+1，也不等于 -f(x)=-x-1，非奇非偶。

2.（中等）f(x) = x² - 4x + 3 的单调递减区间是什么？

**答案：(-∞, 2)**

**解析：** f'(x) = 2x-4。f'(x) < 0 → 2x < 4 → x < 2。单调递减区间为 (-∞, 2)。

3.（挑战）f(x) 是奇函数，在 (0,+∞) 上递增，f(1) = 2，求 f(-3) 的值并判断 f(-3) 与 f(-1) 的大小关系。

**答案：f(-3) = -2，f(-3) < f(-1)**

**解析：** 奇函数 f(-x) = -f(x)，所以 f(-3) = -f(3)，f(-1) = -f(1) = -2。奇函数在 (-∞,0) 上也是递增的。在 (-∞,0) 上 -3 < -1，递增函数中小的自变量对应小的函数值，所以 f(-3) < f(-1)。即 -f(3) < -f(1) → f(3) > f(1)。因为 (0,+∞) 递增，f(3) > f(1) = 2，所以 f(3) > 2，f(-3) = -f(3) < -2 = f(-1)。
