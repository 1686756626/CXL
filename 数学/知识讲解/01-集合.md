# 集合

## 这个知识点是什么？

"集合"听起来很抽象，但其实你每天都在用它。

打个比方：你说"我们班身高超过170的男生"，这句话就已经描述了一个"集合"——把符合条件的那些人**放到一起**，就是一个集合。

就这么简单。**集合就是把满足某些条件的东西放在一起。**

## 一定要记住的几个概念

### 1. 元素
集合里的每一个东西，都叫"元素"。

### 2. 集合的写法
- **大括号** `{}` 用来表示集合：`{1, 2, 3}`
- **花体字母** 或者大写字母给集合起名字：`A = {1, 2, 3}`

### 3. 属于和不属于
- `2 ∈ A`：读作"2 属于 A"，意思是 2 是集合 A 里的元素 ✅
- `4 ∉ A`：读作"4 不属于 A" ❌

### 4. 集合的三大特征（考试常考概念题）

| 特征 | 含义 | 例子 |
|------|------|------|
| **确定性** | 每个东西要么在集合里，要么不在 | "身高170以上"——170在不在？看不等号 |
| **互异性** | 集合里没有重复的元素 | `{1, 1, 2}` = `{1, 2}` |
| **无序性** | 排列顺序无所谓 | `{1, 2, 3}` = `{3, 2, 1}` |

## 常见的集合（一定要认识）

| 符号 | 含义 | 例子 |
|------|------|------|
| `N` | 自然数（从0开始） | 0, 1, 2, 3, ... |
| `Z` | 整数 | ..., -2, -1, 0, 1, 2, ... |
| `Q` | 有理数 | 可以写成分数的数 |
| `R` | 实数 | 所有的数 |

> ⚠️ 数学上 N 包含 0！高考题有时候会考这个。

区间写法：
- `(1, 3)`：大于1且小于3（不包含端点）
- `[1, 3)`：大于等于1且小于3（圆括号=不包含，方括号=包含）

## 集合间的关系

### 子集 A ⊆ B
A 中每个元素都在 B 中。比如 `{1, 2} ⊆ {1, 2, 3}` ✅

### 真子集 A ⊂ B
A ⊆ B 但 A ≠ B。比如 `{1, 2} ⊂ {1, 2, 3}` ✅

> ⚠️ **空集 ∅ 是任何集合的子集**，包括空集本身。这是高考最爱考的细节之一。

---

## 集合的运算（重点！）

### 1. 交集 A ∩ B
两个集合里**都有的**那些元素。

> A 是"会唱歌的同学"，B 是"会跳舞的同学"，A∩B 就是"既会唱歌又会跳舞的同学"。

### 2. 并集 A ∪ B
两个集合里**所有的**元素合在一起。

### 3. 补集 ∁ᵤA
在全集 U 里，**不属于 A** 的那些元素。

> ⚠️ 做补集一定要先看清楚**全集 U 是什么**！

## 做题的时候怎么用？

### 题型一：集合的基本运算（必考，几乎每年Q1）

**例题1（基础，交集+不等式）：**
> 已知 A = {x | x² - 3x + 2 = 0}，B = {x | 1 ≤ x ≤ 4}，求 A ∩ B。

**解题步骤：**

第一步：A 里面有什么。x² - 3x + 2 = (x-1)(x-2) = 0 → x = 1 或 x = 2，A = {1, 2}

第二步：B = [1, 4]

第三步：A ∩ B = 既在 A 里又在 B 里的元素
- 1 ∈ [1, 4] ✅，2 ∈ [1, 4] ✅
- A ∩ B = **{1, 2}** ✅

**例题2（中等，并集+补集）：**
> 已知 U = R，A = {x | x > 1}，B = {x | x < 3}，求 A ∪ B 和 ∁ᵤA。

**A ∪ B：**
- 小于1的数：在 B 里 ✅
- 1到3之间的数：在 A 里（>1）或在 B 里（<3），至少满足一个 ✅
- 大于3的数：在 A 里 ✅
- **A ∪ B = R**（每个实数至少在一个集合里）

**∁ᵤA：**
- A = (1, +∞)
- ∁ᵤA = **(-∞, 1]** ✅（注意1不在A里，所以1在补集里）

### 题型二：韦恩图应用题

**例题3（基础）：**
> 某班50人，参加语文兴趣小组20人，数学兴趣小组25人，两个都参加8人。两个都没参加的有多少人？

只参加语文的：20 - 8 = 12人
只参加数学的：25 - 8 = 17人
两个都参加：8人
都没参加：50 - (12 + 17 + 8) = **13人** ✅

### 题型三：含参集合求范围（高考高频）

**例题4（中等）：**
> 已知 A = {x | x² - 5x + 6 ≤ 0}，B = {x | m < x < m+3}，若 A ⊆ B，求 m 的取值范围。

**第一步：求 A**
x² - 5x + 6 ≤ 0 → (x-2)(x-3) ≤ 0 → A = **[2, 3]**

**第二步：A ⊆ B 意味着区间 [2, 3] 完全在 (m, m+3) 内**

需要：m < 2 且 m+3 > 3，即 m < 2 且 m > 0

**答案：0 < m < 2** ✅

**例题5（挑战，含参集合+讨论）：**
> 已知 A = {x | 1 < x < 3}，B = {x | x² - ax + a - 1 < 0}，若 B ⊆ A，求 a 的取值范围。

**第一步：分析 B**
x² - ax + a - 1 = 0 的两个根：
x = (a ± √(a² - 4(a-1))) / 2 = (a ± √(a²-4a+4)) / 2 = (a ± |a-2|) / 2

**情况一：a ≥ 2**
两根：(a + a-2)/2 = a-1 和 (a - a+2)/2 = 1
B = (1, a-1)
B ⊆ A → a-1 ≤ 3 → a ≤ 4
所以 **2 ≤ a ≤ 4**

**情况二：a < 2**
两根：(a + 2-a)/2 = 1 和 (a - 2+a)/2 = a-1
B = (a-1, 1)
B ⊆ A → a-1 ≥ 1 且 1 ≤ 3... 但 a-1 ≥ 1 → a ≥ 2，与 a < 2 矛盾
**无解**

**答案：2 ≤ a ≤ 4** ✅

## ⚠️ 常见错误

1. **搞混 ∩ 和 ∪**：∩ 是"都要有"，∪ 是"加在一起"
2. **补集忘看全集**：U 不同答案完全不同
3. **空集忘记考虑**：A∩B 可能等于 ∅，A ⊆ B 时要考虑 A=∅ 的情况
4. **区间端点搞混**：圆括号不包含，方括号包含

## 高考常见陷阱

1. **∅ ∈ A vs ∅ ⊆ A**：∅ ∈ A 是错的（∅ 不是元素），∅ ⊆ A 是对的（空集是任何集合的子集）
2. **补集的端点开闭**：A = (1, 3) 的补集在全集 R 下是 (-∞, 1] ∪ [3, +∞)，端点从开变闭
3. **含参集合中 A = ∅ 的情况**：如果 A = {x | mx+1 < 0}，m=0 时 A = {x | 1 < 0} = ∅，需要单独讨论
4. **两个集合的"包含"方向**：A ⊆ B 是 A 中的元素都在 B 中，别搞反

## 小测验

1.（基础）已知 A = {1, 2, 3}，B = {2, 3, 4, 5}，求 A ∩ B、A ∪ B、∁ᵤA（U = {1,2,3,4,5}）

**答案：A ∩ B = {2, 3}，A ∪ B = {1, 2, 3, 4, 5}，∁ᵤA = {4, 5}**

**解析：** 交集就是两边都有的：2和3。并集就是全部合在一起。补集就是U里有但A里没有的：4和5。

2.（中等）已知 A = {x | -1 < x < 2}，B = {x | 0 ≤ x ≤ 3}，求 A ∩ B 和 ∁ᵤ(A ∪ B)（U = R）

**答案：A ∩ B = [0, 2)，∁ᵤ(A ∪ B) = (-∞, -1] ∪ (3, +∞)**

**解析：** A 是 (-1,2)，B 是 [0,3]。A∩B 是两个区间的重叠部分：从0开始（B含0所以结果含0），到2结束（A不含2所以结果不含2），所以是 [0, 2)。A∪B = (-1, 3]，补集就是把 >3 和 ≤-1 的部分取出来，端点开闭互换。

3.（挑战）已知 A = {x | x² - 4x + 3 ≤ 0}，B = {x | m-1 < x < 2m-1}，若 A ∩ B = A，求 m 的范围

**答案：不存在这样的 m（空集）**

**解析：** A ∩ B = A 意味着 A ⊆ B。先解 A：(x-1)(x-3) ≤ 0 → A = [1, 3]（闭区间）。B 是 (m-1, 2m-1)（开区间）。要让闭区间 [1, 3] 完全在开区间里面，需要 m-1 < 1（即 m < 2）且 2m-1 > 3（即 m > 2）。m < 2 且 m > 2，没有数同时满足两个条件，所以答案是空集。**开区间无法完全包含闭区间的端点**，这就是开区间和闭区间的区别。
