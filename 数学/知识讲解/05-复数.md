# 复数

## 这个知识点是什么？

"复数"这个词听着吓人，但其实就是为了解决一个问题：**负数能不能开平方？**

数学家发明了一个"虚拟"的数：**i**，规定 `i² = -1`。复数就是把**实数**和这个**虚数 i** 组合在一起的数。

> 打个比方：实数像是"只用红色颜料画的画"，虚数 i 像是"蓝色颜料"。复数就是"红蓝混在一起画出来的画"——多了 i 这个新工具，能表达的东西更多了。

## 一定要记住的基础知识

### 1. 复数的写法

复数的一般形式：**a + bi**

- **a** 叫"实部"，**b** 叫"虚部"
- `3 + 2i`：实部是 3，虚部是 2
- `5`（即 `5 + 0i`）：就是普通的实数
- `2i`（即 `0 + 2i`）：纯虚数

> ⚠️ 题目问"虚部是多少"，答案是 b（一个实数），不是 bi！`3 + 2i` 的虚部是 **2**，不是 2i。

### 2. i 的乘方规律（必背！）

| i 的幂 | 等于 |
|---------|------|
| i¹ | i |
| i² | -1 |
| i³ | -i |
| i⁴ | 1 |
| i⁵ | i（每4个一循环） |

**做题技巧**：用 4 去除指数看余数。
- `i²⁰²⁵`：2025 ÷ 4 余 1 → 等于 **i**
- `i²⁰²⁶`：2026 ÷ 4 余 2 → 等于 **-1**

### 3. 共轭复数

`a + bi` 的共轭复数是 `a - bi`（只变虚部的符号）

共轭复数的重要性质：
- `z + z̄ = 2a`（两倍实部）
- `z · z̄ = a² + b²`（模的平方）
- `z̄ = z` 当且仅当 z 是实数

## 复数的运算（高考重点）

核心规则：**把 i 当成普通字母正常运算，最后把 i² 换成 -1。**

### 1. 加减法：实部加减实部，虚部加减虚部

`(a + bi) ± (c + di) = (a±c) + (b±d)i`

### 2. 乘法：展开后 i² → -1

`(a + bi)(c + di) = (ac - bd) + (ad + bc)i`

**例题1（基础）：** `(2 + i)(3 - 2i)`

= 6 - 4i + 3i - 2i² = 6 - i + 2 = **8 - i** ✅

### 3. 除法：分子分母同乘分母的共轭复数

```
    a + bi       (a + bi)(c - di)       实部 + 虚部i
   -------  =  -----------------  =  -----------------
    c + di       (c + di)(c - di)          实数
```

**例题2（基础）：** `(3 + 2i) / (1 - i)`

分母：(1-i)(1+i) = 1 - i² = 2
分子：(3+2i)(1+i) = 3 + 3i + 2i + 2i² = 3 + 5i - 2 = 1 + 5i

答案：**1/2 + (5/2)i** ✅

### 4. 复数的模

`|z| = |a + bi| = √(a² + b²)`

> 💡 模的几何意义：复数 z 在复平面上对应点 (a, b)，|z| 就是原点到 (a, b) 的距离。

## 做题的时候怎么用？

### 题型一：复数的基本运算（必考，选择题前几题）

**例题3（基础）：** 已知 z = (1 + 2i)(1 - i)，求 z 的实部和虚部。

(1+2i)(1-i) = 1 - i + 2i - 2i² = 1 + i + 2 = 3 + i

实部 = **3**，虚部 = **1** ✅

### 题型二：共轭复数与模

**例题4（中等）：** 已知 z = 3 - 4i，求 z 的共轭复数 z̄、|z|、z·z̄。

z̄ = **3 + 4i**
|z| = √(9 + 16) = **5**
z·z̄ = (3-4i)(3+4i) = 9 + 16 = **25** ✅

### 题型三：i 的高次幂

**例题5（基础）：** i²⁰²⁴ + i²⁰²⁵ + i²⁰²⁶ + i²⁰²⁷ = ？

四个连续幂刚好是一个完整循环：
= i⁰ + i¹ + i² + i³ = 1 + i + (-1) + (-i) = **0** ✅

### 题型四：复数方程（高考中频）

**例题6（中等）：** 已知复数 z 满足 z + z̄ = 4 且 z·z̄ = 8，求 z。

设 z = a + bi：
- z + z̄ = 2a = 4 → **a = 2**
- z·z̄ = a² + b² = 8 → 4 + b² = 8 → b² = 4 → b = ±2

**z = 2 + 2i 或 z = 2 - 2i** ✅

### 题型五：复数模的几何意义（高考中频）

**例题7（挑战）：** 已知 |z - 1| = 2，求 |z| 的最大值和最小值。

> 💡 |z - 1| = 2 的几何意义：在复平面上，z 到点 1（即 (1,0)）的距离为 2。这是一个以 (1,0) 为圆心、半径为 2 的圆。

|z| = |z - 0| 表示 z 到原点的距离。

圆心 (1,0) 到原点距离 = 1

最大值 = 1 + 2 = **3**
最小值 = 2 - 1 = **1**

**|z| 的范围：[1, 3]** ✅

## ⚠️ 常见错误

1. **虚部写成 bi**：z = 3 + 2i 的虚部是 **2**，不是 2i
2. **除法忘乘共轭复数**：看到复数除法，第一步就是乘分母的共轭
3. **乘法展开忘记 i² = -1**：最后一步检查有没有 i² 没换掉
4. **i 的幂算错**：用"除以4看余数"最保险

## 高考常见陷阱

1. **复数模 vs 绝对值**：|z| 是模，|z|² = z·z̄ = a²+b²，不是 |a|+|b|
2. **共轭公式记混**：z + z̄ = 2a（实部的2倍），不是 2b
3. **复数方程隐含条件**：z·z̄ = |z|² 一定非负，如果算出来 a²+b² < 0 说明方程无解
4. **复数不能比较大小**：两个复数（只要不都是实数）不能说谁大谁小，只能比模

## 小测验

1.（基础）已知 z = 3 - 4i，求 z 的实部、虚部、共轭复数和模。

**答案：实部 = 3，虚部 = 4，z̄ = 3 + 4i，|z| = 5**

**解析：** a + bi 中 a 是实部、b 是虚部。共轭复数变虚部符号。模 = √(3²+4²) = √25 = 5。

2.（中等）已知 z = (2+i)/(1-i)，求 z 的实部和虚部。

**答案：实部 = 1/2，虚部 = 3/2**

**解析：** 分子分母同乘 1+i：分子 (2+i)(1+i) = 2+2i+i+i² = 1+3i，分母 (1-i)(1+i) = 2。所以 z = (1+3i)/2 = 1/2 + (3/2)i。

3.（挑战）已知 |z - i| = 1，求 |z + 1| 的最大值。（提示：画圆，圆心(0,1)半径1，求到(-1,0)的最远距离）

**答案：最大值 = 1 + √5**

**解析：** |z - i| = 1 表示 z 到点 (0,1) 的距离为1，这是一个圆心 (0,1)、半径1的圆。|z + 1| = |z - (-1)| 表示 z 到点 (-1,0) 的距离。圆心 (0,1) 到 (-1,0) 的距离 = √(1+1) = √2。最大值 = √2 + 1 = 1 + √2。
