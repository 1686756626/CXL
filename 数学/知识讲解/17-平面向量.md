# 平面向量（坐标运算+点乘）

## 这个知识点是什么？

向量就是**既有大小又有方向的量**。速度、力、位移都是向量。在高考里，平面向量主要考**坐标运算和点乘**，套路固定，属于"白送分"的题型。

> 打个比方：普通数字（标量）只告诉你"走了多远"，向量还告诉你"往哪个方向走"。高考考的是"怎么用坐标算出来"。

## 知识结构图

```
平面向量
├── 基本概念
│   ├── 向量表示：a = (x, y)（起点在原点）
│   ├── 零向量：(0, 0)，模为0
│   ├── 单位向量：模为1的向量
│   └── 相等向量：方向相同、长度相等
│
├── 坐标运算
│   ├── 加法：(a₁,a₂)+(b₁,b₂) = (a₁+b₁, a₂+b₂)
│   ├── 减法：(a₁,a₂)-(b₁,b₂) = (a₁-b₁, a₂-b₂)
│   ├── 数乘：k(a₁,a₂) = (ka₁, ka₂)
│   └── 向量坐标 = 终点坐标 - 起点坐标
│
├── 模与点乘
│   ├── 模：|a| = √(x²+y²)
│   ├── 点乘：a·b = x₁x₂+y₁y₂
│   ├── 点乘几何意义：|a||b|cosθ
│   └── 用点乘判断垂直：a·b = 0 ⟺ a⊥b
│
├── 高频考法
│   ├── 坐标加减运算（必考）
│   ├── 求向量模
│   ├── 点乘运算+判断垂直

│

    ├── 三角函数（点乘求夹角要用cos）
    ├── 立体几何（空间向量是推广）
    └── 解析几何（向量与直线、圆）
```

## 第一部分：向量的坐标表示

### 向量 a = (x, y) 的含义

- **x**：向量在 x 方向的分量（水平）
- **y**：向量在 y 方向的分量（竖直）

> 💡 如果向量 a 的起点是 A(x₁,y₁)，终点是 B(x₂,y₂)，那么 a = **(x₂-x₁, y₂-y₁)**。

### 例题1（基础）：已知 A(1, 2)，B(4, 6)，求向量 AB 和 |AB|。

AB = B - A = (4-1, 6-2) = **(3, 4)** ✅

|AB| = √(3²+4²) = **5** ✅

> 💡 这就是经典的 3-4-5 直角三角形。

---

## 第二部分：坐标运算

### 加法：对应分量相加

a + b = (a₁+b₁, a₂+b₂)

### 减法：对应分量相减

a - b = (a₁-b₁, a₂-b₂)

### 数乘：每个分量乘以 k

k · a = (ka₁, ka₂)

### 例题2（基础）：已知 a = (2, -1)，b = (-3, 4)，求 a+b、a-b、2a+3b。

a + b = (2+(-3), -1+4) = **(-1, 3)** ✅
a - b = (2-(-3), -1-4) = **(5, -5)** ✅
2a + 3b = (4, -2) + (-9, 12) = **(-5, 10)** ✅

### 例题3（中等）：已知 a = (1, 2)，b = (x, 4)，若 a ∥ b，求 x。

a ∥ b 的条件：x₁y₂ = x₂y₁（对应分量成比例）

1×4 = 2×x → x = **2** ✅

> 💡 a ∥ b ⟺ a = kb（存在实数k使 a 是 b 的 k 倍），即 x₁/x₂ = y₁/y₂。

---

## 第三部分：模与点乘

### 模（向量的长度）

|a| = √(x² + y²)

### 点乘（内积）

a · b = x₁x₂ + y₁y₂

**点乘的几何意义：** a · b = |a| |b| cos θ（θ 是 a 和 b 的夹角）

### 用点乘判断垂直

**a ⊥ b ⟺ a · b = 0 ⟺ x₁x₂ + y₁y₂ = 0**

这个条件高考非常常用！

### 例题4（中等）：已知 a = (2, 1)，b = (-1, k)，若 a ⊥ b，求 k。

a · b = 2×(-1) + 1×k = -2 + k = 0 → k = **2** ✅

### 例题5（中等，求夹角）：已知 |a| = 3，|b| = 4，a·b = 6，求 a 和 b 的夹角 θ。

a · b = |a| |b| cos θ
6 = 3 × 4 × cos θ
cos θ = 6/12 = 1/2
θ = **60°** ✅

### 例题6（挑战，综合）：在△ABC 中，A(0,0)，B(3,0)，C(1,2)，求角 A 的值。

AB = (3, 0)，AC = (1, 2)

AB · AC = 3×1 + 0×2 = 3

|AB| = 3，|AC| = √(1+4) = √5

cos A = (AB · AC) / (|AB| |AC|) = 3 / (3√5) = **1/√5 = √5/5**

A = **arccos(√5/5)** ✅

> 💡 求三角形内角，用两边向量的点乘除以两边模的乘积。

## 做题的时候怎么用？

### 求向量题的通用步骤

```
① 看清楚给了什么（坐标？点？还是向量表达式？）
② 如果给的是点，先算向量（终点-起点）
③ 如果问垂直，用点乘=0
④ 如果问平行，用分量成比例
⑤ 如果问角度，用 cosθ = (a·b)/(|a||b|)
```

## ⚠️ 常见错误

1. **向量和点搞混**：向量 a = (2,3) 不是点 (2,3)。点是位置，向量是位移。
2. **减法方向搞反**：AB = B - A（终点减起点），不是 A - B
3. **平行条件记错**：a∥b 是 x₁y₂ = x₂y₁（交叉相乘相等），不是 x₁x₂ = y₁y₂
4. **点乘结果是标量不是向量**：a·b 是一个数，不是向量

## 高考常见陷阱

1. **零向量与任何向量平行**：a ∥ 0 对所有 a 成立，但 0 不能作为基底
2. **夹角范围是 [0°, 180°]**：cos θ 可能是负数（钝角），要能接受负的 cos 值
3. **|a+b| ≠ |a|+|b|**：只有 a 和 b 同向时才等号成立。一般情况用余弦定理：|a+b|² = |a|²+|b|²+2a·b
4. **向量坐标化后就是普通代数运算**：不要想得太复杂，把 (x,y) 当成坐标算就行

## 小测验

1.（基础）已知 a = (3, -2)，b = (-1, 4)，求 a+b、a-b、|a| 和 a·b。

**答案：a+b = (2, 2)，a-b = (4, -6)，|a| = √13，a·b = -11**

**解析：** a+b = (3+(-1), -2+4) = (2,2)。a-b = (3-(-1), -2-4) = (4,-6)。|a| = √(9+4) = √13。a·b = 3×(-1)+(-2)×4 = -3-8 = -11。

2.（中等）已知 a = (2, -1)，b = (k, 3)，若 a⊥b，求 k；若 a∥b，求 k。

**答案：a⊥b 时 k = 3/2；a∥b 时 k = -6**

**解析：** a⊥b → a·b = 0 → 2k+(-1)×3 = 0 → 2k = 3 → k = 3/2。a∥b → x₁y₂=x₂y₁ → 2×3=(-1)×k → k = -6。

3.（挑战）在△ABC 中，A(1,0)，B(4,0)，C(2,3)，求角 B。（提示：用向量 BA 和 BC 的点乘）

**答案：角 B = arccos(1/√10)**

**解析：** BA⃗ = A-B = (-3, 0)，BC⃗ = C-B = (-2, 3)。BA⃗·BC⃗ = (-3)×(-2)+0×3 = 6。|BA⃗| = 3，|BC⃗| = √(4+9) = √13。cos B = (BA⃗·BC⃗)/(|BA⃗||BC⃗|) = 6/(3√13) = 2/√13。角 B = arccos(2/√13)。

**修正答案：角 B = arccos(2/√13)**
