# 复习讲解稿（集合+命题+指数+薄弱点）

## 一、集合（约10分钟）

### 1. 集合的表示方法
- 列举法：$\{1, 2, 3\}$
- 描述法：$\{x \mid x > 0\}$，竖线前面是代表元素，后面是条件

### 2. 交集与并集
- $A \cap B$：**两个都要满足**，取公共部分
- $A \cup B$：**满足一个就行**，全部合起来

### 3. 区间（⚠️ 易错点）
- **开区间** $(a, b)$：端点**取不到**
- **闭区间** $[a, b]$：端点**能取到**
- 求交集时，**端点能不能取要用 $\leq$ 或 $<$ 仔细检查**

### 4. 空集（⚠️ 易错点）
- 空集 $\emptyset$ 也是集合，它**是任何集合的子集**
- 求子集个数时别忘了算上空集
- **解方程/不等式无解时，答案就是空集**

### 5. 含参问题（⚠️ 薄弱点）
- 解含参不等式时，**参数要分类讨论**
- 例：$x < a$，不知道 $a$ 是几，就要分 $a$ 大于/等于/小于边界值来讨论

## 二、命题与充要条件（约8分钟）

### 1. 四种命题
- 原命题：若 $p$ 则 $q$
- 逆命题：若 $q$ 则 $p$
- 否命题：若 $\neg p$ 则 $\neg q$
- 逆否命题：若 $\neg q$ 则 $\neg p$
- **原命题和逆否命题真假相同**

### 2. 充分条件与必要条件
- 若 $p \Rightarrow q$：$p$ 是 $q$ 的**充分条件**，$q$ 是 $p$ 的**必要条件**
- 记忆口诀：**"谁推谁，谁是充分"**
  - $p$ 推出 $q$，那 $p$ 就充分（有它就够了）
- **充分且必要 = 充要条件**（互相推出，等价）

### 3. 判断方法
- **小范围推大范围**：小是大的充分条件
- 例：$x = 1 \Rightarrow x^2 = 1$，"$x=1$"是"$x^2=1$"的充分条件

## 三、指数（约12分钟）

### 1. 指数运算法则
- $a^m \cdot a^n = a^{m+n}$
- $\frac{a^m}{a^n} = a^{m-n}$
- $(a^m)^n = a^{mn}$
- $(ab)^n = a^n \cdot b^n$

⚠️ **常见错误：**
- $a^m + a^n \neq a^{m+n}$（加法不能合并！）
- $(a+b)^n \neq a^n + b^n$（展开不是这样！）

### 2. 负数指数
- $a^{-n} = \frac{1}{a^n}$，就是**倒过去**

### 3. 分数指数（根式）
- $a^{\frac{1}{n}} = \sqrt[n]{a}$（n次根号）
- $a^{\frac{m}{n}} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m$
- ⚠️ 负数的分数指数要小心，**偶次根号下不能为负**

### 4. 化同底（⚠️ 薄弱点）
- 比较大小时，**先化成同底数**再比较指数
- 例：$2^3$ 和 $4^2$ → $4 = 2^2$，所以 $4^2 = 2^4 = 16$，$16 > 8$
- 常用：$4 = 2^2$，$8 = 2^3$，$9 = 3^2$，$16 = 2^4 = 4^2$

## 四、函数图像平移（⚠️ 薄弱点，约5分钟）

**口诀：左加右减，上加下减**

对 $y = f(x)$：
- $y = f(x + a)$ → 图像向**左**移 $a$ 个单位（加 $a$ 反而左移）
- $y = f(x - a)$ → 图像向**右**移 $a$ 个单位
- $y = f(x) + b$ → 图像向**上**移 $b$ 个单位
- $y = f(x) - b$ → 图像向**下**移 $b$ 个单位

**举例：**
- $y = 2^x$ 向左移1个 → $y = 2^{x+1}$
- $y = 2^x$ 向下移2个 → $y = 2^x - 2$

## 五、快速检测（讲完用，5分钟）

1. $A = \{1, 2\}$，$B = \{2, 3\}$，求 $A \cap B$ 和 $A \cup B$
2. "$x > 2$" 是 "$x > 1$" 的什么条件？（充分/必要/充要）
3. 计算 $8^{\frac{2}{3}}$
4. $y = 2^{x-1}$ 是 $y = 2^x$ 向哪平移了几个单位？
5. $A = \{x \mid -1 < x \leq 2\}$，$B = \{x \mid 0 < x < 3\}$，求 $A \cap B$

---

**参考答案：**
1. $A \cap B = \{2\}$，$A \cup B = \{1, 2, 3\}$
2. 充分条件（$x>2$ 能推出 $x>1$，反过来不行）
3. $8^{\frac{2}{3}} = (\sqrt[3]{8})^2 = 2^2 = 4$
4. 向右平移1个单位
5. $A \cap B = \{x \mid 0 < x \leq 2\}$（注意左端点是开，右端点是闭）
