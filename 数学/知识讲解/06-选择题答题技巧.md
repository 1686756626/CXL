# 选择题答题技巧

## 这个知识点是什么？

这节不是讲某个数学知识点，而是教你**怎么"聪明地"做选择题**。

30多分的同学做选择题，最大的问题不是"不会"，而是**做题方法不对**——很多题其实不用完全算出来，用技巧就能找到答案。

> 打个比方：你要知道一个西瓜甜不甜，有两种方法。一种是把整个西瓜切开吃一口（硬算）；另一种是看瓜蒂新不新鲜、拍一拍听声音（技巧）。选择题就是给你这种"拍一拍"的机会！

## 技巧一：排除法（最常用，优先用）

**什么意思？** 不用找出正确答案，只要排除3个错的，剩下的就是对的。

> 4个选项，排除1个就有33%命中率，排除2个就有50%，排除3个就是100%。

### 什么时候用？
- 看到**"下列说法正确/错误的是"**这种题，逐个判断每个选项
- 有选项明显违背常识或公式的
- 有选项的符号（正负）明显不对的
- 含参集合、含参不等式等需要分类讨论的题

### 例题1（中等，含参集合）：

> 已知集合 A = {x | x² - 5x + 6 < 0}，B = {x | m < x < m+3}，若 A ⊆ B，则实数 m 的取值范围是（　）
> A. m ≤ 1　　B. 1 ≤ m ≤ 2　　C. m ≤ 2　　D. 0 ≤ m ≤ 2

**用排除法：**

先求 A：x² - 5x + 6 < 0 → (x-2)(x-3) < 0 → A = (2, 3)

A ⊆ B 意味着区间 (2,3) 完全包含在 (m, m+3) 里。

需要 m ≤ 2 且 m+3 ≥ 3，即 m ≤ 2 且 m ≥ 0。

所以 0 ≤ m ≤ 2，答案 D ✅

**用排除法快速验证：**
- A: m ≤ 1 包含了 m = -10，(-10, -7) 不包含 (2,3) ❌
- B: m = 1.5 时 B = (1.5, 4.5)，A ⊆ B ✅；但 m = 0 也满足，不在 B 的范围内 ❌（B 是 1≤m≤2）
- C: m = -1 时 B = (-1, 2)，不包含 3 ❌

答案 **D** ✅

### 例题2（中等，函数性质排除）：

> 已知 f(x) = ln(x² - 2x - 3)，则 f(x) 的定义域为（　）
> A. (-3, 1)　　B. (-∞, -1) ∪ (3, +∞)　　C. (3, +∞)　　D. (-1, 3)

**用排除法：**

需要 x² - 2x - 3 > 0 → (x+1)(x-3) > 0 → x < -1 或 x > 3

**快速排除：**
- A: 范围不含 -2，但 f(-2) 有意义（(-2)²-2(-2)-3=5>0）→ A 不完整 ❌
- C: 只有 (3,+∞)，缺了 (-∞,-1) ❌
- D: (-1,3) 里 x=0 代入得 -3<0，无意义 ❌

答案 **B** ✅

### 排除法实战技巧

- **先找最明显的错误**：负号对不对、范围合不合理、特殊值满不满足
- **含参题先代入边界值**：比如 m=0、m=1 试一下
- **多选题先排除"绝对错误"的选项**：找到一个反例就能排除

## 技巧二：特殊值法（最好用的技巧之一）

**什么意思？** 题目里有字母（变量）的时候，**自己随便给这个字母赋一个具体的数**，算出结果，看哪个选项符合。

### 什么时候用？
- 题目有变量，选项也是用变量表示的式子
- 题目说"对任意……成立"或没有限制变量取值
- 题目有抽象的函数关系，但可以代入具体值
- **选项含有"恒成立""恒为正/负"等字样**

### 例题1（中等，高考真题改编）：

> 若 a > b > 0，则下列不等式恒成立的是（　）
> A. a² > b²　　B. 1/a > 1/b　　C. 2^a > 2^b　　D. lg a > lg b

**用特殊值法：** 令 a = -1, b = -2... 不行，题目说 a > b > 0。

令 a = 1/2, b = 1/3（满足 a > b > 0）：
- A: (1/2)² = 1/4 > (1/3)² = 1/9 ✅ 先记下
- B: 1/(1/2) = 2, 1/(1/3) = 3, 2 > 3 ❌
- C: 2^(1/2) ≈ 1.41, 2^(1/3) ≈ 1.26, 1.41 > 1.26 ✅ 先记下
- D: lg(1/2) ≈ -0.3, lg(1/3) ≈ -0.48, -0.3 > -0.48 ✅ 先记下

剩下 A、C、D，再换一组特殊值验证：

令 a = 3, b = 2：
- A: 9 > 4 ✅
- C: 8 > 4 ✅
- D: lg3 ≈ 0.48, lg2 ≈ 0.30, 0.48 > 0.30 ✅

看起来 A、C、D 都成立？再想想有没有反例...

那 A、C、D 都对？这是一道**多选题**！

**答案：ACD** ✅

> 💡 特殊值法能帮你快速排除明显错误的选项，但**不能仅凭一组特殊值就确定所有正确选项**。多选题要换2-3组值验证。

### 例题2（中等，含参函数）：

> 已知 f(x) = x³ - ax² + 1，若 f(1) = 2，则 f(-1) =（　）
> A. 0　　B. -2　　C. 2　　D. 4

**用特殊值法：**

由 f(1) = 2：1 - a + 1 = 2 → a = 0

所以 f(x) = x³ + 1

f(-1) = -1 + 1 = **0**

答案 **A** ✅

### 特殊值法选值技巧

| 题目类型 | 推荐特殊值 | 为什么 |
|----------|-----------|--------|
| 一般代数题 | x = 0, 1, -1 | 最简单，计算量小 |
| 指数对数题 | x = 0, 1, 2 | 避免0的负数次方 |
| 三角函数题 | x = 0, π/6, π/4, π/3 | 常见角的值好算 |
| 分式/不等式 | x = 2, 1/2, -1 | 正负都试试 |
| 多选题验证 | 换2-3组不同的值 | 一组不够，要排除偶然 |

## 技巧三：代入法（把选项代入题目验证）

**和特殊值法的区别：** 特殊值法是给题目里的变量赋值；代入法是**把选项的答案放回题目里**，看成不成立。

### 什么时候用？
- 题目问"……的值为"或"……等于"
- 选项都是具体数值
- 直接计算太复杂但验证答案很简单
- **方程/不等式求根的题**

### 例题1（中等，高考真题改编）：

> 已知 f(x) = { 2^x, x ≤ 0; log₂(x+1), x > 0 }，若 f(a) = 1/2，则 a =（　）
> A. -2　　B. -1　　C. 1　　D. 3

**用代入法**（从中间选项 B、C 开始试）：

**试 B: a = -1**
- a = -1 ≤ 0 → f(-1) = 2^(-1) = 1/2 ✅

答案 **B** ✅

只试了1个选项就找到了！验证 C 也不妨：
**试 C: a = 1**
- a = 1 > 0 → f(1) = log₂(1+1) = log₂2 = 1 ≠ 1/2 ❌

### 例题2（中等，组合技巧）：

> 已知 sin α + cos α = √2，则 sin α cos α =（　）
> A. 0　　B. 1/2　　C. 1　　D. √2/2

**用代入法 + 排除法：**

先排除：sin α + cos α = √2，而 sin α + cos α 的最大值 = √2（因为 sin α + cos α = √2 sin(α+π/4)，最大值 = √2）。

所以这要求 sin α + cos α 取最大值，即 α = π/4。

代入 α = π/4：sin(π/4)·cos(π/4) = (√2/2)² = 1/2

答案 **B** ✅

> 💡 这道题也可以用公式：(sin α + cos α)² = 1 + 2sin α cos α → 2 = 1 + 2sin α cos α → sin α cos α = 1/2。但代入法更快。

## 技巧四：数形结合法（画图！）

**什么意思？** 有些题用式子算很麻烦，但**画个图就一目了然**。

### 什么时候用？
- 涉及函数图像、零点、交点个数
- 不等式和函数结合的题
- 向量、几何相关的选择题
- **三角函数的值域/单调性问题**

### 例题1（中等，函数零点）：

> 函数 f(x) = 2^x - x - 2 的零点个数为（　）
> A. 0　　B. 1　　C. 2　　D. 3

**画图法**：令 g(x) = 2^x 和 h(x) = x + 2，零点个数就是两图交点个数。

- g(x) = 2^x：过(0,1)，单调递增，始终在 x 轴上方
- h(x) = x + 2：直线，过(0,2)和(-2,0)

找交点：
- x = -1 时：g(-1) = 1/2, h(-1) = 1 → g < h
- x = 0 时：g(0) = 1, h(0) = 2 → g < h
- x = 2 时：g(2) = 4, h(2) = 4 → 相等！交点
- x = 3 时：g(3) = 8, h(3) = 5 → g > h

指数函数增长比直线快，所以 x > 2 之后 g(x) > h(x) 永远成立，不会再有交点。

**只有1个交点**，答案 **B** ✅

### 例题2（中等，三角函数值域）：

> 函数 f(x) = sin(2x + π/6) 在区间 [0, π/2] 上的值域为（　）
> A. [-1, 1]　　B. [-√3/2, 1]　　C. [-1/2, 1]　　D. [-√2/2, 1]

**数形结合法**：

2x + π/6 的范围：x ∈ [0, π/2] → 2x ∈ [0, π] → 2x + π/6 ∈ [π/6, 7π/6]

sin 在 [π/6, 7π/6] 上的值：
- 起点 sin(π/6) = 1/2
- 最大值 sin(π/2) = 1
- 终点 sin(7π/6) = -1/2

**值域 = [-1/2, 1]**

答案 **C** ✅

> 💡 画一个正弦函数图像，标出区间 [π/6, 7π/6]，一目了然。

## 技巧五：估算法（不需要算得精确）

**什么意思？** 有些题不需要精确计算，**大概估算一下范围就能排除选项**。

### 什么时候用？
- 计算复杂但选项差距大
- 可以通过量级/范围/符号快速判断
- **对数、根号、阶乘等大数估算**

### 例题1（中等，对数估算）：

> 已知 log₂3 = a，log₂5 = b，则 log₃5 =（　）
> A. a/b　　B. b/a　　C. ab　　D. a+b

**估算法 + 换底公式：**

log₃5 = log₂5 / log₂3 = b/a

答案 **B** ✅

**快速估算验证：**
log₂3 ≈ 1.58, log₂5 ≈ 2.32
log₃5 ≈ 1.46
b/a ≈ 2.32/1.58 ≈ 1.47 ✅（接近）

### 例题2（挑战，裂项相消）：

> (1 + 1/2)(1 + 1/3)(1 + 1/4)…(1 + 1/9) 的值为（　）
> A. 3　　B. 4　　C. 5　　D. 6

**裂项相消：**
= (3/2)(4/3)(5/4)…(10/9) = 10/2 = **5**

> 💡 分子逐项消掉，只留最后分子和第一个分母：(3/2)×(4/3)×…×(10/9) = 10/2 = 5

答案 **C** ✅

## 技巧组合（高考高分的关键）

真正的高考选择题，往往需要**两个甚至三个技巧组合使用**。

### 组合一：排除法 + 特殊值法

**例题（中等，一模Q9类型）：**

> 已知 a > 0, b > 0, a + b = 1，下列说法正确的是（　）
> A. ab ≥ 1/4　　B. a² + b² ≥ 1/2　　C. 1/a + 1/b ≤ 2　　D. ab ≤ 1/4

**先排除明显的错：**
- A: ab ≤ 1/4（基本不等式），所以 ab ≥ 1/4 **错** ❌

**再用特殊值验证：** 令 a = 1/2, b = 1/2
- B: (1/2)²+(1/2)² = 1/2 ≥ 1/2 ✅ 等号成立
- C: 2+2 = 4 ≤ 2 ❌
- D: 1/4 ≤ 1/4 ✅ 等号成立

再令 a = 1/3, b = 2/3 验证：
- B: 1/9 + 4/9 = 5/9 > 1/2 ✅
- D: 2/9 ≤ 1/4 = 0.25, 2/9 ≈ 0.22 ≤ 0.25 ✅

答案 **BD** ✅

### 组合二：数形结合 + 特殊值

**例题（挑战）：**

> 函数 f(x) = |x² - 2x| = a 有3个不同的实数根，则 a 的取值范围是（　）
> A. (0, 1)　　B. (0, 1]　　C. [1, 2)　　D. (1, 2)

**数形结合：** 画 y = |x² - 2x| 的图像。

x² - 2x = x(x-2)，根在 x=0 和 x=2。

当 x ≤ 0 或 x ≥ 2 时，y = x² - 2x（抛物线开口向上）
当 0 < x < 2 时，y = -(x² - 2x) = 2x - x²（开口向下的抛物线，顶点在 x=1, y=1）

图像在 x=1 处有最高点 y=1。

要使 y = a 和图像有3个交点，需要 0 < a < 1（水平线在0和1之间时，穿过图像3次）。

a = 0 时只有2个交点（x=0和x=2），a = 1 时有2个交点（x=1 和另一个）。

**答案 (0, 1) → A** ✅

## 🌟 多选题策略（重要！）

新高考有多选题（每题6分），选全对得6分，**选错得0分**，有部分得分机制。

### 核心原则：**宁可少选，不要多选！**

### 具体策略

1. **每个选项独立判断**：当作4道判断题来做
2. **先排除明显错误的**：找一个反例就能排除
3. **能确定几个选几个**：
   - 4个都能确定 → 全选（6分）
   - 能确定2-3个 → 只选确定的（4-5分）
   - 只能确定1个 → 只选1个（2分）
   - 完全不确定 → 跳过
4. **绝不猜**：多选题猜错代价太大

### 多选题实例（高考级别）：

> 已知函数 f(x) = sin(2x + π/6)，则下列说法正确的是（　）
> A. f(x) 的最小正周期为 π
> B. f(x) 的图像关于直线 x = π/3 对称
> C. f(x) 在区间 [0, π/2] 上单调递增
> D. f(x) 的最大值为 1

逐个判断：
- A: T = 2π/ω = 2π/2 = π ✅
- B: 对称轴在函数取最值处。2x+π/6 = π/2 → x = π/6；2x+π/6 = 5π/2 → x = 7π/3。x=π/3 不在对称轴上... 验证：f(π/3) = sin(2π/3+π/6) = sin(5π/6) = 1/2，不是最值，所以 x=π/3 不是对称轴 ❌
- C: 2x+π/6 ∈ [π/6, 7π/6]。sin 在 [π/6, π/2] 递增，[π/2, 7π/6] 递减。所以 f(x) 先增后减，不是单调递增 ❌
- D: 最大值为 1 ✅

**答案 AD** ✅

> 💡 如果你只确定 A 和 D，就只选 AD（4-5分），不要因为 B 或 C "看起来对"就加上去。

## 做选择题的总体节奏

| 时间 | 做什么 | 策略 |
|------|--------|------|
| 前8题（单选） | 15-20分钟 | 先排除→再特殊值→还不会就跳 |
| Q9-Q11（多选） | 8-10分钟 | 逐项判断，确定几个选几个 |
| 回头检查 | 5分钟 | 用不同方法验证标记的题 |

**遇到不会的题的标准流程：**

```
① 30秒排除法：能排除1-2个选项吗？
② 30秒特殊值/代入：能试出答案吗？
③ 30秒估算/画图：能看出大概范围吗？
④ 还不行 → 标记跳过，绝不卡3分钟
```

---

## ⚠️ 常见错误

1. **看到题目就开始硬算**：选择题不是解答题！先想能不能用技巧。
2. **特殊值只试一组就下结论**：多选题至少换2-3组值。
3. **多选题贪心**：不确定的选项硬选，6分变0分。
4. **跳过一整题不回头看**：标记跳过的题，回头往往有灵感。
5. **数形结合画错图**：关键点（零点、顶点、交点）要算准。

## 高考常见陷阱

1. **"恒成立"不等于"有时成立"**：特殊值法只能排除，不能确认。要证明恒成立需要一般性推导。
2. **多选题"看起来都对"**：高考多选题经常设计"陷阱选项"，看起来像对的但其实有边界情况不满足。
3. **函数题忘看定义域**：比如 ln(x-1) 要求 x>1，不画图容易忽略。
4. **选项之间的细微差别**：比如开区间和闭区间（(0,2) vs [0,2]），差一个端点答案就不同。

## 小测验

1.（中等）已知 f(x) = x² - 2ax + a + 2 在 (-∞, 2) 上单调递减，则 a 的取值范围是（　）
   A. a ≥ 2　　B. a ≤ 2　　C. a > 2　　D. a < 2
   （提示：抛物线对称轴 x=a 要 ≤2）

**答案：A**

**解析：** f(x) = x² - 2ax + a + 2 是开口向上的抛物线，对称轴 x = a。抛物线在对称轴左侧单调递减，所以 (-∞, 2) 必须在对称轴左侧，即 a ≥ 2。用特殊值验证：a=2 时 f(x) = x²-4x+4，对称轴 x=2，(-∞,2) 递减 ✅。a=1 时对称轴 x=1，(1,2) 部分在递增 ❌。

2.（中等，多选）已知 a > 0, b > 0，且 2a + b = 4，下列正确的是（　）
   A. ab ≤ 2　　B. 1/a + 1/b ≥ 3/2　　C. a² + b² ≥ 8　　D. ab ≥ 2

**答案：ABC**

**解析：** 令 a=1, b=2（满足2a+b=4）逐项验证：A. ab=2≤2✅; B. 1+1/2=3/2≥3/2✅; C. 1+4=5≥8❌。等等，C不对。再令 a=b=4/3：ab=16/9>2，说明D不对。重新验证C：a²+b²=(a+b)²-2ab，但a+b不是4。用 2a+b=4→b=4-2a，a²+b²=a²+(4-2a)²=5a²-16a+16，顶点 a=8/5 时最小值=16/5=3.2<8。**C也不对**。再仔细算：a²+b² = a²+(4-2a)² = 5a²-16a+16 = 5(a-8/5)²+16/5，最小值16/5=3.2。所以C的"≥8"是错的。

**修正答案：AB**

**解析（修正）：** 令 a=1, b=2：A. ab=2≤2✅; B. 1+0.5=1.5≥1.5✅; C. 1+4=5<8❌; D. ab=2≥2✅。再令 a=0.5, b=3：D. ab=1.5<2❌。所以只有AB一定成立。

3.（挑战）函数 f(x) = ln x - x + 2 的零点个数为（　）
   A. 0　　B. 1　　C. 2　　D. 3
   （提示：画 y = ln x 和 y = x - 2 的交点）

**答案：C**

**解析：** 令 ln x = x - 2，即 y = ln x 和 y = x - 2 的交点。x=1 时 ln1=0, 1-2=-1，ln x 在上；x=3 时 ln3≈1.1, 3-2=1，ln x 在上；x=4 时 ln4≈1.39, 4-2=2，y=x-2 在上。所以有一个交点在3和4之间。再试 x 接近0：ln x → -∞，x-2 → -2，ln x 在下。x=0.5 时 ln0.5≈-0.69, 0.5-2=-1.5，ln x 在上。所以还有一个交点在0和1之间。**共2个交点**，答案是C。
