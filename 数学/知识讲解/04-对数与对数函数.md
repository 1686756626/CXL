# 对数与对数函数

## 这个知识点是什么？

对数就是**指数的反面**。如果说指数是"2的几次方等于8？"答案是3，那么对数就是反过来问"以2为底，8的对数是多少？"答案还是3。

> 打个比方：指数和对数就像"乘法"和"除法"的关系——2×3=6，6÷2=3。2³=8，log₂8=3。一个正着来，一个反着来。

## 第一部分：对数的基本概念

### 定义

aˣ = N ⟺ x = log_a N

读作：x 等于以 a 为底 N 的对数。

其中：a 叫"底数"，N 叫"真数"，x 叫"对数"。

### 两个特殊值（必记！）

| 对数 | 值 | 原因 |
|------|------|------|
| log_a 1 | 0 | a⁰ = 1 |
| log_a a | 1 | a¹ = a |

### 例题1（基础）：求值

**(a)** log₂ 8 = ？ → 2ˣ = 8 = 2³ → x = **3** ✅

**(b)** log₃ 9 = ？ → 3ˣ = 9 = 3² → x = **2** ✅

**(c)** log₅ 1 = **0** ✅（任何底数，log 1 = 0）

**(d)** log₇ 7 = **1** ✅（log 自己 = 1）

---

## 第二部分：对数运算法则（必背！）

> 💡 这三个公式是对数题的核心，和指数法则刚好"反过来"。

| 法则 | 公式 | 例子 |
|------|------|------|
| 乘法→加法 | log(MN) = logM + logN | log₂(4×8) = log₂4 + log₂8 = 2+3 = 5 |
| 除法→减法 | log(M/N) = logM - logN | log₃(27/3) = log₃27 - log₃3 = 3-1 = 2 |
| 幂→乘法 | log(Mⁿ) = n·logM | log₂(4³) = 3·log₂4 = 3×2 = 6 |

### 例题2（中等）：化简

**(a)** log₃ 18 - log₃ 2 = log₃(18/2) = **log₃ 9 = 2** ✅

**(b)** 2log₂ 3 + log₂ (4/9) = log₂ 9 + log₂(4/9) = log₂(9×4/9) = **log₂ 4 = 2** ✅

**解析：** 2log₂3 = log₂(3²) = log₂9。然后 log₂9 + log₂(4/9) = log₂(9×4/9) = log₂4 = 2。

## 第三部分：换底公式

**log_a b = ln b / ln a = lg b / lg a**

> 💡 什么时候用？当你要求的对数的底数和已知条件不一样的时候。

### 例题3（中等）：求 log₂ 3 的值（用常用对数表示）

log₂ 3 = lg 3 / lg 2 ✅

如果 lg 2 ≈ 0.301，lg 3 ≈ 0.477，那么 log₂ 3 ≈ 0.477/0.301 ≈ **1.585** ✅

### 例题4（中等）：计算 log₈ 2

log₈ 2 = ln 2 / ln 8 = ln 2 / (ln 2³) = ln 2 / (3 ln 2) = **1/3** ✅

> 💡 或者直接想：8^(1/3) = 2，所以 log₈ 2 = 1/3。

## 第四部分：对数函数 y = log_a x

### 关键性质

| | a > 1 | 0 < a < 1 |
|--|-------|-----------|
| 图像 | 从左下到右上 | 从左上到右下 |
| 过点 | (1, 0) | (1, 0) |
| 定义域 | x > 0 | x > 0 |
| 单调性 | 单调递增 | 单调递减 |
| x→0⁺ | y→-∞ | y→+∞ |
| x→+∞ | y→+∞ | y→-∞ |

> 💡 对数函数**真数必须大于0**！logₐ(负数) 无意义。

### 例题5（中等）：解对数方程

log₂ (x-1) = 3

→ x-1 = 2³ = 8
→ **x = 9** ✅

检验：x-1 = 8 > 0 ✅

### 例题6（中等）：解对数不等式

log₂ x > 3

→ x > 2³ = 8
→ **x > 8** ✅（底数2>1，方向不变）

log₀.₅ x < 2

→ x < (0.5)² = 0.25
→ **0 < x < 0.25** ✅（底数0.5<1，方向反转，且真数>0）

### 例题7（挑战）：已知 log₂(log₃ x) = 1，求 x。

log₂(log₃ x) = 1 → log₃ x = 2¹ = 2 → x = 3² = **9** ✅

检验：log₃ 9 = 2，log₂ 2 = 1 ✅

## ⚠️ 常见错误

1. **log(M+N) ≠ logM + logN**：只有乘法可以拆，加法不行！
2. **真数为负**：logₐ(-1) 无意义，真数必须 > 0
3. **对数不等式方向**：底数<1时不等号要反转
4. **log a + log b 和 log(a+b) 搞混**：前者 = log(ab)，后者不能化简

## 高考常见陷阱

1. **对数函数的定义域**：解方程/不等式后必须检验真数>0
2. **底数含变量时**：logₓ a 中 x>0 且 x≠1
3. **换底公式链式应用**：log_a b · log_b c · log_c a = 1（三个换底后约掉）
4. **对数和指数混合**：a^(log_a N) = N，log_a(aⁿ) = n，这两个恒等式经常用来化简

## 小测验

1.（基础）计算：log₂ 16 + log₂ 2

**答案：5**

**解析：** log₂ 16 = log₂(2⁴) = 4，log₂ 2 = 1。4+1 = 5。或者 log₂ 16 + log₂ 2 = log₂(16×2) = log₂ 32 = 5。

2.（中等）解不等式：log₃(2x-1) < 2

**答案：1/2 < x < 5**

**解析：** log₃(2x-1) < 2 → 2x-1 < 3² = 9 → 2x < 10 → x < 5。同时真数>0：2x-1 > 0 → x > 1/2。综合：1/2 < x < 5。

3.（挑战）计算：lg 50 - lg 5 + 2lg 2

**答案：lg 40（约1.602）**

**解析：** lg 50 - lg 5 = lg(50/5) = lg 10 = 1。2lg 2 = lg(2²) = lg 4。1 + lg 4 = lg 10 + lg 4 = lg(10×4) = lg 40。
