# 三角函数图像与基本公式

## 这个知识点是什么？

三角函数就是**描述"波动"的数学工具**。声音的高低、钟摆的摆动、四季的温度变化，都是三角函数。

从一模来看，你**已经能做一些三角函数的题了**（Q7做对了），说明有基础。现在需要的是把**公式记熟、图像看懂**，让这道题从"有时能做"变成"每次都能做"。

## 第一部分：图像（看图就能做题）

### 正弦函数 y = sin x 的关键点

| x | 0 | π/2 | π | 3π/2 | 2π |
|---|---|-----|---|------|-----|
| sin x | 0 | 1 | 0 | -1 | 0 |

> 💡 记忆口诀：**"0起头，到π/2最高，π归零，3π/2最低，2π回起点"**

### 余弦函数 y = cos x 的关键点

| x | 0 | π/2 | π | 3π/2 | 2π |
|---|---|-----|---|------|-----|
| cos x | 1 | 0 | -1 | 0 | 1 |

和 sin x 的形状一样，只是**往左挪了 π/2**。

### 高考最常考的图像题：y = A sin(ωx + φ) + k

| 参数 | 控制什么 | 怎么记 |
|------|----------|--------|
| **A** | 振幅 | 最大值-最小值 / 2 |
| **ω** | 周期 | T = 2π/ω |
| **φ** | 左右平移 | 向左加，向右减 |
| **k** | 上下平移 | 就是整体上移或下移 |

> ⚠️ **ω 越大图像越"挤"，ω 越小图像越"松"**。

## 第二部分：基本公式（必背！）

### 1. 同角三角函数关系

**商数关系：** tan x = sin x / cos x

**平方关系：** sin²x + cos²x = 1

> 💡 这个公式做题频率极高。

**例题1（中等）：** 已知 sin x = 3/5，x 在第二象限，求 cos x 和 tan x。

sin²x + cos²x = 1 → cos²x = 1 - 9/25 = 16/25 → cos x = ±4/5

第二象限 cos x 是**负数** → cos x = **-4/5** ✅

tan x = sin x / cos x = (3/5)/(-4/5) = **-3/4** ✅

### 2. 诱导公式（奇变偶不变，符号看象限）

- **"奇变偶不变"**：π/2 的偶数倍（π, 2π...）→ 函数名**不变**；奇数倍（π/2, 3π/2...）→ 函数名**变化**（sin↔cos）
- **"符号看象限"**：把 x 当锐角，看原角所在象限的符号

**例题2（基础）：** 求 sin(π + α) 和 cos(3π/2 + α)

sin(π + α)：π = π/2×2（偶数倍）→ 函数名不变。π+α 在第三象限，sin为负。→ **-sin α**

cos(3π/2 + α)：3π/2 = π/2×3（奇数倍）→ cos变sin。3π/2+α 在第四象限，cos为正。→ **sin α** ✅

### 3. 二倍角公式

```
sin 2x = 2 sin x cos x
cos 2x = cos²x - sin²x = 1 - 2sin²x = 2cos²x - 1
```

**例题3（中等）：** 已知 sin x = 3/5，x 在第二象限，求 sin 2x 和 cos 2x。

cos x = -4/5（上面算过）

sin 2x = 2 × (3/5) × (-4/5) = **-24/25** ✅
cos 2x = (-4/5)² - (3/5)² = 16/25 - 9/25 = **7/25** ✅

## 第三部分：做题的时候怎么用？

### 题型一：三角函数图像变换

**例题4（基础）：** 将 y = sin x 的图像向右平移 π/3 个单位，得到的函数解析式是？

向右平移 π/3 → 把 x 替换成 x - π/3

答案：**y = sin(x - π/3)** ✅

### 题型二：从图像识别参数

**例题5（中等）：** 函数 y = A sin(ωx + φ)（A > 0, ω > 0）的图像过 (0, √3)，(π/3, 0)，(2π/3, -√3)，(π, 0)。求解析式。

**第一步：求 A**
最大值 = √3，最小值 = -√3 → A = **√3**

**第二步：求 ω（看周期）**
从 (π/3, 0) 到 (π, 0)，经过半周期 T/2 = π - π/3 = 2π/3 → T = 4π/3
ω = 2π/T = 2π/(4π/3) = **3/2**

**第三步：求 φ**
y = √3 sin(3x/2 + φ)，代入 (0, √3)：
√3 = √3 sin(φ) → sin(φ) = 1 → φ = **π/2**

**答案：y = √3 sin(3x/2 + π/2)** ✅

### 题型三：求单调区间和值域

**例题6（中等）：** 求 y = 2sin(x + π/6) 在 [0, π] 上的单调递增区间和值域。

x ∈ [0, π] → x + π/6 ∈ [π/6, 7π/6]

sin 在 [π/6, π/2] 递增，在 [π/2, 7π/6] 递减

单调递增：x + π/6 ∈ [π/6, π/2] → x ∈ **[0, π/3]**

sin 在 [π/6, 7π/6] 上的最大值 = sin(π/2) = 1，最小值 = sin(7π/6) = -1/2

值域：2×[-1/2, 1] = **[-1, 2]** ✅

### 题型四：辅助角公式（挑战）

**例题7（挑战）：** 化简 sin x + √3 cos x。

公式：**a sin x + b cos x = √(a²+b²) sin(x + φ)**

其中 tan φ = b/a。

这里 a=1, b=√3，√(1+3) = 2
tan φ = √3/1 = √3 → φ = **π/3**

**sin x + √3 cos x = 2 sin(x + π/3)** ✅

> 💡 辅助角公式在解答题中非常常用，尤其和正弦定理结合的题。

## ⚠️ 常见错误

1. **象限符号搞混**：画个十字象限图，标出各象限 sin/cos/tan 的正负
2. **ω 和周期搞反**：ω 大周期小（图像挤），ω 小周期大（图像松）
3. **平移方向搞反**：右移减（x → x-a），左移加（x → x+a）
4. **二倍角公式展开时符号错**：cos 2x = 1-2sin²x，不是 1+2sin²x

## 高考常见陷阱

1. **忘记看象限就确定正负号**：sin²x+cos²x=1 给的是绝对值，正负要看象限
2. **图像平移和周期混淆**：先平移再伸缩 vs 先伸缩再平移，结果不同
3. **辅助角公式中 φ 的象限**：φ 不一定在第一象限，要看 a 和 b 的符号
4. **函数名写错**：诱导公式中 sin 和 cos 互变时容易搞混

## 小测验

1.（基础）已知 sin x = 4/5，x 在第二象限，求 cos x 和 tan x。

**答案：cos x = -3/5，tan x = -4/3**

**解析：** sin²x + cos²x = 1 → cos²x = 1 - 16/25 = 9/25 → cos x = ±3/5。第二象限 cos x 是负数，所以 cos x = -3/5。tan x = sin x / cos x = (4/5)/(-3/5) = -4/3。

2.（中等）y = 3sin(2x + π/6) 的振幅、周期分别是多少？在 [0, π/2] 上的值域是多少？

**答案：振幅 = 3，周期 = π，值域 = [-3/2, 3]**

**解析：** A = 3（振幅），ω = 2，周期 = 2π/2 = π。x ∈ [0, π/2] 时，2x+π/6 ∈ [π/6, 7π/6]。sin 在 [π/6, π/2] 递增到1，在 [π/2, 7π/6] 递减到 sin(7π/6) = -1/2。所以 sin 的范围是 [-1/2, 1]，乘以3得值域 [-3/2, 3]。

3.（挑战）化简 √3 sin x + cos x 为 Asin(x+φ) 的形式。

**答案：2sin(x + π/6)**

**解析：** 辅助角公式 a sin x + b cos x = √(a²+b²) sin(x+φ)。这里 a=√3, b=1，√(3+1) = 2。tan φ = b/a = 1/√3 → φ = π/6。所以 √3 sin x + cos x = 2sin(x + π/6)。
