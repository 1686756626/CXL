## 技巧一：排除法

> **核心思想**：不需要算出正确答案，只要能排除3个错误选项，剩下的就是对的。
> **适用场景**：选项有明显错误、计算量大的题。

### 第1题
**难度：** ⭐（基础）
**推荐方法：** 排除法

**题目：** 已知 A = {1, 2, 3, 4}，下列关系中**错误**的是（ ）

A. 1 ∈ A
B. 0 ∈ A
C. {1, 2} ⊂ A
D. ∅ ⊂ A

**答案：** B

**解析：** 题目问的是"错误"的，所以我们要找**不对**的那个。

- A. 1 是 A 的元素，∈ 没问题 ✅（正确的，不是我们要找的）
- B. 0 不是 A 的元素，0 ∈ A ❌（错误的！就是它）
- C. {1,2} 是 A 的子集，⊂ 没问题 ✅
- D. 空集是任何集合的子集，⊂ 没问题 ✅

**答案：B**

**实战技巧**：看到"错误的"和"正确的"要注意审题！高考最爱在这挖坑。另外，∈ 用于元素和集合之间，⊂ 用于集合和集合之间，这个区分一定要会。

---

### 第2题
**难度：** ⭐（基础）
**推荐方法：** 排除法

**题目：** 下列函数中，既是奇函数又在 (0, +∞) 上单调递增的是（ ）

A. y = x²
B. y = |x|
C. y = x³
D. y = 1/x

**答案：** C

**解析：** 两个条件都要满足，用排除法：

先看"奇函数"（关于原点对称）：
- A. y = x² → 关于 y 轴对称，偶函数 ❌ → 排除！
- B. y = |x| → 关于 y 轴对称，偶函数 ❌ → 排除！
- D. y = 1/x → 奇函数 ✅

只剩 C 和 D。再看"在 (0, +∞) 上单调递增"：
- D. y = 1/x → 当 x 增大时 1/x 减小，是递减的 ❌ → 排除！
- C. y = x³ → 当 x 增大时 x³ 增大，递增 ✅

**答案：C**

**实战技巧**：多条件题先看其中一个条件排除选项，缩小范围后再看第二个条件。

### 第3题
**难度：** ⭐⭐（中等）
**推荐方法：** 排除法 + 代入法

**题目：** 已知函数 f(x) = x² - 2x - 3，则 f(x) > 0 的解集为（ ）

A. (-∞, -1)
B. (3, +∞)
C. (-∞, -1) ∪ (3, +∞)
D. (-1, 3)

**答案：** C

**解析：** 排除法 + 代入法结合：

先代入 x = 0：f(0) = -3 < 0，所以 x = 0 不在解集里。看哪个选项包含 0：
- D 包含 0（-1 < 0 < 3）❌ → 排除 D

再代入 x = -2：f(-2) = 4 + 4 - 3 = 5 > 0，x = -2 在解集里。
- B 不包含 -2 ❌ → 排除 B
- A 包含 -2 ✅，C 也包含 -2 ✅

再代入 x = 4：f(4) = 16 - 8 - 3 = 5 > 0，x = 4 在解集里。
- A 不包含 4 ❌ → 排除 A
- C 包含 4 ✅

**答案：C = (-∞, -1) ∪ (3, +∞)**

验证：x² - 2x - 3 = (x+1)(x-3)，开口向上，两个根 -1 和 3，大于0的解集就是 (-∞, -1) ∪ (3, +∞)。✅

**实战技巧**：不等式求解选择题，直接代入选项边界和简单值，不用解不等式。

## 技巧二：特殊值法

> **核心思想**：题目含有变量或参数时，代入具体数值来排除选项。
> **适用场景**：含 x、a、b 等变量的"以下哪个正确"型题目。

### 第4题
**难度：** ⭐（基础）
**推荐方法：** 特殊值法

**题目：** 若 a < 0 < b 且 |a| > b，则下列不等式一定成立的是（ ）

A. a + b > 0
B. a - b > 0
C. ab > 0
D. a² > b²

**答案：** D

**解析：** 代入具体数值：令 a = -3, b = 2（满足 a < 0 < b 且 |a|=3 > b=2）

- A. -3 + 2 = -1 < 0 ❌ → 排除
- B. -3 - 2 = -5 < 0 ❌ → 排除
- C. -3 × 2 = -6 < 0 ❌ → 排除
- D. (-3)² = 9 > 2² = 4 ✅

只剩 D，再验一组确认：a = -5, b = 1
- D. 25 > 1 ✅ 没问题

**答案：D**

**实战技巧**：特殊值法关键是选好几组值！至少选两组确认。选值时要满足题目所有条件。

### 第5题
**难度：** ⭐⭐（中等）
**推荐方法：** 特殊值法

**题目：** 已知 f(x) = x³ + ax² + bx + c，且 f(0) = 0, f(1) = 1, f(-1) = -1，则 f(2) =（ ）

A. 2
B. 3
C. 4
D. 8

**答案：** D

**解析：** 用特殊值法！

f(0) = c = 0 → **c = 0**
f(1) = 1 + a + b + 0 = 1 → a + b = 0 ...①
f(-1) = -1 + a - b + 0 = -1 → a - b = 0 ...②

①+②：2a = 0 → a = 0；则 b = 0

所以 f(x) = x³

f(2) = 2³ = **8**

**答案：D**

**实战技巧**：看到"已知几个点的函数值，求另一个点的值"，先代入简单值确定系数，再算目标值。这比"硬解方程组"快得多。

## 技巧三：代入法（代入选项验证）

> **核心思想**：把选项直接代入题目验证，对的就选。
> **适用场景**：选项是具体数值、方程的根等。

### 第6题
**难度：** ⭐（基础）
**推荐方法：** 代入法

**题目：** 方程 x² - 5x + 6 = 0 的两根为（ ）

A. 1 和 6
B. 2 和 3
C. -2 和 -3
D. 1 和 5

**答案：** B

**解析：** 直接把选项代入验证：

A. 代入 x=1：1 - 5 + 6 = 2 ≠ 0 ❌
B. 代入 x=2：4 - 10 + 6 = 0 ✅；代入 x=3：9 - 15 + 6 = 0 ✅ → 两个都对！
C. 代入 x=-2：4 + 10 + 6 = 20 ≠ 0 ❌
D. 代入 x=1：1 - 5 + 6 = 2 ≠ 0 ❌

**答案：B**

**实战技巧**：从看起来最简单的选项开始代入，找到正确答案就停，不用每个都验。

---

### 第7题
**难度：** ⭐（基础）
**推荐方法：** 代入法

**题目：** 函数 f(x) = √(x - 1) 的定义域为（ ）

A. (-∞, 1]
B. [1, +∞)
C. (-∞, +∞)
D. [0, +∞)

**答案：** B

**解析：** 定义域就是让函数有意义的 x 的范围。

√(x - 1) 有意义 → x - 1 ≥ 0 → x ≥ 1 → [1, +∞)

验证：x = 1 时 √0 = 0，有意义 ✅；x = 0 时 √(-1) 没意义 ❌

**答案：B**

## 技巧四：数形结合

> **核心思想**：画图！很多题看起来复杂，画个图就一目了然。
> **适用场景**：函数题、集合题、不等式题。

### 第8题
**难度：** ⭐⭐（中等）
**推荐方法：** 数形结合

**题目：** 已知 A = {x | -2 < x < 3}，B = {x | x ≥ 1}，则 A ∩ B =（ ）

A. (-2, 3)
B. [1, 3)
C. [1, +∞)
D. (-2, 1)

**答案：** B

**解析：** 画数轴：

```
A:  -2 ←――――→ 3        （不包含-2和3）
B:       1 ←――→ +∞     （包含1）

重叠部分：1 ←――→ 3      （包含1，不包含3）
```

- 左端点：A 从 -2 开始，B 从 1 开始 → 重叠从 1 开始，B 包含 1 → [1
- 右端点：A 到 3 结束，B 到 +∞ → 重叠到 3 结束，A 不包含 3 → 3)

**答案：B = [1, 3)**

**实战技巧**：看到集合交并补、不等式，第一反应就是画数轴！

---

### 第9题
**难度：** ⭐⭐（中等）
**推荐方法：** 数形结合

**题目：** 函数 y = |x - 1| + |x + 2| 的最小值为（ ）

A. 0
B. 1
C. 3
D. 5

**答案：** C

**解析：** 画图思路：y = |x - 1| + |x + 2|

关键点：x = -2 和 x = 1

分三段讨论：
- x ≤ -2：y = -(x-1) + -(x+2) = -2x - 1（递减）
- -2 < x < 1：y = -(x-1) + (x+2) = 3（常数！）
- x ≥ 1：y = (x-1) + (x+2) = 2x + 1（递增）

最小值 = **3**

或者更简单的方法：|x - 1| + |x + 2| 表示数轴上点 x 到 1 和到 -2 的距离之和。当 x 在 -2 和 1 之间时，距离之和 = 1 - (-2) = 3，最小！

**答案：C**

**实战技巧**：看到 |x - a| + |x - b| 型题目，最小值就是 |a - b|（x 在 a 和 b 之间时取到）。

## 技巧五：估算（不需要精确计算）

> **核心思想**：选择题不需要精确答案，估算出大致范围就能排除选项。
> **适用场景**：计算量大但选项差距明显的题。

### 第10题
**难度：** ⭐（基础）
**推荐方法：** 估算

**题目：** 下列数中最大的是（ ）

A. 2^√3
B. 3^√2
C. √2³
D. √3²

**答案：** B

**解析：** 估算每个值：

- C. √2³ = (√2)³ = 2√2 ≈ 2.83
- D. √3² = (√3)² = 3

C 和 D 都不大（≈2.83 和 3），重点比较 A 和 B：
- A. 2^√3，其中 √3 ≈ 1.73 → 2^1.73，在 2^1.5 ≈ 2.83 和 2^2 = 4 之间
- B. 3^√2，其中 √2 ≈ 1.41 → 3^1.41，在 3^1 = 3 和 3^1.5 = 3√3 ≈ 5.2 之间

怎么比 A 和 B？两边取 ln：
- ln(A) = 1.73 × ln2 ≈ 1.73 × 0.693 ≈ 1.20
- ln(B) = 1.41 × ln3 ≈ 1.41 × 1.099 ≈ 1.55

ln(B) > ln(A) → B 更大。

**答案：B（3^√2）**

**实战技巧**：比较指数大小时，两边取对数就变成了乘法比较，简单很多。另外先排除明显小的选项（C≈2.83、D=3），集中比较两个大的。

## 多选题策略（新高考特有）

> **核心规则**：多选题选对部分得3分，选错得0分。所以**只选有把握的，不贪心！**

### 第11题
**难度：** ⭐⭐（中等）
**推荐方法：** 排除法 + 多选策略

**题目：** [多选] 已知 a, b > 0 且 a + b = 1，则下列正确的是（ ）

A. ab ≤ 1/4
B. a² + b² ≤ 1/2
C. 1/a + 1/b ≥ 4
D. a + b ≥ 1

**答案：** A、B、C、D

**解析：** 多选题策略：逐个判断，只选确定的。

**D 选项（最简单）：** a + b = 1（题目给的），所以 a + b ≥ 1 ✅ → 确定选 D

**A 选项：** 基本不等式 ab ≤ (a+b)²/4 = 1/4 ✅ → 确定选 A

**B 选项：** a² + b² = (a+b)² - 2ab = 1 - 2ab，ab ≥ 0，所以 a² + b² ≤ 1 ✅ → 确定选 B

**C 选项：** 1/a + 1/b = (a+b)/ab = 1/ab，ab ≤ 1/4，所以 1/ab ≥ 4 ✅ → 确定选 C

四个都对！全选拿满分。

**多选实战策略：**
1. 先看最简单的选项（D直接从题目条件判断）
2. 不确定的坚决不选（选错得0分，不如只拿3分）
3. 四个都确定才全选

### 第12题
**难度：** ⭐⭐（中等）
**推荐方法：** 排除法 + 多选策略

**题目：** [多选] 关于复数 z = 1 + i，下列说法正确的是（ ）

A. z 的模为 √2
B. z² = 2i
C. z 的共轭复数为 1 - i
D. z 的虚部为 i

**答案：** A、B、C

**解析：** 逐个判断：

**A.** |z| = √(1² + 1²) = √2 ✅ → 确定选 A

**B.** z² = (1+i)² = 1 + 2i + i² = 1 + 2i - 1 = 2i ✅ → 确定选 B

**C.** 共轭复数 = 1 - i（虚部变号）✅ → 确定选 C

**D.** z = 1 + i，虚部是 **1**（不是 i！）❌ → 不选

**答案：A、B、C**

**多选实战策略：** D 是经典陷阱（虚部是 1 不是 i），很多学生会错选 D。不确定的别选！

### 第13题
**难度：** ⭐⭐⭐（稍有挑战）
**推荐方法：** 排除法 + 多选策略

**题目：** [多选] 已知函数 f(x) = sin(2x)，则下列正确的是（ ）

A. f(x) 的最小正周期为 π
B. f(x) 在 [0, π/4] 上单调递增
C. f(π/6) = 1/2
D. f(x) 的图象关于点 (π/2, 0) 对称

**答案：** A、B、D

**解析：** 逐个判断：

**A.** 周期 T = 2π/ω = 2π/2 = π ✅ → 确定选 A

**B.** 在 [0, π/4] 上，2x 从 0 增到 π/2，sin 在 [0, π/2] 上递增 ✅ → 确定选 B

**C.** f(π/6) = sin(2 × π/6) = sin(π/3) = √3/2，不是 1/2 ❌ → 不选

**D.** f(π/2) = sin(π) = 0，检查对称性：
f(π/2 + t) + f(π/2 - t) = sin(π + 2t) + sin(π - 2t) = -sin(2t) + sin(2t) = 0 ✅ → 确定选 D

**答案：A、B、D**

**多选实战策略：** A 最简单先确定。C 看起来简单但要算对，sin(π/3) = √3/2 不是 1/2，这个容易算错。不确定的选项宁可不选。

## 综合运用

### 第14题
**难度：** ⭐⭐（中等）
**推荐方法：** 特殊值法 + 数形结合

**题目：** 已知 f(x) = |x² - 4|，则 f(x) < 3 的解集为（ ）

A. (-1, 1)
B. (-√7, -1) ∪ (1, √7)
C. (-√7, √7)
D. (-√7, 0) ∪ (0, √7)

**答案：** B

**解析：**

**先用特殊值法快速排除：**

代入 x = 0：f(0) = |0 - 4| = 4 > 3，x = 0 不在解集里。
- A 包含 0 ❌ → 排除
- C 包含 0 ❌ → 排除
- D 包含 0 ❌ → 排除

只剩 **B**！但为了保险，再验证一个值：

代入 x = 2：f(2) = |4 - 4| = 0 < 3，x = 2 在解集里。
- B 包含 2 ✅（1 < 2 < √7 ≈ 2.65）

**完整解法（数形结合验证）：**

|x² - 4| < 3 等价于 -3 < x² - 4 < 3

左边：x² - 4 > -3 → x² > 1 → x < -1 或 x > 1
右边：x² - 4 < 3 → x² < 7 → -√7 < x < √7

取交集：
```
左边：(-∞, -1) ∪ (1, +∞)
右边：(-√7, √7)
结果：(-√7, -1) ∪ (1, √7)
```

**答案：B**

**实战技巧：** 本题综合运用了**特殊值法**（快速排除3个选项）和**数形结合**（绝对值不等式转化为两个不等式）。考试时特殊值法先排掉3个就稳了，不用完整计算。

### 第15题
**难度：** ⭐⭐⭐（挑战）
**推荐方法：** 代入法 + 排除法

**题目：** [多选] 已知数列 {aₙ} 满足 a₁ = 1，aₙ₊₁ = 2aₙ + 1，则下列正确的是（ ）

A. a₂ = 3
B. a₃ = 7
C. aₙ = 2ⁿ - 1
D. a₁ + a₂ + a₃ = 12

**答案：** A、B、C

**解析：** 多选题策略：逐个用**代入法**验证。

**A. a₂ = 3：**
a₂ = 2a₁ + 1 = 2 × 1 + 1 = **3** ✅ → 确定选 A

**B. a₃ = 7：**
a₃ = 2a₂ + 1 = 2 × 3 + 1 = **7** ✅ → 确定选 B

**C. aₙ = 2ⁿ - 1：**
用**代入法**验证前几项：
- n = 1：2¹ - 1 = 1 = a₁ ✅
- n = 2：2² - 1 = 3 = a₂ ✅
- n = 3：2³ - 1 = 7 = a₃ ✅

三个都对，公式很可能正确。数学归纳法验证：假设 aₙ = 2ⁿ - 1，则
aₙ₊₁ = 2(2ⁿ - 1) + 1 = 2ⁿ⁺¹ - 2 + 1 = 2ⁿ⁺¹ - 1 ✅ → 确定选 C

**D. a₁ + a₂ + a₃ = 12：**
a₁ + a₂ + a₃ = 1 + 3 + 7 = **11 ≠ 12** ❌ → 不选

**答案：A、B、C**

**多选实战策略：**
- A、B、D 都可以**直接计算**，用代入法逐个验证
- C 需要验证通项公式，先代入前几项确认，再考虑是否需要严格证明
- D 是经典陷阱：1+3+7=11，不是12！看似合理但要仔细算

## 技巧组合（高考常见）

### 第16题
**难度：** ⭐⭐（中等）
**推荐方法：** 排除法 + 特殊值法

**题目：** 已知集合 A = {x | x² - mx + m - 1 < 0}，若 A ⊆ {x | 0 < x ≤ 2}，则 m 的取值范围是（ ）

A. (1, 3]
B. [1, 3]
C. [1, 4)
D. (1, 4]

**答案：** B

**解析：**

**思路：** 含参集合题，先求出不等式对应方程的两根，再根据 A 的范围和子集关系列出条件。

**第一步：求方程 x² - mx + m - 1 = 0 的两根**

Δ = m² - 4(m-1) = m² - 4m + 4 = (m-2)² ≥ 0，恒有实数根。

用求根公式：x = [m ± (m-2)]/2

两根为 x₁ = 1，x₂ = m - 1

**第二步：根据两根大小关系确定 A**

当 m > 2 时，m-1 > 1，A = (1, m-1)。要 A ⊆ (0, 2]，需要 m-1 ≤ 2，即 m ≤ 3。所以 2 < m ≤ 3。

当 m = 2 时，Δ = 0，A = ∅ ⊆ (0, 2] ✅

当 1 ≤ m < 2 时，m-1 ∈ [0,1)，A = (m-1, 1)。要 A ⊆ (0, 2]，需要 m-1 ≥ 0，即 m ≥ 1。所以 1 ≤ m < 2。

当 m < 1 时，m-1 < 0，A = (m-1, 1) 包含负数部分，不满足 A ⊆ (0, 2] ❌

**综合：** m ≥ 1 且 m ≤ 3，即 **[1, 3]**

**第三步：用特殊值验证**

m = 1：A = (0, 1) ⊆ (0, 2] ✅
m = 3：A = (1, 2) ⊆ (0, 2] ✅
m = 4：A = (1, 3) ⊄ (0, 2] ❌（3不在范围内）

验证通过。

**答案：B**

**实战技巧：** 含参集合题先求根，再根据子集关系列不等式组，最后用特殊值验证边界。

### 第17题
**难度：** ⭐⭐（中等）
**推荐方法：** 代入计算 + 多选策略

**题目：** [多选] 已知函数 f(x) = sin(2x + π/6)，则下列正确的是（ ）

A. f(x) 的最小正周期为 π
B. f(x) 的图象关于点 (5π/12, 0) 对称
C. f(x) 在 [0, π/6] 上单调递增
D. f(π/6) = 1

**答案：** A、B、C、D

**解析：** 逐个验证。

**A.** 周期 T = 2π/ω = 2π/2 = π ✅ → 确定选A

**D.** f(π/6) = sin(2·π/6 + π/6) = sin(π/3 + π/6) = sin(π/2) = 1 ✅ → 确定选D

**C.** 在 [0, π/6] 上，2x + π/6 从 π/6 增到 π/2。sin 在 [π/6, π/2] 上单调递增，复合后 f(x) 在 [0, π/6] 上单调递增 ✅ → 确定选C

**B.** 检验 (5π/12, 0) 是否为对称中心：
f(5π/12) = sin(2·5π/12 + π/6) = sin(5π/6 + π/6) = sin(π) = 0 ✅

验证：f(5π/12 + t) + f(5π/12 - t)
= sin(π + 2t) + sin(π - 2t)
= -sin(2t) + sin(2t)
= 0 ✅

所以 (5π/12, 0) 是 f(x) 的对称中心。

**答案：A、B、C、D**

**多选实战策略：** 对称中心的验证需要两步——先确认 f(中心点) = 0，再验证 f(中心+t) + f(中心-t) = 0。每一步都要仔细算。

### 第18题
**难度：** ⭐⭐⭐（挑战）
**推荐方法：** 基本不等式 + 恒等式展开

**题目：** [多选] 已知 a > 0, b > 0, a + b = 1，则下列正确的是（ ）

A. ab ≤ 1/4
B. a² + b² ≥ 1/2
C. 1/a + 1/b ≥ 4
D. a³ + b³ ≥ 1/4

**答案：** A、B、C、D

**解析：**

**A.** 基本不等式：ab ≤ ((a+b)/2)² = 1/4 ✅ → 确定选A

**B.** a² + b² = (a+b)² - 2ab = 1 - 2ab。ab ≤ 1/4，所以 a²+b² ≥ 1 - 2×(1/4) = 1/2 ✅ → 确定选B

**C.** 1/a + 1/b = (a+b)/ab = 1/ab。ab ≤ 1/4，所以 1/ab ≥ 4 ✅ → 确定选C

**D.** 用恒等式展开：
a³ + b³ = (a+b)(a² - ab + b²) = a² - ab + b² = (a+b)² - 3ab = 1 - 3ab
ab ≤ 1/4，所以 a³ + b³ ≥ 1 - 3/4 = 1/4 ✅

等号成立条件：a = b = 1/2 时，ab = 1/4 取最大值，各项同时取等。

**答案：A、B、C、D**

**多选实战策略：** D 选项用立方和公式 a³+b³ = (a+b)(a²-ab+b²) 展开，再利用 a+b=1 的条件化简，最终归结为 1-3ab 的最小值问题，和 A、B 的方法一脉相承。

## 总结：做题口诀

| 技巧 | 一句话 | 什么时候用 |
|------|--------|-----------|
| **排除法** | 不求正确答案，排除错的就行 | 选项有明显错误、多条件题 |
| **特殊值法** | 代具体数字试试 | 含变量的"哪个正确"题 |
| **代入法** | 把选项代进去验 | 选项是具体数值、求根 |
| **数形结合** | 画图就清楚 | 集合、函数、不等式 |
| **估算** | 大概差不多就能选 | 计算量大、选项差距大 |
| **多选策略** | 确定几个选几个，不贪心 | 所有**多选题** |

## 复习题（交叉练习）

### 复1
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 01-集合

**题目：** 已知 A = {x | x² - 4x + 3 < 0}，B = {x | 1 ≤ x ≤ 4}，则 A ∩ B =（ ）

A. (1, 3)　　B. [1, 3]　　C. (1, 4)　　D. (3, 4)

**答案：** A

**解析：**
复习知识点：01-集合

**方法一：直接计算（常规解法）**

化简 A：x² - 4x + 3 = (x - 1)(x - 3)
开口向上，(x-1)(x-3) < 0 → A = (1, 3)

A ∩ B = (1, 3) ∩ [1, 4] = **(1, 3)**

**方法二：特殊值法（排除法，推荐！）**

代入 x = 1：A 中 1²-4+3 = 0，不满足 < 0，所以 x=1 不在 A 中，也不在 A∩B 中。
- B 选项包含 1 → 排除 B

代入 x = 3：A 中 9-12+3 = 0，不满足 < 0，所以 x=3 不在 A 中，也不在 A∩B 中。
- C 选项包含 3 → 排除 C
- D 选项不含 1 到 3 之间的点 → 排除 D

只剩 **A = (1, 3)** ✅

**答案：A**

### 复2
**难度：** ⭐ 基础（5分）
**复习知识点：** 02-复数

**题目：** 计算 (2 + 3i)(1 - 2i) 的值。

**答案：** 8 - i

**解析：**
复习知识点：02-复数

像多项式乘法一样展开：
```
(2 + 3i)(1 - 2i)
= 2 × 1 + 2 × (-2i) + 3i × 1 + 3i × (-2i)
= 2 - 4i + 3i - 6i²
= 2 - i - 6(-1)
= 2 - i + 6
= 8 - i
```

**关键一步：** i² = -1，所以 -6i² = -6×(-1) = +6。

**答案：8 - i**
