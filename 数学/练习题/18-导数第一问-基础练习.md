## 第1题 ⭐ 基础（5分）选择题

**函数 f(x) = x⁴ 的导数 f'(x) = ？**

A. x³  
B. 4x³  
C. 3x⁴  
D. 4x⁴

---

**【答案】B**

**【解析】**

幂函数求导公式：(xⁿ)' = nxⁿ⁻¹

f'(x) = 4x³

**怎么想到的：** "指数下来当系数，指数减 1"。x⁴ 的指数 4 下来做系数，原来位置变成 4-1=3，所以是 4x³。

## 第2题 ⭐ 基础（5分）选择题

**函数 f(x) = x² - 3x + 2 的导数 f'(x) = ？**

A. 2x - 3  
B. 2x + 3  
C. x² - 3  
D. 2x - 2

---

**【答案】A**

**【解析】**

逐项求导（导数的加减法法则）：

f'(x) = (x²)' - (3x)' + (2)' = 2x - 3 + 0 = **2x - 3**

**怎么想到的：** 和的导数等于导数的和，逐项求就行。常数 2 的导数是 0，别漏掉也别多写。

## 第3题 ⭐ 基础（5分）选择题

**f(x) = x² + 1，则 f'(2) = ？**

A. 3  
B. 4  
C. 5  
D. 6

---

**【答案】B**

**【解析】**

先求导：f'(x) = 2x

再代入 x = 2：f'(2) = 2 × 2 = **4**

**怎么想到的：** f'(2) 就是先求导函数，再把 x = 2 代入。顺序不能反——不是先算 f(2) 再求导。

## 第4题 ⭐⭐ 中等（5分）选择题

**f(x) = x³ - 3x² 的单调递增区间是？**

A. (-∞, 0) ∪ (2, +∞)  
B. (0, 2)  
C. (-∞, 2)  
D. (0, +∞)

---

**【答案】A**

**【解析】**

第一步：求导

f'(x) = 3x² - 6x = 3x(x - 2)

第二步：找零点

3x(x - 2) = 0 → x = 0 或 x = 2

第三步：判断正负

| 区间 | (-∞, 0) | (0, 2) | (2, +∞) |
|------|----------|--------|-----------|
| x | - | + | + |
| x - 2 | - | - | + |
| f'(x) | + | - | + |

f'(x) > 0 的区间：**(-∞, 0) ∪ (2, +∞)**

**怎么想到的：** 和前面求单调递减的步骤完全一样，只是这次找 f'(x) > 0 的区间。开口向上的抛物线，两个根外侧为正。

## 第5题 ⭐⭐ 中等（5分）选择题

**曲线 f(x) = x² - x 在 x = 1 处的切线方程是？**

A. y = x - 1  
B. y = x  
C. y = 2x - 2  
D. y = 2x - 1

---

**【答案】A**

**【解析】**

切线方程公式：y - f(x₀) = f'(x₀)(x - x₀)

**第一步：求切点**

f(1) = 1 - 1 = 0，切点为 (1, 0)

**第二步：求斜率**

f'(x) = 2x - 1

f'(1) = 2 × 1 - 1 = 1

**第三步：写切线方程**

y - 0 = 1 × (x - 1)

**y = x - 1**

**怎么想到的：** 切线过曲线上一点，套公式就行。切点就是 (x₀, f(x₀))，斜率就是 f'(x₀)，然后点斜式写方程。

## 第6题 ⭐⭐ 中等（5分）选择题

**曲线 y = x² - 1 过点 (0, -3) 的切线方程是？**

A. y = 2√2 · x - 3  
B. y = -2√2 · x - 3  
C. y = 2√2 · x - 3 或 y = -2√2 · x - 3  
D. y = 2x - 3

---

**【答案】C**

**【解析】**

点 (0, -3) 代入曲线：0 - 1 = -1 ≠ -3，所以 (0, -3) **不在曲线上**，需要设切点。

**第一步：设切点**

设切点为 (a, a² - 1)，则 f'(a) = 2a

**第二步：写切线方程**

y - (a² - 1) = 2a(x - a)

**第三步：代入 (0, -3)**

-3 - (a² - 1) = 2a(0 - a)

-3 - a² + 1 = -2a²

-2 - a² = -2a²

**a² = 2**，a = ±√2

**第四步：写出切线**

a = √2 时：斜率 = 2√2，切点 (√2, 1)

y - 1 = 2√2(x - √2) → **y = 2√2x - 3**

a = -√2 时：斜率 = -2√2，切点 (-√2, 1)

y - 1 = -2√2(x + √2) → **y = -2√2x - 3**

**验证 a = √2：** 切线 y = 2√2x - 3 过 (0, -3)：-3 = 0 - 3 ✅
**验证 a = -√2：** 切线 y = -2√2x - 3 过 (0, -3)：-3 = 0 - 3 ✅

**怎么想到的：** 过曲线**外**一点求切线，关键在于"先设切点 (a, f(a))"，然后利用"切线过已知点"这个条件列方程求 a。可能有多个切点，所以可能有多个切线。

## 第7题 ⭐⭐ 中等（5分）选择题

**f(x) = x³ - 3x 的极大值是？**

A. 2  
B. -2  
C. 3  
D. -3

---

**【答案】A**

**【解析】**

第一步：求导找临界点

f'(x) = 3x² - 3 = 3(x + 1)(x - 1)

临界点：x = -1，x = 1

第二步：判断极值类型

| 区间 | (-∞, -1) | (-1, 1) | (1, +∞) |
|------|-----------|---------|-----------|
| f'(x) | + | - | + |

x = -1：f'(x) 由 + 变 -，取**极大值**

x = 1：f'(x) 由 - 变 +，取极小值

第三步：求极大值

f(-1) = (-1)³ - 3(-1) = -1 + 3 = **2**

**怎么想到的：** 极大值的判断：导数从正变负 = 极大，从负变正 = 极小。找到极大值点后代入原函数算值。

## 第8题 ⭐⭐ 中等（5分）选择题

**下列关于导数的说法，正确的有几项？**

① f'(x) > 0 时 f(x) 单调递增  
② f'(x₀) = 0 则 x₀ 一定是极值点  
③ y = eˣ 的导数是 eˣ  
④ y = lnx 的定义域为 (0, +∞)

A. 2 项  
B. 3 项  
C. 4 项  
D. 1 项

---

**【答案】B**

**【解析】**

逐项分析：

**① f'(x) > 0 时 f(x) 单调递增？**
对。这是单调性判断的基本定理。✅

**② f'(x₀) = 0 则 x₀ 一定是极值点？**
不一定。反例：f(x) = x³，f'(0) = 0，但 x = 0 不是极值点（是拐点）。❌

**③ y = eˣ 的导数是 eˣ？**
对。指数函数 eˣ 的导数等于自身。✅

**④ y = lnx 的定义域为 (0, +∞)？**
对。对数函数真数必须大于 0。✅

正确的有 ①③④ 共 **3 项**。

**怎么想到的：** 这类题主要考概念辨析。②的陷阱最常见——f'(x₀) = 0 只是必要条件，不是充分条件，还要看导数是否变号。

## 第9题 ⭐⭐⭐ 挑战（5分）选择题

**f(x) = x² - 2ax + a² - 1，若 f(x) 在 (1, +∞) 上单调递增，则 a 的取值范围是？**

A. a ≤ 1  
B. a < 1  
C. a ≥ 1  
D. a > 1

---

**【答案】A**

**【解析】**

第一步：求导

f'(x) = 2x - 2a = 2(x - a)

第二步：分析条件

f(x) 在 (1, +∞) 上单调递增 → f'(x) ≥ 0 在 (1, +∞) 上成立

2(x - a) ≥ 0 → x ≥ a

在 (1, +∞) 上 x ≥ a 恒成立，即 a ≤ 1（a 最大不超过区间左端点）。

**验证 a = 1：** f'(x) = 2(x-1)，x > 1 时 f'(x) > 0，满足单调递增。✅
**验证 a = 0：** f'(x) = 2x > 0 当 x > 1，满足。✅
**验证 a = 2：** f'(x) = 2(x-2)，x = 1.5 时 f'(1.5) = -1 < 0，不满足。✅

**怎么想到的：** "在 (1, +∞) 上递增"转化为"f'(x) ≥ 0 在 (1, +∞) 上恒成立"。关键是理解：x ≥ a 对所有 x > 1 成立，等价于 a 不超过这个区间的最小值，即 a ≤ 1。

## 第10题 ⭐⭐⭐ 挑战（5分）选择题

**f(x) = x³ - 3x + m，若 f(x) 有 3 个不同的零点，则 m 的取值范围是？**

A. (-2, 2)  
B. [-2, 2]  
C. (-3, 3)  
D. [-1, 1]

---

**【答案】A**

**【解析】**

第一步：分析函数形态

f'(x) = 3x² - 3 = 3(x + 1)(x - 1)

| 区间 | (-∞, -1) | (-1, 1) | (1, +∞) |
|------|-----------|---------|-----------|
| f'(x) | + | - | + |

极大值点 x = -1，极小值点 x = 1

第二步：求极值

极大值：f(-1) = -1 + 3 + m = 2 + m

极小值：f(1) = 1 - 3 + m = -2 + m

第三步：确定 m 的范围

三次函数有 3 个不同零点的条件：极大值 > 0 且极小值 < 0

2 + m > 0 → m > -2

-2 + m < 0 → m < 2

所以 **-2 < m < 2**

**验证 m = 0：** f(x) = x³ - 3x = x(x² - 3) = x(x - √3)(x + √3)，3 个零点 0, √3, -√3。✅
**验证 m = 2：** f(1) = 0，x = 1 是重根，只有 2 个不同零点。不满足。✅
**验证 m = -2：** f(-1) = 0，x = -1 是重根，只有 2 个不同零点。不满足。✅

**怎么想到的：** 三次函数有 3 个零点 = 图像穿过 x 轴 3 次 = 极大值在 x 轴上方且极小值在 x 轴下方。参数 m 控制图像的上下平移，使得极大值 > 0 且极小值 < 0 就行。

## 复习题（交叉练习）

### 复1
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 14-平面向量-模的计算

**题目：** 已知 $\vec{a} = (2, 1)$，$\vec{b} = (1, k)$，若 $|\vec{a} + \vec{b}| = \sqrt{10}$，则 $k$ 的值为（　）

A. $0$　　B. $-2$　　C. $0$ 或 $-2$　　D. $1$ 或 $-3$

**答案：** C

**解析：**

$\vec{a} + \vec{b} = (3, 1+k)$

$|\vec{a} + \vec{b}|^2 = 9 + (1+k)^2 = 10$

$(1+k)^2 = 1$

$1 + k = \pm 1$

$k = 0$ 或 $k = -2$

**验证：**
- $k = 0$：$\vec{a}+\vec{b}=(3,1)$，$|\vec{a}+\vec{b}|=\sqrt{9+1}=\sqrt{10}$ ✓
- $k = -2$：$\vec{a}+\vec{b}=(3,-1)$，$|\vec{a}+\vec{b}|=\sqrt{9+1}=\sqrt{10}$ ✓

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 13-函数性质-奇函数求值

**题目：** 已知 $f(x)$ 是定义在 $\mathbb{R}$ 上的奇函数，当 $x > 0$ 时 $f(x) = x^2 - 1$，则 $f(-2)$ 的值为（　）

A. $3$　　B. $-3$　　C. $1$　　D. $-1$

**答案：** B

**解析：**

由奇函数性质：$f(-x) = -f(x)$

$f(-2) = -f(2) = -(2^2 - 1) = -(4 - 1) = -3$

**验证：** 奇函数 $f(0)=0$，$f(-2)=-3$，$f(2)=3$，$f(-2)=-f(2)$ ✓

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 09-数列基础-等比求和

**题目：** 在等比数列 $\{a_n\}$ 中，$a_1 = 1$，公比 $q = 2$，前 $n$ 项和 $S_n = 63$，则 $n =$（　）

A. $5$　　B. $6$　　C. $7$　　D. $8$

**答案：** B

**解析：**

等比数列求和公式：$S_n = \dfrac{a_1(1-q^n)}{1-q}$

$63 = \dfrac{1 \times (1 - 2^n)}{1 - 2} = 2^n - 1$

$2^n = 64$

$n = 6$

**验证：** $S_6 = 1+2+4+8+16+32 = 63$ ✓
