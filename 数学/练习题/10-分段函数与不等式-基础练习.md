# 分段函数与不等式 · 解析版

> **专题说明：** 分段函数的核心就是"分段处理"——先看自变量落在哪一段，再用对应公式。解不等式时要每段分别解，再与该段条件取交集，最后取并集。

## 答案与详细解析

---

### 第1题 · 答案：D

**解析：**

求 $f(-1)$：因为 $-1 \le 0$，代入第一段 $f(x)=2x+1$。

$f(-1) = 2\times(-1)+1 = -2+1 = -1$

求 $f(1)$：因为 $1 > 0$，代入第二段 $f(x)=x^2-1$。

$f(1) = 1^2-1 = 0$

$f(-1)+f(1) = -1+0 = \boxed{-1}$

**思路过程：** 分段函数求值的第一步永远是判断自变量落在哪一段。分界点是 $x=0$，第一段用 $\le$（包含0），第二段用 $>$（不包含0），每个 $x$ 只属于一段，不会重复也不会遗漏。

---

### 第2题 · 答案：C

**解析：**

要解 $f(x)=2$，分别在两段中求解。

**段一**（$x < 1$）：$3^x = 2 \Rightarrow x = \log_3 2$

验证：$\log_3 2 \approx 0.631 < 1$ ✓，落在段一范围内，有效。

**段二**（$x \ge 1$）：$2-x = 2 \Rightarrow x = 0$

但 $0 \not\ge 1$，不满足段二的条件，舍去。

**唯一解：$x = \log_3 2$。**

验算：$f(\log_3 2) = 3^{\log_3 2} = 2$ ✓

**思路过程：** 解 $f(x)=c$ 和解不等式的步骤一样——每段分别解，再验证解是否落在该段范围内。段二虽然解出了 $x=0$，但不满足 $x\ge 1$，必须舍去。这是分段函数最容易丢分的地方。

---

### 第3题 · 答案：A

**解析：**

分别求两段的值域，再取并集。

**段一**（$x < 0$）：$f(x)=-x > 0$。

当 $x \to -\infty$ 时 $f(x) \to +\infty$，当 $x \to 0^-$ 时 $f(x) \to 0$。值域为 $(0,+\infty)$。

**段二**（$x \ge 0$）：$f(x)=x+1 \ge 1$。值域为 $[1,+\infty)$。

**并集：** $(0,+\infty) \cup [1,+\infty) = (0,+\infty)$（因为 $[1,+\infty) \subset (0,+\infty)$）

**思路过程：** 求分段函数值域的方法：分别求每段值域，取并集。段一 $x<0$ 不包含 $x=0$，所以 $-x$ 的值域不包含 0。但两段并集是 $(0,+\infty)$，因为第二段从 1 开始，完全被第一段覆盖了。

---

### 第4题 · 答案：B

**解析：**

分两段分别解 $f(x)>1$。

**段一**（$x \le 1$）：$-x+3 > 1 \Rightarrow -x > -2 \Rightarrow x < 2$

与 $x \le 1$ 取交集：$\boxed{(-\infty,1]}$

**段二**（$x > 1$）：$x^2-2 > 1 \Rightarrow x^2 > 3 \Rightarrow x > \sqrt3$ 或 $x < -\sqrt3$

与 $x > 1$ 取交集：$x < -\sqrt3$ 与 $x > 1$ 无交集，舍去；$x > \sqrt3$ 保留。

所以段二结果为 $\boxed{(\sqrt3,+\infty)}$。

**并集：** $(-\infty,1] \cup (\sqrt3,+\infty)$

验算：
- $x=0$：$f(0)=3>1$ ✓（在 $(-\infty,1]$ 中）
- $x=1$：$f(1)=-1+3=2>1$ ✓（$x=1$ 属于段一，$\le$ 包含）
- $x=1.5$：$f(1.5)=2.25-2=0.25<1$ ✓（不在解集中，正确！）
- $x=2$：$f(2)=4-2=2>1$ ✓（$\sqrt3\approx1.732$，$2>\sqrt3$）
- $x=\sqrt3$：$f(\sqrt3)=3-2=1\not>1$ ✓（端点不在）

**思路过程：** 分段解不等式的标准步骤——每段代入公式解出解集，再与该段的条件取交集（最容易遗忘的步骤！），最后所有交集取并集。二次不等式 $x^2>3$ 开口向上，取两根之外。特别注意 $x=1$ 处属于段一（$\le$ 包含），$f(1)=2>1$，包含在解集中。

---

### 第5题 · 答案：A

**解析：**

解 $f(x) \le 1$。

**段一**（$x \le 0$）：$2^{-x} \le 1 = 2^0$

底数 $2>1$，指数函数单调递增，所以 $-x \le 0 \Rightarrow x \ge 0$。

与 $x \le 0$ 取交集：$x=0$。

**段二**（$x > 0$）：$\log_2(x+1) \le 1 = \log_2 2$

底数 $2>1$，对数函数单调递增，所以 $x+1 \le 2 \Rightarrow x \le 1$。

与 $x > 0$ 取交集：$(0,1]$。

**并集：** $\{0\} \cup (0,1] = [0,1]$

验算：
- $x=0$：$f(0)=2^0=1\le1$ ✓
- $x=0.5$：$f(0.5)=\log_2 1.5\approx 0.585\le1$ ✓
- $x=1$：$f(1)=\log_2 2=1\le1$ ✓
- $x=-1$：$f(-1)=2^1=2>1$ ✓（不在解集）
- $x=2$：$f(2)=\log_2 3\approx 1.585>1$ ✓（不在解集）

**思路过程：** 这题同时考指数函数和对数函数不等式。核心技巧：底数 $>1$ 时函数单调递增，解不等式时不等号方向不变。段一解出 $x\ge0$ 但段一条件是 $x\le0$，交集只有 $x=0$ 一个点——这不会遗漏，因为两段恰好"无缝衔接"。

---

### 第6题 · 答案：A

**解析：**

判断分段函数在分界点 $x=1$ 处是否连续。

$x=1$ 属于段一（$x\le 1$），所以 $f(1)=1^2+1=2$。

段二在 $x\to 1^+$ 时的极限：$\lim\limits_{x\to 1^+}(3-x) = 2$。

三者比较：左极限 = 右极限 = 函数值 = 2，**图像在 $x=1$ 处连续**。

**思路过程：** 分段函数在分界点处连续的条件是：左极限 = 右极限 = 函数值。分界点属于 $\le$ 那一段，所以 $f(1)$ 用段一的公式计算。

---

### 第7题 · 答案：B

**解析：**

解 $f(x) > 0$。

**段一**（$x \le 0$）：$x^2-2x > 0$

$x(x-2) > 0$。二次函数开口向上，两根为 0 和 2，取"两根之外"：$x<0$ 或 $x>2$。

与 $x \le 0$ 取交集：$\boxed{(-\infty,0)}$

**段二**（$x > 0$）：$\ln(x+1) > 0$

$\ln(x+1) > 0 \Rightarrow x+1 > 1 \Rightarrow x > 0$。

与 $x > 0$ 取交集：$\boxed{(0,+\infty)}$

**并集：** $(-\infty,0) \cup (0,+\infty)$

验算：
- $x=-1$：$f(-1)=1+2=3>0$ ✓
- $x=-3$：$f(-3)=9+6=15>0$ ✓
- $x=0$：$f(0)=0$（不满足 $>0$）✓
- $x=0.5$：$f(0.5)=\ln 1.5\approx 0.405>0$ ✓
- $x=1$：$f(1)=\ln 2\approx 0.693>0$ ✓

**思路过程：** 段一解二次不等式时注意：$x(x-2)>0$ 的解集是 $x<0$ 或 $x>2$（开口向上取两根之外），再与 $x\le0$ 取交集得到 $(-\infty,0)$。段二 $\ln(x+1)>0$ 等价于 $x+1>1$ 即 $x>0$。两端点 $x=0$ 代入检验恰好为0，不满足严格大于。

---

### 第8题 · 答案：B

**解析：**

解 $f(x) > 0$。

**段一**（$x \le 0$）：$\left(\dfrac{1}{2}\right)^x - 1 > 0$

$\left(\dfrac{1}{2}\right)^x > 1 = \left(\dfrac{1}{2}\right)^0$

底数 $\dfrac{1}{2}<1$，指数函数单调递减，不等号方向反转：$x < 0$。

与 $x \le 0$ 取交集：$\boxed{(-\infty,0)}$

**段二**（$x > 0$）：$x^2-x-1 > 0$

求根公式：$x = \dfrac{1\pm\sqrt{1+4}}{2} = \dfrac{1\pm\sqrt5}{2}$

$\dfrac{1-\sqrt5}{2}\approx -0.618$，$\dfrac{1+\sqrt5}{2}\approx 1.618$

开口向上，取两根之外：$x < \dfrac{1-\sqrt5}{2}$ 或 $x > \dfrac{1+\sqrt5}{2}$。

与 $x > 0$ 取交集：$x < -0.618$ 与 $x > 0$ 无交集，舍去；$x > \dfrac{1+\sqrt5}{2}$ 保留。

所以段二结果为 $\boxed{\left(\dfrac{1+\sqrt5}{2},+\infty\right)}$。

**并集：** $(-\infty,0) \cup \left(\dfrac{1+\sqrt5}{2},+\infty\right)$

验算：
- $x=-1$：$\left(\dfrac{1}{2}\right)^{-1}-1 = 2-1 = 1>0$ ✓
- $x=0$：$1-1=0$（不满足 $>0$）✓
- $x=1$：$1-1-1=-1<0$ ✓（不在解集）
- $x=2$：$4-2-1=1>0$ ✓
- $x=\dfrac{1+\sqrt5}{2}$：代入得 $0$（端点不在）✓

**思路过程：** 这题是挑战题，综合了指数不等式和二次不等式。关键陷阱：底数 $\frac{1}{2}<1$ 时指数函数**单调递减**，$\left(\frac{1}{2}\right)^x > 1$ 要求 $x < 0$（不等号要反转！），这和底数 $>1$ 时恰好相反。二次不等式求根时要小心判别式 $\Delta = 1+4=5$，$\sqrt5$ 不要算错。

---

### 第9题 · 答案：D

**解析：**

$f(x)$ 在 $\mathbb{R}$ 上单调递增，需要满足两个条件。

**条件一：每段内部单调递增。**

段一 $g(x)=-x^2+2x+a=-(x-1)^2+1+a$，对称轴 $x=1$，开口向下。

在 $x\le 1$（对称轴左侧），抛物线单调递增 ✓。

段二 $h(x)=2^x$，单调递增 ✓。

**条件二：分界点处"左端不超过右端"。**

$f$ 在 $\mathbb{R}$ 上单调递增，需要 $f(1) \le f(x)$ 对所有 $x>1$ 成立。

$f(1) = -1+2+a = 1+a$（段一，因为 $x=1 \le 1$）

$\lim\limits_{x\to 1^+}f(x) = 2^1 = 2$（段二在分界点附近的值）

由于段二 $2^x$ 单调递增，对任意 $x>1$ 都有 $2^x > 2$，所以需要 $1+a \le 2$，即 $\boxed{a \le 1}$。

验算：
- $a=1$：$f(1)=2$，$f(1.5)=2^{1.5}\approx 2.828 > 2$ ✓
- $a=0$：$f(1)=1$，$f(1.1)=2^{1.1}\approx 2.143 > 1$ ✓
- $a=-1$：$f(1)=0$，$f(1.5)\approx 2.828 > 0$ ✓
- $a=2$：$f(1)=3$，$f(1.5)\approx 2.828 < 3$ ✗（不满足单调递增）

**思路过程：** 分段函数整体单调递增需要两个条件：① 每段内部各自单调递增；② 在分界点处"左段的最大值 $\le$ 右段的最小值"。段一是开口向下的抛物线，对称轴在 $x=1$，所以 $x\le 1$ 时递增（在对称轴左侧）。关键是求 $a$ 的上界使分界点处衔接。

---

### 第10题 · 答案：B

**解析：**

解 $f(x) < 1$。

**段一**（$x < 0$）：$|x-1| < 1$

$|x-1| < 1 \Rightarrow -1 < x-1 < 1 \Rightarrow 0 < x < 2$

与 $x < 0$ 取交集：空集。

**段二**（$x \ge 0$）：$2x-1 < 1 \Rightarrow 2x < 2 \Rightarrow x < 1$

与 $x \ge 0$ 取交集：$\boxed{[0,1)}$

**并集：** $[0,1)$

验算：
- $x=0$：$f(0)=2\times0-1=-1<1$ ✓
- $x=0.5$：$f(0.5)=0<1$ ✓
- $x=1$：$f(1)=2-1=1\not<1$ ✓（端点不在）
- $x=-1$：$f(-1)=|-1-1|=2\not<1$ ✓（不在解集）

**思路过程：** 段一是绝对值不等式 $|x-1|<1$，化简得 $0<x<2$，但与 $x<0$ 取交集为空。段二是一次不等式，解出 $x<1$，与 $x\ge0$ 交集为 $[0,1)$。最终解集只有第二段的贡献。

## 复习题（交叉练习）

### 复1
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 04-三角函数

**题目：** 求值：2sin²(π/12) - 1

**答案：** -√3/2

**解析：**
复习知识点：04-三角函数

识别结构：cos 2α = 1 - 2sin²α

所以 2sin²α - 1 = -(1 - 2sin²α) = -cos 2α

```
2sin²(π/12) - 1 = -cos(π/6)
```

cos(π/6) = √3/2

所以 2sin²(π/12) - 1 = **-√3/2**

**验证：** sin(π/12) = sin 15° ≈ 0.2588
2 × 0.2588² - 1 = 2 × 0.06699 - 1 = 0.13398 - 1 = -0.8660 ≈ -√3/2 ≈ -0.8660 ✓

**答案：-√3/2**

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 05-二项式定理

**题目：** (1 - 2x)⁶ 展开式中，x³ 的系数是多少？

**答案：** -160

**解析：**
复习知识点：05-二项式定理

**通项公式：** Tₖ₊₁ = C(6,k) · 1⁶⁻ᵏ · (-2x)ᵏ = C(6,k) · (-2)ᵏ · xᵏ

令 k = 3（找 x³ 对应的项）：
```
系数 = C(6,3) · (-2)³
```

C(6,3) = 6!/(3!·3!) = 20
(-2)³ = -8

系数 = 20 × (-8) = **-160**

**答案：-160**

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 06-基本不等式

**题目：** 已知 x > 0，求 2x + 3/x 的最小值。

**答案：** 2√6

**解析：**
复习知识点：06-基本不等式

x > 0 时，2x > 0 且 3/x > 0，且它们的乘积 2x · 3/x = 6 为定值，可以用基本不等式：

```
2x + 3/x ≥ 2√(2x · 3/x) = 2√6
```

等号成立条件：2x = 3/x，即 x² = 3/2，x = √6/2（取正）

**验证：** x = √6/2 时
- 2x = √6
- 3/x = 3/(√6/2) = 6/√6 = √6
- 2x + 3/x = √6 + √6 = 2√6 ✓

**答案：2√6**
