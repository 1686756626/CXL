## 第1题 ⭐ 基础（5分）选择题

**已知 a = (3, -2)，b = (1, 4)，则 a + b = ？**

A. (4, 2)  
B. (2, 6)  
C. (4, -6)  
D. (2, -6)

---

**【答案】A**

**【解析】**

向量加法：对应分量相加。

a + b = (3 + 1, -2 + 4) = **(4, 2)**

**怎么想到的：** 向量加法就像坐标相加，x 加 x，y 加 y，非常直接。

## 第2题 ⭐ 基础（5分）选择题

**已知 a = (2, -3)，则 -3a = ？**

A. (-6, 9)  
B. (6, -9)  
C. (-6, -9)  
D. (6, 9)

---

**【答案】A**

**【解析】**

向量数乘：每个分量乘以 k。

-3a = (-3 × 2, -3 × (-3)) = **(-6, 9)**

**怎么想到的：** 数乘就是把向量的每个分量都乘以同一个数。注意负号也要乘进去，负数乘负数得正。

## 第3题 ⭐ 基础（5分）选择题

**已知 A(1, 3)，B(4, 7)，则 |AB| = ？**

A. 4  
B. 5  
C. 3  
D. 7

---

**【答案】B**

**【解析】**

AB = B - A = (4 - 1, 7 - 3) = (3, 4)

|AB| = √(3² + 4²) = √(9 + 16) = √25 = **5**

**怎么想到的：** 这是经典的 3-4-5 直角三角形。向量模 = √(x² + y²)，和勾股定理一个道理。记住：向量坐标 = 终点 - 起点。

## 第4题 ⭐⭐ 中等（5分）选择题

**已知 a = (2, 1)，b = (3, -2)，则 a · b = ？**

A. 4  
B. 8  
C. -4  
D. 0

---

**【答案】A**

**【解析】**

点乘（内积）公式：a · b = x₁x₂ + y₁y₂

a · b = 2 × 3 + 1 × (-2) = 6 - 2 = **4**

**怎么想到的：** 点乘就是"对应分量相乘再相加"。注意点乘的结果是一个**数**（标量），不是向量。

## 第5题 ⭐⭐ 中等（5分）选择题

**已知 a = (1, 2)，b = (-2, k)，若 a ⊥ b，则 k = ？**

A. 1  
B. 2  
C. -1  
D. -2

---

**【答案】A**

**【解析】**

垂直条件：a · b = 0

a · b = 1 × (-2) + 2 × k = -2 + 2k = 0

解得：2k = 2，**k = 1**

**怎么想到的：** "垂直"和"点乘为零"是等价条件，看到垂直就想到点乘 = 0，这是向量题最高频的考点。

## 第6题 ⭐⭐ 中等（5分）选择题

**已知 |a| = 2，|b| = 3，a · b = 3，则 a 与 b 的夹角为？**

A. 30°  
B. 45°  
C. 60°  
D. 90°

---

**【答案】C**

**【解析】**

点乘的几何意义：a · b = |a| |b| cos θ

3 = 2 × 3 × cos θ

cos θ = 3/6 = 1/2

θ = **60°**

**怎么想到的：** 求向量夹角三步走：①用 a · b = |a||b|cosθ → ②算出 cosθ → ③查常见角度。cos60° = 1/2 是必背的。

## 第7题 ⭐⭐ 中等（5分）选择题

**已知 a = (2, 3)，b = (4, m)，若 a ∥ b，则 m = ？**

A. 3  
B. 6  
C. 5  
D. 8

---

**【答案】B**

**【解析】**

平行条件：对应分量成比例，即 x₁y₂ = x₂y₁

2 × m = 3 × 4

2m = 12，**m = 6**

**验证：** b = (4, 6) = 2 × (2, 3) = 2a，确实平行。✅

**怎么想到的：** 向量平行的坐标条件是"交叉相乘相等"：x₁y₂ = x₂y₁。可以理解为 a 和 b 的分量比相等：2/4 = 3/m → m = 6。

## 第8题 ⭐⭐ 中等（5分）选择题

**在 △ABC 中，A(0, 0)，B(6, 0)，C(0, 4)，则重心 G 的坐标是？**

A. (2, 4/3)  
B. (3, 2)  
C. (2, 3)  
D. (1, 2)

---

**【答案】A**

**【解析】**

重心坐标公式：G = ((x_A + x_B + x_C)/3, (y_A + y_B + y_C)/3)

G = ((0 + 6 + 0)/3, (0 + 0 + 4)/3) = **(2, 4/3)**

**怎么想到的：** 重心就是三角形三条中线的交点，坐标等于三个顶点坐标的平均值。公式直接用就行。

## 第9题 ⭐⭐⭐ 挑战（5分）选择题

**在 △ABC 中，A(0, 0)，B(4, 0)，C(1, 3)，则 △ABC 的面积是？**

A. 5  
B. 6  
C. 7  
D. 8

---

**【答案】B**

**【解析】**

**方法一（叉积法）：**

AB = (4, 0)，AC = (1, 3)

叉积绝对值：|AB × AC| = |4 × 3 - 0 × 1| = 12

面积 = 12/2 = **6**

**方法二（底×高/2）：**

AB 在 x 轴上，|AB| = 4

C(1, 3) 到 x 轴（AB 所在直线）的距离 = 3

面积 = 4 × 3/2 = **6**

**验证：** 两种方法结果一致。✅

**怎么想到的：** 求三角形面积有两种常用方法：①叉积法（向量叉积的一半）；②底×高/2（找一条在坐标轴上的边作底）。这道题 AB 在 x 轴上，方法二更直观。

## 第10题 ⭐⭐⭐ 挑战（5分）选择题

**已知 |a| = 1，|b| = 2，a · b = -1，则 |a + b| = ？**

A. 1  
B. √3  
C. 2  
D. 3

---

**【答案】B**

**【解析】**

**|a + b|² = |a|² + |b|² + 2a · b**

= 1² + 2² + 2 × (-1)

= 1 + 4 - 2

= 3

所以 **|a + b| = √3**

**验证：** 也可以先求 cosθ = a·b/(|a||b|) = -1/2，θ = 120°。
|a + b|² = 1 + 4 + 2×1×2×cos120° = 5 + 4×(-1/2) = 5 - 2 = 3。✅

**怎么想到的：** 求 |a + b| 的万能公式：**|a + b|² = |a|² + |b|² + 2a · b**。这个公式就是向量版的完全平方展开，背下来直接用。

## 复习题（交叉练习）

### 复1
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 13-函数性质-单调性

**题目：** 函数 $f(x) = \dfrac{1}{x}$ 在 $(0, +\infty)$ 上的单调性是（　）

A. 单调递增　　B. 单调递减　　C. 先增后减　　D. 先减后增

**答案：** B

**解析：**

$f'(x) = -\dfrac{1}{x^2} < 0$ 对 $x \in (0, +\infty)$ 恒成立，所以 $f(x)$ 在 $(0, +\infty)$ 上**单调递减**。

**验证：** $f(1) = 1$，$f(2) = 0.5$，$f(3) \approx 0.33$，递减趋势明显 ✓

---

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 15-导数第一问-基本求导与单调性

**题目：** 函数 $f(x) = x^3 + 2x$ 的单调性为（　）

A. 在 $\mathbb{R}$ 上单调递增　　B. 在 $\mathbb{R}$ 上单调递减　　C. 有增有减　　D. 无法确定

**答案：** A

**解析：**

$f'(x) = 3x^2 + 2$

因为 $3x^2 \geq 0$ 且 $2 > 0$，所以 $f'(x) = 3x^2 + 2 > 0$ 恒成立，$f(x)$ 在 $\mathbb{R}$ 上**单调递增**。

**验证：** $f(-1) = -3$，$f(0) = 0$，$f(1) = 3$，递增 ✓

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 12-立体几何第一问-点到面距离

**题目：** 已知正方体 $ABCD-A_1B_1C_1D_1$ 的棱长为 $1$，则点 $A$ 到平面 $A_1BD$ 的距离为（　）

A. $\dfrac{\sqrt{3}}{3}$　　B. $\dfrac{\sqrt{2}}{2}$　　C. $\dfrac{\sqrt{6}}{3}$　　D. $\dfrac{1}{2}$

**答案：** A

**解析：**

以 $A$ 为原点建系：$A(0,0,0)$，$B(1,0,0)$，$D(0,1,0)$，$A_1(0,0,1)$。

平面 $A_1BD$ 的法向量：$\vec{A_1B} = (1,0,-1)$，$\vec{A_1D} = (0,1,-1)$。

设 $\vec{n} = (x,y,z)$：

$\vec{n}\cdot\vec{A_1B} = x - z = 0 \Rightarrow x = z$

$\vec{n}\cdot\vec{A_1D} = y - z = 0 \Rightarrow y = z$

取 $z = 1$：$\vec{n} = (1,1,1)$，$|\vec{n}| = \sqrt{3}$。

点 $A(0,0,0)$ 到平面的距离（平面上点 $A_1(0,0,1)$）：

$d = \dfrac{|\overrightarrow{A_1A}\cdot\vec{n}|}{|\vec{n}|} = \dfrac{|(0,0,-1)\cdot(1,1,1)|}{\sqrt{3}} = \dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}$

**验证：** 等体积法：$V_{A-A_1BD} = \dfrac{1}{3}\times\dfrac{\sqrt{3}}{2}\times d = \dfrac{\sqrt{3}\,d}{6}$，又 $V = \dfrac{1}{3}\times\dfrac{1}{2}\times 1 = \dfrac{1}{6}$，$d = \dfrac{1}{\sqrt{3}}$ ✓
