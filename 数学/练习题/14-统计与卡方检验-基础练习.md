# 统计与卡方检验 · 解析版

> **专题说明：** 这节分两部分：① 条件概率——看行或看列，分子是交集，分母是该行（或列）的总和；② 卡方检验——背公式、算 $ad-bc$、比临界值、写结论。

---

## 核心公式速查

**条件概率：** $P(B|A) = \dfrac{P(AB)}{P(A)}$（看行算就行，分子=交集，分母=该行总和）

**卡方公式：**

$$\chi^2 = \dfrac{n(ad-bc)^2}{(a+b)(c+d)(a+c)(b+d)}$$

其中：

|  | $B_1$ | $B_2$ | 行和 |
|--|-------|-------|------|
| $A_1$ | $a$ | $b$ | $a+b$ |
| $A_2$ | $c$ | $d$ | $c+d$ |
| 列和 | $a+c$ | $b+d$ | $n$ |

**临界值：** $\chi^2>6.635 \Rightarrow$ 99%把握；$\chi^2>3.841 \Rightarrow$ 95%把握

## 答案与详细解析

---

### 第1题 · 答案：A

**解析：**

$P(\text{戴眼镜}|\text{男生})$ 表示"已知是男生的条件下，戴眼镜的概率"。

看"男生"这一行：男生共 30 人，戴眼镜的 15 人。

$P = \dfrac{15}{30} = \boxed{\dfrac{1}{2}}$

**思路过程：** 条件概率 $P(B|A)$ 的做法：找到条件 $A$ 对应的行（或列），分子是 $AB$ 的交集，分母是 $A$ 的总和。这里条件是"男生"，看男生这一行。

---

### 第2题 · 答案：B

**解析：**

$P(\text{男生}|\text{戴眼镜})$ 表示"已知戴眼镜的条件下，是男生的概率"。

看"戴眼镜"这一列：戴眼镜共 $15+10=25$ 人，其中男生 15 人。

$P = \dfrac{15}{25} = \boxed{\dfrac{3}{5}}$

**思路过程：** 条件换了——现在条件是"戴眼镜"，所以看戴眼镜这一列。分母从 30 变成了 25，结果完全不同。$P(B|A)$ 和 $P(A|B)$ 一般不相等！

---

### 第3题 · 答案：A

**解析：**

确定 $a,b,c,d,n$：$a=30$，$b=10$，$c=20$，$d=40$，$n=100$

**第一步：** $ad-bc = 30\times40-10\times20 = 1200-200 = 1000$

**第二步：** $(ad-bc)^2 = 1000000$

**第三步：** 分母 $= (30+10)(20+40)(30+20)(10+40) = 40\times60\times50\times50 = 6000000$

**第四步：** $\chi^2 = \dfrac{100\times1000000}{6000000} = \dfrac{100000000}{6000000} = \dfrac{100}{6} \approx \boxed{16.67}$

**思路过程：** 卡方公式的计算步骤：① 确定 $a,b,c,d$ 的位置（左上、右上、左下、右下）；② 算 $ad-bc$（最容易出错！）；③ 算平方和分母；④ 做除法。

---

### 第4题 · 答案：C

**解析：**

由第 3 题算得 $\chi^2 \approx 16.67$。

比较：$16.67 > 6.635$

所以有 **99% 的把握**认为 $A$ 与 $B$ 有关。

**思路过程：** 算出 $\chi^2$ 后和临界值比较：$\chi^2>6.635$ 就是 99% 把握，$\chi^2>3.841$ 就是 95% 把握。结论的措辞要规范。

### 第5题 · 答案：C

**解析：**

$a=72$，$b=48$，$c=24$，$d=56$，$n=200$

$ad-bc = 72\times56-48\times24 = 4032-1152 = 2880$

$(ad-bc)^2 = 8294400$

分子 $= 200\times8294400 = 1658880000$

分母 $= 120\times80\times96\times104 = 9600\times9984 = 95846400$

$\chi^2 = \dfrac{1658880000}{95846400} \approx \boxed{17.31}$

**思路过程：** 数字变大了，计算更容易出错。建议一步一步写，不要跳步。先算 $ad-bc$，再算 $(ad-bc)^2$，再算分母，最后做除法。

**解析：**

$a=25$，$b=15$，$c=22$，$d=38$，$n=100$

### 第9题 · 答案：B

**解析：**

**表 1：** $a=30,b=10,c=10,d=30,n=80$

$ad-bc=30\times30-10\times10=900-100=800$

$(ad-bc)^2=640000$

$\chi^2 = \dfrac{80\times640000}{40\times40\times40\times40}=\dfrac{51200000}{2560000}=\boxed{20.00}$

**表 2：** $a=35,b=5,c=15,d=25,n=80$

$ad-bc=35\times25-5\times15=875-75=800$

$(ad-bc)^2=640000$

$\chi^2 = \dfrac{80\times640000}{40\times40\times50\times30}=\dfrac{51200000}{2400000}\approx\boxed{21.33}$

$21.33 > 20.00$，所以**表 2 的关联更强**。

**思路过程：** $\chi^2$ 越大说明两个变量之间的关联越强。两张表虽然 $ad-bc$ 相同，但因为行和、列和不同，$\chi^2$ 不同。这说明仅看 $ad-bc$ 的大小不能判断关联强弱，必须完整计算 $\chi^2$。

### 第11题
**难度：** ⭐⭐（中等）

**题目：** 一个袋子中有 3 个红球和 2 个白球，除颜色外完全相同。从中随机取出 2 个球，则取出的 2 个球都是红球的概率为（ ）

A. 3/10　　B. 3/5　　C. 2/5　　D. 1/5

**答案：** A

**解析：**

**思路：** 古典概型，总情况数 = C(5,2)，满足条件的情况数 = C(3,2)。

总情况数：C(5,2) = 5×4/(2×1) = 10

满足条件（2个都是红球）：C(3,2) = 3×2/(2×1) = 3

P = 3/10

**答案：A**

**思路过程：** 古典概型三步：①确认等可能 → ②算总情况数 → ③算满足条件情况数 → P = 满足/总。这里"随机取2个"是等可能的，用组合数算。

### 第12题
**难度：** ⭐⭐（中等）

**题目：** 某校高三学生数学成绩服从正态分布 N(70, 16²)，则成绩在 54 分到 86 分之间的学生比例约为（ ）

A. 68.3%　　B. 81.5%　　C. 95.4%　　D. 99.7%

**答案：** A

**解析：**

**思路：** 正态分布，用 3σ 原则。μ = 70，σ = 16。

54 = 70 - 16 = μ - σ
86 = 70 + 16 = μ + σ

所以 [54, 86] 就是 [μ-σ, μ+σ]，概率为 68.3%。

**思路过程：** 3σ 原则要背：[μ-σ, μ+σ] = 68.3%，[μ-2σ, μ+2σ] = 95.4%，[μ-3σ, μ+3σ] = 99.7%。先算区间对应几个σ范围，再查对应概率。这里 σ=16，54=μ-σ，86=μ+σ，恰好是一个σ范围。

### 第13题
**难度：** ⭐⭐⭐（挑战）

**题目：** 已知 X ~ N(2, 1)，且 P(X ≤ 1) = 0.1587，则 P(1 < X ≤ 3) =（ ）

A. 0.3174　　B. 0.6826　　C. 0.8413　　D. 0.3413

**答案：** B

**解析：**

**思路：** 正态分布关于 μ 对称。μ = 2，σ = 1。

P(X ≤ 1) = 0.1587，即 P(X ≤ μ-σ) = 0.1587

由对称性：P(X ≥ μ+σ) = P(X ≤ μ-σ) = 0.1587

所以 P(X > 3) = P(X > μ+σ) = 0.1587

P(X ≤ 3) = 1 - 0.1587 = 0.8413

P(1 < X ≤ 3) = P(X ≤ 3) - P(X ≤ 1) = 0.8413 - 0.1587 = 0.6826

**答案：B**

**思路过程：** [1, 3] = [μ-σ, μ+σ]，即一个σ范围内的概率。但题目给了 P(X≤1) 而不是直接问3σ范围，所以要用对称性算。P(1<X≤3) = P(X≤3) - P(X≤1) = (1-0.1587) - 0.1587 = 0.6826。

### 复1
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 10-解三角形-余弦定理

**题目：** 在 $\triangle ABC$ 中，$a = 5$，$b = 7$，$C = 60°$，求边 $c$。

**答案：** $c = \sqrt{39}$

**解析：**

已知两边及夹角，用余弦定理：

$c^2 = a^2 + b^2 - 2ab\cos C = 25 + 49 - 2 \times 5 \times 7 \times \dfrac{1}{2} = 74 - 35 = 39$

$c = \sqrt{39}$

**验证：** $\cos C = \dfrac{a^2+b^2-c^2}{2ab} = \dfrac{25+49-39}{70} = \dfrac{35}{70} = \dfrac{1}{2}$，$C = 60°$ ✓

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 13-函数性质-单调性

**题目：** 函数 $f(x) = -x^2 + 4x$ 的单调递增区间为（　）

A. $(-\infty, 0)$　　B. $(-\infty, 2)$　　C. $(2, +\infty)$　　D. $(0, +\infty)$

**答案：** B

**解析：**

$f'(x) = -2x + 4 = -2(x - 2)$

$f'(x) > 0 \Rightarrow -2(x-2) > 0 \Rightarrow x < 2$

所以单调递增区间为 $(-\infty, 2)$。

**验证：** $f(0) = 0$，$f(1) = 3$，$f(2) = 4$（递增）；$f(3) = 3$（递减）。$x=2$ 处取最大值 ✓
