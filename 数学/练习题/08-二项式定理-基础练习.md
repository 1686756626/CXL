### 第1题
**难度：** ⭐（基础）

**题目：** 写出 (a + b)⁴ 的展开式。

**答案：** a⁴ + 4a³b + 6a²b² + 4ab³ + b⁴

**解析：** 查杨辉三角第4行系数：1, 4, 6, 4, 1

逐项写出：
- T₁ = C(4,0)·a⁴ = a⁴
- T₂ = C(4,1)·a³·b = 4a³b
- T₃ = C(4,2)·a²·b² = 6a²b²
- T₄ = C(4,3)·a·b³ = 4ab³
- T₅ = C(4,4)·b⁴ = b⁴

---

### 第2题
**难度：** ⭐（基础）

**题目：** 计算 C(6,2) 和 C(6,4)。

**答案：** C(6,2) = 15，C(6,4) = 15

**解析：**
- C(6,2) = 6! / (2! × 4!) = (6×5) / (2×1) = 15
- C(6,4) = 6! / (4! × 2!) = (6×5) / (2×1) = 15

验证对称性：C(6,2) = C(6,6-2) = C(6,4) ✅

---

### 第3题
**难度：** ⭐（基础）

**题目：** (x + 1)⁵ 展开式中，x³ 的系数是多少？

**答案：** 10

**解析：**
通项：Tₖ₊₁ = C(5,k) · x⁵⁻ᵏ · 1ᵏ = C(5,k) · x⁵⁻ᵏ

令 5 - k = 3 → k = 2

x³ 的系数 = C(5,2) = 10

---

### 第4题
**难度：** ⭐（基础）

**题目：** (x + 1)⁵ 展开式中，所有项的系数之和是多少？

**答案：** 32

**解析：** 技巧：**令 x = 1**

系数之和 = (1 + 1)⁵ = 2⁵ = **32**

---

### 第5题
**难度：** ⭐（基础）

**题目：** (x - 1/x)⁴ 展开式中的常数项是多少？

**答案：** 6

**解析：**
通项：Tₖ₊₁ = C(4,k) · x⁴⁻ᵏ · (-1/x)ᵏ = C(4,k) · (-1)ᵏ · x⁴⁻²ᵏ

常数项：令 4 - 2k = 0 → k = 2

常数项 = C(4,2) · (-1)² = 6 × 1 = **6**

---

### 第6题
**难度：** ⭐⭐（中等）

**题目：** (2x + 1)⁴ 展开式中，x² 的系数是多少？

**答案：** 24

**解析：**
通项：Tₖ₊₁ = C(4,k) · (2x)⁴⁻ᵏ · 1ᵏ = C(4,k) · 2⁴⁻ᵏ · x⁴⁻ᵏ

令 4 - k = 2 → k = 2

x² 的系数 = C(4,2) · 2² = 6 × 4 = **24**

---

### 第7题
**难度：** ⭐⭐（中等）

**题目：** (x - 2)⁵ 展开式中，x² 的系数是多少？

**答案：** -80

**解析：**
通项：Tₖ₊₁ = C(5,k) · x⁵⁻ᵏ · (-2)ᵏ

令 5 - k = 2 → k = 3

x² 的系数 = C(5,3) · (-2)³ = 10 × (-8) = **-80**

**注意**：(-2)³ = -8，k 是奇数时结果为负。

---

### 第8题
**难度：** ⭐⭐（中等）

**题目：** (x² - 2x)⁴ 展开式中，x⁶ 的系数是多少？

**答案：** 24

**解析：** 这就是一模Q12！

通项：Tₖ₊₁ = C(4,k) · (x²)⁴⁻ᵏ · (-2x)ᵏ = C(4,k) · (-2)ᵏ · x⁸⁻ᵏ

令 8 - k = 6 → k = 2

系数 = C(4,2) · (-2)² = 6 × 4 = **24**

### 第9题
**难度：** ⭐⭐⭐（稍有挑战）

**题目：** (x + 1/x)⁶ 展开式中的常数项是多少？展开式中一共有几项？

**答案：** 常数项 = 20，共7项

**解析：**
通项：Tₖ₊₁ = C(6,k) · x⁶⁻ᵏ · (1/x)ᵏ = C(6,k) · x⁶⁻²ᵏ

常数项：令 6 - 2k = 0 → k = 3
常数项 = C(6,3) = 20

展开式项数：k 从 0 到 6，共 **7** 项。

### 第10题
**难度：** ⭐⭐（中等）

**题目：** 求 (1 + x + x²)⁴ 展开式中 x⁵ 的系数。

**答案：** 16

**解析：**

**方法一：利用 (1+x+x²)² 的平方**

先算 (1+x+x²)²：
```
(1+x+x²)² = 1 + 2x + 3x² + 2x³ + x⁴
```
（验证：(1+x+x²)(1+x+x²) = 1+x+x²+x+x²+x³+x²+x³+x⁴ = 1+2x+3x²+2x³+x⁴ ✅）

再算 (1+2x+3x²+2x³+x⁴)² 中 x⁵ 的系数：
需要找到指数和为 5 的项对 (项i, 项j)：

| 项i 的指数 | 项j 的指数 | 系数乘积 |
|-----------|-----------|---------|
| 1         | 4         | 2 × 1 = 2 |
| 2         | 3         | 3 × 2 = 6 |
| 3         | 2         | 2 × 3 = 6 |
| 4         | 1         | 1 × 2 = 2 |

系数之和 = 2 + 6 + 6 + 2 = **16**

**方法二：多项式定理**

(1+x+x²)⁴ 中 x⁵ 的系数 = Σ C(4; a,b,c)，其中 a+b+c=4 且 b+2c=5。

可能情况：
- c=1, b=3, a=0：C(4;0,3,1) = 4!/(0!3!1!) = **4**
- c=2, b=1, a=1：C(4;1,1,2) = 4!/(1!1!2!) = **12**

系数之和 = 4 + 12 = **16** ✅

### 第11题
**难度：** ⭐⭐⭐（挑战）

**题目：** 已知 (1 + x)¹⁰ = a₀ + a₁x + a₂x² + ... + a₁₀x¹⁰，求 a₁ + a₂ + ... + a₉ 的值。

**答案：** 1022

**解析：**

**核心技巧：令 x = 1**

令 x = 1 代入等式：
- (1 + 1)¹⁰ = a₀ + a₁ + a₂ + ... + a₁₀
- 2¹⁰ = a₀ + a₁ + a₂ + ... + a₁₀
- **1024 = 所有系数之和**

**去掉首尾两项：**
- a₀ = C(10,0) = 1（常数项，x=0 时即得）
- a₁₀ = C(10,10) = 1（最高次项系数）

所以：
a₁ + a₂ + ... + a₉ = 1024 - a₀ - a₁₀ = 1024 - 1 - 1 = **1022**

**验证：** 也可以用"令 x=1 减去令 x=0 再减去 x¹⁰ 的系数"的思路，结果一致 ✅

**💡 本题关键：** 展开式中所有系数之和 = 令 x=1 时的值。这个技巧在二项式题中非常常用！

## 复习题（交叉练习）

### 复1
**难度：** ⭐ 基础（5分）
**复习知识点：** 01-集合

**题目：** 已知全集 U = R，集合 A = {x | x² - 5x + 6 ≤ 0}，求 A 在 R 中的补集 ∁ᵤA。

**答案：** (-∞, 2) ∪ (3, +∞)

**解析：**
复习知识点：01-集合

**第一步：化简集合 A**

x² - 5x + 6 = (x - 2)(x - 3)

二次函数开口向上，(x-2)(x-3) ≤ 0 → **A = [2, 3]**

**第二步：求补集**

∁ᵤA = R - A = **(-∞, 2) ∪ (3, +∞)**

注意：A 包含端点 2 和 3（因为 ≤），所以补集不包含 2 和 3（用开区间）。

**答案：(-∞, 2) ∪ (3, +∞)**

---

### 复2
**难度：** ⭐ 基础（5分）
**复习知识点：** 02-复数

**题目：** 已知 z₁ = 1 + 2i，z₂ = 2 - i，求 z₁ + z₂ 的模 |z₁ + z₂|。

**答案：** √10

**解析：**
复习知识点：02-复数

**第一步：复数加法**——实部加实部，虚部加虚部
```
z₁ + z₂ = (1 + 2i) + (2 - i) = 3 + i
```

**第二步：求模**
```
|3 + i| = √(3² + 1²) = √(9 + 1) = √10
```

**答案：√10**

---

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 04-三角函数

**题目：** 求值：sin²75° - cos²75°

**答案：** √3/2

**解析：**
复习知识点：04-三角函数

识别出这是二倍角余弦公式的变形：
cos 2α = cos²α - sin²α

所以 sin²α - cos²α = -cos 2α

```
sin²75° - cos²75° = -cos(2 × 75°) = -cos 150°
```

cos 150° = cos(180° - 30°) = -cos 30° = -√3/2

因此：
```
sin²75° - cos²75° = -(-√3/2) = √3/2
```

**验证：** sin 75° ≈ 0.9659，cos 75° ≈ 0.2588
sin²75° - cos²75° ≈ 0.9330 - 0.0670 = 0.8660 ≈ √3/2 ≈ 0.8660 ✓

**答案：√3/2**
