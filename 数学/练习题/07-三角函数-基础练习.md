### 第1题
**难度：** ⭐（基础）

**题目：** 已知 sin x = 3/5，且 x 是第二象限角，求 cos x。

**答案：** cos x = -4/5

**解析：** 用平方关系：sin²x + cos²x = 1
- cos²x = 1 - (3/5)² = 1 - 9/25 = 16/25
- cos x = ±4/5
- x 在第二象限，第二象限 cos 是**负**的
- 所以 cos x = **-4/5**

---

### 第2题
**难度：** ⭐（基础）

**题目：** 已知 sin x = 3/5，且 x 是第二象限角，求 tan x。

**答案：** tan x = -3/4

**解析：** 利用商数关系：tan x = sin x / cos x
- 由第1题：cos x = -4/5
- tan x = (3/5) / (-4/5) = 3/5 × (-5/4) = **-3/4**

**注意**：第二象限 tan 也是负的（sin正、cos负），结果符号正确 ✅

### 第3题
**难度：** ⭐（基础）

**题目：** 求 sin(π + α) 的值（用 sin α 表示）。

**答案：** -sin α

**解析：** 用诱导公式口诀"奇变偶不变，符号看象限"：
- π = π/2 × 2，是**偶数倍** → 函数名**不变**，还是 sin
- 把 α 当锐角，π + α 在**第三象限**，第三象限 sin 是**负**的
- 所以 sin(π + α) = **-sin α**

---

### 第4题
**难度：** ⭐（基础）

**题目：** 求 cos(π/2 + α) 的值（用 sin α 表示）。

**答案：** -sin α

**解析：** 用诱导公式口诀：
- π/2 = π/2 × 1，是**奇数倍** → 函数名**变化**，cos 变 sin
- 把 α 当锐角，π/2 + α 在**第二象限**，第二象限 cos 是**负**的
- 所以 cos(π/2 + α) = **-sin α**

### 第6题
**难度：** ⭐⭐（中等）

**题目：** 已知 sin x = 4/5，x 是第二象限角，求 sin 2x。

**答案：** sin 2x = -24/25

**解析：**
第一步：求 cos x
- cos²x = 1 - (4/5)² = 1 - 16/25 = 9/25
- x 是第二象限，cos x < 0 → cos x = -3/5

第二步：用二倍角公式
- sin 2x = 2 sin x cos x = 2 × (4/5) × (-3/5) = -24/25

### 第7题
**难度：** ⭐⭐（中等）

**题目：** 将 y = sin x 的图像向右平移 π/3 个单位，得到的函数解析式是（ ）

A. y = sin(x + π/3)
B. y = sin(x - π/3)
C. y = sin x + π/3
D. y = sin x - π/3

**答案：** B

**解析：** 平移规律：**右减左加**
- 向右平移 π/3 → 把 x 替换成 (x - π/3)
- 答案：y = sin(x - π/3)

排除法：C 和 D 是上下平移（加在函数值上），不是左右平移 ❌
A 是向左平移 ❌

### 第8题
**难度：** ⭐⭐（中等）

**题目：** 已知 tan x = 2，求 sin 2x。

**答案：** sin 2x = 4/5

**解析：**
方法一：利用 tan x = 2 构造直角三角形
- 设对边 = 2，邻边 = 1 → 斜边 = √(4+1) = √5
- sin x = 2/√5，cos x = 1/√5

但题目没说 x 在哪个象限，如果 x 在第一象限（tan正）：
- sin 2x = 2 × (2/√5) × (1/√5) = 4/5

方法二：用万能公式
- sin 2x = 2tan x / (1 + tan²x) = 2×2 / (1+4) = 4/5

**答案：4/5**

⚠️ 注意：如果 tan x = 2 且 x 在第三象限，sin x 和 cos x 都是负的，sin 2x = 2 × (-2/√5) × (-1/√5) = 4/5，结果一样！

### 第10题
**难度：** ⭐⭐⭐（稍有挑战）

**题目：** 已知 sin α = 5/13，α 是第二象限角，求 cos(π/2 - α) 的值。

**答案：** 5/13

**解析：**
第一步：求 cos α
- cos²α = 1 - (5/13)² = 1 - 25/169 = 144/169
- α 是第二象限，cos α < 0 → cos α = -12/13

第二步：求 cos(π/2 - α)
- π/2 = π/2 × 1，奇数倍 → 函数名变化，cos 变 sin
- α 是锐角时，π/2 - α 在第一象限，cos 是正的
- cos(π/2 - α) = **sin α = 5/13**

**捷径：** cos(π/2 - α) = sin α 是一个恒等式，直接用！不需要求 cos α。

### 第11题
**难度：** ⭐⭐（中等）

**题目：** 求值：cos²15° - sin²15°

**答案：** √3/2

**解析：** 直接用二倍角的余弦公式：
- cos²α - sin²α = cos 2α

所以 cos²15° - sin²15° = cos(2 × 15°) = **cos 30° = √3/2**

**💡 这个公式是二倍角公式的变体之一，记住三种形式：**
- cos2α = cos²α - sin²α
- cos2α = 2cos²α - 1
- cos2α = 1 - 2sin²α

### 第12题
**难度：** ⭐⭐（中等）

**题目：** 求函数 y = sin(2x + π/3) 的单调递增区间，以及在区间 [0, π/2] 上的值域。

**答案：** 单调递增区间 [-5π/12 + kπ, π/12 + kπ]（k ∈ Z）；值域 [-√3/2, 1]

**解析：**

**第一部分：单调递增区间**

sin 函数的单调递增区间为 [-π/2 + 2kπ, π/2 + 2kπ]（k ∈ Z）。

令 2x + π/3 ∈ [-π/2 + 2kπ, π/2 + 2kπ]：
```
-π/2 + 2kπ ≤ 2x + π/3 ≤ π/2 + 2kπ
-π/2 - π/3 + 2kπ ≤ 2x ≤ π/2 - π/3 + 2kπ
-5π/6 + 2kπ ≤ 2x ≤ π/6 + 2kπ
-5π/12 + kπ ≤ x ≤ π/12 + kπ
```

单调递增区间：**[-5π/12 + kπ, π/12 + kπ]（k ∈ Z）**

**第二部分：值域**

当 x ∈ [0, π/2] 时，2x + π/3 ∈ [π/3, π + π/3] = **[π/3, 4π/3]**

看 sin 在 [π/3, 4π/3] 上的取值：
```
sin(π/3) = √3/2      ← 起始值
sin(π/2) = 1          ← 最大值
sin(π) = 0
sin(4π/3) = -√3/2     ← 终止值（最小值）
```

值域为 **[-√3/2, 1]**

### 第13题
**难度：** ⭐⭐⭐（挑战）

**题目：** 已知 sin α - cos α = √2/2，α ∈ (0, π)，求 tan 2α 的值。

**答案：** -√3/3

**解析：**
第一步：求 sin 2α
- (sin α - cos α)² = (√2/2)²
- sin²α - 2sinαcosα + cos²α = 1/2
- 1 - sin 2α = 1/2
- **sin 2α = 1/2**

第二步：确定 2α 所在的象限，求 cos 2α

由 sin 2α = 1/2，2α ∈ (0, 2π)，所以 2α = π/6 或 5π/6。

需要判断 α 到底是 π/12 还是 5π/12：
- α = π/12（第一象限）：sin(π/12) ≈ 0.259，cos(π/12) ≈ 0.966 → sin - cos < 0 ✗
- α = 5π/12（第二象限）：sin(5π/12) ≈ 0.966，cos(5π/12) ≈ 0.259 → sin - cos > 0 ✅

所以 **2α = 5π/6**（第二象限）。

第三步：求 tan 2α
- cos(5π/6) = -√3/2
- tan(5π/6) = sin(5π/6) / cos(5π/6) = (1/2) / (-√3/2) = **-√3/3**

**💡 关键步骤：** 通过 α ∈ (0, π) 和 sin α > cos α 的条件确定 2α 的具体位置，从而确定 cos 2α 的符号。如果不判断象限，cos 2α = ±√3/2，就会得到两个答案。

## 复习题（交叉练习）

### 复1
**难度：** ⭐ 基础（5分）
**复习知识点：** 01-集合

**题目：** 已知集合 A = {x | x² - x - 6 < 0}，B = {x | -1 ≤ x < 4}，求 A ∪ B。

**答案：** (-2, 4)

**解析：**
复习知识点：01-集合

**第一步：化简集合 A**

x² - x - 6 = (x + 2)(x - 3)

二次函数开口向上，(x+2)(x-3) < 0 → **A = (-2, 3)**

**第二步：与集合 B 取并集**

B = [-1, 4)

画数轴：
```
A: -2 ←→ 3          （不包含端点）
B:   -1 ←――→ 4      （包含 -1，不包含 4）
并集: -2 ←――→ 4      （不包含两端）
```

A 从 -2 开始（不包含），B 到 4 结束（不包含）。两者有重叠，并集取最左到最右：**(-2, 4)**

**答案：(-2, 4)**

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 02-复数

**题目：** 已知复数 z = (2 - i)/(1 + i)，求 z 的模 |z|。

**答案：** √10/2

**解析：**
复习知识点：02-复数

复数除法：分子分母同乘分母的共轭复数 (1 - i)。

**第一步：算分子**
```
(2 - i)(1 - i) = 2 - 2i - i + i²
= 2 - 3i + (-1)
= 1 - 3i
```

**第二步：算分母**
```
(1 + i)(1 - i) = 1 - i² = 1 + 1 = 2
```

**第三步：求 z**
```
z = (1 - 3i) / 2 = 1/2 - (3/2)i
```

**第四步：求模**
```
|z| = √((1/2)² + (3/2)²) = √(1/4 + 9/4) = √(10/4) = √10/2
```

**答案：√10/2**
