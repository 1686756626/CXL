## 一、单选题（每题5分，共30分）

### 第1题（⭐ 基础 · 集合）
已知集合 A = {x | −1 < x ≤ 2}，B = {x | 0 ≤ x < 3}，则 A ∩ B = （　　）

A. (0, 2]　　B. [0, 2]　　C. [0, 2)　　D. (0, 2)

### 第2题（⭐⭐ 中等 · 集合）
已知集合 A = {1, 2, 3}，B = {x | x² − 5x + 6 = 0}，则 A ∪ B = （　　）

A. {1, 2}　　B. {1, 2, 3}　　C. {2, 3}　　D. {1, 3}

### 第3题（⭐⭐ 中等 · 指数对数互化）
下列互化正确的是（　　）

A. $\log_3 9 = 2 \Leftrightarrow 2^3 = 9$
B. $2^{-3} = \dfrac{1}{8} \Leftrightarrow \log_2 \dfrac{1}{8} = -3$
C. $\log_2 8 = 3 \Leftrightarrow 3^2 = 8$
D. $5^2 = 25 \Leftrightarrow \log_5 2 = 25$

### 第4题（⭐⭐ 中等 · 化同底）
比较 $2^{30}$ 与 $3^{20}$ 的大小关系（　　）

A. $2^{30} > 3^{20}$　　B. $2^{30} = 3^{20}$　　C. $2^{30} < 3^{20}$　　D. 无法确定

### 第5题（⭐⭐ 中等 · 函数图像平移）
将函数 $y = 2^x$ 的图像向左平移1个单位，再向下平移2个单位，得到的函数解析式为（　　）

A. $y = 2^{x-1} - 2$　　B. $y = 2^{x+1} - 2$　　C. $y = 2^{x-1} + 2$　　D. $y = 2^{x+1} + 2$

### 第6题（⭐⭐ 中等 · 含参不等式）
不等式 $\log_2(x - 1) > 1$ 的解集为（　　）

A. $x > 2$　　B. $x > 3$　　C. $x > 1$　　D. $x < 3$

## 二、多选题（每题6分，共24分。全部选对得6分，部分选对得3分，有错选得0分）

### 第7题（⭐⭐ 中等 · 集合与空集）
下列关于集合的说法中，正确的有（　　）

A. 空集是任何集合的子集
B. 空集是任何非空集合的真子集
C. {0} 是空集
D. 若 A ∩ B = ∅，则 A = ∅ 或 B = ∅

### 第8题（⭐⭐ 中等 · 命题与充要条件）
下列命题中，真命题有（　　）

A. "x > 0" 是 "$x^2 > 0$" 的充分条件
B. "x = y" 是 "$x^2 = y^2$" 的充分条件
C. "$a > 1$" 是 "$2^a > 2$" 的充要条件
D. "x > 1" 是 "$\log_2 x > 0$" 的充要条件

### 第9题（⭐⭐ 中等 · 对数运算与图像）
关于函数 $y = \log_2(x - 1) + 1$，下列说法正确的有（　　）

A. 定义域为 $(1, +\infty)$
B. 图像可由 $y = \log_2 x$ 向右平移1个单位、再向上平移1个单位得到
C. 过定点 $(2, 1)$
D. 在定义域上单调递减

### 第10题（⭐⭐ 中等 · 综合比较）
设 $a = 0.3^2$，$b = 2^{0.3}$，$c = \log_2 0.3$，则下列大小关系正确的有（　　）

A. $c < a$
B. $a < b$
C. $c < 1$
D. $a > 1$
