## 一、基础题（⭐）

### 第1题：判断命题真假

判断下列命题的真假，并简要说明理由：

（1）若 $x > 2$，则 $x > 1$；

（2）若 $x^2 = 4$，则 $x = 2$；

（3）若 $ab = 0$，则 $a = 0$。

---

**答案：**

（1）**真命题。** 由不等式的传递性，$x > 2$ 必然有 $x > 1$。

（2）**假命题。** $x^2 = 4$ 的解为 $x = 2$ 或 $x = -2$，当 $x = -2$ 时 $x^2 = 4$ 但 $x \neq 2$。

（3）**假命题。** $ab = 0$ 意味着 $a = 0$ 或 $b = 0$，当 $a = 1,\ b = 0$ 时 $ab = 0$ 但 $a \neq 0$。

**解析：**

怎么想到的？判断"若p则q"型命题真假，核心方法是**找反例**——如果能找到一个p成立但q不成立的情况，命题就是假的。第（2）题的关键是 $x^2 = 4$ 有两个解，容易漏掉负根。第（3）题是经典的"或"被当成了"且"，$ab = 0$ 是"或"关系（$a = 0$ 或 $b = 0$），不能只考虑一个。

---

### 第2题：写出否命题并判断真假

写出命题"若 $x > 1$，则 $x^2 > 1$"的**否命题**，并判断原命题和否命题的真假。

---

**答案：**

**否命题：** 若 $x \leq 1$，则 $x^2 \leq 1$。

**原命题：真命题。** 当 $x > 1$ 时，$x^2 > x > 1$（因为 $x > 1$ 乘以 $x > 1$，积更大）。

**否命题：假命题。** 取反例 $x = -2$：$x \leq 1$ 成立，但 $x^2 = 4 > 1$，不满足 $x^2 \leq 1$。

**解析：**

怎么想到的？写否命题的步骤：把条件和结论都**否定**。"若p则q"的否命题是"若¬p则¬q"。这里 $x > 1$ 的否定是 $x \leq 1$，$x^2 > 1$ 的否定是 $x^2 \leq 1$。否命题真假用反例验证最方便，$x = -2$ 是经典反例——负数的平方会变大，超出直觉。注意：原命题和否命题不一定同真同假，只有原命题和**逆否命题**才同真同假。

### 第3题：判断充分/必要条件

设 $p: x > 3$，$q: x > 0$。判断 $p$ 是 $q$ 的什么条件。

---

**答案：**

$p$ 是 $q$ 的**充分不必要条件**。

**解析：**

怎么想到的？分两步验证：先看 $p \Rightarrow q$：$x > 3$ 能推出 $x > 0$ ✅（比3大的数一定比0大），所以 $p$ 是 $q$ 的充分条件。再看 $q \Rightarrow p$：$x > 0$ 不能推出 $x > 3$（比如 $x = 1 > 0$ 但 $1 \not> 3$）❌，所以 $p$ 不是 $q$ 的必要条件。

用集合方法验证：$p$ 对应集合 $A = (3, +\infty)$，$q$ 对应集合 $B = (0, +\infty)$。显然 $A \subset B$（$A$ 是 $B$ 的真子集），所以 $p$ 是 $q$ 的充分不必要条件。

## 二、中等题（⭐⭐）

### 第4题：四种命题

已知原命题："若 $a = 0$，则 $ab = 0$"。

（1）写出原命题的逆命题、否命题、逆否命题；

（2）判断这四种命题的真假。

---

**答案：**

| 命题 | 内容 | 真假 |
|------|------|------|
| 原命题 | 若 $a = 0$，则 $ab = 0$ | ✅ 真 |
| 逆命题 | 若 $ab = 0$，则 $a = 0$ | ❌ 假 |
| 否命题 | 若 $a \neq 0$，则 $ab \neq 0$ | ❌ 假 |
| 逆否命题 | 若 $ab \neq 0$，则 $a \neq 0$ | ✅ 真 |

**解析：**

怎么想到的？四种命题的写法是固定的：
- **逆命题**：条件和结论互换 → "若 $ab = 0$，则 $a = 0$"
- **否命题**：条件和结论分别否定 → "若 $a \neq 0$，则 $ab \neq 0$"
- **逆否命题**：条件结论互换并分别否定 → "若 $ab \neq 0$，则 $a \neq 0$"

真假判断：
- 原命题：$a = 0$ 乘以任何数都是 0 ✅
- 逆命题：$ab = 0$ 时，可能 $a \neq 0$ 而 $b = 0$（如 $a = 3,\ b = 0$）❌
- 否命题：$a \neq 0$ 时，可能 $b = 0$ 使 $ab = 0$（如 $a = 3,\ b = 0$）❌
- 逆否命题：$ab \neq 0$ 说明 $a \neq 0$ 且 $b \neq 0$，所以 $a \neq 0$ ✅

验证规律：原命题和逆否命题同真同假 ✅；逆命题和否命题互为逆否命题，也同真同假 ✅。

### 第5题：用集合方法判断充要条件

设 $p: x^2 - 5x + 6 < 0$，$q: 1 < x < 4$。用集合方法判断 $p$ 是 $q$ 的什么条件。

---

**答案：**

$p$ 是 $q$ 的**充分不必要条件**。

**解析：**

怎么想到的？题目要求用集合方法，所以先把 $p$ 和 $q$ 分别转化为集合，再比较包含关系。

**第一步：解 $p$ 对应的集合。**

$x^2 - 5x + 6 < 0$

用求根公式：$a = 1,\ b = -5,\ c = 6$

$$\Delta = (-5)^2 - 4 \times 1 \times 6 = 25 - 24 = 1$$

$$x = \frac{5 \pm \sqrt{1}}{2 \times 1} = \frac{5 \pm 1}{2}$$

$x_1 = \dfrac{5 - 1}{2} = 2$，$x_2 = \dfrac{5 + 1}{2} = 3$

因为 $a = 1 > 0$，抛物线开口向上，$x^2 - 5x + 6 < 0$ 的解在两根之间：$A = (2,\ 3)$

**第二步：$q$ 对应的集合 $B = (1,\ 4)$。**

**第三步：比较包含关系。**

$A = (2, 3) \subset B = (1, 4)$ ✅（$(2,3)$ 完全在 $(1,4)$ 内部）

验证：任取 $x \in (2,3)$，如 $x = 2.5$，$2.5 \in (1,4)$ ✅。

反过来 $B \not\subset A$，比如 $x = 1.5 \in (1,4)$ 但 $1.5 \notin (2,3)$。

所以 $A \subset B$，即 $p$ 是 $q$ 的**充分不必要条件**。

### 第6题：不等式条件判断

设 $p: |x - 2| < 3$，$q: 0 < x < 5$。用集合方法判断 $p$ 是 $q$ 的什么条件。

---

**答案：**

$p$ 是 $q$ 的**必要不充分条件**。

**解析：**

怎么想到的？绝对值不等式解起来有固定套路，先解出 $p$ 的集合，再和 $q$ 的集合比较。

**第一步：解 $p$ 对应的集合。**

$$|x - 2| < 3 \Leftrightarrow -3 < x - 2 < 3 \Leftrightarrow -1 < x < 5$$

所以 $A = (-1,\ 5)$

**第二步：$q$ 对应的集合 $B = (0,\ 5)$。**

**第三步：比较包含关系。**

$A \subset B$？$(-1, 5) \subset (0, 5)$？不是！$x = -0.5 \in (-1,5)$ 但 $-0.5 \notin (0,5)$ ❌

$B \subset A$？$(0, 5) \subset (-1, 5)$？是的！$(0,5)$ 完全在 $(-1,5)$ 内部 ✅

验证：任取 $x \in (0,5)$，如 $x = 3$，$-1 < 3 < 5$ ✅。

所以 $B \subset A$，即 $q \Rightarrow p$ 成立但 $p \Rightarrow q$ 不成立。

$p$ 是 $q$ 的**必要不充分条件**。

### 第7题：逆否命题的等价性应用

判断命题"若 $x^2 + x - 6 = 0$，则 $x = 2$"的真假。

要求：通过写出该命题的逆否命题来判断原命题的真假。

---

**答案：**

**原命题为假命题。**

**解析：**

怎么想到的？题目要求用逆否命题判断，那就先解方程知道有几个根，再构造逆否命题并验证。

**先解方程确定根：** $a = 1,\ b = 1,\ c = -6$

$$\Delta = 1^2 - 4 \times 1 \times (-6) = 1 + 24 = 25$$

$$x = \frac{-1 \pm \sqrt{25}}{2} = \frac{-1 \pm 5}{2}$$

$x_1 = \dfrac{-1 + 5}{2} = 2$，$x_2 = \dfrac{-1 - 5}{2} = -3$

所以 $x^2 + x - 6 = 0$ 的解为 $x = 2$ 或 $x = -3$。

**写出逆否命题：** "若 $x \neq 2$，则 $x^2 + x - 6 \neq 0$"

**判断逆否命题真假：** 取 $x = -3$，$x \neq 2$ 成立，但 $(-3)^2 + (-3) - 6 = 9 - 3 - 6 = 0$，逆否命题为假。

因为原命题和逆否命题同真同假，所以**原命题为假命题**。

> 直接验证：$x = -3$ 时 $x^2 + x - 6 = 0$ 成立但 $x \neq 2$，原命题确实为假 ✅。

### 第8题：多选题（充分/必要条件判断）

设 $p: x^2 - 1 = 0$。对以下每个 $q$，判断 $p$ 是 $q$ 的什么条件：

（A）$q: x = 1$

（B）$q: x^2 - 1 \leq 0$

（C）$q: |x| = 1$

（D）$q: x > 0$

---

**答案：**

（A）**必要不充分条件**

（B）**充分不必要条件**

（C）**充要条件**

（D）**既不充分也不必要条件**

**解析：**

怎么想到的？用集合方法统一处理，把 $p$ 和每个 $q$ 都化为集合来比较，一目了然。

**$p$ 对应的集合：** $x^2 - 1 = 0$，$x = \pm 1$，$A = \{-1,\ 1\}$

**（A）$q: x = 1$，$B_1 = \{1\}$**

$B_1 \subset A$（$\{1\} \subset \{-1, 1\}$），$p$ 的集合包含 $q$ 的集合 → $p$ 是 $q$ 的**必要不充分条件**。

**（B）$q: x^2 - 1 \leq 0$，即 $-1 \leq x \leq 1$，$B_2 = [-1,\ 1]$**

$A = \{-1, 1\} \subset B_2 = [-1, 1]$ ✅。$p \Rightarrow q$ 成立（$x = \pm 1$ 时 $x^2 - 1 = 0 \leq 0$），$q \Rightarrow p$ 不成立（$x = 0$ 时 $x^2 - 1 = -1 \leq 0$ 但 $x^2 - 1 \neq 0$）。**充分不必要条件**。

**（C）$q: |x| = 1$，即 $x = \pm 1$，$B_3 = \{-1,\ 1\}$**

$A = B_3$，集合相等 → **充要条件**。

**（D）$q: x > 0$，$B_4 = (0, +\infty)$**

$A \not\subset B_4$（$-1 \notin (0,+\infty)$），$B_4 \not\subset A$（$2 \notin \{-1,1\}$），两者互不包含 → **既不充分也不必要条件**。

## 三、挑战题（⭐⭐⭐）

### 第9题：含参的充要条件判断

已知命题 $p$：$ax^2 + x + 1 > 0$ 对任意实数 $x$ 恒成立。求 $a$ 的取值范围。

---

**答案：**

$$a > \dfrac{1}{4}$$

**解析：**

怎么想到的？二次不等式恒成立的条件有两个：①开口向上（$a > 0$）；②与 $x$ 轴没有交点（$\Delta < 0$）。两个条件必须同时满足。如果 $a = 0$ 退化为一次不等式，也需要单独检查。这类问题的解题框架是固定的——先讨论 $a = 0$，再讨论 $a \neq 0$ 时开口方向与判别式。

**当 $a = 0$ 时：** 不等式变为 $x + 1 > 0$，即 $x > -1$，不是对所有实数恒成立（如 $x = -2$ 时不等式不成立）。所以 $a = 0$ 不满足。

**当 $a > 0$ 时：** 抛物线开口向上，恒成立需要 $\Delta < 0$。

$$\Delta = 1^2 - 4 \times a \times 1 = 1 - 4a$$

$$\Delta < 0 \Leftrightarrow 1 - 4a < 0 \Leftrightarrow a > \frac{1}{4}$$

所以当 $a > \dfrac{1}{4}$ 时，抛物线开口向上且在 $x$ 轴上方，对所有 $x$ 都有 $ax^2 + x + 1 > 0$ ✅

**当 $a < 0$ 时：** 抛物线开口向下，当 $|x|$ 很大时 $ax^2 + x + 1 \to -\infty$，不可能恒成立。

**综上：** $a > \dfrac{1}{4}$。

**验算：** 取 $a = 1 > \dfrac{1}{4}$，$f(x) = x^2 + x + 1$，$\Delta = 1 - 4 = -3 < 0$，恒成立 ✅。取 $a = 0$，退化为 $x + 1 > 0$，不恒成立 ✅。取 $a = -1$，$f(x) = -x^2 + x + 1$，$f(10) = -89 < 0$，不恒成立 ✅。

### 第10题：综合题

设 $p$：方程 $x^2 - 3x + 2 = 0$ 成立，$q$：方程 $x^2 - mx + m - 1 = 0$ 成立。

已知 $p \Rightarrow q$，求实数 $m$ 的所有可能取值。

---

**答案：**

$m = 3$

**解析：**

怎么想到的？"$p \Rightarrow q$"意思是：$p$ 成立时 $q$ 一定成立。$p$ 是方程 $x^2 - 3x + 2 = 0$，解出根后，这些根必须也满足 $q$ 的方程。换句话说，$p$ 的根必须是 $q$ 的根。

**第一步：解 $p$。**

用求根公式：$a = 1,\ b = -3,\ c = 2$

$$\Delta = (-3)^2 - 4 \times 1 \times 2 = 9 - 8 = 1$$

$$x = \frac{3 \pm \sqrt{1}}{2} = \frac{3 \pm 1}{2}$$

$x_1 = \dfrac{3 - 1}{2} = 1$，$x_2 = \dfrac{3 + 1}{2} = 2$

所以 $p$ 的根为 $x = 1$ 和 $x = 2$。

**第二步：将 $p$ 的根代入 $q$。**

$q$ 的方程为 $x^2 - mx + m - 1 = 0$，即 $f(x) = x^2 - mx + (m - 1) = 0$。

$p \Rightarrow q$ 要求 $\{1, 2\}$ 都在 $q$ 的解集中。

代入 $x = 1$：$1 - m + m - 1 = 0$，恒成立！这说明 $x = 1$ 是 $f(x) = 0$ 的根，对任何 $m$ 都成立（因为 $f(1) = 1^2 - m \cdot 1 + m - 1 = 0$）。

代入 $x = 2$：$4 - 2m + m - 1 = 0$，即 $3 - m = 0$，$m = 3$。

**所以 $m = 3$ 时，$x = 1$ 和 $x = 2$ 都满足 $q$ 的方程。**

**验算：** 当 $m = 3$ 时，$q$：$x^2 - 3x + 2 = 0$，用求根公式得 $x = 1, 2$，与 $p$ 完全相同，$p \Rightarrow q$ 成立 ✅。

## 四、交叉复习

### 复1：集合综合题（中等）

已知集合 $A = \{x \mid x^2 - 5x + 6 \leq 0\}$，$B = \{x \mid x^2 - 2x - 3 \leq 0\}$。

（1）求 $A \cap B$ 和 $A \cup B$；

（2）若 $C = \{x \mid x < a\}$，且 $A \cap C \neq \emptyset$，求 $a$ 的取值范围。

---

**答案：**

（1）$A \cap B = [2,\ 3]$，$A \cup B = [-1,\ 3]$

（2）$a > 2$

**解析：**

怎么想到的？这是集合运算的复习。先解两个二次不等式得到集合，再用数轴求交和并。第（2）小题要求交集非空，画数轴最直观——$A$ 的最小元素是2，要让 $A$ 和 $C$ 有交集，$C$ 的上界 $a$ 必须大于 $A$ 的最小元素。

**第一步：解集合 $A$。**

$x^2 - 5x + 6 \leq 0$，用求根公式：$a = 1,\ b = -5,\ c = 6$

$$\Delta = 25 - 24 = 1,\quad x = \frac{5 \pm 1}{2}$$

$x_1 = 2,\ x_2 = 3$，开口向上，$A = [2,\ 3]$

**第二步：解集合 $B$。**

$x^2 - 2x - 3 \leq 0$，用求根公式：$a = 1,\ b = -2,\ c = -3$

$$\Delta = 4 + 12 = 16,\quad x = \frac{2 \pm 4}{2}$$

$x_1 = -1,\ x_2 = 3$，开口向上，$B = [-1,\ 3]$

**第三步：求 $A \cap B$ 和 $A \cup B$。**

$A = [2, 3]$，$B = [-1, 3]$

$A \cap B = [2,\ 3]$（两个区间的重叠部分）

$A \cup B = [-1,\ 3]$（两个区间的合并，从 $B$ 的左端到共同的右端）

**第四步：求 $a$ 的范围。**

$C = \{x \mid x < a\}$，$A = [2, 3]$。

$A \cap C \neq \emptyset$ 意味着 $[2, 3]$ 中至少有一个元素 $< a$，即 $a > 2$。

**验算：** 取 $a = 2.5$，$C = (-\infty, 2.5)$，$A \cap C = [2, 2.5) \neq \emptyset$ ✅。取 $a = 2$，$C = (-\infty, 2)$，$A \cap C = \emptyset$（$A$ 的最小元素2不在 $C$ 中）❌。取 $a = 1$，$C = (-\infty, 1)$，$A \cap C = \emptyset$ ❌。
