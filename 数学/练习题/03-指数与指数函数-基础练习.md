# 03-指数与指数函数 · 基础练习

> 10道题 + 1道复习题 | 基础3道、中等5道、挑战2道

---

### 第1题
**难度：** ⭐ 基础
**题目：** 计算 3² × 3⁴ = ______
**答案：** 3⁶ = 729
**解析：** 同底数相乘，指数相加：3² × 3⁴ = 3^(2+4) = 3⁶ = 729。怎么想到的？看到"同底数×同底数"，立刻用公式 aᵐ·aⁿ = aᵐ⁺ⁿ，不要一个个算。

---

### 第2题
**难度：** ⭐ 基础
**题目：** 计算 ⁴√(16) = ______
**答案：** 2
**解析：** ⁴√(16) = 16^(1/4) = (2⁴)^(1/4) = 2。怎么想到的？看到根号，先想"16是哪个数的4次方"，2⁴=16，所以答案是2。

---

### 第3题
**难度：** ⭐ 基础
**题目：** 函数 y = (1/3)ˣ 是递增还是递减的？
**答案：** 单调递减
**解析：** 底数 a = 1/3，满足 0 < a < 1，所以指数函数单调递减。怎么想到的？记住口诀：底大于1往上涨，底小于1往下降。

---

### 第4题
**难度：** ⭐⭐ 中等
**题目：** 化简 (2³ × 4²) ÷ 2⁵ = ______
**答案：** 4
**解析：** 第一步统一底数：4 = 2²，所以 4² = (2²)² = 2⁴。代入原式：(2³ × 2⁴) ÷ 2⁵ = 2^(3+4) ÷ 2⁵ = 2⁷ ÷ 2⁵ = 2² = 4。怎么想到的？看到不同底数（2和4），先统一底数为2，再用指数法则。

---

### 第5题
**难度：** ⭐⭐ 中等
**题目：** 比较 (1/2)^0.3 与 (1/2)^1.5 的大小，用 > 或 < 连接。
**答案：** (1/2)^0.3 > (1/2)^1.5
**解析：** 底数相同都是 1/2，且 0 < 1/2 < 1，递减函数。递减函数中，指数越大值越小。0.3 < 1.5，所以 (1/2)^0.3 > (1/2)^1.5。怎么想到的？先看底数判断递增递减，再比较指数大小。

---

### 第6题
**难度：** ⭐⭐ 中等
**题目：** 解方程 3^(2x-1) = 27，求 x。
**答案：** x = 2
**解析：** 27 = 3³，所以 3^(2x-1) = 3³，因此 2x - 1 = 3，解得 2x = 4，x = 2。怎么想到的？看到方程两边都是3的幂，先把两边统一成同底数，然后让指数相等。

### 第7题
**难度：** ⭐⭐ 中等
**题目：** 解不等式 (1/3)^x < 9，求 x 的范围。
**答案：** x > -2
**解析：** 先把9化成以1/3为底的幂：9 = 3² = (3⁻¹)⁻² = (1/3)^(-2)。所以 (1/3)^x < (1/3)^(-2)。底数 1/3 < 1，递减函数，不等号方向反转：x > -2。怎么想到的？关键步骤是把两边统一成同底的幂，再根据底数<1反转不等号。

### 第8题
**难度：** ⭐⭐ 中等
**题目：** 求函数 y = 2^(x-1) + 1 的值域。
**答案：** y > 1
**解析：** 对任意实数 x，2^(x-1) > 0（指数函数的值域恒正），所以 y = 2^(x-1) + 1 > 0 + 1 = 1。怎么想到的？指数函数 aˣ > 0，再加上常数整体上移。

### 第9题
**难度：** ⭐⭐⭐ 挑战
**题目：** 已知 aˣ > 3，求 x 的范围（分 a > 1 和 0 < a < 1 两种情况讨论）。
**答案：** 当 a > 1 时，x > log_a 3；当 0 < a < 1 时，x < log_a 3。
**解析：** 把 3 写成 a 的幂：3 = a^(log_a 3)。不等式变为 aˣ > a^(log_a 3)。
- 当 a > 1（递增）：x > log_a 3
- 当 0 < a < 1（递减）：x < log_a 3
怎么想到的？底数含参数时必须分类讨论单调性，这是固定套路。

### 第10题
**难度：** ⭐⭐⭐ 挑战
**题目：** 将函数 y = 2ˣ 的图像向右平移 1 个单位，再向上平移 2 个单位，得到的函数解析式是什么？
**答案：** y = 2^(x-1) + 2
**解析：** 向右平移1个单位：x 替换为 x-1，得 y = 2^(x-1)。再向上平移2个单位：整体加2，得 y = 2^(x-1) + 2。怎么想到的？记住"左加右减"（x的位置），"上加下减"（y的整体）。

### 复1（复习题）
**难度：** ⭐⭐ 中等
**题目：** 已知集合 A = {x | 1 < x < 4}，B = {x | x² - 5x + 6 ≤ 0}，求 A ∩ B。
**答案：** A ∩ B = {x | 2 ≤ x ≤ 3}
**解析：** 先求 B。x² - 5x + 6 ≤ 0，用求根公式：x = (5 ± √(25-24))/2 = (5 ± 1)/2，x₁ = 2，x₂ = 3。二次函数开口向上，≤0 的部分是 2 ≤ x ≤ 3，即 B = [2, 3]。A = (1, 4)。A ∩ B = [2, 3]。怎么想到的？分别求出两个集合，取公共部分。二次不等式用求根公式是基本功。
