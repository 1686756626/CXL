# 解三角形 · 解析版

> **专题说明：** 解三角形有两个核心工具：正弦定理 $\dfrac{a}{\sin A}=\dfrac{b}{\sin B}=\dfrac{c}{\sin C}=2R$ 和余弦定理 $a^2=b^2+c^2-2bc\cos A$。面积公式 $S=\dfrac{1}{2}ab\sin C$ 知道两边和夹角就能用。

---

## 核心公式速查

| 定理 | 公式 | 适用场景 |
|------|------|----------|
| 正弦定理 | $\dfrac{a}{\sin A}=\dfrac{b}{\sin B}=\dfrac{c}{\sin C}=2R$ | 两边一对角、两角一边 |
| 余弦定理 | $a^2=b^2+c^2-2bc\cos A$ | 三边求角、两边夹角求第三边 |
| 余弦变形 | $\cos A=\dfrac{b^2+c^2-a^2}{2bc}$ | 求角的余弦值 |
| 面积公式 | $S=\dfrac{1}{2}ab\sin C$ | 知道两边和夹角求面积 |

---

## 答案与详细解析

---

### 第1题 · 答案：A

**解析：**

已知两边 $a=4$，$b=6$ 和夹角 $C=60°$，用余弦定理求 $c$：

$c^2 = a^2+b^2-2ab\cos C = 16+36-2\times4\times6\times\dfrac{1}{2} = 52-24 = 28$

$c = \boxed{2\sqrt7}$

**思路过程：** "已知两边和夹角求第三边"是余弦定理最直接的应用场景。$\cos 60°=\dfrac{1}{2}$ 代入即可。

---

### 第2题 · 答案：B

**解析：**

已知两角 $A=30°$，$B=60°$ 和一边 $a=4$，用正弦定理求 $b$：

$\dfrac{a}{\sin A} = \dfrac{b}{\sin B}$

$b = \dfrac{a\sin B}{\sin A} = \dfrac{4\times\dfrac{\sqrt3}{2}}{\dfrac{1}{2}} = \dfrac{2\sqrt3}{\dfrac{1}{2}} = \boxed{4\sqrt3}$

验算：$C=180°-30°-60°=90°$，$c = \dfrac{a}{\sin A}=\dfrac{4}{\dfrac{1}{2}}=8$。验证勾股定理：$4^2+(4\sqrt3)^2=16+48=64=8^2$ ✓

**思路过程：** "两角一边"用正弦定理。$b=a\sin B/\sin A$，特殊角的三角函数值要背熟：$\sin 30°=\dfrac{1}{2}$，$\sin 60°=\dfrac{\sqrt3}{2}$。

---

### 第3题 · 答案：A

**解析：**

面积公式 $S=\dfrac{1}{2}ab\sin C$，直接代入：

$S = \dfrac{1}{2}\times 2\sqrt3 \times 6 \times \sin 30° = \dfrac{1}{2}\times 2\sqrt3 \times 6 \times \dfrac{1}{2} = \boxed{3\sqrt3}$

**思路过程：** 知道两边和夹角直接用面积公式。$\sin 30°=\dfrac{1}{2}$，计算时不要弄混正弦和余弦。

---

### 第4题 · 答案：C

**解析：**

最大边 $c=7$ 对应最大角 $C$，用余弦定理的变形：

$\cos C = \dfrac{a^2+b^2-c^2}{2ab} = \dfrac{9+25-49}{2\times 3\times 5} = \dfrac{-15}{30} = -\dfrac{1}{2}$

$C = \boxed{120°}$

**思路过程：** 三角形中"大边对大角"，最大角用最大边的余弦定理求。$\cos C=-\dfrac{1}{2}$ 对应 $C=120°$（不是 $60°$！注意负号表示钝角）。

---

### 第5题 · 答案：B

**解析：**

已知两边 $a=\sqrt3$，$b=1$ 和其中一边的对角 $B=30°$，用正弦定理：

$\dfrac{a}{\sin A}=\dfrac{b}{\sin B}$

$\sin A = \dfrac{a\sin B}{b} = \dfrac{\sqrt3\times\dfrac{1}{2}}{1} = \dfrac{\sqrt3}{2}$

$\sin A = \dfrac{\sqrt3}{2} \Rightarrow A = 60°$ 或 $A = 120°$（两个解）

**需要检验两个解是否都有效：**

- $A=60°$：$C=90°$，$c=\sqrt{3+1}=2$ ✓（三角形合法）
- $A=120°$：$C=30°$，$c^2=3+1-2\sqrt3\cos 120°=4+\sqrt3$，$c=\sqrt{4+\sqrt3}$ ✓（三角形合法，$b=c$ 实际上不是等腰... 算一下 $c$：$c=\sin C\cdot a/\sin A=\sin 30°\cdot\sqrt3/\sin 120°=\dfrac{1}{2}\cdot\sqrt3/(\sqrt3/2)=1$，所以 $b=c=1$，等腰三角形 ✓）

两个解都有效，$A = \boxed{60°\text{ 或 }120°}$

**思路过程：** 这是正弦定理的"两解问题"——已知两边和其中一边的对角（且对角为锐角），可能有两个解。$\sin A=\dfrac{\sqrt3}{2}$ 在 $(0°,180°)$ 内有两个值：$60°$ 和 $120°$，都要验证是否构成合法三角形。

---

### 第6题 · 答案：C

**解析：**

$\cos C = \dfrac{a^2+b^2-c^2}{2ab} = \dfrac{49+25-9}{2\times 7\times 5} = \dfrac{65}{70} = \boxed{\dfrac{13}{14}}$

**思路过程：** 直接套余弦定理变形公式。"已知三边求角"是余弦定理变形的标准用法。注意分子是 $a^2+b^2-c^2$（求角 $C$ 对应的边 $c$ 放在减号后面）。

---

### 第7题 · 答案：C

**解析：**

$a^2+b^2+c^2=ab+bc+ca$

两边乘以 2：$2a^2+2b^2+2c^2=2ab+2bc+2ca$

移项：$(a^2-2ab+b^2)+(b^2-2bc+c^2)+(c^2-2ca+a^2)=0$

即：$(a-b)^2+(b-c)^2+(c-a)^2=0$

三个非负数的和为 0，只能每个都是 0：

$a-b=0$，$b-c=0$，$c-a=0$

所以 $a=b=c$，$\triangle ABC$ 是 $\boxed{\text{等边三角形}}$。

**思路过程：** 看到 $a^2+b^2+c^2=ab+bc+ca$，想到完全平方公式：$(a-b)^2=a^2-2ab+b^2$。两边乘 2 配方，得到三个完全平方式的和为 0，由此推出 $a=b=c$。这是判断等边三角形的经典方法。

---

### 第8题 · 答案：B

**解析：**

设 $b+c=4k$，$c+a=5k$，$a+b=6k$。

三个方程联立：

$a+b=6k$ 和 $b+c=4k$ 相减：$a-c=2k$

$c+a=5k$ 和 $b+c=4k$ 相减：$a-b=k$

$a+b=6k$ 和 $a-b=k$：$2a=7k \Rightarrow a=\dfrac{7k}{2}$，$b=\dfrac{5k}{2}$

$b+c=4k$：$c=4k-\dfrac{5k}{2}=\dfrac{3k}{2}$

$\cos A = \dfrac{b^2+c^2-a^2}{2bc} = \dfrac{\dfrac{25k^2}{4}+\dfrac{9k^2}{4}-\dfrac{49k^2}{4}}{2\times\dfrac{5k}{2}\times\dfrac{3k}{2}} = \dfrac{-\dfrac{15k^2}{4}}{\dfrac{15k^2}{2}} = \dfrac{-15}{4}\times\dfrac{2}{15} = \boxed{-\dfrac{1}{2}}$

**思路过程：** 给出边的比例关系，先设参数 $k$ 表示各边，再用余弦定理求 $\cos A$。注意 $a$ 对应角 $A$，余弦定理中 $a^2$ 的系数是 $-1$。

---

### 第9题 · 答案：B

**解析：** 已知边的关系求角，用余弦定理。

**第一步：** 余弦定理求 $\cos A$

$$\cos A = \dfrac{b^2+c^2-a^2}{2bc}$$

**第二步：** 将 $a^2 = b^2+c^2+bc$ 代入

$$\cos A = \dfrac{b^2+c^2-(b^2+c^2+bc)}{2bc} = \dfrac{-bc}{2bc} = -\dfrac{1}{2}$$

**第三步：** 由 $\cos A = -\dfrac{1}{2}$，得 $A = 120°$

**答案：B**

**思路过程：** 已知边的关系 $a^2=b^2+c^2+bc$ 求角 $A$，直接用余弦定理变形。把 $a^2$ 的表达式代入 $\cos A = (b^2+c^2-a^2)/(2bc)$，化简后得到 $\cos A$ 的值，再求角。

---

### 第10题 · 答案：A

**解析：**

$S=\dfrac{1}{2}ab\sin C=\dfrac{1}{2}ab\sin 60°=\dfrac{\sqrt3}{4}ab$

要使 $S$ 最大，需要 $ab$ 最大。

已知 $a+b=10$，由基本不等式（均值不等式）：

$ab \le \left(\dfrac{a+b}{2}\right)^2 = 25$

当且仅当 $a=b=5$ 时取等号。

$S_{\max} = \dfrac{\sqrt3}{4}\times 25 = \boxed{\dfrac{25\sqrt3}{4}}$

验算：$a=b=5$，$c^2=25+25-50\cos 60°=50-25=25$，$c=5$，等边三角形，合法 ✓

**思路过程：** 面积公式 $S=\frac{1}{2}ab\sin C$ 中 $\sin C$ 是定值，$S$ 取决于 $ab$。在 $a+b=10$ 的条件下，$ab$ 的最大值由均值不等式给出：$ab\le(\frac{a+b}{2})^2=25$。

### 第11题 · 答案：B

**解析：**

$2\sin A\cos B = \sin C$

由正弦定理推论：$\sin C = \sin(180°-A-B) = \sin(A+B) = \sin A\cos B + \cos A\sin B$

代入：$2\sin A\cos B = \sin A\cos B + \cos A\sin B$

$\sin A\cos B = \cos A\sin B$

$\sin A\cos B - \cos A\sin B = 0$

$\sin(A-B) = 0$

在三角形中 $A,B \in (0°,180°)$，所以 $A-B=0$，即 $A=B$。

$\triangle ABC$ 是 $\boxed{\text{等腰三角形}}$（$a=b$）。

**思路过程：** 看到 $\sin A\cos B$ 的形式，想到 $\sin(A\pm B)$ 的展开式。$\sin A\cos B - \cos A\sin B = \sin(A-B)$。这是判断三角形形状的经典手法——化简三角恒等式，利用 $\sin(A-B)=0$ 或 $\cos A=\cos B$ 等。

---

### 第12题 · 答案：C

**解析：**

已知 $a=1$，$b=\sqrt3$，$A=30°$，求 $c$。

先用正弦定理求 $B$：

$\dfrac{a}{\sin A}=\dfrac{b}{\sin B}\Rightarrow \sin B = \dfrac{b\sin A}{a}=\dfrac{\sqrt3\times\dfrac{1}{2}}{1}=\dfrac{\sqrt3}{2}$

$\sin B=\dfrac{\sqrt3}{2}\Rightarrow B=60°$ 或 $B=120°$

**情况一：$B=60°$**

$C=180°-30°-60°=90°$

$c = \dfrac{a}{\sin A}\sin C = \dfrac{1}{\dfrac{1}{2}}\times 1 = 2$

验证：$a=1$，$b=\sqrt3$，$c=2$，$1^2+(\sqrt3)^2=4=2^2$ ✓（直角三角形）

**情况二：$B=120°$**

$C=180°-30°-120°=30°$

$c = \dfrac{a}{\sin A}\sin C = \dfrac{1}{\dfrac{1}{2}}\times\dfrac{1}{2} = 1$

验证：$a=1$，$b=\sqrt3$，$c=1$，$a=c=1$（等腰三角形），$A=C=30°$ ✓

$c = \boxed{1\text{ 或 }2}$

**思路过程：** 正弦定理求角时注意"两解问题"。$\sin B=\dfrac{\sqrt3}{2}$ 在 $(0°,180°)$ 内有两个解。两个解都要验证是否能构成合法三角形。这题两种情况都合法，分别得到 $c=2$（直角三角形）和 $c=1$（等腰三角形）。

**复习知识点：** 09-数列基础-等差求和

**题目：** 在等差数列 $\{a_n\}$ 中，$a_1 = 2$，公差 $d = 3$，求前 $10$ 项和 $S_{10}$。

**解析：**

$S_{10} = 10 \times 2 + \dfrac{10 \times 9}{2} \times 3 = 20 + 135 = 155$

**验证：** $a_{10} = 2 + 9 \times 3 = 29$，$S_{10} = \dfrac{10(2+29)}{2} = 155$ ✓

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 三角函数-图像平移

**题目：** 将函数 $y = \sin\left(x + \dfrac{\pi}{6}\right)$ 的图像向右平移 $\dfrac{\pi}{3}$ 个单位后，得到的函数解析式为（　）

A. $y = \sin\left(x - \dfrac{\pi}{6}\right)$　　B. $y = \sin\left(x + \dfrac{\pi}{2}\right)$　　C. $y = \sin\left(x + \dfrac{\pi}{3}\right)$　　D. $y = \cos x$

**答案：** A

**解析：**

函数图像向右平移 $\dfrac{\pi}{3}$ 个单位，将解析式中 $x$ 替换为 $x - \dfrac{\pi}{3}$：

$y = \sin\left(\left(x - \dfrac{\pi}{3}\right) + \dfrac{\pi}{6}\right) = \sin\left(x - \dfrac{\pi}{6}\right)$

**验证：** 原函数在 $x = -\dfrac{\pi}{6}$ 时 $y = 0$，平移后在 $x = \dfrac{\pi}{6}$ 时 $y = 0$，代入新解析式 $\sin\left(\dfrac{\pi}{6} - \dfrac{\pi}{6}\right) = 0$ ✓

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 13-函数性质-奇偶性

**题目：** 判断函数 $f(x) = x^3 + x$ 的奇偶性。

A. 奇函数　　B. 偶函数　　C. 非奇非偶　　D. 既是奇函数又是偶函数

**答案：** A

**解析：**

定义域为 $\mathbb{R}$，关于原点对称。

$f(-x) = (-x)^3 + (-x) = -x^3 - x = -(x^3 + x) = -f(x)$

满足 $f(-x) = -f(x)$，所以 $f(x) = x^3 + x$ 是**奇函数**。

**验证：** $f(0) = 0$ ✓（奇函数且 $0$ 在定义域内，$f(0)$ 必为 $0$）
