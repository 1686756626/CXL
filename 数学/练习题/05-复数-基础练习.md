### 第1题
**难度：** ⭐（基础）

**题目：** 已知复数 z = 3 + 2i，求 z 的实部和虚部。

**答案：** 实部 = 3，虚部 = 2

**解析：** 复数的一般形式是 a + bi：
- a 是实部 → 3
- b 是虚部 → 2（注意：虚部是 2，不是 2i！）

---

### 第2题
**难度：** ⭐（基础）

**题目：** 已知 z = 5 - 7i，求 z 的共轭复数 z̄。

**答案：** z̄ = 5 + 7i

**解析：** 共轭复数就是把虚部的**符号反过来**，实部不变。
- 原来虚部是 -7，变号后是 +7
- z̄ = 5 + 7i

---

### 第3题
**难度：** ⭐（基础）

**题目：** 计算：(3 + 2i) + (1 + 4i)

**答案：** 4 + 6i

**解析：** 加法规则：实部加实部，虚部加虚部。
- 实部：3 + 1 = 4
- 虚部：2 + 4 = 6
- 结果：4 + 6i

---

### 第4题
**难度：** ⭐（基础）

**题目：** 计算：(5 + 3i) - (2 + i)

**答案：** 3 + 2i

**解析：** 减法规则：实部减实部，虚部减虚部。
- 实部：5 - 2 = 3
- 虚部：3 - 1 = 2
- 结果：3 + 2i

---

### 第5题
**难度：** ⭐（基础）

**题目：** 计算：i² + i³

**答案：** -1 + (-i) = -1 - i

**解析：** 直接用 i 的乘方规律：
- i² = -1
- i³ = -i
- i² + i³ = -1 + (-i) = -1 - i

---

### 第6题
**难度：** ⭐⭐（中等）

**题目：** 计算：(2 + i)(1 - i)

**答案：** 3 - i

**解析：** 像多项式乘法一样展开：
```
(2 + i)(1 - i)
= 2×1 + 2×(-i) + i×1 + i×(-i)
= 2 - 2i + i - i²
= 2 - i - (-1)     ← i² 换成 -1
= 2 - i + 1
= 3 - i
```

关键一步：i² = -1，所以 -i² = -(-1) = +1。

---

### 第7题
**难度：** ⭐⭐（中等）

**题目：** 计算：(1 + i)²

**答案：** 2i

**解析：** 方法一：展开
```
(1 + i)(1 + i)
= 1 + i + i + i²
= 1 + 2i + (-1)
= 2i
```

方法二：用完全平方公式 (a+b)² = a² + 2ab + b²
```
= 1² + 2×1×i + i²
= 1 + 2i - 1
= 2i
```

### 第8题
**难度：** ⭐⭐（中等）

**题目：** 计算 i²⁰²⁴ + i²⁰²⁵ + i²⁰²⁶

**答案：** i

**解析：** 用"除以4看余数"的方法：
- 2024 ÷ 4 = 506 余 0 → 和 i⁴ 一样 = 1
- 2025 ÷ 4 = 506 余 1 → 和 i¹ 一样 = i
- 2026 ÷ 4 = 506 余 2 → 和 i² 一样 = -1

相加：1 + i + (-1) = **i**

### 第9题
**难度：** ⭐⭐⭐（稍有挑战）

**题目：** 计算 (1 + 2i) / (1 - i)

**答案：** -½ + (3/2)i

**解析：** 复数除法：分子分母同时乘分母的共轭复数 (1 + i)

第一步：写算式
```
     (1 + 2i)(1 + i)
    -----------------
      (1 - i)(1 + i)
```

第二步：算分母
```
(1 - i)(1 + i) = 1² - i² = 1 - (-1) = 2
```

第三步：算分子
```
(1 + 2i)(1 + i) = 1 + i + 2i + 2i²
= 1 + 3i + 2(-1)
= 1 + 3i - 2
= -1 + 3i
```

第四步：分子除以分母
```
(-1 + 3i) / 2 = -1/2 + (3/2)i
```

**答案：-½ + (3/2)i**

### 第10题
**难度：** ⭐⭐（中等）

**题目：** 已知复数 z 满足 z + z̄ = 4 且 z·z̄ = 8，求 z。

**答案：** z = 2 + 2i 或 z = 2 - 2i

**解析：**
设 z = a + bi（a, b ∈ R），则 z̄ = a - bi。

第一步：由 z + z̄ = 4
- (a + bi) + (a - bi) = 2a = 4
- **a = 2**

第二步：由 z·z̄ = 8
- z·z̄ = (a + bi)(a - bi) = a² + b² = |z|²
- 2² + b² = 8
- b² = 4
- **b = 2 或 b = -2**

第三步：写出 z
- z = 2 + 2i 或 z = 2 - 2i

**验证：**
- z = 2 + 2i：z + z̄ = 4 ✅，z·z̄ = 4 + 4 = 8 ✅
- z = 2 - 2i：z + z̄ = 4 ✅，z·z̄ = 4 + 4 = 8 ✅

**💡 关键公式：** z·z̄ = a² + b² = |z|²，这个恒等式在复数题中经常用到！

### 第11题
**难度：** ⭐⭐（中等）

**题目：** 已知 |z| = 2，求 |z - 1| 的最大值和最小值。

**答案：** 最大值 3，最小值 1

**解析：**

**方法一：几何法（推荐）**

|z| = 2 表示 z 在复平面上以原点为圆心、半径为 2 的圆上。

|z - 1| 的含义：复数 z 到点 1（即 (1, 0)）的距离。

```
       z(圆上的点)
      /|\
     / | \
    /  |  \
   2   d   2
   ----O----1
```

- 最近距离（圆上最接近 (1,0) 的点）：点 (2, 0)，距离 = 2 - 1 = **1**
- 最远距离（圆上最远离 (1,0) 的点）：点 (-2, 0)，距离 = 2 + 1 = **3**

**方法二：三角不等式**

由 ||z| - |1|| ≤ |z - 1| ≤ |z| + |1|：
- |2 - 1| ≤ |z - 1| ≤ 2 + 1
- **1 ≤ |z - 1| ≤ 3**

### 第12题
**难度：** ⭐⭐⭐（挑战）

**题目：** 已知复数 z 满足 |z - 1 + i| = 2，求 |z| 的取值范围。

**答案：** 2 - √2 ≤ |z| ≤ 2 + √2

**解析：**

**几何法：**

|z - (1 - i)| = 2 表示 z 在复平面上以点 (1, -1) 为圆心、半径为 2 的圆上。

|z| 的含义：z 到原点的距离。

```
       (0, 0) O
         \  |
          \ | d  ← 圆心到原点距离
           \|    ← 圆半径 r = 2
            C(1, -1)
```

第一步：求圆心到原点的距离
- d = |1 - i| = √(1² + (-1)²) = **√2**

第二步：利用圆上点到原点的距离范围
- 最远距离 = d + r = √2 + 2
- 最近距离 = |r - d| = |2 - √2| = 2 - √2（因为 2 > √2）

所以 **2 - √2 ≤ |z| ≤ 2 + √2**

**验证：**
- 当 z 在 OC 延长线上时（离原点最远）：|z| = 2 + √2 ≈ 3.414
- 当 z 在 OC 之间时（离原点最近）：|z| = 2 - √2 ≈ 0.586

**💡 本题要点：** 把复数模的问题转化为**几何距离**问题，画出圆和原点的位置关系就一目了然。

## 复习题（交叉练习）

### 复1
**难度：** ⭐ 基础（5分）
**复习知识点：** 01-集合

**题目：** 已知集合 A = {x | x² - 4x + 3 < 0}，B = {x | -1 ≤ x < 4}，求 A ∩ B。

**答案：** (1, 3)

**解析：**
复习知识点：01-集合

**第一步：化简集合 A**

x² - 4x + 3 = (x - 1)(x - 3)

二次函数开口向上，(x-1)(x-3) < 0 的解集在两根之间：**A = (1, 3)**

**第二步：与集合 B 取交集**

B = [-1, 4)

A ∩ B = (1, 3) ∩ [-1, 4) = **(1, 3)**

**画数轴验证：**
```
A:    1 ←→ 3        （不包含 1 和 3）
B: -1 ←――――→ 4      （包含 -1，不包含 4）
重叠: 1 ←→ 3        （都不包含）
```

两端点分析：A 不包含 1 和 3（严格不等式 <），所以交集端点也不包含。

**答案：(1, 3)**
