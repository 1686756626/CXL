### 第1题
**难度：** ⭐（基础）

**题目：** 已知 A = {1, 2, 3, 4}，B = {3, 4, 5, 6}，求 A ∩ B。

**答案：** A ∩ B = {3, 4}

**解析：** 交集就是两个集合里**都有**的元素。逐一检查：
- 1 在 B 里吗？不在 ❌
- 2 在 B 里吗？不在 ❌
- 3 在 B 里吗？在 ✅
- 4 在 B 里吗？在 ✅

所以 A ∩ B = {3, 4}。

---

### 第2题
**难度：** ⭐（基础）

**题目：** 已知 A = {1, 2, 3}，B = {2, 3, 4, 5}，求 A ∪ B。

**答案：** A ∪ B = {1, 2, 3, 4, 5}

**解析：** 并集就是把两个集合的元素**全部合在一起**，重复的只写一次（互异性）。
- A 有 1, 2, 3
- B 有 2, 3, 4, 5
- 合起来就是 1, 2, 3, 4, 5（2和3不重复写）

所以 A ∪ B = {1, 2, 3, 4, 5}。

---

### 第3题
**难度：** ⭐（基础）

**题目：** 已知 U = {1, 2, 3, 4, 5, 6, 7, 8}，A = {2, 4, 6, 8}，求 ∁ᵤA。

**答案：** ∁ᵤA = {1, 3, 5, 7}

**解析：** 补集就是在全集 U 里**不属于 A 的元素**。
- 全集 U 有 1, 2, 3, 4, 5, 6, 7, 8
- A 有 2, 4, 6, 8
- 不在 A 里的就是：1, 3, 5, 7

所以 ∁ᵤA = {1, 3, 5, 7}。

### 第4题
**难度：** ⭐（基础）

**题目：** 下列表示正确的是（ ）

A. 1 ∈ {0, 1, 2}
B. {1} ∈ {0, 1, 2}
C. ∅ ∈ {0, 1, 2}
D. 0 ⊆ {0, 1, 2}

**答案：** A

**解析：** 逐个分析：
- A. 1 是集合 {0,1,2} 里的元素，用 ∈ ✅
- B. {1} 是一个集合，不是元素，不能说"集合属于集合" ❌
- C. ∅（空集）是一个集合，不是元素 ❌
- D. ⊆ 用于集合和集合之间，0 是元素不是集合，应该用 ∈ ❌

### 第5题
**难度：** ⭐（基础）

**题目：** 已知 A = {x | x² - 5x + 6 = 0}，写出集合 A 的所有元素。

**答案：** A = {2, 3}

**解析：** 先解方程 x² - 5x + 6 = 0：
- 因式分解：(x - 2)(x - 3) = 0
- 所以 x = 2 或 x = 3
- A = {2, 3}

### 第6题
**难度：** ⭐⭐（中等）

**题目：** 已知 A = {x | -1 < x ≤ 2}，B = {x | 0 ≤ x < 3}，求 A ∩ B（用区间表示）。

**答案：** A ∩ B = [0, 2]

**解析：** A 是 -1 到 2 的区间（不包含-1，包含2），B 是 0 到 3 的区间（包含0，不包含3）。

交集就是两个区间的**重叠部分**：

画数轴：
```
A:  (-1, 2]    →  -1 ←→ 2
B:  [0, 3)     →    0 ←→ 3
重叠: [0, 2]
```

- 左端点：A 的 -1 < 0，B 包含 0 → 左端点取 0，包含 → [0
- 右端点：A 包含 2，B 的 3 > 2 → 右端点取 2，包含 → 2]

所以 A ∩ B = [0, 2]。

### 第7题
**难度：** ⭐⭐（中等）

**题目：** 已知 A = {x | x² - 3x + 2 = 0}，B = {x | 1 ≤ x ≤ 4, x ∈ Z}，求 A ∩ B。

**答案：** A ∩ B = {1, 2}

**解析：** 第一步：解方程 x² - 3x + 2 = 0
- (x - 1)(x - 2) = 0
- x = 1 或 x = 2
- A = {1, 2}

第二步：B = {x | 1 ≤ x ≤ 4, x ∈ Z}，意思是 1 到 4 的整数
- B = {1, 2, 3, 4}

第三步：A ∩ B = 两个集合都有的元素
- 1 在 B 里吗？1 ≤ 1 ≤ 4，是 ✅
- 2 在 B 里吗？1 ≤ 2 ≤ 4，是 ✅

所以 A ∩ B = {1, 2}。

### 第8题
**难度：** ⭐⭐⭐（稍有挑战）

**题目：** 某班45人，参加语文兴趣小组的有18人，参加数学兴趣小组的有22人，两个都参加的有10人。问：两个兴趣小组都没参加的有多少人？

**答案：** 15人

**解析：** 这是经典的韦恩图题，画两个交叉的圆圈：

- 只参加语文的：18 - 10 = 8人
- 只参加数学的：22 - 10 = 12人
- 两个都参加的：10人
- 参加了至少一个的：8 + 12 + 10 = 30人
- 都没参加的：45 - 30 = **15人**

关键：两个都参加的 10 人不要加两遍！

### 第9题
**难度：** ⭐⭐（中等）

**题目：** 已知 A = {x | 2x - a < 0}，B = {x | x < 3}，若 A ⊆ B，求 a 的取值范围。

**答案：** a ≤ 6（即 a ∈ (-∞, 6]）

**解析：**
第一步：化简集合 A
- 2x - a < 0 → 2x < a → x < a/2
- A = {x | x < a/2} = (-∞, a/2)

第二步：写出集合 B
- B = {x | x < 3} = (-∞, 3)

第三步：A ⊆ B 的含义
- A 是 (-∞, a/2)，B 是 (-∞, 3)
- A ⊆ B 意味着 A 的上界不超过 B 的上界
- 即 a/2 ≤ 3 → **a ≤ 6**

画数轴验证：
```
A:  (-∞, a/2)    ← 取决于 a
B:  (-∞, 3)
```
当 a/2 ≤ 3 时，A 完全在 B 内部 ✅

### 第10题
**难度：** ⭐⭐（中等）

**题目：** 已知 A = {x | x² - x - 6 < 0}，B = {x | 0 < x < 4}，求 A ∩ B。

**答案：** A ∩ B = {x | 0 < x < 3}

**解析：**
第一步：解不等式 x² - x - 6 < 0
- 因式分解：x² - x - 6 = (x - 3)(x + 2)
- 开口向上，取两根之间：(x-3)(x+2) < 0 → **-2 < x < 3**
- A = {x | -2 < x < 3}

第二步：写出 B
- B = {x | 0 < x < 4}

第三步：求交集（画数轴）
```
A:  -2 ←――――→ 3        （不包含-2和3）
B:       0 ←――→ 4       （不包含0和4）
重叠:     0 ←→ 3         （不包含0和3）
```

- 左端点：A 从 -2 开始，B 从 0 开始 → 取 0，不包含 → (0
- 右端点：A 到 3 结束，B 到 4 结束 → 取 3，不包含 → 3)

所以 A ∩ B = {x | 0 < x < 3}。

### 第11题
**难度：** ⭐⭐（中等）

**题目：** 已知 A = {1, 2, 3, 4}，B = {2, 4, 6}，C = {1, 3, 5}，求 A ∩ (B ∪ C)。

**答案：** A ∩ (B ∪ C) = {1, 2, 3, 4}

**解析：**
第一步：先算 B ∪ C
- B = {2, 4, 6}，C = {1, 3, 5}
- 合并所有元素（不重复）：B ∪ C = {1, 2, 3, 4, 5, 6}

第二步：再算 A ∩ (B ∪ C)
- A = {1, 2, 3, 4}
- B ∪ C = {1, 2, 3, 4, 5, 6}
- 交集：A 中每个元素是否在 B ∪ C 中？
  - 1 ∈ B∪C ✅，2 ∈ B∪C ✅，3 ∈ B∪C ✅，4 ∈ B∪C ✅
- 结果：{1, 2, 3, 4}

**💡 注意括号！** 如果没有括号，A ∩ B ∪ C 按运算优先级先算 ∩ 再算 ∪：
- (A ∩ B) ∪ C = {2, 4} ∪ {1, 3, 5} = {1, 2, 3, 4, 5}
- 结果不一样！运算顺序很重要。

### 第12题
**难度：** ⭐⭐⭐（挑战）

**题目：** 已知 U = R，A = {x | x² - 4x + 3 > 0}，求 ∁ᵤA ∩ {x | x ≤ 4}。

**答案：** {x | 1 ≤ x ≤ 3}

**解析：**
第一步：解不等式 x² - 4x + 3 > 0
- 因式分解：x² - 4x + 3 = (x - 1)(x - 3)
- 开口向上，取两根**之外**：(x-1)(x-3) > 0 → x < 1 或 x > 3
- A = {x | x < 1 或 x > 3}

第二步：求补集 ∁ᵤA
- ∁ᵤA = {x | x ≥ 1 且 x ≤ 3} = {x | 1 ≤ x ≤ 3} = **[1, 3]**
- **补集就是把 A 的不等号"反过来"**（<变≥，>变≤）

第三步：与 {x | x ≤ 4} 取交集
```
∁ᵤA:  [1, 3]
{x | x ≤ 4}:  (-∞, 4]
```
重叠部分就是 [1, 3]（完全包含在 (-∞, 4] 内）

所以 ∁ᵤA ∩ {x | x ≤ 4} = **{x | 1 ≤ x ≤ 3}**
