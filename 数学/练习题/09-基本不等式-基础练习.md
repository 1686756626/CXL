## ⭐ 基础题

### 第1题 ⭐

**已知 x > 0，求 $x + \dfrac{1}{x}$ 的最小值，并指出等号成立的条件。**

【解析】

看到 $x + \dfrac{1}{x}$ 这个结构，两个正数的和，立刻想到基本不等式 $a + b \geqslant 2\sqrt{ab}$。

令 $a = x$，$b = \dfrac{1}{x}$，因为 $x > 0$，所以 $a > 0$，$b > 0$，满足使用条件。

$$x + \frac{1}{x} \geqslant 2\sqrt{x \cdot \frac{1}{x}} = 2\sqrt{1} = 2$$

等号成立条件：$x = \dfrac{1}{x}$，即 $x^2 = 1$，$x = 1$（$x > 0$ 舍去负根）。

> **怎么想到的：** $x + \frac{1}{x}$ 是基本不等式最经典的题型。看到"两个正数相加"的结构，且相乘结果恰好是常数 $1$，直接套公式即可。

【答案】最小值为 **2**，当 $x = 1$ 时取得。

### 第2题 ⭐

**已知 $a > 0$，$b > 0$，$a + b = 1$，求 $ab$ 的最大值。**

【解析】

已知两个正数的和为定值 $1$，要求它们的乘积的最大值，这是"和定积最大"的典型场景。

由基本不等式 $a + b \geqslant 2\sqrt{ab}$：

$$1 = a + b \geqslant 2\sqrt{ab}$$

两边平方：$1 \geqslant 4ab$，即 $ab \leqslant \dfrac{1}{4}$。

等号成立条件：$a = b$，结合 $a + b = 1$，得 $a = b = \dfrac{1}{2}$。

> **怎么想到的：** "和定求积"是基本不等式的核心应用之一。关键在于判断：已知的是"和"还是"积"为定值——本题和为 $1$（定值），要求积的最大值，直接套用 $a + b \geqslant 2\sqrt{ab}$。

【答案】最大值为 $\dfrac{1}{4}$，当 $a = b = \dfrac{1}{2}$ 时取得。

### 第3题 ⭐

**下列关于基本不等式的说法，正确的是（ ）**

> A. 若 $a + b = 1$（$a, b > 0$），则 $ab \leqslant \dfrac{1}{4}$
>
> B. 若 $a + b = 1$（$a, b > 0$），则 $\dfrac{1}{a} + \dfrac{1}{b} \leqslant 4$
>
> C. 对任意实数 $a$，都有 $a + \dfrac{1}{a} \geqslant 2$
>
> D. 若 $a^2 + b^2 = 1$，则 $a + b \leqslant 1$

【解析】

逐项判断：

**A.** $a + b = 1$，由基本不等式 $ab \leqslant \left(\dfrac{a+b}{2}\right)^2 = \dfrac{1}{4}$。**正确。**

**B.** $a + b = 1$ 时，$\dfrac{1}{a} + \dfrac{1}{b} = \dfrac{a+b}{ab} = \dfrac{1}{ab}$。因为 $ab \leqslant \dfrac{1}{4}$，所以 $\dfrac{1}{ab} \geqslant 4$，即 $\dfrac{1}{a} + \dfrac{1}{b} \geqslant 4$，方向是 $\geqslant$ 而非 $\leqslant$。**错误。**

**C.** $a + \dfrac{1}{a} \geqslant 2$ 要求 $a > 0$。当 $a < 0$ 时，例如 $a = -1$：$-1 + (-1) = -2 < 2$。题目说"任意实数"，不满足条件。**错误。**

**D.** $a^2 + b^2 = 1$ 时，$a + b \leqslant \sqrt{2(a^2+b^2)} = \sqrt{2} \approx 1.414$。虽然 $a + b$ 确实有一个上限，但这个上限是 $\sqrt{2}$ 而非 $1$。例如 $a = b = \dfrac{\sqrt{2}}{2}$ 时，$a^2 + b^2 = 1$，但 $a + b = \sqrt{2} > 1$。**错误。**

> **怎么想到的：** 判断题的关键是注意基本不等式的使用条件。特别要留意：(1) 正数条件是否满足；(2) 不等号方向是否正确；(3) 等号能否成立。

【答案】**A**

## ⭐⭐ 中等题

### 第4题 ⭐⭐

**已知 $x > 0$，求 $x + \dfrac{9}{x}$ 的最小值。**

【解析】

$x > 0$ 时 $x$ 和 $\dfrac{9}{x}$ 都是正数，且它们的乘积 $x \cdot \dfrac{9}{x} = 9$ 是常数（定值），直接用基本不等式：

$$x + \frac{9}{x} \geqslant 2\sqrt{x \cdot \frac{9}{x}} = 2\sqrt{9} = 6$$

等号成立：$x = \dfrac{9}{x}$，即 $x^2 = 9$，$x = 3$（$x > 0$）。

验证：$x = 3$ 时，$3 + \dfrac{9}{3} = 3 + 3 = 6$。✓

> **怎么想到的：** 这是第1题的变形——把 $\frac{1}{x}$ 换成 $\frac{9}{x}$。核心思路不变：两个正数相加，乘积为定值，直接套基本不等式。关键是识别出"乘积为定值"这个条件。

【答案】最小值为 **6**，当 $x = 3$ 时取得。

### 第5题 ⭐⭐

**已知正数 $x, y$ 满足 $2x + y = 1$，求 $xy$ 的最大值。**

【解析】

已知 $2x + y = 1$（和为定值），要求 $xy$ 的最大值。

关键在于 $xy$ 中 $x$ 的系数是 $1$，但约束条件中 $x$ 的系数是 $2$，不匹配。我们可以把约束条件中的 $2x$ 看作一个整体：

由基本不等式：

$$1 = 2x + y \geqslant 2\sqrt{2x \cdot y} = 2\sqrt{2xy}$$

两边平方：$1 \geqslant 4 \cdot 2xy = 8xy$，即 $xy \leqslant \dfrac{1}{8}$。

等号成立：$2x = y$，结合 $2x + y = 1$，得 $4x = 1$，$x = \dfrac{1}{4}$，$y = \dfrac{1}{2}$。

验证：$xy = \dfrac{1}{4} \times \dfrac{1}{2} = \dfrac{1}{8}$。✓

> **怎么想到的：** 这道题的难点在于 $2x + y$ 中 $x$ 有系数 $2$，不能直接写 $x + y \geqslant 2\sqrt{xy}$。需要把 $2x$ 作为一个整体与 $y$ 配对，再用基本不等式。这就是"配系数"的技巧。

【答案】最大值为 $\dfrac{1}{8}$，当 $x = \dfrac{1}{4}$，$y = \dfrac{1}{2}$ 时取得。

### 第6题 ⭐⭐

**已知正数 $a, b$ 满足 $2a + 3b = 1$，求 $\dfrac{1}{a} + \dfrac{1}{b}$ 的最小值。**

【解析】

要求 $\dfrac{1}{a} + \dfrac{1}{b}$ 的最小值，而约束是 $2a + 3b = 1$。两个式子的变量结构不同，需要"搭桥"。

**乘1法：** 利用 $1 = 2a + 3b$：

$$\frac{1}{a} + \frac{1}{b} = \left(\frac{1}{a} + \frac{1}{b}\right) \times 1 = \left(\frac{1}{a} + \frac{1}{b}\right)(2a + 3b)$$

展开：

$$= \frac{2a}{a} + \frac{3b}{a} + \frac{2a}{b} + \frac{3b}{b} = 2 + \frac{3b}{a} + \frac{2a}{b} + 3 = 5 + \frac{3b}{a} + \frac{2a}{b}$$

对 $\dfrac{3b}{a} + \dfrac{2a}{b}$ 使用基本不等式（均为正）：

$$\frac{3b}{a} + \frac{2a}{b} \geqslant 2\sqrt{\frac{3b}{a} \cdot \frac{2a}{b}} = 2\sqrt{6}$$

因此：

$$\frac{1}{a} + \frac{1}{b} \geqslant 5 + 2\sqrt{6}$$

等号成立条件：$\dfrac{3b}{a} = \dfrac{2a}{b}$，即 $3b^2 = 2a^2$，$\dfrac{b}{a} = \sqrt{\dfrac{2}{3}}$。

结合 $2a + 3b = 1$：$a\!\left(2 + 3\sqrt{\dfrac{2}{3}}\right) = 1$，$a = \dfrac{1}{2 + \sqrt{6}} = \dfrac{2 - \sqrt{6}}{-2} = \dfrac{\sqrt{6}-2}{2}$。

$b = a\sqrt{\dfrac{2}{3}} = \dfrac{\sqrt{6}-2}{2} \cdot \dfrac{\sqrt{6}}{3} = \dfrac{6-2\sqrt{6}}{6} = \dfrac{3-\sqrt{6}}{3}$。

验证（取 $a = \dfrac{\sqrt{6}-2}{2} \approx 0.2247$，$b = \dfrac{3-\sqrt{6}}{3} \approx 0.1835$）：
$2a + 3b = \sqrt{6}-2 + 3 - \sqrt{6} = 1$。✓
$\dfrac{1}{a} = \dfrac{2}{\sqrt{6}-2} = \sqrt{6}+2 \approx 4.449$
$\dfrac{1}{b} = \dfrac{3}{3-\sqrt{6}} = 3+\sqrt{6} \approx 5.449$
$\dfrac{1}{a} + \dfrac{1}{b} = 5 + 2\sqrt{6} \approx 9.899$。✓

> **怎么想到的：** 当目标和约束条件的变量"长得很不一样"时，用"乘1法"把两者联系起来——乘以约束条件等于1，展开后出现交叉项，对交叉项再用基本不等式。这是基本不等式中非常实用的技巧。

【答案】最小值为 $5 + 2\sqrt{6}$，当 $a = \dfrac{\sqrt{6}-2}{2}$，$b = \dfrac{3-\sqrt{6}}{3}$ 时取得。

### 第7题 ⭐⭐

**已知正数 $a, b$ 满足 $a + b = 1$，求 $\log_2 a + \log_2 b$ 的最大值。**

【解析】

利用对数的运算性质，先把加法变乘法：

$$\log_2 a + \log_2 b = \log_2(ab)$$

因为 $a + b = 1$，由基本不等式：$ab \leqslant \left(\dfrac{a+b}{2}\right)^2 = \dfrac{1}{4}$。

又因为 $\log_2 x$ 是增函数（底数 $2 > 1$），所以：

$$\log_2(ab) \leqslant \log_2\!\left(\frac{1}{4}\right) = \log_2 1 - \log_2 4 = 0 - 2 = -2$$

等号成立：$a = b = \dfrac{1}{2}$。

验证：$a = b = \dfrac{1}{2}$ 时，$\log_2 \dfrac{1}{2} + \log_2 \dfrac{1}{2} = -1 + (-1) = -2$。✓

> **怎么想到的：** 这道题结合了对数和基本不等式。关键步骤是先用 $\log_a x + \log_a y = \log_a(xy)$ 把对数加法变成对数乘法，然后用基本不等式放缩 $ab$，最后利用对数函数的单调性传递不等号方向。

【答案】最大值为 **$-2$**，当 $a = b = \dfrac{1}{2}$ 时取得。

### 第8题 ⭐⭐

**下列命题中正确的个数是（ ）**

> ① 对任意实数 $a, b$，$a^2 + b^2 \geqslant 2ab$
>
> ② 对任意正实数 $a$，$a + \dfrac{1}{a} \geqslant 2$
>
> ③ 已知 $a, b > 0$，$a + b = 1$，则 $ab$ 的最大值为 $\dfrac{1}{2}$
>
> ④ 已知 $a, b > 0$，则 $\dfrac{a}{b} + \dfrac{b}{a} \geqslant a + b$

【解析】

逐条判断：

**①** $a^2 + b^2 - 2ab = (a-b)^2 \geqslant 0$，对任意实数成立。**正确。** ✅

**②** $a > 0$ 时 $a$ 和 $\frac{1}{a}$ 都是正数，由基本不等式：$a + \frac{1}{a} \geqslant 2\sqrt{a \cdot \frac{1}{a}} = 2$。**正确。** ✅

**③** $ab \leqslant \left(\dfrac{a+b}{2}\right)^2 = \dfrac{1}{4}$，最大值是 $\dfrac{1}{4}$ 而非 $\dfrac{1}{2}$。**错误。** ❌

**④** 取 $a = 1, b = 2$：$\dfrac{1}{2} + \dfrac{2}{1} = \dfrac{5}{2} = 2.5$，而 $a + b = 3$。$\dfrac{5}{2} < 3$。**错误。** ❌

> **怎么想到的：** 判断题的核心是检查三个要点：(1) 使用条件是否满足（正数？任意实数？）；(2) 等号/最值是否正确；(3) 不等号方向是否搞反。对于③，$\frac{1}{2}$ 是常见错误答案（与 $\frac{a+b}{2}$ 混淆），实际最大值是 $\frac{(a+b)^2}{4} = \frac{1}{4}$。

【答案】正确的个数是 **2**（①②）。

## ⭐⭐⭐ 挑战题

### 第9题 ⭐⭐⭐

**已知正数 $a, b$ 满足 $a + 2b = 3$，求 $\dfrac{1}{a} + \dfrac{4}{b}$ 的最小值。**

【解析】

目标和 $\dfrac{1}{a} + \dfrac{4}{b}$ 与约束 $a + 2b = 3$ 的结构不同，使用**乘1法**：

$$\frac{1}{a} + \frac{4}{b} = \left(\frac{1}{a} + \frac{4}{b}\right) \times 1 = \frac{1}{a + 2b}\left(\frac{1}{a} + \frac{4}{b}\right)(a + 2b)$$

$$= \frac{1}{3}\left[\frac{a}{a} + \frac{2b}{a} + \frac{4a}{b} + \frac{8b}{b}\right]$$

$$= \frac{1}{3}\left[1 + \frac{2b}{a} + \frac{4a}{b} + 8\right]$$

$$= \frac{1}{3}\left[9 + \frac{2b}{a} + \frac{4a}{b}\right]$$

对交叉项 $\dfrac{2b}{a} + \dfrac{4a}{b}$ 用基本不等式：

$$\frac{2b}{a} + \frac{4a}{b} \geqslant 2\sqrt{\frac{2b}{a} \cdot \frac{4a}{b}} = 2\sqrt{8} = 4\sqrt{2}$$

因此：

$$\frac{1}{a} + \frac{4}{b} \geqslant \frac{9 + 4\sqrt{2}}{3}$$

**等号验证：** $\dfrac{2b}{a} = \dfrac{4a}{b}$，即 $2b^2 = 4a^2$，$b^2 = 2a^2$，$b = a\sqrt{2}$（取正）。

代入 $a + 2a\sqrt{2} = 3$：$a(1 + 2\sqrt{2}) = 3$，$a = \dfrac{3}{1 + 2\sqrt{2}} = \dfrac{3(2\sqrt{2}-1)}{7}$。

$b = \dfrac{3\sqrt{2}(2\sqrt{2}-1)}{7} = \dfrac{3(4-\sqrt{2})}{7} = \dfrac{12 - 3\sqrt{2}}{7}$。

**数值验证：**
$a \approx 0.7839$，$b \approx 1.1085$。
$2a + 3b$，验证约束条件 $a + 2b$：
$a + 2b = 0.7839 + 2 \times 1.1085 = 0.7839 + 2.2170 = 3.0009 \approx 3$（精确值恰为3）。✓
$\dfrac{1}{a} + \dfrac{4}{b} = \dfrac{2\sqrt{2}+1}{3} + \dfrac{8+2\sqrt{2}}{3} = \dfrac{9+4\sqrt{2}}{3} \approx 4.886$。✓

> **怎么想到的：** 这道题的关键是"乘1法"——目标和与约束的结构不一致，需要通过乘以约束条件（等于1）来搭桥。展开后会出现交叉项 $\frac{2b}{a} + \frac{4a}{b}$，它们乘积为定值 $\frac{2b}{a} \cdot \frac{4a}{b} = 8$，正好可以用基本不等式。

【答案】最小值为 $\dfrac{9 + 4\sqrt{2}}{3}$，当 $a = \dfrac{3(2\sqrt{2}-1)}{7}$，$b = \dfrac{12-3\sqrt{2}}{7}$ 时取得。

### 第10题 ⭐⭐⭐

**已知 $x \geqslant \dfrac{1}{2}$，求 $f(x) = \dfrac{x^2 - x + 1}{2x - 1}$ 的最小值。**

【解析】

这道题的形式不像基本不等式的标准形式，需要先变形。

**第一步：配方变形**

分子 $x^2 - x + 1$：

$$x^2 - x + 1 = x^2 - x + \frac{1}{4} + \frac{3}{4} = \left(x - \frac{1}{2}\right)^2 + \frac{3}{4}$$

注意到分母 $2x - 1 = 2\!\left(x - \dfrac{1}{2}\right)$，令 $t = x - \dfrac{1}{2} \geqslant 0$：

$$f(x) = \frac{t^2 + \frac{3}{4}}{2t} = \frac{t^2}{2t} + \frac{3}{4 \cdot 2t} = \frac{t}{2} + \frac{3}{8t} = \frac{1}{2}\!\left(t + \frac{3}{4t}\right)$$

**第二步：基本不等式**

对 $t + \dfrac{3}{4t}$（$t > 0$，当 $t = 0$ 时原式无意义）：

$$t + \frac{3}{4t} \geqslant 2\sqrt{t \cdot \frac{3}{4t}} = 2\sqrt{\frac{3}{4}} = \sqrt{3}$$

因此 $f(x) \geqslant \dfrac{\sqrt{3}}{2}$。

**等号验证：** $t = \dfrac{3}{4t}$，即 $t^2 = \dfrac{3}{4}$，$t = \dfrac{\sqrt{3}}{2}$（$t \geqslant 0$）。

$x = \dfrac{1}{2} + \dfrac{\sqrt{3}}{2} = \dfrac{1 + \sqrt{3}}{2} \geqslant \dfrac{1}{2}$。✓

**数值验证：** $x = \dfrac{1+\sqrt{3}}{2} \approx 1.366$
$x^2 = \dfrac{1+2\sqrt{3}+3}{4} = 1 + \dfrac{\sqrt{3}}{2}$
$x^2 - x + 1 = 1 + \dfrac{\sqrt{3}}{2} - \dfrac{1+\sqrt{3}}{2} + 1 = 2 + \dfrac{\sqrt{3}}{2} - \dfrac{1}{2} - \dfrac{\sqrt{3}}{2} = \dfrac{3}{2}$
$2x - 1 = \sqrt{3}$
$f(x) = \dfrac{3/2}{\sqrt{3}} = \dfrac{3}{2\sqrt{3}} = \dfrac{\sqrt{3}}{2}$。✓

> **怎么想到的：** 这道题表面上是求函数最值，但本质是基本不等式与函数值的结合。关键在于：
> 1. **配方**：把分子凑成 $(x-\frac{1}{2})^2 + \frac{3}{4}$ 的形式，与分母 $2(x-\frac{1}{2})$ 对应；
> 2. **换元**：令 $t = x - \frac{1}{2}$，把分式拆成两项之和；
> 3. **用基本不等式**：拆开的两项之积恰好是常数 $\frac{3}{4}$。
>
> 整个过程体现了"凑结构"的思想——通过代数变形把不熟悉的式子化成基本不等式能处理的形式。

【答案】最小值为 $\dfrac{\sqrt{3}}{2}$，当 $x = \dfrac{1+\sqrt{3}}{2}$ 时取得。

## 复习题（交叉练习）

### 复1
**难度：** ⭐ 基础（5分）
**复习知识点：** 02-复数

**题目：** 已知复数 z = (1+i)²/(1-i)，求 z 的实部。

**答案：** -1

**解析：**
复习知识点：02-复数

**第一步：计算分子**
```
(1+i)² = 1 + 2i + i² = 1 + 2i - 1 = 2i
```

**第二步：复数除法**
```
z = 2i / (1 - i)
```

分子分母同乘 (1+i)：
```
z = 2i(1+i) / ((1-i)(1+i))
= 2i(1+i) / (1+1)
= 2i(1+i) / 2
= i(1+i)
= i + i²
= i - 1
= -1 + i
```

z 的实部 = **-1**

**验证：** (1+i)²/(1-i) = 2i/(1-i)
= 2i(1+i)/2 = i + i² = -1 + i，实部 = -1 ✓

**答案：-1**

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 04-三角函数

**题目：** 已知 sin α + cos α = 1/2，求 sin α · cos α 的值。

**答案：** -3/8

**解析：**
复习知识点：04-三角函数

将 sin α + cos α = 1/2 两边平方：
```
(sin α + cos α)² = 1/4
```

展开左边：
```
sin²α + 2sinαcosα + cos²α = 1/4
```

利用 sin²α + cos²α = 1：
```
1 + 2sinαcosα = 1/4
2sinαcosα = 1/4 - 1 = -3/4
sinαcosα = -3/8
```

**验证：** 将结果代回检查：
sin²α + cos²α + 2sinαcosα = 1 + 2×(-3/8) = 1 - 3/4 = 1/4 = (1/2)² ✓

**答案：-3/8**

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 05-二项式定理

**题目：** (x² + 1/x)⁵ 展开式中，x⁴ 的系数是多少？

**答案：** 10

**解析：**
复习知识点：05-二项式定理

**通项公式：** Tₖ₊₁ = C(5,k) · (x²)⁵⁻ᵏ · (1/x)ᵏ

化简 x 的指数：
```
(x²)⁵⁻ᵏ · x⁻ᵏ = x²⁽⁵⁻ᵏ⁾⁻ᵏ = x¹⁰⁻²ᵏ⁻ᵏ = x¹⁰⁻³ᵏ
```

令 10 - 3k = 4（找 x⁴ 对应的项）：
```
3k = 6
k = 2
```

系数 = C(5,2) = 5!/(2!·3!) = **10**

**答案：10**
