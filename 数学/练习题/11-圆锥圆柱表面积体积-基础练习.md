# 圆锥圆柱表面积体积 · 解析版

> **专题说明：** 这节的核心是"背公式+找关系+套公式"。圆锥的 $r,h,l$ 三个量通过 $r^2+h^2=l^2$（勾股定理）联系。注意区分"侧面积"和"全面积"。

---

## 必背公式速查

| 几何体 | 侧面积 | 全面积 | 体积 |
|--------|--------|--------|------|
| 圆柱 | $2\pi rh$ | $2\pi r^2 + 2\pi rh$ | $\pi r^2 h$ |
| 圆锥 | $\pi rl$ | $\pi r^2 + \pi rl$ | $\dfrac{1}{3}\pi r^2 h$ |
| 球 | — | $4\pi R^2$ | $\dfrac{4}{3}\pi R^3$ |

---

## 答案与详细解析

---

### 第1题 · 答案：B

**解析：**

圆柱侧面积公式：$S_{\text{侧}} = 2\pi rh$

代入 $r=3$，$h=4$：

$S_{\text{侧}} = 2\pi \times 3 \times 4 = \boxed{24\pi}$

**思路过程：** 纯套公式题。圆柱侧面展开是长方形，长 = 底面周长 $2\pi r$，宽 = 高 $h$，所以侧面积 = $2\pi rh$。

---

### 第2题 · 答案：C

**解析：**

球的表面积：$S = 4\pi R^2 = 36\pi$

$R^2 = 9$，$R = 3$

球的体积：$V = \dfrac{4}{3}\pi R^3 = \dfrac{4}{3}\pi \times 27 = \boxed{36\pi}$

**思路过程：** 先从表面积反求半径 $R$，再代入体积公式。$R^2=9 \Rightarrow R=3$（半径取正值）。

---

### 第3题 · 答案：C

**解析：**

轴截面是等边三角形，边长为 6。

圆锥轴截面中：底边 $= 2r$（直径），两腰 $= l$（母线），高 $= h$。

等边三角形：$2r = 6 \Rightarrow r = 3$，$l = 6$。

圆锥侧面积：$S_{\text{侧}} = \pi rl = \pi \times 3 \times 6 = \boxed{18\pi}$

**思路过程：** 从轴截面三角形提取 $r$ 和 $l$ 的关键：底边 = $2r$，腰 = $l$。等边三角形的三条边相等，所以 $l = 6$，$r = 3$。然后直接套侧面积公式 $\pi rl$。

---

### 第4题 · 答案：C

**解析：**

圆柱侧面展开为长方形：

- 长 $= 2\pi r = 8\pi \Rightarrow r = 4$
- 宽 $= h = 6$

圆柱体积：$V = \pi r^2 h = \pi \times 16 \times 6 = \boxed{96\pi}$

**思路过程：** 圆柱侧面展开的长方形中，长 $=$ 底面周长 $2\pi r$，宽 $=$ 高 $h$。从展开图反推 $r$ 和 $h$，再代入体积公式。

---

### 第5题 · 答案：B

**解析：**

从展开图提取信息：

扇形半径 $= l = 6$（母线长）

扇形弧长 $= 2\pi l \times \dfrac{120°}{360°} = 2\pi \times 6 \times \dfrac{1}{3} = 4\pi$

弧长 $=$ 底面周长 $= 2\pi r$，所以 $2\pi r = 4\pi \Rightarrow r = 2$。

求高：$r^2 + h^2 = l^2 \Rightarrow 4 + h^2 = 36 \Rightarrow h^2 = 32 \Rightarrow h = 4\sqrt2$

体积：$V = \dfrac{1}{3}\pi r^2 h = \dfrac{1}{3}\pi \times 4 \times 4\sqrt2 = \boxed{\dfrac{16\sqrt2\pi}{3}}$

**思路过程：** 展开图问题的关键桥梁：**扇形弧长 $=$ 圆锥底面周长**。扇形半径 $=$ 母线 $l$。求出 $r$ 后用勾股定理求 $h$，最后套体积公式（别忘了 $\frac{1}{3}$！）。

---

### 第6题 · 答案：C

**解析：**

球内切于圆柱的条件：球和圆柱的上下底面、侧面都相切。

- 球和侧面相切 $\Rightarrow$ 球半径 $R = $ 圆柱底面半径 $r = 3$
- 球和上下底面相切 $\Rightarrow$ 球直径 $2R = $ 圆柱高 $h = 6$

验证：$2R = 6 = h$ ✓，$R = r = 3$ ✓

球的表面积：$S = 4\pi R^2 = 4\pi \times 9 = \boxed{36\pi}$

**思路过程：** 球内切于圆柱的条件是 **$2R = h = 2r$**，即圆柱高等于球的直径，底面半径等于球半径。题目给的数据恰好满足这个条件（$h=6=2\times3$），可以直接得出 $R=3$。

---

### 第7题 · 答案：B

**解析：**

**圆柱全面积：**

$S_{\text{柱全}} = 2\pi r^2 + 2\pi rh = 2\pi \times 9 + 2\pi \times 3 \times 4 = 18\pi + 24\pi = 42\pi$

**圆锥：** 先求母线 $l = \sqrt{r^2+h^2} = \sqrt{9+16} = 5$

$S_{\text{锥全}} = \pi r^2 + \pi rl = 9\pi + 15\pi = 24\pi$

**差：** $42\pi - 24\pi = \boxed{18\pi}$

**思路过程：** 分别计算两个几何体的全面积再做差。圆锥的全面积 $= $ 底面积 $+$ 侧面积 $= \pi r^2 + \pi rl$，注意要先由 $r$ 和 $h$ 用勾股定理求出 $l$。

---

### 第8题 · 答案：B

**解析：**

轴截面为等腰直角三角形，斜边 $= 4$，直角在顶点。

圆锥轴截面中：底边 $= 2r$（直径），两腰 $= l$（母线）。

等腰直角三角形中，直角在顶点，则：
- 两腰 $= l$ 是直角边
- 底边 $= 2r$ 是斜边 $= 4$

所以 $l = \dfrac{4}{\sqrt2} = 2\sqrt2$（等腰直角三角形中，腰 $= \dfrac{\text{斜边}}{\sqrt2}$）

$r = \dfrac{4}{2} = 2$

侧面积：$S_{\text{侧}} = \pi rl = \pi \times 2 \times 2\sqrt2 = \boxed{4\sqrt2\pi}$

验证：$h = l/\sqrt2 = 2$，$r^2+h^2 = 4+4 = 8 = l^2 = 8$ ✓

**思路过程：** 等腰直角三角形的直角位置很关键——"直角在顶点"意味着两腰（母线 $l$）是直角边，底边（$2r$）是斜边。如果直角在底角，结果完全不同。题目说"直角在顶点"要牢记。

---

### 第9题 · 答案：C

**解析：**

已知 $r = h$，体积 $V = \pi r^2 h = \pi r^3 = 27\pi$

$r^3 = 27 \Rightarrow r = 3$，$h = 3$

全面积：$S_{\text{全}} = 2\pi r^2 + 2\pi rh = 2\pi \times 9 + 2\pi \times 9 = 18\pi + 18\pi = \boxed{36\pi}$

**思路过程：** 这题的技巧在于"同底等高"实际上是 $r=h$，体积公式化为 $V=\pi r^3$，可以直接解出 $r$。全面积代入 $r=h=3$ 即可。

---

### 第10题 · 答案：A

**解析：**

组合体不含底面公共部分，表面积包括：

- **圆柱的下底面**：$\pi r^2 = \pi \times 4 = 4\pi$
- **圆柱的侧面**：$2\pi rh = 2\pi \times 2 \times 3 = 12\pi$
- **圆锥的侧面**：$\pi rl$

先求圆锥母线：$l = \sqrt{r^2+h^2} = \sqrt{4+16} = \sqrt{20} = 2\sqrt5$

圆锥侧面积：$\pi \times 2 \times 2\sqrt5 = 4\sqrt5\pi$

**总表面积：** $4\pi + 12\pi + 4\sqrt5\pi = \boxed{(16+4\sqrt5)\pi}$

**思路过程：** 组合体表面积问题的关键是搞清楚哪些面要算、哪些面不要算。圆柱和圆锥"同底"，公共底面不露在外面，所以不算。需要算的是：圆柱下底面 + 圆柱侧面 + 圆锥侧面。

## 复习题（交叉练习）

### 复1
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 06-基本不等式

**题目：** 已知正数 a, b 满足 a + 2b = 6，求 ab 的最大值。

**答案：** 9/2

**解析：**
复习知识点：06-基本不等式

利用基本不等式：a + 2b ≥ 2√(a · 2b) = 2√(2ab)

代入 a + 2b = 6：
```
6 ≥ 2√(2ab)
3 ≥ √(2ab)
9 ≥ 2ab
ab ≤ 9/2
```

等号成立条件：a = 2b，结合 a + 2b = 6，得 4b = 6，b = 3/2，a = 3。

**验证：** a = 3，b = 3/2 时
- ab = 3 × 3/2 = 9/2 ✓
- a + 2b = 3 + 3 = 6 ✓

**答案：9/2**

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 07-分段函数与不等式

**题目：** 已知 f(x) = { x + 3, x ≤ 1; 2x - 1, x > 1 }，求不等式 f(x) ≤ 3 的解集。

**答案：** (-∞, 0] ∪ (1, 2]

**解析：**
复习知识点：07-分段函数与不等式

分段解不等式，每段分别求解，再与该段条件取交集。

**段一**（x ≤ 1）：x + 3 ≤ 3 → x ≤ 0
与 x ≤ 1 取交集：**(-∞, 0]**

**段二**（x > 1）：2x - 1 ≤ 3 → 2x ≤ 4 → x ≤ 2
与 x > 1 取交集：**(1, 2]**

**并集：** **(-∞, 0] ∪ (1, 2]**

**验证：**
- x = 0：f(0) = 0 + 3 = 3 ≤ 3 ✓（在解集中）
- x = -1：f(-1) = -1 + 3 = 2 ≤ 3 ✓（在解集中）
- x = 0.5：f(0.5) = 0.5 + 3 = 3.5 > 3 ✓（不在解集中）
- x = 1.5：f(1.5) = 2×1.5 - 1 = 2 ≤ 3 ✓（在解集中）
- x = 2：f(2) = 2×2 - 1 = 3 ≤ 3 ✓（在解集中）
- x = 3：f(3) = 2×3 - 1 = 5 > 3 ✓（不在解集中）

**答案：(-∞, 0] ∪ (1, 2]**

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 01-集合

**题目：** 已知 A = {x | x² - x - 2 > 0}，B = {x | x ≤ 3}，求 A ∩ B。

**答案：** (-∞, -1) ∪ (2, 3]

**解析：**
复习知识点：01-集合

**第一步：化简集合 A**

x² - x - 2 = (x + 1)(x - 2)

二次函数开口向上，(x+1)(x-2) > 0 → **A = (-∞, -1) ∪ (2, +∞)**

**第二步：与 B 取交集**

B = (-∞, 3]

A ∩ B = ((-∞, -1) ∪ (2, +∞)) ∩ (-∞, 3]
= ((-∞, -1) ∩ (-∞, 3]) ∪ ((2, +∞) ∩ (-∞, 3])
= **(-∞, -1) ∪ (2, 3]**

**画数轴验证：**
```
A:  -∞ ←→ -1    2 ←→ +∞
B:  -∞ ←――――――→ 3
交集: -∞ ←→ -1    2 ←→ 3]
```

A 的 (2, +∞) 与 B 的 (-∞, 3] 交集是 (2, 3]。注意 B 包含 3（因为 ≤），所以交集到 3 且包含。

**答案：(-∞, -1) ∪ (2, 3]**
