# 04-对数与对数函数 · 基础练习

> 10道题 + 1道复习题 | 基础3道、中等5道、挑战2道

---

### 第1题
**难度：** ⭐ 基础
**题目：** 计算 log₂ 8 = ______
**答案：** 3
**解析：** log₂ 8 就是问"2的几次方等于8"。2³ = 8，所以 log₂ 8 = 3。怎么想到的？对数就是问"底数的几次方等于真数"，直接找幂次。

---

### 第2题
**难度：** ⭐ 基础
**题目：** 计算 log₅ 1 = ______，log₅ 5 = ______
**答案：** log₅ 1 = 0，log₅ 5 = 1
**解析：** 任何数的0次方等于1，所以 logₐ 1 = 0；底数自己的1次方等于自己，所以 logₐ a = 1。怎么想到的？这是两个特殊值，必须背下来：log 1 = 0，log 自己 = 1。

---

### 第3题
**难度：** ⭐ 基础
**题目：** 把 3⁴ = 81 改写成对数形式。
**答案：** log₃ 81 = 4
**解析：** aˣ = N ⟺ x = logₐ N。这里 a = 3，x = 4，N = 81，所以 log₃ 81 = 4。怎么想到的？指数和对数互化就是"底数不变，等号左边指数变成对数，右边变成真数"。

---

### 第4题
**难度：** ⭐⭐ 中等
**题目：** 化简 log₂ 4 + log₂ 8 = ______
**答案：** 5
**解析：** log₂ 4 = 2（因为 2²=4），log₂ 8 = 3（因为 2³=8），2 + 3 = 5。也可以用公式：log₂ 4 + log₂ 8 = log₂(4×8) = log₂ 32 = log₂(2⁵) = 5。怎么想到的？同底对数相加，既可以直接算值相加，也可以用 log(MN) = log M + log N 合并再算。

---

### 第5题
**难度：** ⭐⭐ 中等
**题目：** 用换底公式计算 log₈ 2 的值。
**答案：** 1/3
**解析：** log₈ 2 = lg 2 / lg 8 = lg 2 / lg(2³) = lg 2 / (3 lg 2) = 1/3。怎么想到的？看到底数和真数都是2的幂（8=2³，2=2¹），也可以直接想：8的几次方等于2？8^(1/3) = 2。

---

### 第6题
**难度：** ⭐⭐ 中等
**题目：** 解方程 log₂(x - 1) = 3，求 x。
**答案：** x = 9
**解析：** log₂(x - 1) = 3，改写成指数形式：x - 1 = 2³ = 8，所以 x = 9。检验：x - 1 = 8 > 0 ✓。怎么想到的？对数方程直接改写成指数形式，解出 x 后一定要检验真数>0。

---

### 第7题
**难度：** ⭐⭐ 中等
**题目：** 解不等式 log₂ x > 3，求 x 的范围。
**答案：** x > 8
**解析：** log₂ x > 3 改写成指数形式：x > 2³ = 8。同时真数 x > 0 自动满足。怎么想到的？对数不等式改写成指数不等式，底数2>1方向不变。

---

### 第8题
**难度：** ⭐⭐ 中等
**题目：** 求函数 y = log₂(x - 3) 的定义域。
**答案：** x > 3
**解析：** 对数的真数必须大于0，所以 x - 3 > 0，解得 x > 3。怎么想到的？对数函数定义域就是"真数>0"，列不等式解即可。

---

### 第9题
**难度：** ⭐⭐⭐ 挑战
**题目：** 计算 lg 2 + lg 5 + lg 10 的值。
**答案：** 2
**解析：** lg 2 + lg 5 = lg(2×5) = lg 10 = 1。再加上 lg 10 = 1，共 1 + 1 = 2。怎么想到的？看到 lg 2 和 lg 5 就要想到 lg 10 = 1，这是常用凑整技巧。

---

### 第10题
**难度：** ⭐⭐⭐ 挑战
**题目：** 解不等式 logₐ x > 1，求 x 的范围（分 a > 1 和 0 < a < 1 两种情况讨论）。
**答案：** 当 a > 1 时，x > a；当 0 < a < 1 时，0 < x < a。
**解析：** logₐ x > 1 = logₐ a（因为 logₐ a = 1）。
- 当 a > 1（递增）：x > a
- 当 0 < a < 1（递减）：0 < x < a（不等号反转，且真数>0）
怎么想到的？底数含参数必须分类讨论，把1改写成 logₐ a 是关键一步。

---

### 复1（复习题）
**难度：** ⭐⭐ 中等
**题目：** 判断命题"若 x > 1，则 x² > 1"的真假，并说明其逆命题的真假。
**答案：** 原命题为真，逆命题为假。
**解析：** 原命题：x > 1 ⇒ x² > 1。x > 1 时 x² > 1，这是对的。逆命题是"若 x² > 1，则 x > 1"。当 x = -2 时，x² = 4 > 1 但 x < 1，所以逆命题为假。怎么想到的？判断命题真假，举例子验证是最直接的方法。逆命题是否命题的"条件和结论互换"。
