## 第1题 ⭐ 基础（5分）选择题

**椭圆 x²/16 + y²/9 = 1 的离心率是？**

A. √7/4  
B. 3/4  
C. √7/3  
D. 4/5

---

**【答案】A**

**【解析】**

从方程读出：a² = 16，a = 4；b² = 9，b = 3

椭圆关系：c² = a² - b² = 16 - 9 = 7，c = √7

离心率：e = c/a = **√7/4**

**怎么想到的：** 椭圆求离心率三步：①从方程读 a² 和 b²（分母大的那个是 a²）；②c² = a² - b²（减号！）；③e = c/a。这道题 16 > 9，所以 a² = 16。

---

## 第2题 ⭐ 基础（5分）选择题

**双曲线 x²/9 - y²/16 = 1 的离心率是？**

A. 5/4  
B. 5/3  
C. 4/3  
D. 7/4

---

**【答案】B**

**【解析】**

从方程读出：a² = 9，a = 3；b² = 16

双曲线关系：c² = a² + b² = 9 + 16 = 25，c = 5

离心率：e = c/a = **5/3**

**怎么想到的：** 双曲线求离心率和椭圆几乎一样，唯一的区别是 c² = a² + b²（**加号！**不是减号）。这是最容易搞混的地方。

**椭圆 x²/4 + y²/3 = 1 的离心率是？**

---

**【答案】A**

c² = a² - b² = 4 - 3 = 1，c = 1

e = c/a = 1/2

**怎么想到的：** 和第 1 题一样的步骤。这题的特点是 c = 1，结果很简洁。

## 第5题 ⭐⭐ 中等（5分）选择题

**双曲线的渐近线方程为 y = ±3x/4，一个焦点为 (5, 0)，则离心率是？**

A. 5/4  
B. 5/3  
C. 4/3  
D. 3/2

---

**【答案】A**

**【解析】**

第一步：从渐近线求 a 和 b 的关系

渐近线 y = ±(b/a)x = ±3x/4，所以 b/a = 3/4

设 a = 4k，b = 3k

第二步：从焦点求 c

c² = a² + b² = 16k² + 9k² = 25k²，c = 5k

焦点 (5, 0) → c = 5，所以 5k = 5，k = 1

第三步：求离心率

a = 4，c = 5

e = c/a = **5/4**

**怎么想到的：** 渐近线给出 b/a 的比值，焦点给出 c 的值。两个条件联立就能求出 a、b、c，再算 e。

## 第6题 ⭐⭐ 中等（5分）选择题

**已知椭圆的焦点为 F₁(-3, 0)，F₂(3, 0)，且经过点 (0, 4)，则椭圆的标准方程是？**

A. x²/25 + y²/16 = 1  
B. x²/16 + y²/25 = 1  
C. x²/9 + y²/16 = 1  
D. x²/25 + y²/9 = 1

---

**【答案】A**

**【解析】**

第一步：从焦点得 c

c = 3

第二步：从经过的点得 b

点 (0, 4) 在椭圆上：0/a² + 16/b² = 1 → b² = 16，b = 4

第三步：求 a

c² = a² - b² → 9 = a² - 16 → **a² = 25**

椭圆方程：**x²/25 + y²/16 = 1**

**验证：** a = 5, b = 4, c = 3。代入点 (0, 4)：0 + 16/16 = 1 ✅

**怎么想到的：** 焦点在 x 轴上 → 椭圆长轴在 x 轴 → 标准方程 x²/a² + y²/b² = 1。"经过某点"就是代入方程，得到一个关于 a、b 的条件。

## 第7题 ⭐⭐ 中等（5分）选择题

**双曲线 x²/a² - y²/b² = 1 (a > 0, b > 0) 的渐近线方程为 y = ±2x，且过点 (3, 0)，则双曲线方程是？**

A. x²/9 - y²/36 = 1  
B. x²/36 - y²/9 = 1  
C. x²/9 - y²/16 = 1  
D. x²/16 - y²/9 = 1

---

**【答案】A**

**【解析】**

第一步：从经过的点求 a

(3, 0) 代入 x²/a² - y²/b² = 1：9/a² = 1 → **a² = 9**，a = 3

第二步：从渐近线求 b

渐近线 y = ±(b/a)x = ±2x → b/a = 2 → b = 2a = 6，**b² = 36**

双曲线方程：**x²/9 - y²/36 = 1**

**怎么想到的：** 经过点 (3, 0) 是最简单的——y = 0，只剩 x²/a² = 1，直接得出 a = 3。渐近线给出 b/a 的比值，有了 a 就能算 b。

## 第8题 ⭐⭐ 中等（5分）选择题

**下列关于圆锥曲线的说法，正确的有几项？**

① 椭圆的离心率满足 0 < e < 1  
② 双曲线满足 c² = a² + b²  
③ 抛物线 y² = 2px (p > 0) 的焦点在 x 轴上  
④ 椭圆 x²/a² + y²/b² = 1 (a > b > 0) 的焦点在 y 轴上

A. 2 项  
B. 3 项  
C. 4 项  
D. 1 项

---

**【答案】B**

**【解析】**

逐项分析：

**① 椭圆的离心率满足 0 < e < 1？**
对。c < a（因为 c² = a² - b² 且 b > 0），所以 e = c/a < 1。✅

**② 双曲线满足 c² = a² + b²？**
对。这是双曲线的基本关系。✅

**③ 抛物线 y² = 2px (p > 0) 的焦点在 x 轴上？**
对。焦点 (p/2, 0) 在 x 轴上。✅

**④ 椭圆 x²/a² + y²/b² = 1 (a > b > 0) 的焦点在 y 轴上？**
错。a > b > 0，a 在 x 下面（x²/a²），所以**长轴在 x 轴上，焦点在 x 轴上**。❌

正确的有 ①②③ 共 **3 项**。

**怎么想到的：** ④的陷阱在于：x 下面是 a²（大的那个）还是 b²（小的那个），决定了焦点在哪个轴。a > b > 0 且 x 下面是 a² → 焦点在 x 轴。

## 第9题 ⭐⭐⭐ 挑战（5分）选择题

**已知椭圆 x²/25 + y²/m² = 1 与双曲线 x²/9 - y²/m² = 1 有相同的焦点，则 m = ？**

A. 2√2  
B. 2  
C. 4  
D. 3

---

**【答案】A**

**【解析】**

**第一步：分别写出两种曲线的 c²**

椭圆：c₁² = a₁² - b₁² = 25 - m²

双曲线：c₂² = a₂² + b₂² = 9 + m²

**第二步：利用"相同焦点"列方程**

c₁² = c₂²

25 - m² = 9 + m²

25 - 9 = 2m²

16 = 2m²

**m² = 8**，m = 2√2（m > 0）

**验证：** 椭圆 c² = 25 - 8 = 17，双曲线 c² = 9 + 8 = 17。✅ 相同。

**怎么想到的：** "相同焦点"意味着 c 值相同。椭圆的 c² = a² - b²（减法），双曲线的 c² = a² + b²（加法），令它们相等就能解出 m。注意两种曲线共用同一个 b²（都是 m²）。

## 第10题 ⭐⭐⭐ 挑战（5分）选择题

**椭圆 x²/4 + y² = 1 的左焦点为 F₁，右焦点为 F₂，P 为椭圆上一点，且 PF₁ ⊥ PF₂，则 △F₁PF₂ 的面积是？**

A. 1/2  
B. 1  
C. √3  
D. 2

---

**【答案】B**

**【解析】**

**第一步：求椭圆的基本量**

a² = 4，a = 2；b² = 1，b = 1

c² = a² - b² = 4 - 1 = 3，c = √3

F₁(-√3, 0)，F₂(√3, 0)

**第二步：利用 PF₁ ⊥ PF₂ 建关系**

设 |PF₁| = d₁，|PF₂| = d₂

PF₁ ⊥ PF₂ → △F₁PF₂ 是直角三角形（直角在 P）

由勾股定理：d₁² + d₂² = |F₁F₂|² = (2√3)² = **12**

椭圆性质：d₁ + d₂ = 2a = **4**

**第三步：求面积**

(d₁ + d₂)² = d₁² + 2d₁d₂ + d₂²

16 = 12 + 2d₁d₂

d₁d₂ = 2

面积 = d₁d₂/2 = **1**

**验证：** d₁² + d₂² = 12，d₁ + d₂ = 4
(d₁ + d₂)² = 16 = d₁² + d₂² + 2d₁d₂ = 12 + 2d₁d₂ → d₁d₂ = 2
t² - 4t + 2 = 0，Δ = 16 - 8 = 8 > 0，有实数解。✅

**怎么想到的：** 椭圆上一点 P 到两焦点 F₁、F₂ 的连线构成焦点三角形。PF₁ ⊥ PF₂ → 用勾股定理 + 椭圆定义（d₁ + d₂ = 2a）联立求解。"焦点三角形"是椭圆的经典考法。

## 复习题（交叉练习）

### 复1
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 15-导数第一问-切线方程

**题目：** 曲线 $y = x^3$ 在 $x = 1$ 处的切线方程为（　）

A. $y = 3x - 2$　　B. $y = 3x + 2$　　C. $y = 3x$　　D. $y = x + 2$

**答案：** A

**解析：**

切点：$(1, f(1)) = (1, 1)$

斜率：$f'(x) = 3x^2$，$f'(1) = 3$

切线方程：$y - 1 = 3(x - 1)$，即 $y = 3x - 2$

**验证：** 切线过 $(1, 1)$：$3\times 1 - 2 = 1$ ✓；斜率 = 导数 ✓

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 14-平面向量-模的运算

**题目：** 已知 $|\vec{a}| = 3$，$|\vec{b}| = 4$，$\vec{a}$ 与 $\vec{b}$ 的夹角为 $60°$，则 $|2\vec{a} - \vec{b}| =$（　）

A. $2\sqrt{7}$　　B. $\sqrt{28}$　　C. $10$　　D. $2\sqrt{10}$

**答案：** A

**解析：**

$|2\vec{a} - \vec{b}|^2 = 4|\vec{a}|^2 + |\vec{b}|^2 - 4\vec{a}\cdot\vec{b}$

$\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos 60° = 3 \times 4 \times \dfrac{1}{2} = 6$

$|2\vec{a} - \vec{b}|^2 = 4 \times 9 + 16 - 4 \times 6 = 36 + 16 - 24 = 28$

$|2\vec{a} - \vec{b}| = 2\sqrt{7}$

**验证：** $2\sqrt{7} \approx 5.29$，介于 $2|\vec{a}|-|\vec{b}|=2$ 和 $2|\vec{a}|+|\vec{b}|=10$ 之间，合理 ✓

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 10-解三角形-面积与余弦定理

**题目：** 在 $\triangle ABC$ 中，$a = \sqrt{3}$，$B = 60°$，$\triangle ABC$ 的面积为 $\dfrac{\sqrt{3}}{2}$，求边 $b$。

**答案：** $b = \dfrac{\sqrt{21}}{3}$

**解析：**

由面积公式 $S = \dfrac{1}{2}ac\sin B$：

$\dfrac{\sqrt{3}}{2} = \dfrac{1}{2} \times \sqrt{3} \times c \times \dfrac{\sqrt{3}}{2}$

$\dfrac{\sqrt{3}}{2} = \dfrac{3c}{4}$

$c = \dfrac{2\sqrt{3}}{3}$

由余弦定理：

$b^2 = a^2 + c^2 - 2ac\cos B = 3 + \dfrac{4}{3} - 2 \times \sqrt{3} \times \dfrac{2\sqrt{3}}{3} \times \dfrac{1}{2} = 3 + \dfrac{4}{3} - 2 = 1 + \dfrac{4}{3} = \dfrac{7}{3}$

$b = \sqrt{\dfrac{7}{3}} = \dfrac{\sqrt{21}}{3}$

**验证：** $b^2 = 7/3 \approx 2.33$，$b \approx 1.53$。$a \approx 1.73$，$c \approx 1.15$，$B = 60°$，$S = \dfrac{1}{2}\times 1.73\times 1.15\times 0.866 \approx 0.86 \approx \dfrac{\sqrt{3}}{2}$ ✓
