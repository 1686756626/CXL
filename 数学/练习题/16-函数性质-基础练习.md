## 第1题 ⭐ 基础（5分）选择题

**判断函数 f(x) = x³ 的奇偶性。**

A. 奇函数  
B. 偶函数  
C. 非奇非偶  
D. 既是奇函数又是偶函数

---

**【答案】A**

**【解析】**

直接代入 -x 验证：

f(-x) = (-x)³ = -x³ = -f(x)

满足 f(-x) = -f(x)，所以 f(x) = x³ 是**奇函数**。

**怎么想到的：** 判断奇偶性就三步：①定义域对称（R显然对称）→ ②算 f(-x) → ③和 f(x) 或 -f(x) 比较。这题 f(-x) 算出来正好等于 -f(x)，一步到位。

## 第2题 ⭐ 基础（5分）选择题

**函数 f(x) = 2x + 1 的单调性是？**

A. 在 R 上单调递增  
B. 在 R 上单调递减  
C. 在 (-∞,0) 递增，(0,+∞) 递减  
D. 在 (-∞,0) 递减，(0,+∞) 递增

---

**【答案】A**

**【解析】**

求导：f'(x) = 2

因为 f'(x) = 2 > 0 恒成立，所以 f(x) 在 R 上**单调递增**。

**怎么想到的：** 判断单调性最快的方法就是求导。f'(x) > 0 就递增，f'(x) < 0 就递减。一次函数的导数就是斜率，斜率为正当然递增。

## 第3题 ⭐ 基础（5分）选择题

**下列函数中，图像关于 y 轴对称的是？**

A. f(x) = x  
B. f(x) = x²  
C. f(x) = x³  
D. f(x) = 1/x

---

**【答案】B**

**【解析】**

逐一验证：

- A. f(x) = x：f(-x) = -x = -f(x)，奇函数，关于**原点**对称。❌
- B. f(x) = x²：f(-x) = (-x)² = x² = f(x)，偶函数，关于**y 轴**对称。✅
- C. f(x) = x³：f(-x) = -x³ = -f(x)，奇函数，关于**原点**对称。❌
- D. f(x) = 1/x：f(-x) = -1/x = -f(x)，奇函数，关于**原点**对称。❌

**怎么想到的：** "关于 y 轴对称"就是偶函数。偶函数满足 f(-x) = f(x)，代进去看看等不等于 f(x) 就行。

## 第4题 ⭐⭐ 中等（5分）选择题

**已知 f(x) 是定义在 R 上的奇函数，在 (0,+∞) 上单调递减，f(2) = 5，则 f(-3) 与 f(-1) 的大小关系是？**

A. f(-3) > f(-1)  
B. f(-3) < f(-1)  
C. f(-3) = f(-1)  
D. 无法比较

---

**【答案】A**

**【解析】**

奇函数在对称区间上单调性相同：f(x) 在 (0,+∞) 递减 → f(x) 在 (-∞,0) 也递减。

在 (-∞,0) 上：-3 < -1，递减函数中大数对应小值，

所以 f(-3) > f(-1)。

**验证：** 也可以用奇函数性质转换：f(-3) = -f(3)，f(-1) = -f(1)。
在 (0,+∞) 递减：3 > 1 → f(3) < f(1) → -f(3) > -f(1) → f(-3) > f(-1)。✅

**怎么想到的：** 奇函数比较大小有两个思路：①利用"对称区间单调性相同"直接在负半轴上比；②用 f(-x) = -f(x) 转到正半轴上比。两种方法殊途同归。

## 第5题 ⭐⭐ 中等（5分）选择题

**判断分段函数 f(x) 的奇偶性：**

$$f(x) = \begin{cases} x^2 + x, & x \geq 0 \\ x^2 - x, & x < 0 \end{cases}$$

A. 奇函数  
B. 偶函数  
C. 非奇非偶  
D. 既是奇函数又是偶函数

---

**【答案】B**

**【解析】**

分段函数需要分段验证 f(-x) 是否等于 f(x)。

**当 x ≥ 0 时：** -x ≤ 0，用第二段公式：

f(-x) = (-x)² - (-x) = x² + x = f(x) ✅

**当 x < 0 时：** -x > 0，用第一段公式：

f(-x) = (-x)² + (-x) = x² - x = f(x) ✅

两段都满足 f(-x) = f(x)，所以 f(x) 是**偶函数**。

**怎么想到的：** 分段函数判断奇偶性的关键：x 在哪一段，-x 就在另一段（或同一段），要用对应的公式算 f(-x)。不能只看一个分段。

## 第6题 ⭐⭐ 中等（5分）选择题

**f(x) = x³ - 3x 的单调递减区间是？**

A. (-∞, -1)  
B. (-1, 1)  
C. (1, +∞)  
D. (-∞, -1) ∪ (1, +∞)

---

**【答案】B**

**【解析】**

第一步：求导

f'(x) = 3x² - 3 = 3(x + 1)(x - 1)

第二步：解 f'(x) = 0

3(x + 1)(x - 1) = 0 → x = -1 或 x = 1

第三步：判断正负

| 区间 | (-∞, -1) | (-1, 1) | (1, +∞) |
|------|-----------|---------|-----------|
| x + 1 | - | + | + |
| x - 1 | - | - | + |
| f'(x) | + | - | + |

f'(x) < 0 的区间是 **(-1, 1)**，即单调递减区间。

**怎么想到的：** 求单调区间就是"求导→找零点→看正负"三步。遇到二次导数（抛物线），两个根之间为负（抛物线开口向上时），两侧为正。

## 第7题 ⭐⭐ 中等（5分）选择题

**已知 f(x) 是定义在 R 上的偶函数，在 (-∞, 0] 上单调递增，且 f(3) = 1，则不等式 f(x) ≤ 1 的解集是？**

A. [-3, 3]  
B. (-∞, -3] ∪ [3, +∞)  
C. [0, 3]  
D. (-∞, 3]

---

**【答案】B**

**【解析】**

**第一步：确定各区间的单调性**

偶函数在对称区间单调性相反：
- (-∞, 0] 递增 → (0, +∞) **递减**

**第二步：分别在两个区间解不等式**

f(x) ≤ 1 = f(3)

**在 (0, +∞) 上递减：** f(x) ≤ f(3) → x ≥ 3

**在 (-∞, 0) 上：** 偶函数 f(x) = f(-x)，所以 f(x) ≤ 1 即 f(-x) ≤ 1 = f(3)。
在 (0, +∞) 递减：f(-x) ≤ f(3) → -x ≥ 3 → x ≤ -3

**第三步：检查分界点 x = 0**

f(x) 在 (-∞, 0] 递增，f(-3) = f(3) = 1。
当 -3 < x ≤ 0 时，f(x) ≥ f(-3) = 1，即 f(x) ≥ 1，不满足 f(x) ≤ 1。

**综合解集：** (-∞, -3] ∪ [3, +∞)

**怎么想到的：** 偶函数解不等式的套路：①先确定对称区间的单调性（相反）；②分别在正半轴和负半轴上解；③负半轴用 f(x) = f(-x) 转化到正半轴处理。

## 第8题 ⭐⭐ 中等（5分）选择题

**下列关于函数奇偶性的说法，正确的有几项？**

① 奇函数的图像一定过原点  
② 偶函数的定义域一定关于原点对称  
③ 若 f(x) 是奇函数，则 f(0) = 0（0 在定义域内时）  
④ 若 f(-x) = -f(x)，则 f(x) 一定是奇函数

A. 2 项  
B. 3 项  
C. 4 项  
D. 1 项

---

**【答案】B**

**【解析】**

逐项分析：

**① 奇函数的图像一定过原点？**
不一定。f(x) = 1/x 是奇函数，但定义域不含 0，图像不过原点。❌

**② 偶函数的定义域一定关于原点对称？**
对。定义域关于原点对称是函数具有奇偶性的**前提条件**。✅

**③ 奇函数且 0 在定义域内，则 f(0) = 0？**
对。f(0) = f(-0) = -f(0) → 2f(0) = 0 → f(0) = 0。✅

**④ f(-x) = -f(x) 能推出 f(x) 是奇函数？**
对。只要 f(-x) = -f(x) 对定义域内所有 x 成立，定义域自然关于原点对称（x 能代，-x 也能代）。✅

正确的有 ②③④ 共 **3 项**。

**怎么想到的：** 这类判断题关键是抠细节。①的陷阱是"一定"——奇函数不一定过原点，因为 0 可能不在定义域内。③要注意"0 在定义域内时"这个前提条件。

## 第9题 ⭐⭐⭐ 挑战（5分）选择题

**已知 f(x) = x³ - ax，讨论 f(x) 的单调性，下列结论正确的是？**

A. a ≤ 0 时在 R 上递增；a > 0 时在 (-√(a/3), √(a/3)) 递减，其余递增  
B. a > 0 时在 R 上递增  
C. a < 0 时在 R 上递增；a ≥ 0 时有增有减  
D. a > 0 时在 (-√(a/3), √(a/3)) 递增，其余递减

---

**【答案】A**

**【解析】**

f'(x) = 3x² - a

**情况一：a ≤ 0**

f'(x) = 3x² - a = 3x² + |a| ≥ 0 恒成立（且只在 x = 0 时可能等于 0）。

所以 f(x) 在 **R 上单调递增**。

**情况二：a > 0**

f'(x) = 3(x² - a/3) = 3(x - √(a/3))(x + √(a/3))

令 f'(x) = 0：x = -√(a/3) 或 x = √(a/3)

| 区间 | (-∞, -√(a/3)) | (-√(a/3), √(a/3)) | (√(a/3), +∞) |
|------|----------------|---------------------|-----------------|
| f'(x) | + | - | + |

- 递增：(-∞, -√(a/3)) 和 (√(a/3), +∞)
- 递减：(-√(a/3), √(a/3))

**验证：** 取 a = 3，f'(x) = 3x² - 3 = 3(x+1)(x-1)。
递减区间 (-1, 1)，-√(3/3) = -1，√(3/3) = 1。公式正确。✅

**怎么想到的：** 含参单调性讨论的核心是：导数中出现参数 → f'(x) = 0 的根的个数和大小可能随参数变化 → 需要分类讨论。关键分界点是导数判别式为零时。

## 第10题 ⭐⭐⭐ 挑战（5分）选择题

**已知 f(x) 是定义在 R 上的奇函数，在 (0,+∞) 上单调递减，f(x) > 0 的解集为 (0, 2)，则不等式 f(x² - 4x + 3) > 0 的解集是？**

A. (1, 3)  
B. (2 - √3, 2 + √3)  
C. (2 - √3, 1) ∪ (3, 2 + √3)  
D. (-∞, -2) ∪ (0, 2)

---

**【答案】C**

**【解析】**

**第一步：确定 f(x) > 0 的完整解集**

奇函数 f(0) = 0。在 (0,+∞) 递减：
- (0, 2)：f(x) > 0
- x = 2：f(2) = 0
- (2, +∞)：f(x) < 0

奇函数在 (-∞, 0) 也递减（对称区间单调性相同）：
- (-2, 0)：f(x) = -f(-x)，-x ∈ (0,2)，f(-x) > 0，所以 f(x) < 0
- x = -2：f(-2) = -f(2) = 0
- (-∞, -2)：f(x) = -f(-x)，-x ∈ (2,+∞)，f(-x) < 0，所以 f(x) > 0

所以 **f(x) > 0 的解集 = (-∞, -2) ∪ (0, 2)**

**第二步：解 f(x² - 4x + 3) > 0**

即 x² - 4x + 3 ∈ (-∞, -2) ∪ (0, 2)

**条件①：x² - 4x + 3 < -2**

x² - 4x + 5 < 0，判别式 Δ = 16 - 20 = -4 < 0，**无解**。

**条件②：0 < x² - 4x + 3 < 2**

下半部分：x² - 4x + 3 > 0 → (x-1)(x-3) > 0 → x < 1 或 x > 3

上半部分：x² - 4x + 3 < 2 → x² - 4x + 1 < 0

x² - 4x + 1 = 0 的根：x = (4 ± √12)/2 = 2 ± √3

所以 2 - √3 < x < 2 + √3

**取交集：** ((−∞, 1) ∪ (3, +∞)) ∩ (2-√3, 2+√3) = **(2-√3, 1) ∪ (3, 2+√3)**

**验证：** 取 x = 0.5：x²-4x+3 = 0.25-2+3 = 1.25 ∈ (0,2) ✅
取 x = 3.5：x²-4x+3 = 12.25-14+3 = 1.25 ∈ (0,2) ✅
取 x = 2：x²-4x+3 = 4-8+3 = -1 ∉ 解集 ✅（不在答案中）

**怎么想到的：** 这类抽象函数不等式的套路：①先根据奇偶性+单调性画出 f(x) > 0 的完整解集（不要忘记负半轴）；②把不等式转化为 t ∈ 解集（令 t = x²-4x+3）；③解关于 t 的不等式，再解关于 x 的不等式。

## 复习题（交叉练习）

### 复1
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 12-立体几何第一问-线面垂直

**题目：** 在四棱锥 $P-ABCD$ 中，底面 $ABCD$ 是矩形，$AB \perp AD$，$AB=4$，$AD=3$，$PA \perp$ 底面 $ABCD$，$PA=6$。求三棱锥 $P-ABD$ 的体积。

**答案：** $V = 12$

**解析：**

$V_{P-ABD} = \dfrac{1}{3} \times S_{\triangle ABD} \times PA$

$S_{\triangle ABD} = \dfrac{1}{2} \times AB \times AD = \dfrac{1}{2} \times 4 \times 3 = 6$

$V = \dfrac{1}{3} \times 6 \times 6 = 12$

**验证：** 也可以先算 $S_{\triangle PAD} = \dfrac{1}{2}\times 3\times 6 = 9$，$V = \dfrac{1}{3}\times 9\times 4 = 12$ ✓（换底等体积）

### 复2
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 10-解三角形-正弦定理

**题目：** 在 $\triangle ABC$ 中，$a = 2$，$b = 2\sqrt{3}$，$A = 30°$，求角 $B$。

**答案：** $B = 60°$ 或 $B = 120°$

**解析：**

由正弦定理：$\dfrac{a}{\sin A} = \dfrac{b}{\sin B}$

$\sin B = \dfrac{b \sin A}{a} = \dfrac{2\sqrt{3} \times \dfrac{1}{2}}{2} = \dfrac{\sqrt{3}}{2}$

$\sin B = \dfrac{\sqrt{3}}{2}$ 在 $(0°, 180°)$ 内有两个解：$B = 60°$ 或 $B = 120°$。

**验证：**
- $B = 60°$：$C = 90°$，$c = \sqrt{4+12} = 4$，勾股定理 $2^2+(2\sqrt{3})^2=16=4^2$ ✓
- $B = 120°$：$C = 30°$，$a = c$，等腰三角形 ✓

### 复3
**难度：** ⭐⭐ 中等（5分）
**复习知识点：** 基本不等式-求最值

**题目：** 已知正实数 $x$、$y$ 满足 $xy = 4$，则 $x + 2y$ 的最小值为（　）

A. $2\sqrt{2}$　　B. $4\sqrt{2}$　　C. $8$　　D. $6$

**答案：** B

**解析：**

$x + 2y = x + \dfrac{2 \times 4}{x} = x + \dfrac{8}{x}$

由基本不等式（$x > 0$）：

$x + \dfrac{8}{x} \geq 2\sqrt{x \cdot \dfrac{8}{x}} = 2\sqrt{8} = 4\sqrt{2}$

当且仅当 $x = \dfrac{8}{x}$ 即 $x = 2\sqrt{2}$，$y = \dfrac{4}{2\sqrt{2}} = \sqrt{2}$ 时取等号。

**验证：** $x = 2\sqrt{2}$，$y = \sqrt{2}$，$x + 2y = 2\sqrt{2} + 2\sqrt{2} = 4\sqrt{2}$，$xy = 2\sqrt{2}\times\sqrt{2}=4$ ✓
