## 一、单选题（每题5分，共30分）

### 第1题 【答案】B

$A = (-1, 2]$，$B = [0, 3)$

- 左端点：取较大者 $0$，$B$ 在 $0$ 处闭 → $[0$
- 右端点：取较小者 $2$，$A$ 在 $2$ 处闭 → $2]$

**$A ∩ B = [0, 2]$**，选 B。

### 第2题 【答案】B

$B = \{x \mid x^2 - 5x + 6 = 0\} = \{2, 3\}$

$A ∪ B = \{1, 2, 3\} ∪ \{2, 3\} = \{1, 2, 3\}$

选 B。

### 第3题 【答案】B

- A: $\log_3 9 = 2$ 对应 $3^2 = 9$，不是 $2^3 = 9$ ✗
- B: $2^{-3} = \dfrac{1}{8} \Leftrightarrow \log_2 \dfrac{1}{8} = -3$ ✓
- C: $\log_2 8 = 3$ 对应 $2^3 = 8$，不是 $3^2 = 8$ ✗
- D: $5^2 = 25 \Leftrightarrow \log_5 25 = 2$，不是 $\log_5 2 = 25$ ✗

选 **B**。

### 第4题 【答案】C（$2^{30} < 3^{20}$）

化同底：$2^{30} = (2^3)^{10} = 8^{10}$，$3^{20} = (3^2)^{10} = 9^{10}$

因为 $8 < 9$，所以 $8^{10} < 9^{10}$

因此 $2^{30} < 3^{20}$，选 **C**。

### 第5题 【答案】B

$y = 2^x$ 向左平移1个单位 → $y = 2^{x+1}$（左加）

再向下平移2个单位 → $y = 2^{x+1} - 2$（下减）

选 **B**。

### 第6题 【答案】B

$\log_2(x - 1) > 1 = \log_2 2$

底数 $2 > 1$，单调递增：$x - 1 > 2$，$x > 3$

选 **B**。

## 二、多选题（每题6分，共24分。全部选对得6分，部分选对得3分，有错选得0分）

### 第7题 【答案】AB

- A: $\varnothing \subseteq S$ 对任意集合 $S$ 成立 ✓
- B: $\varnothing \subsetneq S$ 当 $S \neq \varnothing$ ✓
- C: $\{0\}$ 含一个元素，不是空集 ✗
- D: $A = \{1\}, B = \{2\}$，$A ∩ B = \varnothing$ 但 $A \neq \varnothing, B \neq \varnothing$ ✗

选 **A、B**。

### 第8题 【答案】ABCD

- A: $x > 0 \Rightarrow x \neq 0 \Rightarrow x^2 > 0$ ✓（充分条件）
- B: $x = y \Rightarrow x^2 = y^2$ ✓（充分条件）
- C: $a > 1 \Leftrightarrow 2^a > 2$（$y = 2^x$ 单调递增）✓（充要条件）
- D: $x > 1 \Leftrightarrow \log_2 x > 0$（$\log_2 x$ 单调递增）✓（充要条件）

选 **A、B、C、D**。

### 第9题 【答案】ABC

- A: $x - 1 > 0 \Rightarrow x > 1$，定义域 $(1, +\infty)$ ✓
- B: $y = \log_2 x$ 右移1个 → $y = \log_2(x-1)$，上移1个 → $y = \log_2(x-1) + 1$ ✓
- C: $x = 2$ 时 $y = \log_2 1 + 1 = 1$，过点 $(2, 1)$ ✓
- D: 底数 $2 > 1$，单调递增，不是递减 ✗

选 **A、B、C**。

### 第10题 【答案】ABC

$a = 0.3^2 = 0.09$

$b = 2^{0.3}$，$\because 2^0 = 1 < 2^{0.3} < 2^1 = 2$，$\therefore 1 < b < 2$

$c = \log_2 0.3$，$\because 0 < 0.3 < 0.5 = 2^{-1}$，$\therefore c < -1$

大小：$c < -1 < 0.09 = a < 1 < b$

- A: $c < a$ ✓
- B: $a < b$ ✓
- C: $c < 1$ ✓
- D: $a > 1$ ✗（$a = 0.09 < 1$）

选 **A、B、C**。

## 三、填空题（每题5分，共15分）

### 第11题 【答案】$b - a$

$3^a = 5$，$3^b = 10 = 2 \times 5$

$b = \log_3 10 = \log_3 2 + \log_3 5 = \log_3 2 + a$

所以 $\log_3 2 = b - a$

### 第12题 【答案】$(-\infty, \dfrac{3-\sqrt{17}}{2}) \cup (\dfrac{3+\sqrt{17}}{2}, +\infty)$

$2^{x^2 - 3x} > 4 = 2^2$

底数 $2 > 1$，单调递增：$x^2 - 3x > 2$

$x^2 - 3x - 2 > 0$

$\Delta = 9 + 8 = 17$，$x = \dfrac{3 \pm \sqrt{17}}{2}$

所以 $x < \dfrac{3 - \sqrt{17}}{2}$ 或 $x > \dfrac{3 + \sqrt{17}}{2}$

### 第13题 【答案】$[1, 3)$

- 左端点：取较大者 $1$，$A$ 在 $1$ 处闭 → $[1$
- 右端点：取较小者 $3$，$B$ 在 $3$ 处开 → $3)$

$A ∩ B = [1, 3)$

## 四、解答题（第14题10分，第15题12分）

### 第14题 【指数对数综合】

**（1）**（4分）

$\log_2 8 + \log_2 4 = \log_2(8 \times 4) = \log_2 32 = 5$

> 因为 $2^5 = 32$。

**（2）**（6分）

$\log_2 20 = \log_2(4 \times 5) = \log_2 4 + \log_2 5 = 2 + \log_2 5$

因为 $2^x = 5$，所以 $\log_2 5 = x$

因此 $\log_2 20 = 2 + x$

### 第15题 【含参不等式】

**（1）**（3分）

要使 $\log_a(x - 1)$ 有意义：$x - 1 > 0$，即 $x > 1$

所以定义域为 $(1, +\infty)$。

**（2）**（4分）

$a = 2$ 时，$f(x) = \log_2(x - 1) \leq 2 = \log_2 4$

底数 $2 > 1$，单调递增：$x - 1 \leq 4$，$x \leq 5$

结合定义域：$1 < x \leq 5$

解集为 $(1, 5]$。

**（3）**（5分）

$0 < a < 1$ 时，$f(x) = \log_a(x - 1) > 0 = \log_a 1$

底数 $0 < a < 1$，单调递减：$x - 1 < 1$，$x < 2$

结合定义域：$1 < x < 2$

解集为 $(1, 2)$。
