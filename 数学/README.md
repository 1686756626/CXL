### 学习顺序

```
第一阶段（01-10）→ 阶段一小测（80分/45分钟）
第二阶段（11-12, 16-17）→ 阶段二小测（90分/55分钟）
第三阶段（13-15, 18-19）→ 阶段三小测（100分/65分钟）
第四阶段 → 综合模拟 → 高考
```

详细顺序见 [`学习指南.md`](学习指南.md)，详细每日安排见 [`每日学习计划.md`](每日学习计划.md)。
