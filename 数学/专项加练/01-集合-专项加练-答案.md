# 01-集合 专项加练 — 答案解析

## 第一部分：求根公式

### 加1 x² - 7x + 10 = 0
a=1, b=-7, c=10
判别式 = (-7)² - 4×1×10 = 49 - 40 = 9
x = (7 ± 3) / 2
**答案：x = 5 或 x = 2**

### 加2 x² + 3x - 4 = 0
a=1, b=3, c=-4
判别式 = 3² - 4×1×(-4) = 9 + 16 = 25
x = (-3 ± 5) / 2
**答案：x = 1 或 x = -4**

### 加3 x² - 6x + 9 = 0
a=1, b=-6, c=9
判别式 = (-6)² - 4×1×9 = 36 - 36 = 0
x = 6 / 2 = 3
**答案：x = 3（只有一个解，判别式为0）**

### 加4 2x² - 5x - 3 = 0
a=2, b=-5, c=-3
判别式 = (-5)² - 4×2×(-3) = 25 + 24 = 49
x = (5 ± 7) / 4
**答案：x = 3 或 x = -1/2**

### 加5 x² - 3x - 4 > 0
先求根：a=1, b=-3, c=-4
判别式 = 9 + 16 = 25
x = (3 ± 5) / 2 → x=4 或 x=-1
开口向上，>0 取两根之外
**答案：x < -1 或 x > 4**

## 第二部分：区间交集

### 加6
A = (-2, 4]，B = [1, 6)
数轴上重叠部分：从1到4
A含4（闭），B不含6但包含大于4的
左端点取大的：B含1 → 含1
右端点取小的：A含4 → 含4
**答案：A ∩ B = [1, 4]**

### 加7
A = [0, 5]，B = (-3, 3)
数轴上重叠部分：从0到3
左端点：A含0 → 含0
右端点：B不含3 → 不含3
**答案：A ∩ B = [0, 3)**

### 加8
分两步：
① A = [-1, 5)，B = (2, 7]
A ∩ B = (2, 5)：B不含2所以不含2，A不含5所以不含5

② A ∪ B = [-1, 7]
∁ᵤ(A ∪ B) = (-∞, -1] ∪ (7, +∞)

③ A ∩ B ∪ ∁ᵤ(A ∪ B) = (2, 5) ∪ ((-∞, -1] ∪ (7, +∞))
**答案：(2, 5) ∪ (-∞, -1] ∪ (7, +∞)**

## 第三部分：含参不等式

### 加9
3x - a < 0 → 3x < a → x < a/3
A = (-∞, a/3)，B = (-∞, 2)
A ⊆ B → a/3 ≤ 2 → **a ≤ 6**

## 第四部分：空集

### 加10
A = {x | x² + 1 = 0}，x² = -1，无实数解，**A = ∅**
空集和任何集合的交集都是空集
**答案：A ∩ B = ∅**
