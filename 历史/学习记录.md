### 新文化运动与五四运动（4月7日）✅ 过关

**掌握情况：** 5题全对 ✅

```
历史：新文化运动与五四运动

新文化运动（1915年）：
- 标志：陈独秀创办《新青年》
- 口号：民主（德先生）、科学（赛先生）
- 三提倡：民主反专制、科学反迷信、新文学反旧文学
- 人物：陈独秀、胡适（白话文）、鲁迅（狂人日记）、李大钊（马克思主义）

五四运动（1919年）：
- 导火索：巴黎和会，山东权益转给日本
- 5月4日北京学生游行，"外争主权，内除国贼"
- 扩大：工人罢工、商人罢市
- 结果：拒签巴黎和约
- 意义：新民主主义革命开端，工人阶级登上政治舞台

变革链条：辛亥革命（制度）→ 新文化运动（思想）→ 五四运动（群众）
```

**滚动复习重点：**
- 1915新文化运动：陈独秀+《新青年》+民主科学
- 1919五四运动：巴黎和会→学生游行→工人罢工→新民主主义革命开端
- 变革趋势：制度→思想→群众

### 第4课：中共成立+国共合作+土地革命（4月9日）✅ 7/8过关

**教学内容：**
- 中共一大（1921·上海）：开天辟地，第一个马克思主义政党
- 第一次国共合作（1924-1927）：国民党一大→新三民主义→北伐→四一二政变破裂
- 土地革命（1927-1937）：南昌起义→井冈山→长征→遵义会议

**故事链：** 合作→北伐→叛变→起义→上山→被围→长征→转折

**测试记录：**
- 8道选择题
- 首轮：7/8（87.5%）✅ 过关
- 错误：第1题中共一大时间地点选了A（1920北京），实际是B（1921上海）
- 补充：容易跟五四运动（1919北京）混淆

**笔记本内容：**
```
历史第4课：中共成立+国共合作+土地革命

中共成立 1921·上海（中共一大）
  意义：开天辟地，第一个马克思主义政党

第一次国共合作 1924-1927
  标志：国民党一大（广州）→ 新三民主义
  成果：北伐战争，推翻北洋军阀
  破裂：1927 四一二政变（蒋介石叛变）

土地革命 1927-1937
  南昌起义1927.8.1 — 武装反抗第一枪
  井冈山 — 农村包围城市
  长征1934-1936 — 第五次反围剿失败
  遵义会议1935 — 确立毛泽东领导，生死攸关转折点

故事链：合作→北伐→叛变→起义→上山→被围→长征→转折
```

**薄弱点：** 中共一大时间地点（1921上海）容易跟五四运动（1919北京）混淆

**教学反馈：** 新灵喜欢故事链+联系小说剧情的讲解方式，后续历史课多用

### 第5课：抗日战争（4月13日）✅ 测试5/6（83%）

**抗战故事链：**
九一八(1931)→东北沦陷 → 七七事变(1937)→全面抗战开始 → 第二次国共合作 → 台儿庄(1938)→正面首次大捷 → 百团大战(1940)→敌后主动出击 → 日本投降(1945.8.15)

**两次国共合作对比：**
第一次(1924)=打军阀  第二次(1937)=打日本

**两个战场：** 正面（国民党：淞沪、台儿庄、武汉）+ 敌后（共产党：游击、百团大战）

**抗战胜利意义：** ①近代以来第一次完全胜利 ②提高国际地位 ③为民主革命奠定基础

**薄弱点：**
- 七七事变≠东北沦陷（东北在九一八就沦陷了，七七是打北平全面侵华）
- 九一八=局部侵略起点，七七=全面抗战开始
- 大测试遗留：辛亥革命功绩=推翻封建帝制，五四性质=反帝反封建

### 第6课：解放战争（4月14日）✅ 测试6/6满分

**解放战争故事链：**
重庆谈判(1945)→双十协定→蒋介石撕毁协议(1946)→内战爆发→刘邓挺进大别山(1947)→战略反攻→三大战役(1948-49)→渡江战役(1949.4)→解放南京→国民党统治覆灭→新中国成立(1949.10.1)

**三大战役：**
- 辽沈战役（东北全境解放）
- 淮海战役（规模最大，歼敌最多）
- 平津战役（和平解放北平）

**解放战争意义：** ①结束半殖民地半封建社会 ②结束国民党反动统治 ③新民主主义革命胜利

**教学反馈：** 用《某某》类比"明知被为难还是去了"，新灵理解重庆谈判的主动性
