## 二、考试概况

### 试卷结构
| 题型 | 题量 | 分值 | 新灵目标 |
|------|------|------|---------|
| 单选题 | 15-16题 | 45-48分 | 27分（对9个） |
| 非选择题（材料分析+小论文） | 3-4题 | 52-55分 | 33分 |
| **合计** | | **100分** | **60分卷面→赋分70+** |

### 考点分布
| 内容 | 占分 | 说明 |
|------|------|------|
| 中国古代史 | ~35分 | 政治制度演变+经济发展+思想文化 |
| 中国近现代史 | ~30分 | 列强侵略+救亡探索+革命建设 |
| 世界史 | ~25分 | 政治革命+工业革命+两次大战+冷战 |
| 选择性必修融合 | 融入以上 | 政治制度+经济贸易+文化交流 |

### 得分策略
- **选择题（45→27分）：** 时间定位→阶段特征→排除法。控制在15分钟内。
- **材料分析题：** "背景（政治+经济+思想）→ 内容/过程 → 影响/评价"
- **小论文题（12分必考）：** 三段论——论题+史实论证+总结升华

## 三、第一阶段：中国近现代史（8次课）

> 占分最大，新灵已学3个事件（辛亥革命、新文化运动、五四运动），接上继续。
> 教学方法：故事链+逻辑链，新灵对故事性内容接受好。

### 第1次课：列强侵略四战争（上）——鸦片战争+第二次鸦片战争

**教学目标：** 掌握两次鸦片战争的时间、条约、影响，建立"半殖民地半封建"概念

**教学内容：**

```
【鸦片战争】1840-1842
原因：英国为打开中国市场，倾销鸦片→林则徐虎门销烟→借口开战
条约：《南京条约》
  - 割香港岛
  - 赔款2100万银元
  - 开放广州、厦门、福州、宁波、上海（五口通商）
  - 协定关税
影响：中国开始沦为半殖民地半封建社会（近代史起点）

【第二次鸦片战争】1856-1860
原因：列强不满足既有权益，要求进一步开放
条约：《天津条约》+《北京条约》
  - 增开通商口岸（深入长江）
  - 外国人可内地游历、通商、传教
  - 赔款增加
  - 火烧圆明园（1860，英法联军）
影响：半殖民地化程度加深
      中外反动势力开始勾结
```

**记忆链：** 英国卖鸦片→林则徐禁烟→英国开战→签《南京条约》→列强不满足→再打→圆明园被烧→更深陷入

**过关标准：** 能说出两次战争的条约核心内容和"半殖民地半封建"的含义，做3道选择题

**笔记本内容：**
```
历史·列强侵略（上）
鸦片战争1840→《南京条约》→开始沦为半殖民地半封建
第二次鸦片1856→《天津/北京条约》→半殖民地化加深
关键词：五口通商、割香港、协定关税、火烧圆明园
```

---

### 第2次课：列强侵略四战争（下）——甲午战争+八国联军

**教学内容：**

```
【甲午中日战争】1894-1895
原因：日本明治维后对外扩张，朝鲜问题引发
条约：《马关条约》
  - 割辽东半岛、台湾、澎湖列岛
  - 赔款白银2亿两
  - 开放沙市、重庆、苏州、杭州
  - 允许日本在通商口岸设厂（资本输出开始！）
影响：大大加深半殖民地化
      刺激列强瓜分中国
      民族意识开始普遍觉醒

【八国联军侵华】1900
原因：义和团运动威胁列强利益
条约：《辛丑条约》
  - 赔款4.5亿两（本息9.8亿）
  - 设使馆区（东交民巷），中国人不得居住
  - 拆毁大沽炮台，允许外国驻军
  - 严禁中国人民参加反帝活动
影响：完全沦为半殖民地半封建社会
```

**四战争升级链：**
```
鸦片战争→开始沦为
第二次鸦片→加深
甲午战争→大大加深
八国联军→完全沦为
（一次比一次惨）
```

**过关标准：** 能按顺序说出四战争递进关系，能区分每个条约的特殊条款

**笔记本内容：**
```
历史·列强侵略（下）
甲午1894→《马关条约》→割台湾+设厂（资本输出）→大大加深
八国联军1900→《辛丑条约》→赔巨款+使馆区+严禁反帝→完全沦为
递进：开始→加深→大大加深→完全（一次比一次惨）
```

### 第3次课：救亡探索五运动——洋务运动+戊戌变法+太平天国+义和团

**教学目标：** 梳理各阶层救亡探索，建立"谁在探索→怎么探索→为什么失败"的逻辑链

**教学内容：**

```
探索逻辑：面对侵略，不同阶层想出不同办法——

【地主阶级抵抗派】
林则徐、魏源："师夷长技以制夷"——学技术，但没真正实施

【地主阶级洋务派】洋务运动（1860s-1890s）
口号："自强""求富"
内容：办军事工业（江南制造总局）+民用工业（轮船招商局）+建新式海军+办学堂
结果：甲午战败，北洋水师覆灭→洋务运动失败
评价：引进了西方技术，刺激了民族资本主义，但没有触动封建制度（中体西用）

【农民阶级】太平天国（1851-1864）
纲领：《天朝田亩制度》（平均主义，空想）+《资政新篇》（资本主义方案，未实施）
局限：农民阶级的局限性，没有科学理论指导

【农民阶级】义和团（1899-1901）
口号："扶清灭洋"
评价：反帝爱国，但盲目排外，被清政府利用后出卖

【资产阶级维新派】戊戌变法（1898）
核心：君主立宪
代表：康有为、梁启超
结果：只维持103天（百日维新），被慈禧太后镇压
教训：改良道路在半殖民地中国走不通

【资产阶级革命派】辛亥革命（1911）——已学
```

**故事链记忆：**
```
打不过→"学技术！"→洋务运动→还是打不过（甲午）
       →"要变法！"→戊戌变法→被太后镇压（103天）
       →"要革命！"→辛亥革命→推翻清朝但被袁世凯偷家
       →"要启蒙！"→新文化运动→改变思想
       →"要群众！"→五四运动→全民觉醒

规律：从学技术→学制度→学思想→动员群众，一步步深入
```

**过关标准：** 能说出5次探索的先后顺序、各自主张和失败原因

### 第4次课：国共关系与革命（上）——建党+国民革命+土地革命

**教学内容：**

```
【中共成立】1921·上海·中共一大
意义：中国有了以马克思主义为指导的无产阶级政党
1922·中共二大：第一次提出反帝反封建的民主革命纲领

【第一次国共合作】1924-1927
背景：反帝反封建的共同目标
标志：国民党一大（1924，广州）→ 新三民主义
成果：北伐战争（1926-1928），基本推翻北洋军阀
破裂：1927蒋介石发动"四一二"政变，屠杀共产党人

【土地革命】1927-1937
南昌起义（1927.8.1）：武装反抗第一枪
秋收起义→井冈山革命根据地：农村包围城市道路
长征（1934-1936）：第五次反围剿失败→战略转移
遵义会议（1935）：确立毛泽东领导地位→生死攸关的转折点
长征胜利（1936.10）：三大主力会师
```

**过关标准：** 能说出国共第一次合作→破裂→土地革命的故事链

---

### 第5次课：国共关系与革命（下）——抗日战争+解放战争

**教学内容：**

```
【抗日战争】1931-1945
1931·九一八事变→东北沦陷（14年抗战起点）
1937·七七事变→全面抗战开始
第二次国共合作：抗日民族统一战线
正面战场：淞沪会战、台儿庄战役、武汉会战
敌后战场：共产党领导的游击战争
1945·日本投降→抗战胜利

抗战胜利的意义：
(1) 近代以来第一次完全胜利的民族解放战争
(2) 提高了中国的国际地位
(3) 为民主革命胜利奠定基础

【解放战争】1946-1949
重庆谈判（1945）→政协会议→蒋介石撕毁协议→内战爆发
1947·刘邓大军挺进大别山→战略反攻
1948-49·三大战役（辽沈、淮海、平津）→消灭国民党主力
1949·渡江战役→占领南京→国民党统治覆灭
1949.10.1·中华人民共和国成立
```

**过关标准：** 能区分两次国共合作的背景和结果

### 第6次课：新中国建设与改革开放

**教学内容：**

```
【过渡时期】1949-1956
建国初期：土地改革（1950-1952）→废除封建土地制度
抗美援朝（1950-1953）→提高国际地位
三大改造（1953-1956）→农业/手工业/资本主义工商业→社会主义公有制确立
一五计划（1953-1957）→工业化起步

【探索时期】1956-1978
中共八大（1956）：正确分析国内主要矛盾
大跃进+人民公社化（1958）：急于求成，严重失误
文化大革命（1966-1976）：十年动乱
教训：必须正确认识国情，不能脱离实际

【改革开放】1978至今
十一届三中全会（1978）：工作重心转移到经济建设→改革开放开端
农村：家庭联产承包责任制→安徽小岗村
城市：国有企业改革
对外开放：经济特区（深圳等）→沿海开放城市→全方位开放
南方谈话（1992，邓小平）：确立社会主义市场经济体制目标

改革开放的意义：
(1) 极大解放和发展了生产力
(2) 综合国力显著增强
(3) 人民生活水平显著提高
(4) 开辟了中国特色社会主义道路
```

**过关标准：** 能说出新中国成立后的三个阶段及各阶段标志性事件

## 四、第二阶段：中国古代史（5次课）

> 不追求每个朝代都详细学，抓住三条主线：政治制度+经济发展+思想文化。

### 第9次课：政治制度演变

**教学内容：**

```
【核心线索：权力越来越集中】

先秦：分封制+宗法制→天子权力有限
秦朝：皇帝制度+三公九卿+郡县制→中央集权开端
      （从"分封亲戚"变成"皇帝任命官员"）
汉朝：承秦制+推恩令（削弱诸侯）+察举制（选官）
隋唐：三省六部制→分工制约+科举制→用考试选官（影响1300年）
宋朝：二府三司→进一步分割宰相权力
元朝：行省制→地方管理新模式
明朝：废丞相、设内阁→皇权直接管理六部
清朝：军机处→皇权达到顶峰

总结：皇权不断加强，相权不断削弱
      中央权力加强，地方权力削弱
      选官从血缘→推荐→考试（科举制是进步）
```

**过关标准：** 能按朝代说出政治制度的演变方向

---

### 第10次课：经济发展演变

**教学内容：**

```
【核心线索：从自给自足到商品经济繁荣】

原始社会：刀耕火种
商周：井田制（土地国有）+青铜器
春秋战国：铁犁牛耕→私田出现→井田制瓦解
汉朝：丝绸之路→中外贸易开始
唐宋：经济重心南移（安史之乱后加速）
      宋代商品经济空前繁荣→出现纸币（交子）
      坊市制度瓦解→夜市出现
明清：商品经济继续发展→资本主义萌芽出现
      但重农抑商+海禁/闭关锁国→限制了发展

农业：铁犁牛耕是主线
手工业：官营→民营+家庭手工业
商业：受抑制→逐渐繁荣→资本主义萌芽
```

**过关标准：** 能说出经济重心南移的时间线和宋代商业的特点

---

### 第11次课：思想文化演变

**教学内容：**

```
春秋战国：百家争鸣
  儒家（孔子/孟子/荀子）：仁+礼+民贵君轻
  道家（老子/庄子）：无为而治+顺应自然
  法家（韩非子）：以法治国+中央集权
  墨家（墨子）：兼爱+非攻

秦朝：焚书坑儒→法家独尊
汉朝：汉武帝"罢黜百家，独尊儒术"→儒学成为正统
      董仲舒：天人感应+君权神授
宋明：理学/心学→儒学哲学化
  程朱理学：存天理，灭人欲
  王阳明心学：致良知
明清之际：批判君主专制
  黄宗羲：天下为主，君为客
  顾炎武：天下兴亡，匹夫有责
  王夫之：唯物主义
```

**过关标准：** 能说出思想演变的三个大阶段

### 第12-13次课：古代史综合+测试

第12次课用故事链串联中国古代史全貌：

```
秦统一→汉巩固→三国两晋南北朝分裂→隋唐统一繁荣
→五代辽宋夏金分裂→元统一→明清专制极盛

每个朝代的阶段特征（用关键词）：
秦：统一+制度
汉：巩固+丝路
唐：开放+繁荣
宋：经济巅峰+军事弱势
元：疆域最大+行省制
明：专制+海禁
清：极盛转衰+闭关
```

第13次课测试：6道选择题+1道材料分析题（古代政治制度或经济）

## 五、第三阶段：世界史（4次课）

> 世界史占分约25%，不追全覆盖，只抓3条主线。

### 第14次课：政治革命+启蒙运动

```
【英国】1688光荣革命→1689《权利法案》→君主立宪制
  特点：和平过渡，议会权力至上

【美国】1775独立战争→1776《独立宣言》→1787宪法
  特点：三权分立，联邦制，总统制

【法国】1789大革命→《人权宣言》→拿破仑时代
  特点：最彻底的资产阶级革命，影响深远

【启蒙运动】17-18世纪
  核心思想：理性主义
  代表人物：伏尔泰（自由）+孟德斯鸠（三权分立）+卢梭（社会契约）
  意义：为资产阶级革命提供了思想武器
```

---

### 第15次课：两次工业革命

```
【第一次工业革命】1760s-1840s·英国开始
标志：蒸汽机的广泛应用
代表：纺织机+蒸汽机+火车
影响：生产力飞跃→工厂制度→城市化→资产阶级和无产阶级对立

【第二次工业革命】1870s-20世纪初·多国同时
标志：电力的广泛应用
代表：发电机+电动机+汽车+飞机+电话
影响：垄断资本主义出现→世界市场最终形成→列强瓜分世界

两次工业革命的比较：
  第一次：蒸汽→英国领先→工厂→自由资本主义
  第二次：电力→多国竞争→垄断组织→垄断资本主义
```

---

### 第16次课：两次世界大战+冷战

```
【一战】1914-1918
原因：帝国主义政治经济发展不平衡→萨拉热窝事件（导火索）
结果：同盟国失败→《凡尔赛和约》→国际联盟成立
影响：削弱帝国主义→推动民族解放运动→为二战埋下隐患

【二战】1939-1945
原因：凡尔赛体系的不合理+经济大危机+法西斯上台
结果：轴心国失败→联合国成立
影响：确立雅尔塔体系→美苏两极格局→推动殖民体系瓦解
      中国抗战是世界反法西斯战争的重要组成部分

【冷战】1947-1991
标志：杜鲁门主义（1947）→北约vs华约
表现：柏林墙、朝鲜战争、越南战争、古巴导弹危机
结束：1991苏联解体
影响：两极格局终结→多极化趋势
```

## 六、第四阶段：答题模板+综合（3次课）

### 第18次课：材料分析题答题模板

**教学目标：** 掌握历史大题的通用答题框架

**六大题型及模板：**

#### 1. 原因/背景类

```
模板：政治+经济+思想文化（三角度）

政治角度：政治制度、政府政策、民族关系、国际局势
经济角度：生产力水平、经济政策、商业贸易、技术进步
思想角度：思想运动、文化传统、理论创新

示例："分析鸦片战争的背景"
→ 政治：英国完成工业革命，急需市场；清朝闭关锁国，政治腐败
→ 经济：英国资本主义发展需要原料和市场；中国自然经济自给自足
→ 思想：英国经历启蒙运动；中国天朝上国观念
```

#### 2. 影响/意义/评价类

```
模板：积极+消极（辩证分析）

积极：促进了...推动了...开创了...为...奠定基础
消极：没有完成...存在局限...导致...

注意：影响要从多角度写（政治+经济+思想+对外）
```

#### 3. 特点/特征类

```
模板：从材料中提炼
常见角度：时间（早/晚/长/短）、范围（广/窄）、主导者、方式（和平/暴力）
         内容、过程（曲折/顺利）、结果（彻底/不彻底）
```

#### 4. 比较类

```
模板：找共同点+找不同点
比较角度：背景、过程、内容、结果、影响、性质
```

#### 5. 启示/认识类

```
模板：从历史事件中提炼普遍规律
常见答题角度：
- 改革要顺应时代潮流
- 要立足本国国情
- 经济发展是社会进步的基础
- 开放交流促进发展
```

**过关标准：** 给一道大题能判断题型并用对应模板组织答案

### 第19次课：小论文题专项（12分必考题）

**教学目标：** 掌握小论文三段论答题法

**教学内容：**

```
小论文三段论：

第一段：论题（2分）
- 必须是明确的陈述句，不能是问句
- 观点要正确（符合唯物史观）
- 最好单向角度，不要太宏观
- 格式："我认为..."/"...是...的重要原因"

第二段：论证（8分，最核心）
- 至少用2个史实论证
- 每个史实：先叙述史实→再分析如何证明你的观点
- 分段论述（不要分点，要分段写成小论文）
- 史论结合：史实为论点服务

第三段：结论（2分）
- 重申观点（不能简单重复，要升华）
- 用"综上所述""由此可知""总之"开头
- 最好能联系现实或提炼规律
```

**万能论题角度：**

```
如果材料涉及改革：论题 = "改革是推动社会进步的重要动力"
如果涉及对外交流：论题 = "对外开放促进文明发展"
如果涉及科技：论题 = "科技创新推动社会变革"
如果涉及制度：论题 = "制度创新是国家发展的关键"
如果涉及民族觉醒：论题 = "民族觉醒是抗争胜利的精神力量"
```

**训练：** 做2道真题小论文，严格按三段论写

**过关标准：** 论题明确+至少2个史实+有总结升华

## 七、过关验证机制

### 每次课的标准流程

```
1. 快速复习上节课（5分钟）→ 用故事链串联核心事件
2. 新知识讲解（25分钟）→ 故事链+逻辑链+阶段特征
3. 当堂练习（20分钟）→ 选择题+小材料分析
4. 总结+笔记本（10分钟）→ 时间线+关键词
```

### 过关三步走

```
记（时间线+故事链）→ 练（选择题+判断时间段）→ 写（按模板做大题）
```

### 没过关怎么处理

1. 用故事链重新串一遍
2. 第二次还错 → 画时间线图，视觉记忆
3. 第三次还错 → 只背最核心的5个事件，先拿基础分
