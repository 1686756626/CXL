# 定时任务配置表

> 智能体接管后，按本文件设置所有cron定时任务。
> 每个任务对应排课总表中的一个时间段，到点自动提醒新灵开始学习。

## 一、前置条件

设置cron之前，需要先确认：

1. **新灵的钉钉会话已建立** — 先在钉钉和新灵对话一次，建立session
2. **获取 target-user 和 target-session**
3. **agent-id**: `Dh6Jpf`

---

## 二、核心原则

### 周六 = 测试日
- 周六上午8:45-12:00为本周统一大测试，纯考试不讲解
- 所有学习任务只在**周一至周五**触发
- 周六只有大测试一个任务

### 高考真题风格
- 所有测试和练习必须贴近高考真题，不用人造题
- 英语：完形填空式选词、句子翻译、近义词辨析
- 数学：直接用高考真题
- 政史地：材料分析题，像高考大题
- 语文：文言文放到高考真题语境里考

### 任务1：早间交流（周一至周五 8:30）

```
名称：早间交流
Cron：30 8 * * mon-fri
Prompt：现在是早间交流时间（8:30-8:45）。读一下README、今日任务和排课总表，了解今天的安排，然后跟新灵聊几句，聊聊昨天学得怎么样、今天状态怎么样。记得语气自然。不要追问太多，3-5分钟搞定。
```

## 四、特殊情况处理

### 陈老师数学时段（不设cron）

| 时段 | 星期 | 处理 |
|------|------|------|
| 10:15-12:00 | 周一 | cron已排除 |
| 10:15-12:00 | 周四 | cron已排除 |
| 19:00-21:00 | 周三 | 文言文和语文cron已排除周三 |

### 周六测试日
- 只有"周六统一大测试"一个任务（8:45）
- 其他学习任务全部不触发
- 新灵下午自由安排

### 周日休息
- 只有"每周排课校准"（20:00）触发
- 新灵全天休息，鼓励读笔记

### 临时调整

跳过某个时段：
```bash
copaw cron pause <job_id> --agent-id Dh6Jpf
```
恢复：
```bash
copaw cron resume <job_id> --agent-id Dh6Jpf
```
