## 经济生活

### 板块一：市场与政府的关系（4月6日）✅ 过关

**掌握情况：** 第二轮测试全对 ✅

**第一轮错误：**
- 把"行政手段"说成"政策手段"
- 市场调节优点（价格、供求、竞争）没记住

**笔记本内容：**

```
政治·经济生活：市场与政府

一、市场调节
- 优点：通过价格、供求、竞争配置资源
- 缺陷三个：
  自发性 → 为赚钱不择手段（假冒伪劣、污染）
  盲目性 → 信息不灵一窝蜂跟风
  滞后性 → 反应过来已经晚了

二、政府宏观调控（因为市场有缺陷）
- 经济手段 → 用钱（财政政策、货币政策）
- 法律手段 → 用法（立法、执法）
- 行政手段 → 用权（限价、审批）
```

**滚动复习重点：**
- 市场调节三机制：价格、供求、竞争
- 宏观调控三手段：经济（钱）、法律（法）、行政（权）
- 三缺陷辨析：自发性 vs 盲目性 vs 滞后性（容易混）
