### 板块二：价格与消费（4月8日）✅ 测试7/8，补考全对

**影响价格：**
- 价值决定价格（社会必要劳动时间）
- 供求影响价格（供不应求涨，供过于余跌）
- 价格围绕价值上下波动

**价格变动影响：**
- 生活必需品：影响小
- 高档耐用品：影响大
- 对生产：调节产量 + 调节生产要素

**消费：**
- 前提：收入（当前收入+未来预期+收入差距）
- 类型：钱货两清/贷款/租赁；生存/发展/享受
- 四原则：量入为出、理性消费、绿色消费、勤俭节约

**滚动复习重点：**
- 价值决定 vs 供求影响（两个不同层面）
- 价格对生产的影响：调节产量（涨了多生产）+ 调节生产要素（哪个便宜用哪个）
- 四个消费原则辨析

**掌握情况：** 第二轮测试全对 ✅

**第一轮错误：**
- 把"行政手段"说成"政策手段"
- 市场调节优点（价格、供求、竞争）没记住

**笔记本内容：**

```
政治·经济生活：市场与政府

一、市场调节
- 优点：通过价格、供求、竞争配置资源
- 缺陷三个：
  自发性 → 为赚钱不择手段（假冒伪劣、污染）
  盲目性 → 信息不灵一窝蜂跟风
  滞后性 → 反应过来已经晚了

二、政府宏观调控（因为市场有缺陷）
- 经济手段 → 用钱（财政政策、货币政策）
- 法律手段 → 用法（立法、执法）
- 行政手段 → 用权（限价、审批）
```

**滚动复习重点：**
- 市场调节三机制：价格、供求、竞争
- 宏观调控三手段：经济（钱）、法律（法）、行政（权）
- 三缺陷辨析：自发性 vs 盲目性 vs 滞后性（容易混）
