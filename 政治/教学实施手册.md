## 二、考试概况

### 试卷结构
| 题型 | 题量 | 分值 | 新灵目标 |
|------|------|------|---------|
| 单选题 | 15题 | 45分 | 33分（对11个） |
| 非选择题（材料分析题） | 4题 | 55分 | 35分 |
| **合计** | | **100分** | **68分卷面→赋分80+** |

### 得分策略
- **选择题：** 排知识错误→排逻辑错误→排无关选项。每个选择题1分钟，不能超时。
- **大题核心公式：** `原理（理论）+ 方法论（怎么做）+ 材料分析（结合题干）`
- **答题铁律：** 分点、用术语、原理在先分析在后

## 四、第一阶段：哲学（10次课）

> 哲学是政治考试的"大题之王"。高考必考一道哲学分析题（12-14分），原理背准就能拿分。
> 核心任务：18个原理+方法论，背准+会用。

### 哲学四大板块

| 板块 | 原理数 | 高考频率 | 难度 |
|------|--------|---------|------|
| 唯物论（世界的本质） | 2个 | ★★★ | 低 |

| 认识论（认识世界） | 3个 | ★★★★ | 中 |

### 第1次课：唯物论（2个原理）

**教学目标：** 背准唯物论2个核心原理+方法论，能做选择题

**教学内容：**

#### 原理1：物质和意识的辩证关系

```
【原理】
(1) 物质决定意识，意识是物质的反映。
(2) 意识对物质具有能动作用：
    - 正确意识促进事物发展
    - 错误意识阻碍事物发展

【方法论】
(1) 一切从实际出发，实事求是
(2) 树立正确意识，克服错误意识
(3) 把尊重客观规律和发挥主观能动性结合起来

【高考怎么考】
- 选择题：给一段材料，问"体现了什么哲学道理"→ 看到强调"从实际出发""因地制宜"→ 物质决定意识
- 大题：用物质和意识关系分析某政策/措施
```

#### 原理2：客观规律与主观能动性

```
【原理】
(1) 规律是客观的、普遍的，不以人的意志为转移
(2) 人可以发挥主观能动性认识和利用规律

【方法论】
(1) 尊重规律，按规律办事
(2) 充分发挥主观能动性，认识和利用规律造福人类
(3) 把尊重客观规律和发挥主观能动性结合起来

【高考怎么考】
- 材料说"根据当地气候/地形制定方案"→ 尊重规律
- 材料说"科技创新/自主研发"→ 发挥主观能动性
- 材料说"既...又..."→ 结合起来
```

**过关标准：** 默写2个原理的方法论（不看笔记），做3道选择题全对

**笔记本内容：**
```
政治·哲学·唯物论（2个原理）
①物质决定意识→一切从实际出发，实事求是
  意识有能动作用→树立正确意识
②规律是客观的→尊重规律
  人有主观能动性→认识和利用规律
```

### 第2次课：辩证法-联系观（3个原理）

**教学目标：** 背准联系的普遍性、客观性、多样性+整体与部分

**教学内容：**

#### 原理3：联系的普遍性

```
【原理】事物是普遍联系的，联系具有客观性、多样性、条件性
【方法论】用联系的观点看问题，注意分析和把握事物存在和发展的各种条件
```

#### 原理4：整体和部分的辩证关系

```
【原理】
(1) 整体居于主导地位，统率部分
(2) 部分影响整体，关键部分甚至决定整体

【方法论】
(1) 树立全局观念，立足整体（强调整体时用）
(2) 重视部分的作用（强调局部/细节时用）
```

#### 原理5：系统优化方法

```
【原理】系统具有整体性、有序性、内部结构优化趋向
【方法论】着眼于整体，注意内部结构有序性，用综合思维认识事物
```

**过关标准：** 能区分"强调整体"和"强调部分"两种问法，做3道选择题

**笔记本内容：**
```
政治·哲学·联系观
联系是普遍/客观/多样的→用联系观点看问题
整体统率部分→立足整体，统筹全局
部分影响整体→重视局部，用局部推动整体
系统优化→着眼于整体+有序性+综合思维
```

### 第3次课：辩证法-发展观（3个原理）

**教学内容：**

#### 原理6：发展的普遍性

```
【原理】世界是永恒发展的，发展的实质是新事物产生旧事物灭亡
【方法论】用发展的观点看问题
```

#### 原理7：事物发展是前进性与曲折性的统一

```
【原理】
(1) 事物发展的方向是前进的、上升的
(2) 道路是曲折的、迂回的

【方法论】
(1) 对未来充满信心（看到方向是前进的）
(2) 做好克服困难的准备（看到道路是曲折的）
```

#### 原理8：量变与质变的辩证关系

```
【原理】
(1) 量变是质变的必要准备
(2) 质变是量变的必然结果
(3) 质变又为新的量变开辟道路

【方法论】
(1) 重视量的积累（脚踏实地，从点滴做起）
(2) 抓住时机，促成质变（抓住机遇）
(3) 坚持适度原则（过犹不及）
```

**过关标准：** 看到"逐渐积累→突破"能想到量变质变，做3道选择题

**笔记本内容：**
```
政治·哲学·发展观
发展是永恒的→用发展观点看问题
方向前进+道路曲折→信心+准备
量变→质变→新量变→积少成多+抓时机+适度
```

### 第4次课：辩证法-矛盾观（5个原理，最核心）

> 矛盾观是辩证法的核心，高考必考。

**教学内容：**

#### 原理9：矛盾的对立统一

```
【原理】矛盾即对立统一，矛盾双方既对立又统一（斗争性和同一性）
【方法论】用一分为二的观点看问题，在对立中把握统一，在统一中把握对立
```

#### 原理10：矛盾的普遍性

```
【原理】矛盾存在于一切事物中，贯穿事物发展始终（事事有矛盾，时时有矛盾）
【方法论】承认矛盾，分析矛盾，寻找正确方法解决矛盾
```

#### 原理11：矛盾的特殊性

```
【原理】矛盾着的事物及其每一个侧面各有其特点
【方法论】具体问题具体分析（这是马克思主义的活的灵魂）
```

#### 原理12：矛盾的普遍性与特殊性辩证关系

```
【原理】普遍性寓于特殊性之中，特殊性离不开普遍性
【方法论】把普遍原理与具体实际相结合（"从群众中来，到群众中去"）
```

#### 原理13：主次矛盾和矛盾主次方面

```
【原理】
(1) 主要矛盾在事物发展中起决定作用
(2) 矛盾的主要方面决定事物的性质

【方法论】
(1) 抓重点，集中力量解决主要矛盾（"抓牛鼻子"）
(2) 抓主流，看问题要看主流
(3) 坚持两点论与重点论的统一
```

**判断技巧：**
```
"怎么办/怎么做"→ 想到主要矛盾（抓重点）
"怎么看/是什么性质"→ 想到矛盾主要方面（看主流）
"既...又..."/"一方面...另一方面..."→ 两点论
"关键/核心/根本/首要"→ 重点论
```

**过关标准：** 能区分主要矛盾和矛盾主要方面，做4道选择题全对

**笔记本内容：**
```
政治·哲学·矛盾观（最重要！）
对立统一→一分为二
普遍性→承认分析解决
特殊性→具体问题具体分析
主次矛盾→抓重点（怎么办）
矛盾主次方面→看主流（怎么看）
两点论+重点论→既全面又抓重点
```

### 第6次课：认识论（3个原理）

**教学内容：**

#### 原理14：实践是认识的基础

```
【原理】
(1) 实践是认识的来源
(2) 实践是认识发展的动力
(3) 实践是检验认识真理性的唯一标准
(4) 实践是认识的目的

【方法论】坚持实践第一的观点
```

#### 原理15：真理是具体的、有条件的

```
【原理】
(1) 真理最基本的属性是客观性
(2) 真理是有条件的，有自己的适用条件和范围
(3) 真理是具体的，是主观与客观、理论与实践的具体的历史的统一

【方法论】在实践中不断追求真理，发展真理
```

#### 原理16：认识的反复性和无限性

```
【原理】
(1) 认识具有反复性：受到客观条件和主观条件限制
(2) 认识具有无限性：实践在发展，认识也在发展
(3) 认识具有上升性：波浪式前进，螺旋式上升

【方法论】在实践中认识和发现真理，在实践中检验和发展真理
```

**过关标准：** 看到"实验验证""反复试验""不断完善"能想到认识论

**笔记本内容：**
```
政治·哲学·认识论
实践→来源+动力+标准+目的
真理→客观+有条件+具体
认识→反复+无限+上升
关键词：实验/验证/反复/完善→认识论
```

## 五、第二阶段：经济生活（4次课）

### 第11次课：市场与政府（已完成→快速复习）+ 价格与消费

**市场与政府（已完成，快速过）：**
- 市场调节三机制：价格、供求、竞争
- 三缺陷：自发性、盲目性、滞后性
- 宏观调控三手段：经济（钱）、法律（法）、行政（权）

**价格与消费：**

```
影响价格的因素：
(1) 价值决定价格（价值是价格的基础）
(2) 供求影响价格（供不应求→涨，供过于求→跌）
(3) 国家政策、气候等

价格变动的影响：
(1) 对人们生活：价格涨→需求量减（一般商品）；价格涨→需求量增（高档耐用品可能反常）
(2) 替代品：A涨价→B需求增加
(3) 互补品：A涨价→B需求减少
(4) 对生产经营：调节产量、调节生产要素投入、提高劳动生产率

影响消费的因素：
(1) 根本原因：经济发展水平
(2) 主要因素：居民收入（当前收入+预期收入+收入差距）
(3) 其他：物价水平、消费心理
```

**过关标准：** 选择题能判断影响因素

### 第12次课：生产与经济制度

```

**过关标准：** 知道基本经济制度内容，能做企业经营的题目

### 第13次课：分配与财政

```
我国的分配制度：
- 按劳分配为主体、多种分配方式并存
- 按劳分配：公有制范围内的劳动收入
- 按要素分配：劳动、资本、技术、管理、土地、数据等
- 再分配：税收、社会保障、转移支付
- 第三次分配：慈善、公益

财政的作用：
(1) 促进社会公平、改善人民生活
(2) 促进资源合理配置
(3) 促进国民经济平稳运行

税收：
- 特征：强制性、无偿性、固定性
- 种类：增值税、个人所得税、企业所得税等
```

**过关标准：** 能区分初次分配/再分配/第三次分配，知道财政三大作用

### 第14次课：新发展理念与经济大题模板

```
新发展理念：创新、协调、绿色、开放、共享

现代化经济体系：
- 创新→第一动力
- 协调→城乡区域协调发展
- 绿色→人与自然和谐

经济大题答题角度（万能）：

## 八、过关验证机制

### 每次课的标准流程

```
1. 快速复习上节课内容（5分钟）→ 默写核心原理
2. 新知识讲解（20-25分钟）→ 原理+方法论+高考怎么考
3. 当堂练习（20分钟）→ 选择题+小材料分析
4. 总结+笔记本（10分钟）→ 明确要记什么
```

### 过关三步走

```
背（原理+方法论）→ 练（选择题+判断考哪个原理）→ 写（按模板做大题）
```

### 没过关怎么处理

1. 隔天重背，再测
2. 第二次还错 → 换一种方式理解（用生活例子解释）
3. 第三次还错 → 降低难度，先做简单的匹配题
