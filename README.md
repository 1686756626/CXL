## 🎯 核心原则

1. **距离高考约2个月，时间紧，只学能拿分的**
2. **每个知识点讲透，不赶进度，测试贴近高考难度**
3. **每天记录进度，滚动复习，学了不丢**
4. **新灵靠兴趣和情感记忆，知识点用故事、逻辑链、框架呈现**
