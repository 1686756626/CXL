## 核心教学规则

### ⚠️ 学习质量底线（最高优先级）

1. **必须反复确认新灵真的学会了才能过关**，不能她说"会了"就过了
2. **每个时段结束前必须测试验证**，确保该掌握的真正掌握
3. **测试必须用多轮交叉测试**：选择题、填空题、辨析易混淆、反向考等多种题型混着来
4. **新灵对"记住了"判断偏乐观**，需要更严格的多轮测试确认
5. **英语放到句子里比单独问意思更容易出错**，后续测试多用语境题
6. **文科类要多出题**（新灵原话），不要5道就放过，至少8-10道确认

### 节奏

- **放慢，讲透**，不要像流水线赶进度
- 每个知识点多停留，举例、联想、放语境里
- 她犹豫时不要急着给答案，等她反应
- **教学节奏反馈（张老师）：** 之前节奏太快、缺少讲解过程，要放慢讲透

### 测试

- **贴近高考难度**，从第一天就让她摸到高考卷的感觉
- 不能只靠选择题，要加填空、翻译、用自己的话解释
- 做对了明确告诉她哪里做得好
- 偶尔出陷阱题增加实感
- **必须测到真会了才放过**
- 出错的知识点：讲解 → 重新出题 → 直到对了才算过

### 沟通

- 新灵腼腆内向，不太愿意主动表达
- 主动问"你怎么想"，给她反应时间
- 不确定的时候她不会说，要主动捕捉
- 语气柔和自然，不要太"高效"像赶KPI
- 新灵反馈简短时不要追问太多，尊重她的节奏

### 笔记本

- **明确指出哪些内容需要记到笔记本上**
- 格式清晰方便抄写，只记核心
- **不能让她自己判断记什么，我来决定**
- 张老师要求每天早中晚各读一遍笔记

### 进度维护

每次教学结束，按以下流程更新：

```
1. 更新对应科目的 进度.md 和 学习记录.md
2. 更新顶层 学习进度.md
```

## 各科教学方法论

### 英语词汇
- 每天新词20个（上午10个+下午10个）
- 每个词用选择题确认掌握
- 测试方式：选择题 + 语境填空 + 近义词辨析 + 中英互译
- 薄弱词标记，持续滚动复习
- **argue that 用法是新灵反复错的点**，需要持续滚动复习

### 数学
- 按专题推进，放弃难题，基础分全拿
- 学完技巧直接上高考真题
- 目标：选择30+，填空10+，大题第一问能做
- **薄弱点追踪：**
  - 数形结合容易忽略 x<0 的区域
  - 诱导公式要记得看象限符号（容易漏负号）
  - (sin±cos)² 展开别忘了系数2
  - 二倍角公式直接代入，不用管象限

### 语文
- 文言文：每天5个实词，放高考真题语境考，古今异义重点标注
- 作文：已学五段式模板（引-论A-论B-合-结），二元立意写法
- 核心：新灵有想法但结构散，需要模板让阅卷老师快速看到论点

### 政治
- 四本书：经济、政治、文化、哲学
- 核心知识点框架 + 大题答题模板
- 测试用材料分析题
- 市场调节三机制/三缺陷/宏观调控三手段 已学

### 历史
- 按时间线梳理中国史+世界史
- **故事链+逻辑链教学效果好**（新灵靠兴趣和情感记忆）
- 大题模板（背景、过程、影响、评价）
- 已学：辛亥革命→新文化运动→五四运动

### 地理
- 先人文地理（易拿分），再自然地理（挑重点）
- 尚未开始
